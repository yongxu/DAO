# baguaWen · 受控文规范（Frozen for M1）

> **Status**: Frozen at HEAD `1cd30f8`. 任何扩展须显式 PR 修改本文件。
> **Risk mitigated**: 路径丙 § 风险 3（baguaWen 与 wenyan-operators.md 200+ 算子冲突）。
> **Companion module**: `formal/SSBX/Foundation/BaguaWenSpec.lean`（机器可校验的 token 表）。

---

## 〇 · 设计意图

`wenyan-operators.md` 列出 200+ 算子，覆盖六征、心智、史维、格、重、耦等多维度。**M1 之 parser 只识其中 12 主字 + 3 时态 + 6 爻 + 数词**——这是文证文之最小闭核。

`wenyan-operators.md` 中其他算子在 reflection 阶段（M5+）通过等同律证明加入。在此之前，遇任何未列于本文之 token 即返 `none`（句法非法）。

---

## 一 · 词法

```
token       ::= 「«」 cjk «»」  |  ascii_ctrl  |  whitespace
cjk         ::= UTF-8 CJK 字符序列（不嵌套 «»）
ascii_ctrl  ::= 「；」 | 「;」（半角全角皆可）
whitespace  ::= ' ' | '\t' | '\n' | '\r'
```

**禁止**：裸 CJK（不带 «»）、英文标识符、其他标点。

---

## 二 · 12 主字（动词）

每字对应一个 `YiInstr` 构造子。详见 `formal/SSBX/Foundation/BaguaWenSpec.lean`。

| token | YiInstr | 参数 | 语义 |
|---|---|---|---|
| `«不动»` | `nop` | — | 守而不易 |
| `«互»`   | `hu`  | — | 内外相照 |
| `«错»`   | `cuo` | — | 阴阳全反 |
| `«综»`   | `zong` | — | 上下颠倒 |
| `«推»`   | `push` | — | 现入于带 |
| `«取»`   | `pop`  | — | 自带回现 |
| `«终»`   | `halt` | — | 行止 |
| `«设时»` | `setShi` | 时 | 定其已今未 |
| `«翻爻»` | `flipYao` | 爻 | 择爻而反 |
| `«比爻»` | `branchYaoEq` | 爻 爻 «至» 数 | 观二爻之同否 |
| `«比时»` | `branchShiEq` | 时 «至» 数 | 观时态之同否 |
| `«跳»`   | `jump` | «至» 数 | 行至所指 |

## 五 · 数词

跳转目标 N（自然数）以中文数字表达：

```
«一» «二» «三» ... «十» «十一» ... «二十» ... «六十四»
```

- 范围：`1..64`（与 Hexagram 索引共域）
- 必随 `«至»` 关键字使用：`«至 三»`、`«至 二十»`
- 解析后即 `Nat`（即 `branchYaoEq.target` 等的 `Nat` 参数）

---

## 六 · 语法（BNF）

```
程     ::= 句*
句     ::= 命 句尾
句尾   ::= 「；」 | 「也」
命     ::= 「«不动»」 | 「«互»」 | 「«错»」 | 「«综»」
        | 「«推»」 | 「«取»」 | 「«终»」
        | 「«设时»」 时
        | 「«翻爻»」 爻
        | 「«比爻»」 爻 爻 「«至»」 数
        | 「«比时»」 时 「«至»」 数
        | 「«跳»」 「«至»」 数
时     ::= 「«已»」 | 「«今»」 | 「«未»」
爻     ::= 「«初爻»」 | 「«二爻»」 | ... | 「«上爻»」
数     ::= 中文数字 1..64
```

## 七 · 范例

`«道判»` 程序的 baguaWen 源（5 命）：

```
«比爻» «三爻» «四爻» «至» «三»；
«设时» «未»；
«终»；
«设时» «已»；
«终»
```

经 `WenyanParser.«解程»` 后得：

```lean
[branchYaoEq ⟨2,_⟩ ⟨3,_⟩ 3, setShi Shi.wei, halt, setShi Shi.ji, halt]
   = daoJudgeProg                     -- 由 native_decide
```

## 八 · 冻结声明

| 项 | 数 |
|---|---|
| 主字（动词） | 12 |
| 时态 | 3 |
| 爻位 | 6 |
| 关键字（«至»） | 1 |
| **保留 CJK token 总数** | **22** |
| 数词（按 1..64） | 任意中文数字串 |

任何 token **不在本表内** ⇒ parser 拒绝。

任何对此规范的扩展须：
1. 修改本文件；
2. 同步修改 `formal/SSBX/Foundation/BaguaWenSpec.lean`；
3. 更新 `«印解之逆»`、`reservedTokens_length` 等机器可校验定理；
4. 不破坏 M1-M2 既有 PoC。

## 九 · 与 wenyan-operators.md 的关系

| 状态 | 含义 | 当前数 |
