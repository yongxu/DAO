## Audit Reading

- Six single-position flips reach `64/64` hexagram states from any start; connected = `true`.
- Therefore the T6 bit-state layer has no missing lateral flip operator for reachability.
- `错` is redundant for reachability because it is the composite of all six flips.
- `综` is a position-reversal symmetry; it is not generated by single flips because it permutes slots rather than values.
- `互` is directed and non-invertible: it drops y1/y6 and duplicates middle positions. It is a semantic extraction operator, not a reachability generator.
- For `Cell192 = Hexagram × Shi`, hexagram edges alone split into three time components. A time operator (`shiNext`/`shiPrev`, or YiInstr `setShi`) is required for full 192-cell connectivity.

## Missing-Operator Interpretation

- Missing for T6 state reachability: none, given `flip1..flip6`.
- Missing for full Cell192 reachability: none if the time edge is included; missing time connectivity if only hexagram operators are admitted.
- Missing for full position-permutation algebra: adjacent swap or rotation is not primitive. Current code has `综` but no generator for arbitrary S6 permutations. This is only a gap if the intended operator set must cover all position permutations, not just Yi/Bagua state reachability.
