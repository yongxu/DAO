## 6. 4 aspects 全 empirical

### 6.1 4 aspects derivation

```
公理: 道理 unified.

定理 (4-aspect derivation):
  道理 之 manifest articulation requires 4 modes:
    (A) atomic carriers → 物 (discrete entities)
    (B) continuous-dynamic carriers → 動 (continuous transformation)
    (C) relational carriers → 間 (relational topology)
    (D) actualized carriers → 事 (events / happenings)

  此 4-mode 必 (transcendental: 完整 + 一致 + 结构).
  全 empirical, 全 visible, 全 mutually accessible.
```

### 6.2 4 aspects' empirical features

```
物 → 质 + 位 (substance + location)
動 → 势 (tendency / dynamic disposition)
間 → 关系 (relations / topological connections)
事 → 结构 (structure / patterning)

每 feature observable empirically:
  质: heft, substantive identity over time.
  势: tendency to continue, gravitational pull, momentum.
  关系: connections, network structure.
  结构: patterns, regularities, organizational form.
  位: location, position.
```

注: 物 之 features 包 质 + 位 (双 feature). 其他 aspects 各 一 primary feature.

### 6.3 全 empirical — 没 transcendent

```
v12 commit:
  4 aspects 全 visible, 全 empirical, 全 mutually accessible.
  没 形而上 / transcendent dimension.

理由:
  1. framework 是 empirical structure, 为真.
  2. 名实合一 要求 framework = reality. 不 separate worlds.
  3. 道法自然 之 immanentist: 道 在 reality 内.
  4. 每 aspect 已 commit empirical features.

庄子 "天地与我并生, 万物与我为一" — 一 reality, 全 visible from within.
```

### 6.4 Mutual perspective accessibility

```
法则 57 (perspective accessibility):
  ∀ aspects X, Y ∈ {物, 動, 間, 事}, X ≠ Y,
    X 之 perspective 可 study Y.

例:
  从 物 视: 学 動 (势 之 manifestation) + 間 (relations of 物) + 事 (involving 物).
  从 動 视: 学 物 (entities undergoing 動) + 間 (connections through 動) + 事 (动态 events).
  从 間 视: 学 物 (nodes in 間) + 動 (dynamics across 間) + 事 (events through 間).
  从 事 视: 学 物 (involved entities) + 動 (event dynamics) + 間 (event relations).

4 perspectives 全 valid, 全 empirical.
没 perspective "above" or "transcendent".
```

### 6.5 Aspects 与 frames orthogonal

```
4 aspects (territory) 与 8 primary 面 orthogonal.
任 面 之 articulation 在 4 aspects 内 manifest.

例: 因果 面 之 4-aspect manifestation:
  因果 in 物: discrete cause-effect (个体 因果)
  因果 in 動: continuous causation flow (process 因果)
  因果 in 間: network causation (relational 因果)
  因果 in 事: actualized causation (event 因果)

8 面 × 4 aspects = 32 specific articulation cells.
具体 theories 选 specific cells.
```

### 7.4 式量 面 (functional)

### 7.5 阴阳 面 (polar)

### 7.7 名实 面 (semantic)

### 7.8 体用 面 (substantive-functional)

```

universal: applies to 心 / 物 / 系统 / 道 / etc. — any entity has 体用.
```

注: 体 / 用 二字 polysemous (体 包 substance, body, manifest, experience; 用 包 use, function, apply). proximity dispatch handles polysemy.
