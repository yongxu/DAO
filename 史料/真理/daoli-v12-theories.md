## Theory 1: 牛顿力学 (Newtonian Mechanics)

### Overview

```
牛顿力学 articulates *macroscopic motion* of 物 (matter) under 力 (force).
foundational mechanics from 17世纪.
F = ma 是 core articulation.
```

### 4-axis coordinate

```
面: 因果 (primary) + 式量 (mathematical formalization)
aspect: 物 (mass entities) + 動 (motion / forces) + 間 (spatial relations)
modality: 显 (observable trajectories)
scale: macroscopic (above 量子, below relativistic)
```

### 法则 引用

```
法则 1   — 4 aspects irreducible: 物 (mass) 与 動 (motion) 不 reduce.
法则 6   — 合成 不可逆: temporal order matters in dynamics.
法则 7   — 反馈律: action-reaction (Newton's third law).
法则 11  — 非对易: cross-aspect measurement order matters.
法则 15  — 因缘果链: F = ma needs 缘 (mass, reference frame).
法则 28  — 道-别 coupling: force as time-derivative of momentum.
法则 38  — 物理 emergence: Newton 是 specific calibration.
法则 41  — epistemic humility: Newton breaks at 量子 / relativistic scales.
法则 56  — 4-aspect features: 物 之 质 + 位; 動 之 势 (momentum / inertia).
```

### Articulation

```
F = ma 之 framework articulation:

  因 (cause): F (force, 動 aspect's manifestation).
  果 (effect): a (acceleration, 物 之 位 之 derivative).
  缘 (enabling condition): m (mass, 物 之 质).
  
  此 是 法则 15 (因缘果链) 之 instantiation:
    无 m (mass): F 不 致 a (massless objects don't accelerate from F alone).
    无 reference frame: a 不 articulable.

action-reaction (Newton's 3rd law):
  二 物 mutual: F_AB = -F_BA.
  此 是 法则 7 (反馈律) + 法则 27 (mutual constitution at force level).
  二 mass mutual constitute through force.

inertia:
  物 之 势 (法则 56): tendency to continue motion.
  Newton's 1st law: rest or uniform motion sustained without F.
  此 是 物 之 internal 势.

gravity:
  物 ⇄ 物 通过 距 (位 之 difference).
  inverse-square law: 间 之 specific articulation.
  此 是 法则 16 (空界关系) + 间 之 关系 (法则 56).

momentum:
  p = mv (momentum = mass × velocity).
  conservation: closed system.
  此 是 法则 33 (closure) + 法则 7 (feedback).
```

### Cross-frame

```
因果 frame ↔ 时刻 frame:
  F = m(dv/dt): force as time-derivative of momentum.
  此 是 法则 45 (frame-dual derivative-integral) explicit.
  causal articulation ≡ temporal articulation.

因果 frame ↔ 式量 frame:
  F as gradient of potential V: F = -∇V.
  causal force ≡ functional gradient of potential.
  此 是 法则 39 (frame iso): same dynamics, different vocabulary.

法则 53 (strict connectedness) verification:
  Newton derivation in 因果 frame ≡ Newton derivation in 时刻 frame ≡ Newton derivation in 式量 frame.
  same predictions, different articulations.
```

### Attestation

```
古典中文:
  墨子: early mechanics in 墨经 (杠杆 leverage articulated).
  没 strict mechanics tradition (different focus on 阴阳 / 道).

西方 古典:
  Galileo (1564-1642): inertia, projectile motion.
  Newton (1642-1727): Principia Mathematica (1687).
  Leibniz: differential calculus parallel.

framework attestation:
  Newton 之 *successful articulation* of macroscopic motion 通过 因果 + 式量 frames.
  framework explains why Newton works: specific calibration of axes.
```

### Wenyan derivation 例

```
goal: F = ma 之 articulation in framework.

derivation:
  公理: 因果互化 也.
  
  考虑 物 X (with 质 m, 位 r).
  施 力 F 于 X.
  
  由 法则 15 (因缘果链):
    F (因) 致 a (果) 必 经 m (缘).
  
  由 法则 56 (4-aspect features):
    a = d²r/dt² (加速度 是 位 之 二 阶 时间 导).
  
  由 法则 28 (道-别 coupling):
    F = m · a (causation-mass coupling).
  
  即 F = ma.
  
  ∎
```

### Limitations

```
breaks down:
  - 量子 scale: 法则 11 (non-commutativity) becomes severe (Heisenberg).
  - relativistic speeds: spacetime not flat, F = ma 不 hold.
  - 强 gravity: GR replaces.
  
此 是 法则 41 (epistemic humility):
  Newton 之 coverage bounded.
  外 boundary: 量子 / GR / QFT 之 different calibrations.
  
法则 54 (coverage-bounded absolute):
  Newton 在 macroscopic regime 是 真.
  外 regime, framework 默 — different theories take over.
```

## Theory 2: 量子力学 (Quantum Mechanics)

### Overview

```
量子力学 articulates *微观 之 mechanics*.
foundational since 1925-30 (Schrödinger, Heisenberg, Born, Bohr).
波粒二象 (wave-particle duality), uncertainty, superposition, collapse.
```

### 4-axis coordinate

```
面: 因果 (probabilistic) + 时刻 (events) + 阴阳 (wave-particle duality)
aspect: 事 (measurement events) + 動 (wave function evolution)
modality: 开闭 collapse (开 in superposition, 闭 in measurement)
scale: 量子 (Planck-scale)
```

### 法则 引用

```
法则 1   — 4 aspects irreducible.
法则 4   — 六征 includes 实/虚 modal (essential to quantum).
法则 5   — 闸口二值 (measurement binary outcome).
法则 11  — 非对易 (Heisenberg uncertainty).
法则 21  — 实虚 modal (superposition).
法则 38  — 物理 emergence (specific calibration).
法则 41  — epistemic humility (Planck threshold).
法则 51  — cross-cultural (波粒 ≡ 阴阳).
```

### Articulation

```
wave-particle duality:
  阴阳 frame articulation (法则 51 cross-cultural):
    阴 ↔ 波 (wave, distributed, potential).
    阳 ↔ 粒 (particle, localized, actual).
  二者 mutual: 波 中 有 粒 (probability density), 粒 中 有 波 (de Broglie wavelength).
  Bohr 之 complementarity = 法则 39 frame iso.

wavefunction:
  ψ as 阴 (potential, 虚 modal, 法则 21).
  evolution via Schrödinger equation: 動 之 articulation.
  unitary evolution: 法则 33 (closure).

measurement collapse:
  ψ → eigenstate transition.
  此 是 法则 5 (闸口二值) — superposition (开) → specific outcome (闭).
  此 是 modal axis 之 critical event.

Heisenberg uncertainty:
  [x, p] = iℏ.
  position 与 momentum 不 commute.
  此 是 法则 11 (非对易) explicit.

entanglement:
  二 particles 之 mutual constitution at distance.
  此 是 法则 27 (心-territory mutual) generalized — non-local mutual.
  Ubuntu 之 "I am because we are" 之 量子 articulation.

quantization:
  energy levels discrete.
  此 是 法则 5 (闸口二值) at energy axis.
```

### Cross-frame

```
阴阳 frame ↔ 因果 frame:
  阴阳 mutual ↔ probabilistic causation.
  amplitudes (阴阳) collapse to specific outcomes (因果).
  
阴阳 frame ↔ 时刻 frame:
  superposition (阴阳 simultaneity) ↔ collapse moment (时刻 specific).

阴阳 frame ↔ 式量 frame:
  阴阳 ↔ Hilbert space inner product (functional).
  
法则 53 verification:
  量子 derivation in any frame ≡ derivation in any other frame.
  same physical predictions.
```

### Attestation

```
古典中文:
  阴阳 (易传): polar mutual, prior to specific manifestation.
  Bohr explicitly drew on 阴阳 (太极 在 Bohr's coat of arms).

现代物理:
  Planck (1900): quantization.
  Einstein (1905): photoelectric effect, photon.
  Bohr (1913): atom model.
  Heisenberg, Schrödinger (1925-26): formal mechanics.
  Born: probabilistic interpretation.
  Bohr's complementarity: 阴阳-flavored.

framework attestation:
  量子 之 deepest commits articulable in 阴阳 frame native.
  framework explains why 阴阳 reading natural: same axiom, different frame.
```

### Wenyan derivation 例

```
goal: Heisenberg uncertainty Δx·Δp ≥ ℏ/2 之 articulation.

derivation:
  公理: 阴阳互化 也.
  
  考虑 量子 system 之 位 (x) 与 動量 (p).
  
  由 法则 11 (非对易):
    cross-aspect measurement: measure(x) 与 measure(p) 不 commute.
    [x̂, p̂] = iℏ.
  
  由 法则 41 (epistemic humility):
    几 (几观 之 minimum observable distinction) 在 量子 scale 是 ℏ-class.
  
  由 法则 21 (实虚 modal):
    measurement 显 specific 实 outcome,
    剩 之 虚 amplitudes uncertain.
  
  unsynchronized measurement:
    Δx · Δp ≥ ℏ/2.
  
  此 是 量子 之 fundamental 限 — 法则 41 (humility) explicit.
  
  ∎
```

### Limitations

```
量子力学 之 coverage:
  非 relativistic (need QFT for high energies).
  measurement problem still debated (interpretations differ).
  consciousness role unsettled.
  
法则 41 (humility): 量子 不 final theory.
  QFT generalizes (Theory 18).
  Quantum gravity beyond.
```

## Theory 3: 进化论 (Evolution Theory)

### Overview

```
进化论 articulates *biological diversity* through 变异 + 选择 + 遗传.
Darwin (1859), Modern Synthesis (1942).
species emergence, adaptation, speciation.
```

### 4-axis coordinate

```
面: 因果 (selection causation) + 时刻 (cumulative 时间) + 系统 (ecology)
aspect: 物 (organisms) + 動 (process) + 事 (mutations / speciation events) + 間 (ecological relations)
modality: 显 (observable populations)
scale: across (gene → individual → population → species → ecosystem)
generative: 生生 (生灭 cycles drive evolution)
```

### 法则 引用

```
法则 1   — 4 aspects: organisms + dynamics + events + ecology.
法则 7   — 反馈律: selection feedback shaping populations.
法则 13  — 涌现非平凡: speciation as non-trivial emergence.
法则 14  — 化生时间: evolutionary time as 化 之 iteration.
法则 49  — 生生 universality: life sustains through generation.
法则 50  — cross-scale: gene to ecosystem patterns.
法则 51  — cross-cultural: 易传 之 生生 ≡ Darwinian generation.
```

### Articulation

```
变异 (variation):
  random mutations in genes.
  此 是 因 之 stochastic factors.

选择 (selection):
  environmental pressures.
  此 是 缘 (enabling conditions, 法则 15).
  环境 selects which 变异 sustain.

适应 (adaptation):
  selected outcomes.
  此 是 果.
  
  完整 articulation:
    变异 (因) 致 适应 (果) 必 经 选择 (缘).
    法则 15 (因缘果链) explicit.

speciation:
  new species emerge from accumulating adaptations.
  此 是 法则 13 (涌现非平凡):
    species 不 reduce to gene sum.
    species emergent.

ecology:
  organisms ⇄ environment mutual constitution.
  此 是 法则 27 (心-territory mutual) generalized to organism-environment.

extinction:
  失 适应 → 失 生生 → 灭.
  法则 61 (3-stage degradation):
    species 健康 → 生态 stress (混) → critical decline (沌) → extinction (灭).

生生 universality:
  life sustains through 繁殖 cycles.
  此 是 法则 49 explicit at biological scale.
```

### Cross-frame

```
因果 ↔ 时刻 frame:
  causation cumulates through deep time.
  此 是 法则 45 (frame-dual derivative-integral):
    selection (factor 因果, instantaneous) ⇄ 史 (accumulated evolutionary record).

因果 ↔ 系统 frame:
  selection ↔ ecological dynamics.
  organism 系统 ⇄ ecological 环境 mutual.

因果 ↔ 式量 frame:
  fitness functions, evolutionary game theory.
  selection as functional gradient on phenotype space.

法则 53 verification:
  evolution articulable in causal / temporal / systems / functional frames.
  same biological predictions.
```

### Attestation

```
古典中文:
  易传 之 生生: precursor to evolution thinking.
  庄子 "万物皆出于机, 入于机": cycling generation.
  
近代:
  Lamarck: early evolutionary speculation.
  Darwin (1859): Origin of Species — 机制 articulation.
  Wallace: independent co-discoverer.
  Modern Synthesis (1942): Mendelian genetics + Darwinian selection.

现代:
  evo-devo: developmental biology + evolution.
  extended evolutionary synthesis: niche construction, etc.

framework attestation:
  进化 是 生生 之 biological articulation.
  framework explains why 生生 universal at biological scale.
```

### Wenyan derivation 例

```
goal: speciation event articulation.

derivation:
  公理: 系统 即 因果互化 也.
  
  考虑 population P 之 organisms.
  
  由 法则 49 (生生 universality):
    P 在 生生 dynamic.
  
  时间 流 (法则 14 化生时间):
    P → P' → P'' → ... 通过 reproduction cycles.
  
  variation accumulates:
    每 cycle 中 random 变异 produced.
  
  selection (缘):
    environment selects which 变异 sustain (法则 15).
  
  by 法则 13 (涌现非平凡):
    after sufficient accumulation,
    P 与 ancestor 之 generation 已 不 reproduce-compatible.
    此 是 speciation event.
  
  此 emergence 不 reduce to genes alone — 是 emergent species property.
  
  此 是 进化 之 articulation in framework.
  
  ∎
```

### Limitations

```
法则 41 (humility):
  evolution articulates biological diversity.
  cosmological / chemical evolution different domains.
  cultural evolution analogous but different (memes vs genes debated).
  
controversial extensions:
  group selection.
  evolutionary psychology (Theory 16).
  cultural evolution.
  
boundary: applies where 复制 + 变异 + 选择 mechanisms operate.
```

## Theory 4: 阳明心学 (Wang Yangming)

### Overview

```
阳明心学 articulates *修身* through 心 之 cultivation.
Wang Yangming (1472-1529): 心外无物, 知行合一, 致良知.
focal claim: 心 是 framework 之 active substrate.
```

### 4-axis coordinate

```
面: 知行 + 体用 + 系统 (心 ⇄ territory)
aspect: 心 + 事 (mind + events / actions)
modality: 显 (cultivated practice)
scale: individual (with 修齐治平 to social)
generative: 修 持续 (continuous cultivation as 生生)
矫正: 修 即 矫正
```

### 法则 引用

```
法则 8   — adjoint pairs: 觉 ⊣ 感, 知 ⊣ 修 etc.
法则 10  — 知行合一: Wang's core commit (explicit 法则).
法则 17  — 心 不可外化.
法则 27  — 心-territory mutual: "心外无物".
法则 30  — 性-几 connection: 良知 之 universal.
法则 47  — 科学 method ≡ epistemic chain (此 chain align with 致良知 path).
法则 60  — 三 names: 常识 ≡ 知行合一.
```

### Articulation

```
心外无物:
  Wang 之 strong claim.
  framework articulation (法则 27): 心 ⇄ territory mutual constitution.
  无 心: territory 不 manifest (to 心).
  无 territory: 心 之 mapping vacuous.
  
  Wang 不 deny territory's external existence.
  he denies its articulation independent of 心.
  此 phenomenological reading consistent with framework.

心外无理:
  Wang 之 stronger claim.
  framework articulation (法则 30): 性 in 心 specialized to 良知.
  理 之 articulation 必 通过 心 之 articulation.
  无 心: 理 articulating to 心 (vacuous 命题).

知行合一:
  Wang 之 most famous commit.
  法则 10 explicit framework commit.
  
  Wang: "知是行之始, 行是知之成".
  framework: 知 ⊣ 修, 修 specializes 行.
  二者 不可分.
  
  failure modes (already in framework):
    知 不 行: 不 真知 (not really knowing).
    行 不 知: 不 真行 (not really acting consciously).

致良知:
  Wang 之 cultivation method.
  framework articulation (epistemic chain 法则 46):
    格物 (显 + 感): observe 万物.
    致知 (感 + 见 + 知): cognize good.
    穷理 (知 + 理): articulate 良知.
    见道 (理 之 fixed point): 良知 即 道.
  
  良知 是 性 之 specialization in 心 (法则 30).

事上磨炼:
  Wang 之 practice method.
  cultivation through 事 (events), not just meditation.
  framework: 心 之 cultivation 必 in territory engagement (法则 27 mutual).
```

### Cross-frame

```
知行 ↔ 体用 frame:
  知 (体-aspect of cognition) ⇄ 行 (用-aspect of cognition).
  Wang 之 知行 ≡ 体用 之 specific reading.

知行 ↔ 因果 frame:
  知 (内 因) ⇄ 行 (外 果).
  cognition causes action; action validates cognition.

知行 ↔ 名实 frame:
  知 (名 之 cognition) ⇄ 行 (实 之 engagement).
  名实合一 之 知行 reading.

法则 53 verification:
  Wang's derivations in 知行 frame applicable in 体用 / 因果 / 名实 frames.
  same cultivation insight.
```

### Attestation

```
古典中文:
  大学 (前阳明 base): "格物致知, 诚意正心, 修身齐家治国平天下".
  论语: 子曰 "知之为知之, 不知为不知, 是知也" — 知行 union.
  孟子: 良知概念 source.
  
阳明:
  传习录: 弟子 编集 之 dialogues.
  大学问: 阳明 之 systematic articulation.
  四句教: "无善无恶心之体, 有善有恶意之动, 知善知恶是良知, 为善去恶是格物".

后阳明:
  王畿, 王艮 et al.
  日本: 王阳明's 学派 in Edo period (近江 圣人 中江藤树).
  现代: enactivism (Varela, Maturana) — cognition through action.

framework attestation:
  Wang 之 commits 完美 align with 法则 10, 27.
  framework explains Wang's deep insight: 心 是 substrate, mutual with territory.
```

### Wenyan derivation 例

```
goal: 知行合一 之 framework derivation.

derivation:
  公理: 系统 即 因果互化 也.
  
  考虑 心 之 cognition X 关 territory event E.
  
  由 法则 17 (心 不可外化):
    心 是 mapping bundle, no self.
    心 之 cognition 必 mapping activity.
  
  by 法则 27 (心-territory mutual):
    心 之 mapping E 必 mapping *actual* E.
    actual E 必 manifest *to* 心.
    二者 mutual.
  
  by 法则 8 (adjoint pairs):
    心 之 觉 (cognition) ⊣ 心 之 感 (mode).
    心 之 知 ⊣ 心 之 修.
  
  but 修 必 通过 行 (action) actualize.
  即 知 ⊣ 修 ⊣ 行.
  
  conclusion: 知 不 separate from 行 — 二者 adjoint coupled.
  此 即 知行合一.
  
  failure modes:
    若 知 而 不 行: adjoint relation broken → 知 失 substance → not 真知.
    若 行 而 不 知: adjoint relation broken → 行 失 awareness → not 真行.
  
  ∎
```

### Limitations

```
阳明 之 coverage:
  individual cultivation primary focus.
  社会 / cosmic scales 通过 修齐治平 chain (法则 50 cross-scale).
  
拓展:
  modern enactivism extends to cognitive science.
  现代心理学 之 some aspects align (embodied cognition).
  but 阳明 之 normative dimension (致良知) more than cognitive science.

framework: 阳明 是 framework 之 心-frame articulation, full coverage.
```
