# Tradition 1: 中国传统 (Chinese Tradition)

## 1.1 Overview

```
中国传统 articulates *道* — the Way — as foundational reality.
spans millennia: 易经 (商周) → 易传 (战国汉) → 老庄道家 (战国) → 儒家 (春秋) → 玄学 (魏晋) → 理学 (宋明) → 现代.

framework's 古典 base draws extensively from Chinese tradition.
```
