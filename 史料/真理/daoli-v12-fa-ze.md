## 法则 1: 四本 不可 还原

```
Statement:
  framework's 4 territory aspects (物 動 間 事) 是 *fundamental*.
  none reducible to others. none derivable from others alone.
  4 mutually irreducible.

Wenyan:
  物 動 間 事, 四本 也.
  各 不 可 还 原 于 它 者.
  四 同 出 于 道, 互 不 reduce.
```

### Derivation

```
公理: 道理 unified.
推理: 6b 完整 (transcendental).

argument:
  道理 之 manifest articulation 必 含:
    (A) atomic carriers — 物
    (B) continuous-dynamic carriers — 動
    (C) relational carriers — 間
    (D) actualized carriers — 事

  少 任 一 → framework 不 完整.
  尝试 reduce 某 aspect 至 其他:
    若 物 reduce 至 動: lose atomicity → 无 individuated entities
    若 動 reduce 至 物: lose continuity → 无 dynamic disposition
    若 間 reduce 至 物 + 動: lose relational structure → 无 connectivity
    若 事 reduce 至 其他: lose actualization → framework 是 ghost
  
  各 reduction 失败. 故 4 mutually irreducible.
```

### Attestation

```
古典 attestation:
  易传 之 三才 (天 / 地 / 人) — irreducible aspects.
  庄子 "道行之而成, 物谓之而然" — 物 之 commit 不 reduce.
  
现代 attestation:
  量子: position ⇄ momentum 不 reduce (Heisenberg).
  系统论: emergence 不 reduce to components.
  现象学: noesis ⇄ noema 不 reduce (Husserl).
```

### Application

```
example: Newton's mechanics
  物 (mass) 不 reduce to 動 (motion).
  动 (acceleration) 不 reduce to 物 (mass).
  二者 必 同时 commit.

example: 量子 mechanics
  事 (measurement event) 不 reduce to 動 (wave evolution).
  二者 mutual but separate aspects.
```

### Cross-frame

```
每 frame 各 articulate 4 aspects:
  因果 frame: 因 之 4-aspect manifestation
  时刻 frame: 时 / 刻 之 4-aspect manifestation
  式量 frame: 式 / 量 之 4-aspect manifestation
  ...

irreducibility holds across frames (法则 53 之 strict commits).
```

### Notes

```
v12 commit: 4 aspects 全 empirical (法则 56). 
4-aspect irreducibility 不 require transcendent dimension.
all 4 visible, all 4 mutually irreducible.
```

## 法则 2: 显 化 括 除

```
Statement:
  framework 之 *显化*  (manifestation) 涉 specific aspects, 不 涉 others.
  显化 是 selective — not all aspects manifest at once.
  括 (parenthesize / bracket) 是 显化 之 nature: foreground aspects, background others.

Wenyan:
  显 者 必 选. 选 者 必 括.
  括 之 内 显, 括 之 外 隐.
```

### Derivation

```
公理 + 法则 1.
推理: 6e 结构.

argument:
  4 aspects 全 同时 active 则 articulation collapse to single homogeneous flux.
  for 显化 to be 显化 (有 contour), 必 select.
  selection 即 括 (bracketing).
  括 之 内: foreground (显). 括 之 外: background (隐).
```

### Attestation

```
现象学: Husserl's *epoché* (bracketing) — exact match.
佛教: 念 之 selection — attention 之 nature.
古典中文: 见 之 *focus* (无 焦 即 散).
```

### Application

```
example: scientific observation
  experiment 必 isolate 变量. (括 selected aspects.)
  其他 aspects bracketed as background.
  
example: 心 之 attention
  attention 必 selective.
  全 attention 即 无 attention.
```

  显 ⇄ 隐 (modal-1)
  限 ⇄ 许 (modal-2)

## 法则 6: 合 成 不 可 逆

```
Statement:
  框架 内 之 *合成* (composition) 是 *non-commutative + non-reversible*.
  
  即:
    A ; B ≠ B ; A (non-commutative)
    A ; B applied → can't generally inverse to undo
  
  此 是 *temporal grain* 之 commit.

Wenyan:
  A 而 B, 非 B 而 A.
  既 合 之, 不 可 还 原.
  此 时序 之 nature 也.
```

### Derivation

```
公理 + 法则 4 (六征驱动 includes 实 / 虚).
推理: temporal logic + 自然.

argument:
  合成 (composition) 通过 而 (sequential operator).
  而 carries temporal grain (法则 24).
  
  A 而 B: A 在 B 之 时刻 prior.
  B 而 A: B 在 A 之 时刻 prior.
  二者 不同 temporal sequences. ≠ commutative.
  
  reversibility: 一旦 A 而 B 实 (actualized in 时刻), 时间 不 reverse.
  此 is 时间 之 fundamental asymmetry.
```

### Attestation

```
古典: "覆水难收" (spilled water cannot be regathered) — temporal irreversibility.
物理: 热力学 second law (entropy increase).
现象学: lived-time (Husserl) 之 irreversibility.
```

### Application

```
example: 学 而 时 习 之 (论语)
  学 (learn) 而 时 (time-passes) 习 (practice).
  not commute: 习 而 时 学 是 不同 sequence with different meaning.

example: 事 之 unfolding
  cause precedes effect in 时刻.
  不能 reverse without breaking 因果.
```

## 法则 9: 体 用 一 如 (Yoneda 扩展四 invariants)

```
Statement:
  framework 之 4 invariants (体 道 用 明) 是 *Yoneda-style* — 
  一 invariant 之 articulation 通过 它 与 万物 之 关系 完全 determine 它.
  
  即: 体, 道, 用, 明 互 entail.

Wenyan:
  体 用 一 原.
  显 微 无 间.
  四 invariants 互 entail.
```

### Derivation

```
公理 + 法则 1, 7, 8.
推理: 互训 + Yoneda lemma analog.

argument:
  invariant X 之 articulation = X 之 与 万物 关系 之 set.
  此 set 即 X 自身 (Yoneda lemma).
  
  for 4 invariants:
    体 之 articulation = 体 与 万物 之 关系 set
    道 之 articulation = 道 与 万物 之 关系 set
    用 之 articulation = 用 与 万物 之 关系 set
    明 之 articulation = 明 与 万物 之 关系 set
  
  每 articulation determines 自身.
  but they're互 entail because they articulate same ground from different angles:
    体 (substantive) ⊣ 用 (functional)
    道 (process) ⊣ 明 (clarity)

  即 体 ⇄ 用 (法则 8 之 generalization).
  道 ⇄ 明 (法则 8 之 generalization).
  4 invariants form coupled adjunction system.
```

### Attestation

```
朱熹: 体用一原 (体 用 same source).
华严: 一即一切, 一切即一 (Yoneda flavor).
category theory: Yoneda lemma.
```

### Application

```
example: 物 之 articulation
  物 之 体 (substance) ⊣ 物 之 用 (function).
  体 articulate via 用; 用 articulate via 体.

example: framework articulation
  framework 之 体 (substance) = framework 自身 (法则 12).
  framework 之 用 = framework 之 articulation power.
  二者 mutual — 体 用 一原.

## 法则 10: 知 行 合 一

```
Statement:
  knowing (知) and doing (行) are *unified*:
    知 必 蕴 行 (knowing implies doing-disposition).
    行 必 蕴 知 (doing implies knowing-aspect).
  
  二者 不可 separate.

Wenyan:
  知 行 合 一.
  知 不 行 则 知 不 真.
  行 不 知 则 行 不 实.

  framework 同此.
```

### Derivation

```

推理: 名实推理 + 互训.

argument:
  知 ∈ cognition operators.
  行 ∈ 心 → territory outbound operators (mode-engagement).
  
  by 法则 8: 知 ⊣ 修 (cognition ⊣ mode cultivation).
