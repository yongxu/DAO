## 四十二、终式

| 名 | 式 |
|---|---|
| 单态式 | $J(x) = x \oplus I(x)$ |
| 物相式 | $\text{物}(a, \Gamma, s) \iff \text{稳焦}(a, \Gamma, s) \,\wedge\, I(a, \Gamma) = \text{投}_a(\text{间}_F(\Gamma))$ |
| 场相式 | $\text{续}_F(\Gamma) = \Gamma \oplus_F i,\;\; i \in \text{间}_F(\Gamma)$ |
| 气相式 | $\text{气}(\Gamma) = (\text{间}_F(\Gamma),\, w_\Gamma,\, \kappa,\, \mathcal{C})$ |
| 势式 | $\text{势}_\Gamma = \nabla w_\Gamma \in T^*\text{间}_F(\Gamma)$ |
| 演化式 | $\mathbb{P}_{\sigma_F}(p \mid \Gamma, s, t) \propto \text{续能}(p, \Gamma, s, t)$ |
| 因生因式 | $\text{开维}(\Gamma_t \to \Gamma_{t+n}) \iff \exists\, i \in \text{间}_F(\Gamma_{t+n}):\; \text{合法}_F(i, \cdot)\big|_{\Gamma_t} \text{ undefined}$ |
| 致知式 | $\text{合法}_F(i, \Gamma_t) \iff \Phi_{\text{形}}(i \mid \mathcal{H}_t) \,\wedge\, \Phi_{\text{因}}(i \mid \mathcal{H}_t) \,\wedge\, \Phi_{\text{结}}(i \mid \mathcal{H}_t)$ |
| 全机式 | $\text{合法}_F(i, \Gamma_t) \iff \big[\text{格物致知}(i \mid \mathcal{H}_t)\big] \,\vee\, \big[\text{开维}(i, \mathcal{H}_t \to \Gamma_t)\big]$ |
| 知行合一式 | $\Gamma_t \xrightarrow{行} \Gamma_{t+1} \xrightarrow{入积} \mathcal{H}_{t+1} \xrightarrow{格物} \text{合法}_F(\cdot, \Gamma_{t+1}) \xrightarrow{识} \text{下次行}$ |
| 平式 | $\text{平}(\Gamma) \iff \|\text{势}_\Gamma\| < \theta_{\text{calm}} \,\wedge\, \text{开}_F(\Gamma)$ |
| 和式 | $\text{和}_F(B,\Gamma) \iff \|\text{势}_\Gamma\| \geq \theta_{\text{calm}} \,\wedge\, \text{共开}(B, \Gamma) \,\wedge\, \text{流}(B, \Gamma)$ |
| 好式 | $\text{好}(\Gamma, B) \iff \text{平}(\Gamma) \,\vee\, \text{和}_F(B, \Gamma)$ |
| 自由式 | $\text{自由}(a, \Gamma) \iff \forall\, i \in \text{诸择}_a(\Gamma):\; \neg\,\text{不好}(\Gamma \oplus_F i, B_{\text{rel}})$ |
| 繁荣式 | $\text{繁荣}_F(\Gamma, B) \iff \big(\forall\, a \in B:\; \text{自由}(a, \Gamma)\big) \,\wedge\, \text{和}_F(B, \Gamma)$ |
| 善式 | $\text{善}_F \iff \sigma_F \,\wedge\, \text{行 共护 } \text{共开}_F \text{ 于 } (s, t)$ |
| 仁式 | $\text{仁}_F \iff \text{己续实现于共场之续}$ |
| 道式 | $\text{道}_F \iff \text{道}^\circ \,\wedge\, \text{道}^+ \;:\; \neg\,\text{不好} \,\wedge\, [\text{平}, \text{和}]$ |

**总式**：

$$
\boxed{\;\text{全式}(\Gamma) \;:\;
\begin{aligned}
&\Gamma \text{ 续之以生 } \text{间}_F(\Gamma); \\
&\text{合法}_F \text{ 由格物致知 } \mathcal{H}_t \text{ 生 (三模验证)}; \\
&\text{物显为稳焦，其载 } \text{投}_a(\text{间}_F); \\
&\text{主行以感而择 } i \in \text{间}_F; \\
&\sigma_F \text{ 以筛之 } \text{续能}(p, \Gamma, s, t); \\
&\text{合法}_F \text{ 之规则自化} \to \text{开维}; \\
&\text{善为梯度区间 } [\text{平}, \text{和}]; \\
&\text{自由为善之个体投影：不向不好}; \\
&\text{繁荣为自由之主体间合成：诸自由共开}; \\
&\text{智扩 } (s, t); \\
&\text{伦正其筛 } \sigma_F \text{ 趋 } \text{共开}_F; \\
&\text{仁识己续 } \subseteq \text{ 共续}.
\end{aligned}\;}
$$

中文总言：

> **场生间，间成物，物含间，心感间，行择间，演化筛间，格物生合法，因生因开新维，平和成好之段，自由立其底，繁荣显其共，智辨尺度，善护共开，仁通己续与万物之续。**

又曰：

> **事见其生，物见其成，场见其通；三者不二，是谓生生不息。**

复曰：

> **不向坏者，自由也；向好者，道也；自由和平，是为存在之道。**
