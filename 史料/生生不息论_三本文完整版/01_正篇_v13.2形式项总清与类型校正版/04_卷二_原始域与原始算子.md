## 卷二 · 原始域与原始算子

### 二一、基本论域

$$
\begin{aligned}
\mathfrak{F} &:= \text{场态域}\\
\mathcal{A} &:= \text{聚焦域}\\
\mathcal{M} &:= \text{间内容域}\\
\mathcal{R} &:= \text{关系域}\\
\mathcal{S} &:= \text{度域}\\
\mathcal{W} &:= \text{期域}\\
\mathcal{K} &:= \text{间域结构域}\\
\mathcal{L} &:= \text{谓词格或判准格}\\
\mathbb{T} &:= \text{时域}
\end{aligned}
$$

场态：

$$
\Gamma=(A,R,\sigma,\rho,\mathcal{C})
$$

其中：

$$
A\subseteq\mathcal{A},\quad R\subseteq\mathcal{R},\quad \sigma:A\to X,\quad \rho:R\to Y
$$

解释：

- $A$：场中可焦者；
- $R$：焦与焦之间之关系与超关系；
- $\sigma$：焦之态赋值；
- $\rho$：关系态赋值；
- $\mathcal{C}$：约束与积，含资源、法、界、注意、噪声、迹、制度条件。

### 二二、原始算子表

下列为模型接口，不强行字元化：

| 算子 | 定用 |
|---|---|
| $I_F$ | 场之合法间域 |
| $K_\Gamma$ | 间域结构 |
| $w_\Gamma$ | 间权重 |
| $D_{K_\Gamma}$ | 结构差分 |
| $\|\cdot\|_{K_\Gamma}$ | 势强度 |
| $\oplus_F$ | 生合算子 |
| $\kappa$ | 间间耦合 |
| $\kappa^\ast$ | 集合耦合 |
| $\Omega$ | 开度合成 |
| $\Omega_B$ | 共开度合成 |
| $\mu$ | 度期权重 |
| $\Pr$ | 率 |
| $\Phi_{\mathrm{形}},\Phi_{\mathrm{因}},\Phi_{\mathrm{结}}$ | 史法三据接口 |
| $\Pi_{\mathrm{开}}$ | 开势投影或开度判准接口 |

> **原始算子非黑箱；虽不由字生，亦须受校。**


### 二三、原始接口契约

原始算子不由字元生成，但须有契约。契约不足，则其输出只能入待校。

| 算子 | 签名 | 必守约束 | 败式 |
|---|---|---|---|
| $I_F$ | $\mathbf{F}\times\mathbf{S}\times\mathbf{W}\to\mathcal{P}(\mathbf{M})$ | 输出皆为候间且合法 | 含非法间、漏显要间 |
| $K_\Gamma$ | $\mathbf{F}\to\mathcal{K}$ | 给出邻接、距离、可比结构 | 无结构而言势 |
| $w_\Gamma$ | $\mathbf{M}\to\mathbb{R}_{\ge0}$ | 权重来源可查、可随期变 | 权源不可校 |
| $D_{K_\Gamma}$ | $(\mathcal{K},w)\to\text{差分场}$ | 只比较同结构中可比之间 | 跨型比较 |
| $\|\cdot\|_{K_\Gamma}$ | 差分场 $\to\mathbb{R}_{\ge0}$ | 强度与结构相容 | 强度不可复算 |
| $\oplus_F$ | $\mathbf{F}\times\mathbf{M}\rightharpoonup\mathbf{F}$ | 仅对合法间有定义，且留迹 | 无迹生合 |
| $\kappa$ | $\mathbf{M}\times\mathbf{M}\to[-1,1]$ | 明示助、碍、无关 | 耦合不可查 |
| $\kappa^\ast$ | $\mathcal{P}(\mathbf{M})\to\text{耦合结构}$ | 集合耦合可分解至间间耦合 | 聚合失真 |
| $\Omega$ | $[0,1]^4\to[0,1]$ | 单调、短板敏感、可解释 | 一项归零仍称强开 |
| $\Omega_B$ | 群体开度资料 $\to[0,1]$ | 单调、反夺、存异敏感 | 以均值掩夺 |
| $\mu$ | 路径集或间集 $\to\mathbb{R}_{\ge0}$ | 度期显式、归一规则明示 | 度期漂移 |
| $\Pr$ | 事件 $\to[0,1]$ | 样本、模型或主观先验须标明 | 率源不明 |
| $\Phi_{\mathrm{形}},\Phi_{\mathrm{因}},\Phi_{\mathrm{结}}$ | 历史迹 $\to\mathbf{V}$ | 形、因、结三据可分查 | 以一据冒三据 |
| $\Pi_{\mathrm{开}}$ | $\mathbf{F}\times\mathbf{S}\times\mathbf{W}\to\text{开判准}$ | 与 $\Omega$ 或开判定相容 | 投影与定义相悖 |

文言：

> **算子可原始，不可任意；接口可抽象，不可无约。**

### 二四、类型别名与偏函数

偏函数记作：

$$
X\rightharpoonup Y
$$

表示非处处有定义。凡用偏函数，须列明有定义条件。

例：

$$
\oplus_F:\mathbf{F}\times\mathbf{M}\rightharpoonup\mathbf{F},\qquad
\mathrm{dom}(\oplus_F)=\{(\Gamma,i)\mid i\in I_F(\Gamma;s,\omega)\}
$$

若输入未满足定义域，则输出为 $\mathsf{U}$，不得强作 $\bot$。
