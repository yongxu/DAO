## 〇六、审校数据结构

审校数据不是附属说明，而是判定的一部分。最小事件记录为：

$$
e
:=
(
\mathrm{id},
t,
s,
\omega,
a,
B,
\Gamma^-,
i,
\Gamma^+,
\mathsf{claim},
\mathsf{trace},
\mathsf{response},
\mathsf{modelCheck},
\mathsf{outcome},
\mathsf{source}
)
$$

字段表：

| 字段 | 义 |
|---|---|
| $\Gamma^-$ | 行动前场态 |
| $i$ | 所选间或行动 |
| $\Gamma^+$ | 行动后场态 |
| $\mathsf{claim}$ | 本次判定声称，如正行、邪行、护闭、义、善 |
| $\mathsf{trace}$ | 可复查迹 |
| $\mathsf{response}$ | 受者回应 |
| $\mathsf{modelCheck}$ | 多模型互证结果 |
| $\mathsf{outcome}$ | 后果与度期周 |
| $\mathsf{source}$ | 数据来源与独立性 |

### 审校分

定义审校质量：

$$
Q_{\mathrm{audit}}(e)
:=
\min
\left(
Q_{\mathrm{迹}},
Q_{\mathrm{众}},
Q_{\mathrm{互}},
Q_{\mathrm{源}},
Q_{\mathrm{期}}
\right)
$$

其中：

| 分量 | 对应 |
|---|---|
| $Q_{\mathrm{迹}}$ | 迹查是否可复算 |
| $Q_{\mathrm{众}}$ | 受者回应是否足够覆盖 |
| $Q_{\mathrm{互}}$ | 多模型是否互证 |
| $Q_{\mathrm{源}}$ | 来源是否独立、非同执 |
| $Q_{\mathrm{期}}$ | 是否覆盖相关度期 |

若：

$$
Q_{\mathrm{audit}}(e)<\theta_{\mathrm{审}}^-
$$

则相关价值判定降为：

$$
\mathsf{U}
$$
