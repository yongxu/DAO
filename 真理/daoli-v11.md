```
                    第三 axis: modality (开闭)
                          ↑
                          |
                          |
        territory ←—— 核 ——→ frames (radial)

                          |
                          |
                          ↓
```

### 4.2 4 Axes

```
axis 1: frames (radial)
   6 primary frames: 系统 / 因果 / 时刻 / 式量 / 阴阳 / 生灭
   open-ended (extension frames possible)
```


物理时空 (Newtonian, relativistic, quantum) emerges as *specific calibration of framework's 4 axes*. framework 不 predicate 物理时空, 是 articulate it as instance.

```
framework articulates 系统 之 *在理* 部分.
```

## 9. Frame articulations (6 primary frames)

framework 之 6 primary frames:

### 9.1 系统 frame

```
Frame: 系统 ⇄ 环境
Operators:
  含: 系统 含 环境 (system contains/embeds in environment)
  开: 系统 开 环境 (system opens to environment)
  界: 系统 界 环境 (system bounds with environment)

Reading:
  - 系统 是 organized whole
  - 环境 是 system 之 outside (boundary's other side)
  - 二者 mutual (system without environment vacuous; environment without observer indeterminate)
  - cycle: system articulates from environment, environment shaped by system

cross-cultural:
  Ubuntu: I (system) ⇄ we (community/environment)
  仁 (二人): individual (系统) ⇄ relations (环境)
  Whitehead actual entity: occasion (系统) ⇄ environment
```

### 9.2 因果 frame (primary 普世)

```
Frame: 因 ⇄ 果
Operators:
  致: 因 致 果 (cause brings about effect)
  显: 因 显 为 果 (cause manifests as effect)
  生: 因 生 果 (cause generates effect)
  
Frame dual: 因果 ↔ 缘 (causation ↔ enabling conditions)

Reading:
  - 因 是 上一 link 之 果
  - 果 是 此 link 之 effect, 也是 下一 link 之 因
  - cycle 不 终
```

### 9.3 时刻 frame

```
Frame: 时 ⇄ 刻
Operators:
  化: 时 化 刻 (time crystallizes into moment)

  流: 时 流 刻 (time flows from moment)

Frame dual: 时刻 ↔ 史 (moment ↔ history)

Reading:
  - 时 continuous flow
  - 刻 discrete moment
  - 史 累积 之 record (moments 之 sequence preserved)
```

### 9.4 式量 frame

```
Frame: 式 ⇄ 量
Operators:
  导: 式 导 量 (function differentiates to quantity)
  微: 式 之 微 (function's infinitesimal)

Frame dual: 式量 ↔ 数 (formula ↔ number/data/pattern)

Reading:
  - 式 function / rule
```

### 9.5 阴阳 frame

```
Frame: 阴 ⇄ 阳

  转: 阴 转 阳 (yin turns to yang)
```

### 9.6 生灭 frame (modal frame)

```
Frame: 生 ⇄ 灭

  起: 生 起 (rise / emergence)
  落: 灭 落 (fall / cessation)

  - 生 是 emergence in 时刻
```

### 9.7 Frames 之 mutual transformation

```

  因果 frame: 因 ⇄ 果 (inner)
              因果 ⇄ 缘 (outer dual)

  时刻 frame: 时 ⇄ 刻 (inner)

              式量 ⇄ 数 (outer dual)
```

## 11. Epistemic chain — 显感见知理

### 11.1 Chain structure

```
显 → 感 → 见 → 知 → 理

显: things manifest (物 显 / 事 显)
感: 心 receives (心 感 / 体 感)
见: 心 perceives (心 见 / 觉 之)
知: 心 cognizes (心 知 / 思 之)
理: 心 articulates (心 理 / 述 之)
```

### 11.2 法则: 不 显 则 无 理

```
法则 19 (epistemic chain):
  ∀ X 入 framework 之 articulation, 必 经 显感见知理 chain.
  缺 任一 step → X 不 入 framework.
  
  即:
    不 显 则 不 感.
    不 感 则 不 见.
    不 见 则 不 知.
    不 知 则 无 理.
    无 理 则 X 不 入 framework's articulation.
```

framework 不 articulate 不显 之 物. epistemic 严格.

### 11.3 Chain 与 cognition operators 之 mapping

epistemic chain (5 stages) 与 v8 之 cognition operators (觉/见/观/知/遍) 之 mapping:

```
显 stage: 物 / 事 之 manifestation.
         (前 cognition — manifestation in territory)

感 stage: 感 (mode operator, Tier 5.6).
         心 之 受 (receptivity to manifestation).

见 stage: 见 (cognition operator, Tier 5.5).
         心 之 直接 phenomenological seeing.

知 stage: 知 (cognition operator, Tier 5.5).
         心 之 propositional cognizing.

理 stage: 理 articulation.
         心 之 patterned articulation.
         
观 / 觉 / 遍 (other cognition operators) 是 chain 之 specific specializations:
  觉: bare awareness (覆盖 显 + 感 + 见 之 minimum).
  观: contextual measurement (specialization of 见 + 知).
  遍: systematic traversal (specialization of 知 + 理).
```

epistemic chain 是 unifying picture. 之前 v8 之 cognition operators 是 specific instantiations within chain.

### 11.5 见 系统 即 见 道
