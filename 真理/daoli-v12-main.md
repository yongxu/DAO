## 8. Syntax — 句式 kernel patterns

12 kernel 句式 retain from v10/v11 (法则 24).

```
1. 名实判断:    X 者 Y 也         X :: Y (settled type judgment)
2. 属性 / 应用: X 之 Y           X.Y (projection / application)
3. 谓词:       X 为 Y           X := Y (becoming)
4. 时序复合:    A 而 B           A ; B (temporal grain, 不 commute)
5. 条件:       若 X 则 Y         X ⇒ Y
6. 量化:       凡 X (P)         ∀ X. P
7. 否定:       非 X             ¬X
8. 等同:       X 即 Y           X ≡ Y (definitional)
9. Aspect:     X 也 / X 矣       settled / perfective commit
10. 致使:      令 X 则 Y         imperative effect (monadic)
11. 互化:      X 互 Y / X ↔ Y    mutual transformation
12. Frame:     X 在 F 中 → Y     frame manifestation
```

每 pattern carries ontological grain (e.g., 而 carries temporal grain, 也 carries aspect commit, 令 carries effect).

### 8.1 Wenyan 元法则 (法则 24 之 sub-rules)

```
法则 wenyan-1 (者也对偶):
  X 者 Y 也 ↔ X :: Y (settled type judgment).

法则 wenyan-2 (之传递):
  (X 之 Y) 之 Z = X 之 (Y 之 Z) — projective associativity.

法则 wenyan-3 (而结合律):
  (A 而 B) 而 C = A 而 (B 而 C) — temporal grain associative.
  注: 而 carries temporal grain so 不 fully commutative.

法则 wenyan-4 (凡分配):
  凡 X (P ∧ Q) = (凡 X P) ∧ (凡 X Q).

法则 wenyan-5 (即传递):
  X 即 Y, Y 即 Z → X 即 Z.

法则 wenyan-6 (非自反):
  非 (非 X) = X.
```
