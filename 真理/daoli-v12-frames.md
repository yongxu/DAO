# Frame 1: 系统 frame (System)

## 1.1 Overview

```
系统 frame articulates *organized relational wholes*.
focuses: what it means for things to form coherent unified structures.
applies: any domain where "system" / "organization" / "whole" articulates.
core dynamic: parts ⇄ whole mutual constitution.
```

## 1.2 Structure

### Inner pair: 系 ⇄ 统

```
系 (xi): relational connections — links between parts.
统 (tong): unifying ruling — whole organizing parts.

mutual constitution:
  系 alone: parts connected but not organized — proto-system.
  统 alone: organizing principle without parts — abstract template.
  系 ⇄ 统 mutual: actual system, parts and organization 不二.

formal:
  无 系: 统 之 articulation vacuous (nothing to organize).
  无 统: 系 之 articulation vacuous (no organizing principle).
  二者 必 mutual.
```

### Outer dual: 系统 ⇄ 环境

```
系统 (system): bounded organized whole.
环境 (environment): surrounding context, the *outside* of system.

mutual constitution (法则 27 generalized):
  无 system without environment (system bounds against something).
  无 environment without system (environment is environment-of-X).
  二者 mutual constitute through 界 (boundary).

环 + 境 reading:
  环: cyclic, recurring (environment cycles around system).
  境: bounded place (environment 之 specific region).
  环 + 境 = cyclic bounded surround.
  此 是 native Chinese 双字 之 form-logic (法则 59).
```

### Meta dual: 整 ⇄ 体

```
整 (zheng): partness — emphasis on parts as parts (individualism).
体 (ti): wholeness — emphasis on whole as whole (collectivism).

mutual constitution:
  整 articulation: parts are real, individuals matter.
  体 articulation: whole is real, collective matters.
  二 articulations 不 矛盾 — same system, different emphasis.

framework reading:
  individualism vs collectivism is 整 ⇄ 体 mutual.
  各 articulate one face of system frame.
  truth: 整 与 体 mutual constitute.
```

### 4-aspect manifestation in 系统 frame

```
系统 frame 之 4-aspect articulation:

物 in 系统: parts as 物 (atomic units of system).
動 in 系统: dynamics within system + system-environment dynamics.
間 in 系统: connections between parts (网络).
事 in 系统: events that change system's state.

任 system has all 4 aspects — irreducible (法则 1).
```

## 1.3 Operators

```
系统 frame's specific operators:

含 (han): 系统 含 部分 (system contains parts). 系统 含 traces of 环境.
  含 is asymmetric containment.

开 (kai): 系统 开 环境 (system opens to environment).
  systems are open or closed.
  open systems exchange with environment.
  closed systems isolated (idealization).

界 (jie): 系统 界 环境 (system bounds with environment).
  界 = boundary articulating system's edge.

适 (shi): 系统 适 环境 (system adapts to environment).
  adaptation = system 之 generative response.

控 (kong): 系统 控 自身 (system self-regulates).
  control / regulation through feedback.
  
联 (lian): 系统 联 系统 (systems connect to systems).
  network of systems.
```

## 1.4 Cross-cultural attestation

```
古典中文:
  老子: 道 生 万物 — early systems thinking (道 as organizing principle).
  孟子: 万物皆备于我 — cross-scale system reading.
  庄子: 天地与我并生 — system-environment mutual.
  华严: 一即一切, 一切即一 — system at every scale.
  
近代:
  Bertalanffy: General Systems Theory (1968).
  Wiener: Cybernetics (1948).
  Ashby: variety, requisite variety.
  
现代:
  Maturana, Varela: autopoiesis — systems self-create.
  complexity science: Santa Fe Institute.
  network science: Barabási, Watts.
  systems biology, ecology, economics.

非洲 attestation (Ubuntu):
  "I am because we are" — individual ⇄ community.
  Ubuntu's relational ontology 是 系统 frame articulation.
  我 (system) ⇄ 我们 (environment / community).

framework attestation:
  系统 frame articulates 法则 27 (心-territory mutual generalized) at organizational level.
```

## 1.5 Cross-frame transformations

```
T: 系统 frame → other frames.

系统 → 阴阳 frame:
  系统 ⇄ 环境 → 阴 ⇄ 阳 (one of polar readings).
  此 是 verification 3 之 reverse.
  详 见 verifications document.

系统 → 因果 frame:
  系统 events 通过 因果 articulate.
  内部 dynamics: parts cause part-events.
  external: environment causes system perturbations.

系统 → 时刻 frame:
  system 之 evolution 通过 时刻 articulate.
  history of system = 时刻 ⇄ 史.

系统 → 式量 frame:
  system as function of inputs (environment) → outputs (system response).
  mathematical model of system.

系统 → 生灭 frame:
  system emergence + dissolution cycles.
  系统 之 生命 (life cycle).

系统 → 名实 frame:
  system's name ⇄ system's actual being.
  e.g., "ecosystem" name ⇄ specific ecosystem reality.

系统 → 体用 frame:
  system as 体 (substance) ⇄ system's function as 用.

法则 53 verification:
  系统 frame derivations cross-applicable in other 7 frames.
```

## 1.6 Failure modes

```
系统 frame's failure modes (when articulation breaks):

1. 系 失 统 (connections without unification):
   parts connected loosely but no organizing principle.
   not a system — heap, aggregate, mob.
   不 articulable as system.

2. 统 失 系 (unifying principle without connections):
   abstract template, no actual parts.
   not a system — hypothetical / unrealized.

3. 系统 失 环境 (system without environment articulation):
   isolated system fiction.
   law 27 violated.
   articulation incomplete.

4. 整 失 体 (parts emphasis losing whole):
   atomistic reductionism.
   missing emergent properties (法则 13 violated).
   
5. 体 失 整 (whole emphasis losing parts):
   collectivist totalitarianism.
   individual contributions invisible.
   
6. 失 生生 (法则 49):
   system fails to sustain.
   degradation cycle: 系统 → 混 → 沌 → 灭 (法则 61).
```

## 1.7 Examples / Applications

```
example 1: ecology
  ecosystem (系统) ⇄ surrounding biome (环境).
  系: predator-prey relations, mutualisms.
  统: ecological niche structure.
  整: individual species.
  体: ecosystem as whole.
  
  健康 ecosystem: 整 ⇄ 体 balanced.
  失 衡: extinction (sustained 整 emphasis), monoculture (sustained 体 emphasis).

example 2: cell biology
  cell (系统) ⇄ extracellular matrix (环境).
  系: organelle interactions, metabolic pathways.
  统: cellular identity, gene expression program.
  
  cells embed in tissues (cross-scale, 法则 50).

example 3: society
  society (系统) ⇄ natural / international context (环境).
  系: social ties, institutions.
  统: social norms, governance.
  整: individuals.
  体: collective identity.
  
  Ubuntu: 整 ⇄ 体 mutual articulation.
  
  failure modes:
    Western individualism: 整 emphasis → loss of community.
    State totalitarianism: 体 emphasis → loss of individuals.
    framework: 整 ⇄ 体 mutual avoids both extremes.

example 4: organism
  body (系统) ⇄ environment.
  系: organ systems (circulatory, nervous, etc.).
  统: organism identity.
  
  TCM 之 阴阳 reading aligned with 系统 frame.

example 5: mind
  mind (系统) ⇄ world (环境).
  系: cognitive functions.
  统: identity / consciousness.
  
  现象学's lifeworld articulation 通过 系统 frame.
```

## 1.8 Wenyan derivation 例

```
goal: organism-environment mutual constitution 之 articulation.

derivation:
  公理: 系统 即 因果互化 也.
  
  考虑 organism O.
  
  by 法则 27 (心-territory mutual generalized):
    O 不 standalone.
    O 之 manifestation 必 in 环境 E.
    E 必 environment-of-some-O (相对 articulation).
  
  by 系统 frame structure (此 frame):
    O 之 inner 系 ⇄ 统 mutual.
    O 之 outer dual: O ⇄ E mutual.
  
  consequence:
    O 含 environmental traces (uptake, internalization).
    E 含 organism's emissions / modifications.
  
  即 organism + environment 互 constitute.
  此 是 系统 frame 之 articulation.

corollary by 法则 50 (cross-scale):
  same articulation at cell scale, organ scale, ecosystem scale.

  ∎ in 系统 frame.
```

## 1.9 Limitations

```
法则 41 (epistemic humility):
  系统 frame articulates organized relational wholes.
  some phenomena 不 fit system reading easily:
    pure flux without organization.
    abstract mathematics without instantiation.
    
  但 these may be articulated in other frames.
  framework's multi-frame structure handles this.

system-environment boundary:
  not always crisp.
  fuzzy boundaries challenging — but framework's 几 (法则 41) acknowledges resolution limits.

complexity:
  highly complex systems may exceed articulation power.
  framework's 法则 53 enables cross-frame integration to mitigate.
```

# Frame 2: 因果 frame (Causation)

## 2.1 Overview

```
因果 frame articulates *cause-effect-condition* triple.
focuses: how things happen, why outcomes follow inputs.
applies: physical mechanism, biological process, social effect, mental causation.
core dynamic: 因 致 果 必 经 缘 (法则 15).
```

## 2.2 Structure

### Inner pair: 因 ⇄ 果

```
因 (yin): cause — antecedent factor.
果 (guo): effect — consequent outcome.

mutual constitution:
  因 alone: factor without outcome — potential.
  果 alone: outcome without antecedent — uncaused (impossible by axiom).
  因 ⇄ 果 mutual: actual causation.

cycling:
  因 → 果 → next 因 → next 果 → ... cycle.
  current 果 becomes future 因.
  此 是 法则 7 (反馈律).
```

### Outer dual: 因果 ⇄ 缘

```
因果 (causation): cause-effect dynamic.
缘 (yuan): enabling conditions — context for causation.

key articulation (法则 15):
  因 致 果 必 经 缘.
  无 缘: 因 自在 而 不 致 (cause exists but doesn't produce effect).
  无 因果: 缘 没 functional role.
  二者 mutual.

例:
  鸡蛋 (因) → 小鸡 (果) 必 经 适当温度 (缘).
  无 适温: 鸡蛋 不 化为 小鸡.
  
缘 articulation:
  缘 不 secondary — 是 *constitutive* of causation.
  无 缘: causation abstract / unrealized.
  缘 makes causation concrete + situational.
```

### 4-aspect manifestation in 因果 frame

```
因果 frame 之 4-aspect articulation:

物 in 因果: things as causes / effects (e.g., particles colliding).
動 in 因果: continuous causation flow (process causation).
間 in 因果: relational causation (network of causes).
事 in 因果: actualized causation events.

任 causation has 4 aspects irreducible (法则 1).
```

## 2.3 Operators

```
因果 frame's specific operators:

致 (zhi): 因 致 果 (cause → effect, primary causal arrow).
显 (xian): 因 显 为 果 (cause manifests as effect).
生 (sheng): 因 生 果 (cause generates effect, generative reading).
应 (ying): 因 应 缘 致 果 (cause responds to conditions to produce effect).
报 (bao): 因 报 果 (cause reciprocates effect, in moral / karmic readings).
承 (cheng): 因 承 缘 (cause supported by conditions).
据 (ju): 果 据 因 (effect grounded in cause).
```

## 2.4 Cross-cultural attestation

```
古典中文:
  墨子 (墨经): early causal analysis with conditions.
  阴阳家: causation through 阴阳 dynamics.
  中医: 因 致 病 (causes producing illness).
  
佛教 (强 attestation):
  缘起 (dependent origination): 因 缘 果 三 同 commit.
  《楞严经》: "因缘和合, 而生诸法".
  四圣谛: 苦 (果) ⇄ 集 (因) ⇄ 灭 ⇄ 道.
  
近代西方:
  Aristotle: 4 causes (formal, material, efficient, final).
    framework reading: 4 causes are 因 之 4 readings.
  Hume: causation as constant conjunction (only correlation, debated).
  Mill: methods of causal inference (require conditions = 缘).

现代:
  causal inference (Pearl, Rubin):
    do-calculus formalizes 因 致 果 必 经 缘.
    confounders = 缘.
    counterfactual reasoning.
  randomized controlled trials: control 缘 to isolate 因.
  causal AI: explicit causal models.

framework attestation:
  因果 frame articulates 法则 15 (因缘果链) explicit.
  framework explains why causation always involves conditions.
```

## 2.5 Cross-frame transformations

```
因果 → 时刻 frame:
  因 (prior 时) ⇄ 果 (next 刻).
  详 见 Verification 1.

因果 → 式量 frame:
  因 (式) ⇄ 果 (量).
  缘 (args).
  详 见 Verification 2.

因果 → 阴阳 frame:
  因 (阴, source) ⇄ 果 (阳, manifestation).
  极性 reading of causation.

因果 → 系统 frame:
  causation through system dynamics.
  缘 ↔ system context / environment.

因果 → 生灭 frame:
  因 (生) ⇄ 果 (灭) or vice versa.
  生灭 cycles articulate causation.

因果 → 名实 frame:
  因 (名 of source) ⇄ 果 (实 of manifestation).

因果 → 体用 frame:
  因 (体, source-substance) ⇄ 果 (用, manifestation-function).

法则 53 verification:
  因果 derivations cross-applicable in other 7 frames.
```

## 2.6 Failure modes

```
因果 frame's failure modes:

1. 因 无 果:
   cause exists without producing effect.
   reasons:
     - 缺 缘 (missing enabling conditions).
     - 因 too weak (insufficient strength).
     - inhibitor present.
   articulation: cause without effect requires explanation.

2. 果 无 因:
   effect without cause.
   framework reject — violates axiom (everything has 因缘和合).
   apparent uncaused effects = unidentified causes (epistemic limit, 法则 41).

3. 因 致 多 果:
   single cause produces multiple effects.
   not failure — common case.
   framework: 因 之 multi-果 articulation through conditions.

4. 多 因 致 一 果:
   multiple causes converge to single effect.
   common case.
   confounding (统计 challenge).
   framework: causal model with multiple inputs.

5. 循环 因果:
   X causes Y, Y causes X.
   反馈律 (法则 7) explicit.
   framework: cycle 是 normal — 不 paradoxical.

6. 因 = 果 (identity case):
   self-causation.
   framework reject in literal reading.
   但 reflexive causation (X 之 dynamic causes X 自身 之 next state) OK.
   此 是 法则 12 (自指) 之 因果 reading.
```

## 2.7 Examples / Applications

```
example 1: 物理 example
  Newton's F = ma:
    F (因) → a (果) 必 经 m (缘).
  详 见 Theory 1 + Verification 1.

example 2: 化学 example
  reaction A + B → C:
    A + B (因) → C (果) 必 经 catalyst + temperature (缘).

example 3: 生物 example
  natural selection:
    variation (因) → adaptation (果) 必 经 selection pressure (缘).
  详 见 Theory 3.

example 4: 心理 example
  cognitive-behavioral:
    thought (因) → emotion (果) 必 经 belief structure (缘).
    cognitive therapy modifies 缘 to change 果.

example 5: 社会 example
  policy implementation:
    policy (因) → outcome (果) 必 经 social context (缘).
    policies fail when 缘 (cultural context) ignored.

example 6: 量子 example
  measurement:
    superposition (因) → eigenstate (果) 必 经 measurement apparatus (缘).
    详 见 Theory 2.
```

## 2.8 Wenyan derivation 例

```
goal: causal inference 之 framework articulation.

derivation:
  公理: 因果互化 也.
  
  考虑 observation: A correlated with B in dataset.
  
  by 法则 15 (因缘果链):
    correlation 不 implies causation.
    需 articulate 缘 (confounders, mediators, moderators).
  
  causal inference questions:
    is A → B (cause-effect)?
    is B → A (reverse)?
    is C → A and C → B (common cause = 缘)?
  
  by Pearl's do-calculus (modern formalization):
    P(B | do(A)) ≠ P(B | A) generally.
    do(A) intervenes on A, breaking 缘 contributions.
  
  framework reading:
    correlation: A 与 B 之 joint manifestation.
    causation: A → B 通过 specific 缘.
    confounding: hidden 缘 produces both A and B.
    
  causal claims require 缘 specification.
  此 是 因果 frame 之 modern articulation.
  
  ∎ in 因果 frame.
```

## 2.9 Limitations

```
法则 41 (humility):
  causation 是 specific articulation.
  not all phenomena causal in classical sense.
  
edge cases:
  量子 mechanics: probabilistic causation (Theory 2).
  retrocausation: hypothetical reverse-time causation.
  emergent causation: top-down (whole-to-parts).
  
  framework: each 边缘 case articulable in framework, may need other frames complementary.

强 causation vs weak causation:
  强: deterministic (Newton).
  weak: probabilistic (statistics).
  framework's 因果 frame articulates both via 缘 specification.

但 因果 frame primary articulation 是 universal — any change has 因缘果 articulation.
```

# Frame 3: 时刻 frame (Temporal)

## 3.1 Overview

```
时刻 frame articulates *temporality* through continuous-discrete mutual.
focuses: time, change, history, evolution.
applies: any temporal phenomenon.
core dynamic: 时 ⇄ 刻 mutual; accumulated history.
```

## 3.2 Structure

### Inner pair: 时 ⇄ 刻

```
时 (shi): continuous time — flow, duration, becoming.
刻 (ke): discrete moment — punctuation, instant, snapshot.

mutual constitution:
  时 alone: continuous flux without distinct moments — undifferentiated.
  刻 alone: discrete instants without flow — no becoming.
  时 ⇄ 刻 mutual: actual temporality.

formal:
  时 与 刻 的 mutual:
    时 之 articulation needs 刻 之 punctuations.
    刻 之 articulation needs 时 之 continuous flow between.
  
易传: "穷则变, 变则通, 通则久" — 时 + 刻 mutual cycle.
```

### Outer dual: 时刻 ⇄ 史

```
时刻 (moment): instantaneous temporal articulation.
史 (shi): history — accumulated temporal record.

mutual constitution:
  时刻 之 sequence accumulates → 史.
  史 之 articulation reveals 时刻 之 sequence.
  二者 mutual.

法则 45 (frame-dual derivative-integral):
  时刻 (微) ⇄ 史 (积).
  instantaneous moments ⇄ accumulated history.
  类 derivative ⇄ integral.
```

### 4-aspect manifestation in 时刻 frame

```
时刻 frame 之 4-aspect articulation:

物 in 时刻: things across time — diachronic identity.
動 in 时刻: dynamic processes through time.
間 in 时刻: temporal relations (before, after, simultaneous).
事 in 时刻: events at specific moments.

irreducible 4-aspect at 时刻 frame.
```

## 3.3 Operators

```
时刻 frame's specific operators:

化 (hua): 时 化 刻 (continuous flow crystallizes into moment).
经 (jing): 时 经 刻 (time passes through moments).
流 (liu): 时 流 刻 (flow articulates moments).
往 (wang): 时刻 往 史 (moments flow into history).
来 (lai): 时刻 来 自 史 (moments emerge from history).
今 (jin): 现 之 时刻 (the now).
昔 (xi): 过 之 时刻 (the then).
将 (jiang): 来 之 时刻 (the will-be).
```

## 3.4 Cross-cultural attestation

```
古典中文:
  易传: 时 之 重要性 (timing in 易经).
  论语: "逝者如斯夫, 不舍昼夜" (time as flowing river).
  庄子: 时 之 articulation through 化.
  
近代西方:
  Augustine: time as paradox.
  Bergson: durée (continuous duration) — close to 时 articulation.
  Husserl: internal time consciousness.
  Heidegger: temporality of Dasein.

现代物理:
  Newton: absolute time.
  Einstein: relative time (frame-dependent).
  量子重力: time emerges from underlying structure.
  
现代哲学:
  Whitehead: process philosophy — actual occasions (时刻).
  Deleuze: time and duration.

framework attestation:
  时刻 frame articulates 法则 14 (化生时间) explicit.
  time emerges from 化 之 iteration.
  时 ⇄ 刻 mutual articulation.
```

## 3.5 Cross-frame transformations

```
时刻 → 因果 frame:
  时 (prior) → 刻 (subsequent) ↔ 因 → 果.
  详 见 Verification 1.

时刻 → 式量 frame:
  时 (continuous variable t) ↔ 式 (function of time).
  刻 (discrete value at t) ↔ 量.
  
时刻 → 阴阳 frame:
  时 之 cycles articulate 阴 ⇄ 阳 alternation.
  例: day-night cycle.

时刻 → 系统 frame:
  system evolution through time.
  时 之 dynamics 是 系统 之 dynamics.

时刻 → 生灭 frame:
  生 之 timing + 灭 之 timing.
  生灭 cycles in 时.

时刻 → 名实 frame:
  name's 时 (temporal extension of name).
  实 之 时刻 (actual instances at moments).

时刻 → 体用 frame:
  体 之 持续 (continuous identity through time).
  用 之 actualization at specific 时刻.

法则 53 verification:
  时刻 derivations cross-applicable in other 7 frames.
```

## 3.6 Failure modes

```
时刻 frame's failure modes:

1. 时 失 刻:
   pure flux without distinct moments.
   indeterminate articulation.
   reading: chaos before time emerges (法则 14 violated).

2. 刻 失 时:
   discrete moments without connecting flow.
   disjoint events (no temporal relation).
   reading: unstable without 时 articulation.

3. 时刻 失 史:
   moments without accumulation.
   amnesiac universe.
   reading: 不 sustainable (法则 45 violated).

4. 史 失 时刻:
   history without specific moments.
   abstract narrative without instants.
   reading: incomplete articulation.

5. 不 reversible articulation:
   forward (时 → 史): natural.
   backward (史 → original 时刻): partial reconstruction only.
   此 是 法则 6 (composition irreversibility).

6. 时 之 paradoxes:
   time travel, causal loops.
   framework: handles via 法则 7 反馈律 + careful articulation.
   no real paradox if framework articulated correctly.
```

## 3.7 Examples / Applications

```
example 1: 物理 dynamics
  classical mechanics evolution.
  state s(t) → s(t+dt).
  详 见 Verification 1.

example 2: biological aging
```


## 8.2 Structure

### Inner pair: 体 ⇄ 用

```

  体用 is *perspectival*, not ontological.
  same entity has both 体 + 用.
  体 is "from outside" view (substance-like).
  用 is "from action" view (function-like).
  二 perspectives on *one entity*.

  体 alone: lifeless substance, no function — abstract.
  用 alone: function without substance — abstract action.
  体 ⇄ 用 mutual: actual entity.

  "体用一原, 显微无间" (substance-function one source, manifest-subtle no gap).
  framework's 法则 9 explicit.
```
