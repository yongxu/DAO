
道 (R₈ origin) 在 (Z/2)⁸ 中是唯一选定之 zero element: `(Hexagram.qian, Shi.dao)` = `OX["oooooooo"]` = `Cell256.origin`。Cayley `ι: c ↦ (· ⊕ c)` + `ε: f ↦ f(道)` 互逆给出 `R₈ ≅ XOR(R₈)` — 元 ≡ 算子 严格同构 (详 yi-RO-hierarchy.md 第五部分)。

| 位 | 字符 | 含义 | identity 位 / set 位 |
|---|---|---|---|
| 1 (`y₁`) | `o` 阳 / `x` 阴 | 初爻 (inner trigram bit 1) | `o` = identity (yang) |
| 2 (`y₂`) | `o` / `x` | 二爻 (inner bit 2) | 同上 |
| 3 (`y₃`) | `o` / `x` | 三爻 (inner bit 3) | 同上 |

| 6 (`y₆`) | `o` / `x` | 上爻 (outer bit 3) | 同上 |
| 7 | `o` / `x` | YinBit (因 axis, R₇ atom) | `o` = false, `x` = true |
| 8 | `o` / `x` | GuoBit (果 axis, R₈ atom) | `o` = false, `x` = true |

末两位 (因, 果) ∈ Bool² 通过双射 `Shi.ofYinGuo` ↔ Shi V₄:

| 末 2 位 | (因, 果) | Shi |
|---|---|---|
| `oo` | (false, false) | **道 (dao)** — V₄ identity |
| `xo` | (true, false) | **已 (ji)** — past, P-like |
| `ox` | (false, true) | **未 (wei)** — future, T-like |
| `xx` | (true, true) | **今 (jin)** — present, PT central |

故 `OX["oooooooo"]` = `(Hexagram.qian, Shi.dao)` = 道 (R₈ origin = identity = no-op = 永真 cell, 一个 8-bit 字符串承担五重身份); `OX["xxxxxxxx"]` = `(Hexagram.kun, Shi.jin)` = 坤·今 = R₈ 之最远 anti-pole。

## 4 · 256 格全表 — 序卦 (King Wen) 序 × Shi V₄

下表逐格枚举全 **256 cells = 64 卦 × 4 Shi**, 按 BenZheng 4 象限分组 (本本 / 本征 / 征本 / 征征 各 16 卦 × 4 Shi = 64 cells), 卦序按 `xuGua` (Cell256.lean § 4) King Wen 序排列。 每格一行, 10 列:

- `#` — 全表序号 (1..256, 按象限+卦序+Shi 序)
- `King Wen` — 序卦传序号 (1..64)
- `Hex` — 卦名中文
- `内卦` / `外卦` — Trigram 符号 + 名称 (内 = y₁y₂y₃, 外 = y₄y₅y₆, 各自之低位在下)
- `6-yao` — y₁..y₆ 内→外 OX 串 (前 6 char of OX 字面量)
- `Shi` — V₄ Klein 时态 (道/已/未/今), 道 = identity
- `(因, 果)` — Shi 之 (因 bit, 果 bit) ∈ Bool²
- `OX 字面量` — 8-char `o`/`x` OXNotation term-level Lean 字面量
- `简称` — `卦·时` 二字标签

Shi 顺序为 V₄ 标准: 道 (identity) → 已 (P / 错) → 未 (T / 综) → 今 (PT / 错综)。

## 5 · 道 cell — 跨时空 anchor

道 cell `OX["oooooooo"]` 在 256 格中是**唯一**之 V₄ identity, 落在 (乾, 道) 处, 即本本象限 #1 行第一列。其特殊性:

| 身份 | 形式见证 |
|---|---|
| (Z/2)⁸ identity / 群单位元 | `Cell256.origin = (Hexagram.qian, Shi.dao)`; `origin_xor : ∀ c, xor origin c = c` (Cell256.lean § 7.4) |
| Cayley 评估 origin | `Cell256.epsAtOrigin (cayley c) = c` (§ 7.6); fusion ε ∘ ι = id |
| no-op operator | `c • origin = c`; left-translation by origin 是恒等映射 |
| 跨时空永真 cell | Shi V₄ 单位元, 既无因 (因=0) 又无果 (果=0); 其它 3 个 Shi (已 / 未 / 今) 全可由道 + 印 + 投 + 印·投 得到 |
| 太极 R₀ 在 R₈ 之 落地 | 太极 (R₀ Unit) 是 absolute "无分" ground, 道 (R₈ origin) 是 (Z/2)⁸ 之 origin choice — 同一 ontological zero anchor 在两个不同 R-layer 之表达 |

道 之**跨象限唯一性**: 在本表 4 象限 × 4 时态 = 16 个 (象限, 时态) bucket 中, 道 cell 唯一落在 (本本, 道) bucket; 其它 15 个 bucket 各有 16 个 cell, (本本, 道) 也是 16 cell 之一, 但只有 (乾, 道) = `OX["oooooooo"]` 是 V₄ identity。其它 15 个本本-道格 (如 `(坤, 道) = OX["xxxxxxoo"]`) 是「乾 axis 上之 V₄ identity 平移」, 不是 (Z/2)⁸ 单位元。

## 6 · V₄ Klein involution structure on Shi (and lifted to Cell256)

Shi V₄ 之三 involution (Cell256.lean § 1):

| 算子 | mask 后 2 位 (V₄ 元) | dao→ | ji→ | wei→ | jin→ | 物理对应 |
|---|---|---|---|---|---|---|
| `Shi.cuo` | `xo` (factor `(1, 0)`) | ji | dao | wei | jin (固) — wait actually | P (parity, 翻 因 axis) |
| `Shi.zong` | `ox` (factor `(0, 1)`) | wei | jin | dao | ji | T (time-reversal, 翻 果 axis) |
| `Shi.cuoZong` | `xx` (factor `(1, 1)`) | jin | wei | ji | dao | PT (双轴翻) |
| `id` | `oo` | dao | ji | wei | jin | 恒等 |

(精确表见 `Shi.cuo / zong / cuoZong` 之 pattern match 定义; 上文「物理对应」按 P/T/PT 之 V₄ Klein 群表示。)

Lift 到 Cell256 (`shiCuo / shiZong / shiCuoZong`, Cell256.lean § 3) 是 (id_hex × cuo_shi) 之 tensor; 与 hex-side V₄ (`hexCuo / hexZong / hexHu`) 互相交换 (§ 3 之 `hexCuo_shiCuo_comm` 等)。

印 / 投 mask form (Cell256.lean § 8):

```lean
yin_mask : Cell256 := (Hexagram.qian, Shi.ji)   -- 只 YinBit (因 axis) set
tou_mask : Cell256 := (Hexagram.qian, Shi.wei)  -- 只 GuoBit (果 axis) set

yin (c) = xor c yin_mask     -- toggle 因
tou (c) = xor c tou_mask     -- toggle 果
yin_mask ⊕ tou_mask = (qian, jin)   -- V₄ central element = PT
```

印 / 投 在 Shi 上是 cuo / zong 之 alias; 在 Cell256 上是 mask XOR involution。二者皆 self-inverse, 互相交换 (§ 8 之 `yin_yin / tou_tou / yin_tou_comm / yin_tou_eq_central`)。

## 8 · 单字覆盖统计 — 道行加入后

按 v2 之单字策略 (见 v2 之 [`史/义理-pre-v3/表六_192格全表_旧Z3版.md`](../史/义理-pre-v3/表六_192格全表_旧Z3版.md) § 单字覆盖统计), 192 格皆需 ≥ 2 字 (时态字 + 卦名字)。 v3 加了 64 个「道-行」 (道 × 64 卦), 此 64 行之单字覆盖与 v2 同结构:
