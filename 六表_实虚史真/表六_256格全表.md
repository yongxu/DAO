## 256 之来源

**256 = 64 卦 × 4 时态（道 / 已 / 今 / 未）= (Z/2)⁶ × (Z/2)² = (Z/2)⁸**

依《周易·序卦》之 64 卦次序，每卦在「道（永真 anchor）/ 已（过去）/ 今（现在）/ 未（未来）」四时态投影。
共 256 格。此表查每格之单字覆盖、(因, 果) bits、OX 字串、与 BenZheng 象限位置。

**8-bit layout (OX["..."] 编码)**：

```
位置 1..6 = 六爻 (内→外)，o = yang/false, x = yin/true
位置 7    = 因 bit (YinBit / R₅ atom)：o = 无因, x = 有因
位置 8    = 果 bit (GuoBit / R₆ atom)：o = 无果, x = 有果
```

**Shi 由 (因, 果) 双轴恢复**：
- `oo` → 道 (dao,  V₄ identity)
- `xo` → 已 (ji,   parity-like)
- `ox` → 未 (wei,  time-reversal-like)
- `xx` → 今 (jin,  PT)

例：`OX["oooooooo"]` = 乾·道 = (Hexagram.qian, Shi.dao) = Cell256.origin = (Z/2)⁸ 单位元。

无论何种分解，**256 之共性是「(Z/2)⁸ 笛卡尔积 / Cayley 自作用」**——汉字之单字策略对单轴可精确覆盖（如六征、史四态、真假三态各有单字），但对多轴乘积**不再造新字**，而是用「轴字 + 轴字」之复合词。

## 总判断（回答原问题）

> **「中国汉字是否对这 256 个都有精确单字对应？」**

**否。**

汉字策略是：
1. **单轴层**有完整单字覆盖（六征 3 对 × 3 位 = 9 单元，史四态各有单字 道/已/今/未，真假各 3 字皆有）
2. **三轴乘积层**部分有单字（表五之 27 格中 **20 格有，7 格无**）
3. **多轴乘积层**（如此 256 格 = 64 卦 × V₄ 时态）**普遍不再造新字**，而采用「轴字 + 轴字（+ 轴字）」之复合表示

> **「有些是由真假判断的，那不知真假的是不是也有此？」**

**有。**「不知真假」之单字候选：
- **疑**（认识侧主字）
- **玄**（本体侧）、**悬**（程序侧）、**惑**（心理侧）、**待**（时序侧）

每维各有一字承担——汉字以「多字分维」代替「单字统一」，反而更精微。
