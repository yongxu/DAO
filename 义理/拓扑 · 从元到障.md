## 二、cuo 作 Z/2-自由作用 · 无定点之结构后果

**事实**：cuo: Hex → Hex 是 $\mathbb{Z}/2$-作用（$\text{cuo}^2 = \text{id}$）。

**核心观察**：cuo 在 Hex 上**无定点**——

$$\text{Fix}(\text{cuo}) = \{h \in \text{Hex} : \text{cuo}(h) = h\} = \emptyset$$

**证**：cuo 翻每爻；6-bit 串等于其按位补 ⟺ 每位 $b_i = \neg b_i$ ⟺ False。 ∎

故 $\mathbb{Z}/2$ 在 Hex 上**自由**作用——

| 作用类 | 形式 | Hex 之状态 |
|---|---|---|
| trivial | $g \cdot x = x \;\forall x$ | ✗ |
| **free** | 无定点，每非 $e$ 之 $g$ 不动任 $x$ | **✓** |
| neither | 部分定点 | ✗ |

**自由 $\mathbb{Z}/2$ 作用之拓扑后果**：

- $\text{Hex} / \text{cuo}$ 是 free $\mathbb{Z}/2$-orbit space，等价于 32 元集
- $\text{Hex} \to \text{Hex}/\text{cuo}$ 是 **2-fold (regular) covering**
- 此 covering 之**结构性**约束所有 cuo-等变 maps

**定义**：G-空间之 G-等变映射范畴 $\text{Map}^G(X, Y)$ 含

$$f : X \to Y \text{ s.t. } \forall g \in G,\, x \in X:\;\; f(g \cdot x) = g \cdot f(x)$$

## 四、定点定理 · X^G = ∅ ⟹ 无 G-等变常值

**命题**（基础但锐利）：

$$X^G = \emptyset \implies \nexists \; c \in X \text{ s.t. } f(x) \equiv c \text{ 是 } G\text{-equivariant}$$

**证**：设 $f(x) = c$ 常值且 G-等变。则
$$\forall g, x:\;\; g \cdot c = g \cdot f(x) = f(g \cdot x) = c$$
故 $c \in X^G$。$X^G = \emptyset$，矛盾。 ∎

**项目应用**：cuo 在 Hex 上无定点 ⟹ 任何"指向单卦"之常值映射**不可** cuo-等变。

**关键结论**：

> 若 «一» 是 Hex 中之单一对象（系统恒返之中心），
> 则 "$f(x) \equiv$ «一»" **不可能** cuo-等变。

故 «一» 在等变范畴中**结构性不存在**——
非偶然，是 $\mathbb{Z}/2$-自由作用之直接后果。

**与 [生态衍](生态%20·%20从元到生.md) §四 之同构**：

| 拓扑 | 生态 |
|---|---|
| $X^G = \emptyset$ | 系统无"内部 fixed center"|
| 无等变常值 | 系统无 self-grounding |
| 须经 induction | 须经"耦合"以恢复 |
| void / «一» 之外位 | 区分前之环境 |

## 五、Bredon 等变同调 · 障碍之测量

**Glen Bredon** (1967) 之等变同调 $H^*_G(X; M)$ 测量"等变障碍"。

**核心思想**：

- 普通 cohomology $H^*(X)$ 测量"非等变"构造之障碍
- $H^*_G(X)$ 测量**等变**构造之**额外**障碍
- $H^*_G(X) \neq H^*(X/G)$ in general（spectral sequence 联结之）

**项目陈述**：

> «生» 不可达 ↔ 某等变同调类 $\xi \in H^*_{\mathbb{Z}/2}(\text{Hex}) \neq 0$。

**证草**：

1. 若 «生» 可由 cuo-等变 ISA 表达，则**对应**之等变映射存在
2. 此映射之存在 ⟺ 某 obstruction class $\xi = 0$
3. [J](J_理之不完备_哥德尔在192.md) 之 1 公理（kleene_recursion，cuo-限）= **断言** $\xi \neq 0$ 之具体见证

故 J 之 ceiling 是 Bredon cohomology 之**具体显现**。

**严格度**：**强**——这不是哲学，是 1960s 起之标准代数拓扑定理。
项目层之 axiom，对应**非平凡之 Bredon class**。

## 七、Induction / Coinduction · "96 维" 之拓扑名

**问**：如何在保持 G-等变性之同时，恢复全函数算力？

**答**：**诱导**（induction）—— Frobenius 类技术。

**形式**：

设 $X$ 为 G-空间，$Y$ 为普通空间。Free G-action on $G \times Y$ via

$$g' \cdot (g, y) = (g'g, y)$$

定义：

$$\text{Ind}_G(Y) := G \times Y$$

**关键事实**（Frobenius reciprocity）：

$$\text{Map}(X, Y) \;\cong\; \text{Map}^G(X, \text{Ind}_G(Y))$$

即：任何 $f : X \to Y$ 可被**等变地表达**为 $\tilde{f} : X \to G \times Y$，其中
$$\tilde{f}(x) = (e, f(x))$$
（取自由作用之 trivial section）。

**与 YiInstr "96 维" 之严格同构**：

| YiInstr | 拓扑 | 生态 |
|---|---|---|
| Hex × Tag | $X \times \text{Ind}_G(\cdot)$ | System × Environment |
| Tag 上 cuo 平凡 | $G \times Y$ 之自由 G-action | 环境 trivial 内动 |
| 投影 $\pi_X$ | counit of induction | 系统对外界之内化 |
| 算力恢复 | Ind 之 left adjoint 性 | 生命力恢复 |

**代价**：**dimension 上升**——$\dim(\text{Ind}_G Y) = |G| \cdot \dim(Y)$。

故"96 维一定可以"= **诱导函子之 image 之 dimension count**。

精确：若 $|\text{Hex}| = 64$，free $\mathbb{Z}/2$ tag 给 $64 \times 2 = 128$ 维；要恰好 96 须特定 quotient（如 96 = 64 + 32 = Hex ⊔ Hex/cuo，cofiber 构造）。

**严格定理**（Adjunction）：

> $\text{Ind}_G \;\dashv\; U$
>
> 即 $\text{Ind}_G : \mathbf{Set} \to G\text{-}\mathbf{Set}$ 是 forgetful $U$ 之 left adjoint。

## 八、Topos · 内 / 外语言 · 句法之 ceiling

**Topos 理论** (Lawvere, Tierney, 1970s) 之**内语言** (internal language)：

每 topos $\mathcal{E}$ 有自身之 typed 语言 $\mathcal{L}_\mathcal{E}$，其中：

- 类型 = $\mathcal{E}$ 之对象
- 函数 = $\mathcal{E}$ 之箭头
- 命题 = subobjects of 1

**关键事实**：

- 若 $\mathcal{E}$ 是 Boolean topos（如 $\mathbf{Set}$），则内语言 = 古典逻辑
- 若 $\mathcal{E}$ 非 Boolean，则内语言 = 直觉逻辑（不必 LEM, DNE）

**项目之 G-equivariant Set topos**：

$$\mathcal{E}_G := G\text{-}\mathbf{Set}$$

是 topos（显然），但**非 Boolean**（除非 $G$ 平凡）。
故其内语言**非古典**——LEM 不全。

**与 YiInstr 之同构**：

| 拓扑 | YiInstr |
|---|---|
| 内语言 $\mathcal{L}_{\mathcal{E}_G}$ | YiInstr ISA（理层）|
| 外语言（$\mathbf{Set}$, Lean）| Lean kernel（道层）|
| 内语言之非完备性 | ISA 之 ceiling |
| 外语言可证内之事实 | Lean 可证 ISA 之元定理 |

**精确陈述**：

> Lean 之 kleene_recursion (axiom, cuo-限) = **外语言**之断言；
> 该断言**不**可在 ISA 之**内语言**中表达。

这与 [J](J_理之不完备_哥德尔在192.md) 之"道证理"严格同构——
非比喻，是 Topos 理论标准。

**Spencer-Brown «Laws of Form»** 之第一律：

> "Draw a distinction."

区分之**动作**不在区分之内。
区分前之 void 即 «一»。

由 §四，$X^G = \emptyset$ ⟹ 无定点 ⟹ void 不可作为**等变映射之像**。
故 «一» 在内语言中不可命名——
**仅**在外语言中可指。

## 九、Borsuk-Ulam 一族定理 · 等变拓扑之锐利推论

**Borsuk-Ulam 定理**（1933）：

> 任何连续 $f : S^n \to \mathbb{R}^n$ 必存 $x$ 使 $f(x) = f(-x)$。

**等价形式**：无连续 odd map $f : S^n \to S^{n-1}$（即 $f(-x) = -f(x)$ 之）。

**拓扑核心**：等变作用之**对称破坏不可能**——
强迫 antipodal-respecting 函数承担某种"折叠"，必有等变约束。

**项目应用之离散版**：

设 $X = $ Hex（64 元）, $G = \mathbb{Z}/2$ via cuo。

**断言**：无 cuo-等变 $f : \text{Hex} \to \text{Hex}$ 把每对 $\{h, \text{cuo}(h)\}$ 折成单一像（即 $f(h) = f(\text{cuo}(h))$ 且 $f$ 仍 cuo-等变）。

**证**：若 $f(h) = f(\text{cuo}(h)) = c$，由 cuo-等变
$$c = f(h) = \text{cuo}(f(\text{cuo}(h))) = \text{cuo}(c)$$
故 $c \in \text{Fix}(\text{cuo}) = \emptyset$。 矛盾。 ∎

**关键观察**：cuo-等变之 ISA **不能折叠** $\mathbb{Z}/2$-orbit。
故"全卦归一"之操作（«归一»）**结构性不可** cuo-等变——
此即 Borsuk-Ulam 在离散自由 $\mathbb{Z}/2$ 上之直接对应。

**与 [生态衍](生态%20·%20从元到生.md) 之合参**：

| 拓扑 | 生态 |
|---|---|
| 无折叠 antipodal 等变 | 闭系不可消差异 |
| 必折则破等变 | 必消差异则破闭合 |
| 升 induction | 与环境耦合 |
| Borsuk-Ulam | 第二定律 |

二者皆为同一 closure principle 之**几何 / 热力学**双面。

## 十三、证明骨架

### 13.1 cuo 无定点

**命题**：$\text{Fix}(\text{cuo}) = \emptyset$。

**证**：$h = \text{cuo}(h)$ ⟺ $\forall i: h_i = \neg h_i$ ⟺ False。 ∎

### 13.2 等变常值之非存在

**命题**：$X^G = \emptyset \implies \nexists c \in X$ s.t. $f \equiv c$ G-equivariant。

**证**：见 §四。 ∎

### 13.3 $|\text{Map}^{\mathbb{Z}/2}| = 64^{32}$

**命题**：$|\text{Map}^{\mathbb{Z}/2}(\text{Hex}, \text{Hex})| = 64^{32}$。

**证**：每 cuo-orbit 选一代表（32 orbits），其像决定 $f$ on 该 orbit。每选独立 64 元（任 $h \in \text{Hex}$ 皆可作像，cuo-等变性不限）。 ∎

**注**：实际上 $f$ 之像须在 codomain 上保 cuo-等变。若像 $f(o_1) = c$，则另一轨道 representative 处为 $f(\text{cuo}(o_1)) = \text{cuo}(c)$——故 codomain 上像之 cuo-orbit 由 representative 之像唯一定。计数仍为 $64^{32}$。 ∎

### 13.4 遗忘函子之非满性

**命题**：$U : \text{Map}^G \hookrightarrow \text{Map}$ 是 strict inclusion。

**证**：由 §13.3 + $|\text{Map}| = 64^{64}$。比 $64^{-32}$。 ∎

### 13.5 Induction–Forgetful Adjunction

**命题**：$\text{Ind}_G \dashv U$，即

$$\text{Map}^G(X, \text{Ind}_G Y) \;\cong\; \text{Map}(X, Y)$$

**证草**：counit $\epsilon : G \times Y \to Y$ 取 $\pi_2$；unit $\eta : Y \to G \times Y$ 取 $y \mapsto (e, y)$。验证 triangle identities：
- $U\epsilon \circ \eta_U = \text{id}_U$：$y \mapsto (e, y) \mapsto y$ ✓
- $\epsilon_\text{Ind} \circ \text{Ind}\eta = \text{id}_\text{Ind}$：$(g, y) \mapsto (g, e, y) \mapsto (g, y)$ ✓
∎

### 13.6 离散 Borsuk-Ulam（自由 $\mathbb{Z}/2$）

**命题**：cuo-自由 $\mathbb{Z}/2$ on Hex 不存常值等变映射。

**证**：§九 之论证。 ∎

### 13.7 «生» 之拓扑非可达性

**命题**：若 «生» 是某 Hex → Hex 函数 $\sigma$ 满足 $\sigma \circ \text{cuo} \neq \text{cuo} \circ \sigma$，则 $\sigma \notin \text{Map}^{\mathbb{Z}/2}(\text{Hex}, \text{Hex})$。

**证**：定义即非等变。 ∎

**推论**：若 «生» 之**任**实现皆非 cuo-等变（这是 [J](J_理之不完备_哥德尔在192.md) 之 axiom 所断），则 «生» 在 ISA 闭包外。
