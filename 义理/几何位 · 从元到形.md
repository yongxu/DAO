## 二、点 · 几何之元

### 算子：点

| 项 | 内容 |
|---|------|
| 单字 | **点** |
| 别字 | 元、止、住 |
| 类型 | $\{*\} \cong \mathrm{Unit}$，cardinality 1 |
| 出处 | 几何原本（欧氏）；中文「点」字本义"小痕"|

形式定义：

$$\mathbf{点} = \mathrm{Unit} = \{*\}$$

**点 = 太极**：项目主线之太极一级（T₀）即此。
不可再分之最小空间原子——既无延展，亦无方向。

> 「极小者无内」——《墨经》经下 P-2

### 多点空间

| 形式 | 结构 |
|------|-----|

| List $S$ | 有序点列 |
| Vec $S$ $n$ | 长 n 之有序点列（fin 长）|
| ℕ → $S$ | 无穷点序列 |

**字元承担**：

| 字 | 偏向 |
|---|------|
| 集 | 无序之全 |
| 列 | 有序之延 |
| 串 | 有限之续 |

## 三、序 · 一维之元

### 算子：序

| 项 | 内容 |
|---|------|
| 单字 | **序** |
| 别字 | 列、次、第 |
| 类型 | 偏序 / 全序关系 $\leq$ |
| 出处 | 系辞「天尊地卑，乾坤定矣；卑高以陈，贵贱位矣」|

**两种序**：

| 形式 | 例 |
|------|---|
| 偏序 (poset) | $\leq$ 自反传递反对称 | 集合包含、整除关系 |
| 全序 (total order) | 偏序 + 任意两元可比 | ℕ, ℤ, ℝ |
| 良序 (well-order) | 全序 + 任非空子集有最小 | ℕ, 序数 |

**v14 之"位"含序**：易爻自下而上排序（初、二、三、四、五、上），即良序长 6。

### 一维空间之雏形

$$\text{Bool} = \{0, 1\} = \{\text{阴}, \text{阳}\}$$

最简一维空间——**两位**：阴、阳。
作为**序**：阴 < 阳（按此约定，但易经无严格高低之分）。
作为**对**：阴↔阳（项目主线之"两翼"）。

## 五、距离 · 度量

### 算子：距

| 项 | 内容 |
|---|------|
| 单字 | **距** |
| 别字 | 远、近、长 |
| 类型 | $X \times X \to \mathbb{R}^+$ |
| 出处 | 《墨经》经下 P-3「中，同长也」（距离平等）|

### 度量空间公理

度量空间 $(X, d)$：

| 公理 | 形式 |
|------|------|
| 非负 | $d(x, y) \geq 0$ |
| 自距零 | $d(x, x) = 0$ |
| 同一 | $d(x, y) = 0 \Rightarrow x = y$ |
| 对称 | $d(x, y) = d(y, x)$ |
| 三角 | $d(x, z) \leq d(x, y) + d(y, z)$ |

### 汉明距离（八卦上）

$$d_H : \mathrm{Bool}^n \times \mathrm{Bool}^n \to \mathbb{N}, \quad d_H(a, b) = |\{i : a_i \neq b_i\}|$$

**已 Lean 验证**：见 `BaguaAlgebra.lean` 之 `hammingDist`：

| 定理 | 内容 |
|------|------|
| `hammingDist_self` | $d_H(t, t) = 0$ |
| `hammingDist_eq_zero_iff` | $d_H(a, b) = 0 \iff a = b$ |
| `hammingDist_le_3` | $\forall a, b, d_H(a, b) \leq 3$（八卦层）|
| `hexHammingDist_le_6` | $d_H(a, b) \leq 6$（64 卦层）|

**对称与三角不等式**：易证（`xor` 之性质），但 Lean 中未单列定理——可补。

### 欧氏距离（ℝⁿ 上）

$$d_E(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}$$

由数与算术 §10 之 ℝ 之 Cauchy 完备承担"开方"。

### 字元之精细分

| 字 | 偏向 |
|---|------|
| 距 | 通用度量值 |
| 远 | 距离大 |
| 近 | 距离小 |
| 长 | 一维线段之距 |
| 广 | 二维面之展 |

## 六、拓扑 · 开闭与邻

### 算子：开 / 闭 / 邻

| 项 | 内容 |
|---|------|
| 单字 | **开 / 闭** |
| 别字 | 邻、围、围 |
| 类型 | $\tau \subseteq \mathcal{P}(X)$ |
| 出处 | 老子 11「凿户牖以为室，当其无，有室之用」（开/闭之辩证）|

### 拓扑空间公理

$(X, \tau)$ 满足：

| 公理 | 形式 |
|------|------|
| 全体属开 | $X \in \tau$ |
| 空属开 | $\emptyset \in \tau$ |
| 任并属开 | $\{U_i\} \subseteq \tau \Rightarrow \bigcup_i U_i \in \tau$ |
| 有限交属开 | $U, V \in \tau \Rightarrow U \cap V \in \tau$ |

闭集 = 开集之补。
邻域 $N$ of $x$：存在 $U \in \tau, x \in U \subseteq N$。

### 离散拓扑（八卦层）

$\mathrm{Bool}^3$ 上**离散拓扑** $\tau = \mathcal{P}(\mathrm{Bool}^3)$（所有子集皆开）：
- 每个卦自成一开集
- 任意子集之并仍是子集——闭合
- 度量 $d_H$ 诱导此拓扑（球 $B(t, 1/2) = \{t\}$）

> **故八卦作为离散拓扑空间是 trivial**——拓扑结构在此层无新信息。
> 拓扑之**真正用处**在**连续空间**（ℝⁿ 之上）。

### ℝ 之标准拓扑

由开区间 $(a, b)$ 生成。等价描述：

$$U \in \tau_\mathbb{R} \iff \forall x \in U, \exists \epsilon > 0, (x - \epsilon, x + \epsilon) \subseteq U$$

由数与算术 §10 之 ℝ 完备性承担。

## 七、连续 · 保形之映

### 算子：连

| 项 | 内容 |
|---|------|
| 单字 | **连** |
| 别字 | 通、续、贯 |
| 类型 | $f : (X, \tau_X) \to (Y, \tau_Y)$ 满足开集回拉条件 |
| 出处 | 系辞「显诸仁，藏诸用」「贯而通之」|

### 连续映射

$$f : (X, \tau_X) \to (Y, \tau_Y) \text{ 连续} \iff \forall V \in \tau_Y, f^{-1}(V) \in \tau_X$$

即"开集之原像皆开"。

等价（度量空间上）：

$$\forall x \in X, \forall \epsilon > 0, \exists \delta > 0, d_X(x, y) < \delta \Rightarrow d_Y(f(x), f(y)) < \epsilon$$

### 八卦上之"连续"

由于 $\mathrm{Bool}^3$ 离散，**任何**函数 $f : \mathrm{Bool}^3 \to \mathrm{Bool}^3$ 皆连续——
拓扑层无信息。

但**度量层**（汉明距离）有约束：

| 算子 | 是否汉明保 1 | 释 |
|------|------------|---|
| 反爻（变 / 化 / 动） | ✗ 增 1 | 但保**距离** $d_H(a, b) = d_H(\bar{\partial}_i a, \bar{\partial}_i b)$ |
| 错（三反复合） | ✗ 增 3 | 同样保距离 |
| 综（位置反序） | ✗ 不规则 | 不保距离一般情况 |

**保距离即 isometry**——本文之**几何对称**。

### 字元之精细分

| 字 | 几何意 |
|---|------|
| 连 | 连续映射（拓扑层）|
| 通 | 连通空间（不可分）|
| 续 | 序列收敛之连续 |
| 贯 | 全程不断 |
| 保 | 保结构（isometry / homeo / iso）|

## 八、易经之位 · 中、应、比、当

易经之"位"是本文**核心衍承**——四种位关系，皆有现代几何对应。

### 8.1 当位（proper position）

| 字 | 单字承担 |
|---|------|
| **当** | "当其位"——阳爻在阳位 / 阴爻在阴位 |

**位之奇偶**：位 $i \in \{1, 2, 3, 4, 5, 6\}$
- 阳位：$i$ 奇（1, 3, 5）
- 阴位：$i$ 偶（2, 4, 6）

形式：

$$\text{当位}(h, i) = \begin{cases}
\text{真} & \text{若 } y_i = \text{阳} \land i \text{ 奇} \\
\text{真} & \text{若 } y_i = \text{阴} \land i \text{ 偶} \\
\text{假} & \text{其他}
\end{cases}$$

**Lean 已实现**：`Yi.lean` 之 `wellPos` 函数。

**几何对应**：当位 = 顶点与位标签之**类型匹配**——类似类型论中 type-judging。

### 8.2 中位（centrality）

| 字 | 单字承担 |
|---|------|
| **中** | "中道"——位居中心 |

**中位**：六爻中之第 2 位（内卦中）与第 5 位（外卦中）。

形式：

$$\text{中}(i) = (i = 2) \lor (i = 5)$$

**Lean 已实现**：`Yi.lean` 之 `isZhongPos`。

**几何对应**：中位 = 一维段之**重心 / 中点**——三爻段 $\{1, 2, 3\}$ 之中是 2，外卦段 $\{4, 5, 6\}$ 之中是 5。

### 8.3 应位（correspondence）

| 字 | 单字承担 |
|---|------|
| **应** | "相应"——内外对应位之阴阳异 |

**应位对**：(初, 四), (二, 五), (三, 六)。
**应**：对应位之阴阳**异**（互补）。

形式：

$$\text{应}(h, i) = (y_i \neq y_{i+3}) \quad i \in \{1, 2, 3\}$$

**Lean 已实现**：`Yi.lean` 之 `yingResponds`。

**几何对应**：应位 = **轴对称之异**——内外卦在第 i 爻位置之值相对（XOR）。
等价于：内卦 = ¬外卦（按位）⟺ 全应（即"错"位之半）。

### 8.4 比位（adjacency）

| 字 | 单字承担 |
|---|------|
| **比** | "比邻"——相邻爻之关系 |

**比位对**：(初, 二), (二, 三), (三, 四), (四, 五), (五, 六)。

形式：

$$\text{比}(h, i, i+1) = \text{相邻爻 } y_i, y_{i+1} \text{ 之关系}$$

**Lean 已实现**：`Yi.lean` 之 `biAdj`。

**几何对应**：比位 = **图论邻接**——爻位序列之 1-skeleton 路径图。

### 8.5 四位之结构定理

> 易传论卦之"得失"四标准：**当、中、应、比**——
> 形式上对应：**类型** × **重心** × **对偶** × **邻接** 四种位关系。

**总结**：

| 易经 | 现代 | 形式 |
|------|-----|------|
| 当位 | 类型匹配 | $y_i \sim \text{parity}(i)$ |
| 中位 | 重心 | $i \in \{2, 5\}$ |
| 应位 | 反对偶 | $y_i \neq y_{i+3}$ |
| 比位 | 邻接 | $\|i - j\| = 1$ |

## 九、立方体之 Cayley 图（八卦层）

### 9.1 立方体作为图

$\mathrm{Bool}^3$ 之 **1-skeleton**：

- **顶点**（0-cell）：8 个 = 八卦
- **边**（1-cell）：每对汉明距离 1 之顶点 = 12 条
- **面**（2-cell）：每四元子立方体 = 6 个面
- **体**（3-cell）：整个立方体 = 1 个

字元承担：

| 字 | 几何 |
|---|------|
| 点 | 顶点 = 卦 |
| 边 | 一爻之变 = 翻一位 |
| 面 | 二爻不变 = 二维子立方体 |
| 体 | 整体八卦集 |

### 9.2 边即「变」

每条边对应**单一反爻算子**之作用：
- 沿 $y_1$ 轴：动（动）
- 沿 $y_2$ 轴：化
- 沿 $y_3$ 轴：变

总边数 = $3 \times 4 = 12$（三轴各 4 条）。

### 9.3 任意两卦之最短路

$$d_H(a, b) \leq 3 = \dim(\mathrm{Bool}^3)$$

**Lean 已证**：`hammingDist_le_3` + `pathFromTo_length_eq_hammingDist`（路径长度紧度）。

故"任两卦皆可在 ≤ 3 步互通"是**几何陈述**——立方体之直径 = 3（汉明）。

### 9.4 64 卦层（6 维超立方体）

$\mathrm{Bool}^6$ 之 1-skeleton：
- 顶点 64
- 边 $6 \times 32 = 192$
- 直径 = 6

**Lean 已证**：`hexHammingDist_le_6`。

> **巧合**：64 × 3 时态 = 192 = 6 维超立方体之边数——
> 此非偶然，而是 $\binom{6}{1} \times 2^{6-1} = 192$（一爻可变 × 其余固定之总）。

**实际数字**：
- 6 维超立方体顶点数 = $2^6 = 64$
- 边数 = $\binom{6}{1} \cdot 2^{6-1} = 6 \cdot 32 = 192$
- 此处 64 × 3 = 192 是 J 之 Cell192 之**时态扩展数**（64 × 3 时态），非超立方体边数

故二者皆 192 但**起源不同**：
- $J/Cell192$ 之 192 = $|\mathrm{Hexagram}| \times |\mathrm{Shi}|$
- 几何之 192 = 6-cube 之边数

数字相同，结构异——参 K §四之"巧合表"（待补）。

## 十、连续空间 ℝⁿ · 几何之底

### 10.1 ℝ¹ · 实数线

由数与算术 §10 给出。

| 项 | 内容 |
|---|------|
| 集合 | $\mathbb{R}$ = Cauchy 序列等价类 |
| 拓扑 | 由开区间生成 |
| 度量 | $d(x, y) = \|x - y\|$ |

### 10.2 ℝⁿ · 高维欧氏空间

$$\mathbb{R}^n = \underbrace{\mathbb{R} \times \cdots \times \mathbb{R}}_{n}$$

| 项 | 形式 |
|------|-----|
| 点 | $\mathbf{x} = (x_1, \ldots, x_n)$ |
| 内积 | $\langle \mathbf{x}, \mathbf{y} \rangle = \sum_i x_i y_i$ |
| 范数 | $\|\mathbf{x}\| = \sqrt{\langle \mathbf{x}, \mathbf{x} \rangle}$ |
| 距离 | $d(\mathbf{x}, \mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|$ |

### 10.3 立方体嵌入 ℝⁿ

$$\mathrm{Bool}^n \hookrightarrow \mathbb{R}^n, \quad (b_1, \ldots, b_n) \mapsto (b_1, \ldots, b_n)$$

即把 $\{0, 1\}^n \subset \mathbb{R}^n$ 视为单位立方体之 $2^n$ 顶点。
**汉明距离** $\to$ **欧氏距离平方**：

$$d_H(\mathbf{a}, \mathbf{b}) = d_E(\mathbf{a}, \mathbf{b})^2 \quad (\text{当 } \mathbf{a}, \mathbf{b} \in \mathrm{Bool}^n)$$

故"汉明距离"是"欧氏距离平方"在二进顶点上之投影。

## 十一、流形 · 局部同胚

### 11.1 流形（manifold）

n 维流形 $M$：

> 每点 $p \in M$ 有邻域 $U \ni p$，存在同胚 $\phi : U \to V \subseteq \mathbb{R}^n$（开集）。

$\phi$ 称为**局部坐标卡**（chart）。
**流形** = 一族卡之相容覆盖。

### 11.2 字元

| 字 | 几何 |
|---|------|
| 局 | 局部 / chart |
| 形 | shape / 流形整体 |
| 面 | 二维流形 |
| 体 | 三维流形 |
| 域 | 开集 / domain |

### 11.3 与八卦之关系

**八卦本身**是离散有限集，**不是流形**（无局部欧氏结构）。
但 $\mathrm{Bool}^n$ **可嵌入**实流形 $[0,1]^n \subset \mathbb{R}^n$，
作为单位立方体之**顶点 0-skeleton**。

> **故"八卦几何"是离散拓扑 + 嵌入连续流形之顶点集**——
> 项目主线之**离散与连续之桥**。

## 十五、证明骨架

### 15.1 三角不等式（汉明距离）

**断言**：$\forall a, b, c \in \mathrm{Bool}^n, d_H(a, c) \leq d_H(a, b) + d_H(b, c)$.

**证明**：设 $a, b, c \in \{0, 1\}^n$。
对每位 $i$：
- 若 $a_i = c_i$，则 $|a_i - c_i| = 0 \leq |a_i - b_i| + |b_i - c_i|$（trivial）
- 若 $a_i \neq c_i$，则必 $a_i \neq b_i \lor b_i \neq c_i$（否则 $a_i = b_i = c_i$ 矛盾 $a_i \neq c_i$），
  故 $|a_i - b_i| + |b_i - c_i| \geq 1 = |a_i - c_i|$.

逐位求和：

$$d_H(a, c) = \sum_i |a_i - c_i| \leq \sum_i (|a_i - b_i| + |b_i - c_i|) = d_H(a, b) + d_H(b, c). \quad \blacksquare$$

### 15.2 立方体直径（汉明距离上界）

**断言**：$\forall a, b \in \mathrm{Bool}^n, d_H(a, b) \leq n$.

**证明**：$d_H(a, b) = \sum_{i=1}^n |a_i - b_i| \leq \sum_{i=1}^n 1 = n$（每位差异最多 1）。$\blacksquare$

**取等**：$a = \mathbf{0}, b = \mathbf{1}$（即坤 ↔ 乾），$d_H = n$。

### 15.3 连续映射之等价定义

**断言**：度量空间 $(X, d_X), (Y, d_Y)$，$f : X \to Y$。下二者等价：

(a) $\forall V \in \tau_Y, f^{-1}(V) \in \tau_X$
(b) $\forall x \in X, \forall \epsilon > 0, \exists \delta > 0, d_X(x, x') < \delta \Rightarrow d_Y(f(x), f(x')) < \epsilon$

**证明**：(a) ⟹ (b)：取 $V = B_Y(f(x), \epsilon)$ 开。则 $f^{-1}(V) \ni x$ 开，故有 $\delta > 0$ 使 $B_X(x, \delta) \subseteq f^{-1}(V)$，
即 $d_X(x, x') < \delta \Rightarrow f(x') \in V \Rightarrow d_Y(f(x), f(x')) < \epsilon$.

(b) ⟹ (a)：设 $V \in \tau_Y$ 任开，$x \in f^{-1}(V)$。$f(x) \in V$ 开，故 $\exists \epsilon > 0, B_Y(f(x), \epsilon) \subseteq V$。
由 (b) 取相应 $\delta$。则 $B_X(x, \delta) \subseteq f^{-1}(V)$。故 $f^{-1}(V)$ 开。$\blacksquare$

### 15.4 八卦层离散拓扑之 trivial 性

**断言**：在 $\mathrm{Bool}^3$ 上，每个单点集 $\{t\}$ 是开集。

**证明**：取 $\epsilon = 1/2$。则 $B_H(t, 1/2) = \{s : d_H(t, s) < 1/2\} = \{t\}$（因 $d_H$ 取整数值，$< 1/2$ 即 $= 0$ 即 $s = t$）。
故 $\{t\} = B_H(t, 1/2)$ 是基础开集。$\blacksquare$

**推论**：$\mathrm{Bool}^3$ 上**所有函数**皆连续（任意子集皆开）——拓扑层无信息，
**约束在度量层**（各反爻保汉明距离）。

### 15.5 反爻之等距性（isometry）

**断言**：$\forall i, \forall a, b \in \mathrm{Bool}^3, d_H(\bar{\partial}_i a, \bar{\partial}_i b) = d_H(a, b)$.

**证明**：$\bar{\partial}_i$ 翻第 $i$ 位。设 $a, b$ 位 $i$ 同则翻后仍同，位 $i$ 异则翻后仍异——故各位之差异不变，求和不变。$\blacksquare$

**推论**：$(\mathbb{Z}/2)^3$ 群作用是 $(\mathrm{Bool}^3, d_H)$ 上之**等距群**——
即"几何对称群"。

### 15.6 当位 / 中位 / 应位 / 比位之独立性

**断言**：四种位关系**两两独立**（在概率均匀分布下，知一不蕴涵另一）。

**证明**：构造例：
- 知"当位"不蕴涵"中位"：六爻全阳 ⟹ 1, 3, 5 当位（3 个），但中位 = 2, 5（2 个），非全等。
- 知"应位"不蕴涵"比位"：内坤外乾 (0,0,0,1,1,1) 全应（三对），比位结构无确定。
- 等等。

形式：四关系定义在不同 indexing 之上，组合上独立。$\blacksquare$

### 15.7 立方体 Cayley 图之连通性

**断言**：$(\mathrm{Bool}^n, \text{汉明距 1})$ 1-skeleton 图连通。

**证明**：$\forall a, b$，可逐位翻转 $a$ 之异于 $b$ 之位，得序列 $a = c_0, c_1, \ldots, c_k = b$，每步汉明距 1，$k = d_H(a, b)$。$\blacksquare$

**Lean 已证**：`bagua_intercommunication` (T₃) + `hex_intercommunication` (T₆)。

### 15.8 8 顶点立方体之 Euler 特征

**断言**：3-cube 之 Euler 特征 $\chi = V - E + F - C = 8 - 12 + 6 - 1 = 1$.

**证明**：直接计数：
- $V = 8$（八卦）
- $E = 12$（每轴 4 边 × 3 轴）
- $F = 6$（每轴对 face × 3 轴对）
- $C = 1$（整体一个 3-cube）

$\chi = 8 - 12 + 6 - 1 = 1$（与一般 contractible 空间之 Euler 特征 1 相符）。$\blacksquare$

**意义**：八卦立方体是**可缩**的——可连续变形为单点（**回归太极**）。

### 15.9 6-cube 之 Euler 特征

**断言**：6-cube 之 Euler 特征 $\chi = \sum_{k=0}^6 (-1)^k \binom{6}{k} 2^{6-k} = (2-1)^6 = 1$.

**证明**：使用 cubical complex 之 face count 公式：

$$\chi(I^n) = \sum_{k=0}^n (-1)^k \binom{n}{k} 2^{n-k}$$

代 $n = 6$：

$$\chi = 64 - 192 + 240 - 160 + 60 - 12 + 1 = 1$$

亦可由 $(2-1)^n = 1$ 之代数恒等。$\blacksquare$

**意义**：64 卦立方体亦可缩——亦回归太极。**立方体之"归一"是几何必然**。
