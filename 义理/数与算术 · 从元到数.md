# 数与算术 · 从元到数

四级生成 A–G 七体系给出之元数皆止于 2ⁿ（1, 2, 4, 8, 64）。
本文追加**自然数 ℕ、负数 ℤ、实数 ℝ** 之发生链与四则运算，
以项目自身之「道」为脊，以字元落地为底，以八卦为发生点。

## 二、零 与 一 · 本体始位

### 一 = 太极 = `Unit`

老子：「道生一」。系辞：「易有太极」。
形式：

$$\mathbf{一} = \text{Unit} = \{*\}, \quad |\mathbf{一}| = 1$$

「一」是**最小非空类型**——单点集，cardinality 1。
项目主线之「太极」即此。

### 零 = 空 = `Empty`

零非中国本土字。中国传统先有「一」（道之始），再有「无」（道之先），
**「零」字与零位记数法**自印度入华，约 12 世纪经阿拉伯—南宋传入。

形式：

$$\mathbf{零} = \text{Empty} = \emptyset, \quad |\mathbf{零}| = 0$$

中文承字：

| 字 | 偏向 |
|---|------|
| 空 | 容器无内容（佛家） |
| 无 | 本体之未发（道家） |
| 无极 | 太极之先（周敦颐） |
| 零 | 算术意义之 0（外来） |

**项目宜用**：`空`（中性），`无`（哲学侧），`零`（算术侧）。
**注**：此三字位置上同一（都指向 cardinality 0），但**语义层不同**——`空` 是容器之相，`无` 是本体之相，`零` 是计数之位。

### 三种构造哲学并列

| 构造 | 形式 | 意 |
|------|-----|---|
| von Neumann 序数 | $n = \{0, 1, \ldots, n-1\}$ | n **包含**所有前驱 |
| Zermelo 序数 | $n+1 = \{n\}$ | n+1 **仅含**最近前驱 |
| Peano 公理 | `0, succ` 公设 | n+1 **由** succ 生成 |
| 四级生成 2ⁿ | $2^n = $ 分 $^n($Unit$)$ | n 步**双分**之总数 |

四种构造皆合法，前三给 ℕ，第四仅给 2ⁿ 子集。
**本文主用 Peano**（最贴 `生生不息` 之「续」字），辅以列长（`|List Unit| = n`）作直观对照。

## 三、后继 · 计数之核心算子

### 算子：生

| 项 | 内容 |
|---|------|
| 单字 | **生** |
| 别字 | 续、再、又 |
| 类型 | $\text{ℕ} \to \text{ℕ}$，即 $n \mapsto n+1$ |
| 出处 | 系辞「生生之谓易」；老子 42「一生二，二生三，三生万物」 |
| 等价于 | Peano `succ`、Y combinator 之有限化 |

### 生 ≠ 分（关键澄清）

二者皆为「太极发生」之合法算子，**语义截然不同**：

| 算子 | 类型 | 卡数变换 | 给出 |
|------|-----|---------|-----|
| **分** | $A \to A \times \text{Bool}$ | $\|A\| \to 2\|A\|$ | 2ⁿ 子集（A–G 主用） |
| **生** | $A \to A + \text{Unit}$ | $\|A\| \to \|A\| + 1$ | 全 ℕ（本文主用） |

四级生成 (A 体系) 之「太极→两仪→四象→八卦」用 `分`；
本文之「一→二→三→…」用 `生`。

二者作为 endofunctor 不同：

$$\text{分} = - \times \text{Bool}, \qquad \text{生} = - + \text{Unit}$$

在 cardinality 上一者乘 2，一者加 1。

**老子之「三生万物」**——`生` 迭代直至「万物」（不限定基数），不再 2ⁿ。

## 四、自然数 ℕ 之构造

### 形式

$$\text{ℕ} \cong \mu X. \text{Unit} + X$$

即 `inductive ℕ := zero | succ (n : ℕ)`。
等价表述：

$$\text{ℕ} \cong \text{List Unit}, \qquad n \cong \underbrace{[*, *, \ldots, *]}_{n \text{ 个}}$$

「数」即「列长」——v14 之「位 → 列 → 量 → 数」具体落实。

### 与八卦之衔接

八卦给 ℕ 之**特定切片**：

| 级 | n | 卡数 |
|----|---|-----|
| 太极 | 0 步分 | 1 |
| 两翼 | 1 步分 | 2 |
| 四象 | 2 步分 | 4 |
| 八卦 | 3 步分 | 8 |
| 64 卦 | 6 步分 | 64 |

但 ℕ ⊋ {2ⁿ}：3, 5, 6, 7, 9, 10, ... 皆在 ℕ 中，不在分之像中。
此正是**本文超出八卦之处**——`生` 给非 2ⁿ 之数。

### 卡数 vs 序数

| 维度 | 字元 | 释 |
|-----|------|---|
| 基数 (cardinality) | 量 | 「有几个」（无序）|
| 序数 (ordinal) | 位 | 「第几个」（有序）|

二者在有限 ℕ 上同构（n 个 = 第 n 位之总），在 ω 之上分歧。
项目截于有限，故**ℕ 之卡数与序数同**。

## 五、加 + · 单字「合」

### 算子：合（加 / 益）

| 项 | 内容 |
|---|------|
| 单字 | **合** |
| 别字 | 加、益 |
| 类型 | $\text{ℕ} \times \text{ℕ} \to \text{ℕ}$ |
| 出处 | 益卦（42）；I-7 之「合二归一」 |

### Peano 定义

$$a + 0 = a, \qquad a + \text{生}(b) = \text{生}(a + b)$$

直观：合就是「列连」——将两列首尾相接。

### 道理

- 「同声相应，同气相求」（《易·乾·文言》）——同质之量可累加
- 「积少成多」——`生` 之有限多次复合即合

### 性质（皆 Peano 归纳可证）

| 律 | 表 |
|---|---|
| 交换 | $a + b = b + a$ |
| 结合 | $(a + b) + c = a + (b + c)$ |
| 零元 | $a + 0 = a$ |

## 六、减 − · 单字「损」（含 ℕ-截断与 ℤ-延伸）

### 算子：损（减 / 余）

| 项 | 内容 |
|---|------|
| 单字 | **损** |
| 别字 | 减、除去、亏、欠、余 |
| 类型 | 见下两版 |
| 出处 | 损卦（41）；老子 48「损之又损，以至于无为」 |

### 两版减法

**版一：ℕ 截断减**（道之「至无而止」）

$$a \stackrel{.}{-} b = \begin{cases} a - b & a \geq b \\ 0 & a < b \end{cases}$$

道理：损之极是「无为」（老子 48）——损至 0 即止，不复进。
此版与 ℕ 闭合，但**非群运算**（无逆）。

**版二：ℤ 延伸减**（道之「损过无极」）

允许损过 0 进入**负数**——见下节。
此版需扩域：ℕ ↪ ℤ。

### 与加之对偶（Galois 伴随）

ℕ 截断减是加法之**左伴随**：

$$\text{损}_b \dashv \text{合}_b \quad\text{即}\quad a \stackrel{.}{-} b \leq c \iff a \leq c + b$$

证：
- (⇒) 设 $a - b \leq c$（截断意义）。若 $a \geq b$，则 $a \leq c + b$ 直接由 $a - b \leq c$ 得；若 $a < b$，则 $a < b \leq c + b$。
- (⇐) 设 $a \leq c + b$。若 $a \geq b$，$a - b \leq c$；若 $a < b$，$a \stackrel{.}{-} b = 0 \leq c$。

**注意方向**：是 `损 ⊣ 合`，不是 `合 ⊣ 损`——
减是「左推」，加是「右拉」；这与日常直觉「减是加之逆」一致，但伴随之**精确方向**须以 Galois connection 之定义读。

## 七、负数 ℤ · 道之「反」

### 何须负数

ℕ 截断减损失信息：在 `3 - 5 = 0`（截断）下无法回到 `5 - 0 = 5`，故损非可逆。
若要损成群运算，须扩 ℕ 至 **ℤ**——加入「负」。

### 字元

| 字 | 偏向 |
|---|------|
| **负** | 计数侧主字（负 = 反正）|
| 反 | 易经主字（错卦之「全反」）|
| 亏 | 经济侧（亏空、亏欠）|
| 欠 | 债侧（欠债、欠收）|
| 异 | 抽象差异（v14 表中「自然数 → 异」即此一萌芽）|

中国传统在《九章算术·方程章》（成书约公元 1 世纪）已明列**正负术**：
「同名相除，异名相益，正无入负之，负无入正之」——
红筹表正、黑筹表负，加减规则形式化。
Brahmagupta（印度，约 628 CE）系统讨论负数之运算，
故**中国之负数实践早印度约六个世纪**。

### 形式构造

**Grothendieck 商集**：

$$\text{ℤ} = (\text{ℕ} \times \text{ℕ}) / {\sim}, \qquad (a, b) \sim (c, d) \iff a + d = b + c$$

直观：每整数表为「两 ℕ 之差」`(a, b) ↔ a - b`。

| ℤ 元 | 代表对 |
|------|-------|
| 0 | (0, 0)、(1, 1)、… |
| +n | (n, 0)、(n+1, 1)、… |
| -n | (0, n)、(1, n+1)、… |

### 与「错」之结构对应

G 体系之**错卦**算子：阴阳全反，作用空间 (Z/2)³（cardinality 8）。
ℤ 上之相应算子：**取反 negation**，作用空间 ℤ（cardinality ℵ₀）。

$$\text{错}(b_1, b_2, b_3) = (1{-}b_1, 1{-}b_2, 1{-}b_3), \qquad \text{反}(n) = -n$$

**作用空间 cardinality 不同，但作为算子皆是 order-2 involution**：

$$\text{错}^2 = \text{id}_{(\mathbb{Z}/2)^3}, \qquad \text{反}^2 = \text{id}_{\mathbb{Z}}$$

二者非 ℤ negation 与错卦之**同一算子**，而是**「order-2 involution」之两个具体实现**：
- 错卦：在 (Z/2)³ 上之全坐标 negation（无不动点）
- ℤ 反：在 ℤ 上之 additive inverse（仅 0 为不动点）

故「错卦」是 ℤ 之负元算子在**有限离散结构**上之**类似物**，
形式同——皆 involution；
作用域异——一者 (Z/2)³，一者 ℤ。

### 道理

- 阴阳之**全对偶**——非「以 0 为底，以正为尊」，而「正负互含，互为对偶」
- 老子「反者道之动」（40 章）——`反` 即 ℤ 之核心算子
- 损之深引：「损之又损」过 0 入负，即「亏」之入

### 性质：(ℤ, 合, 重) 是环

| 律 | 表 |
|---|---|
| (ℤ, +) 阿贝尔群 | 加法交换、结合、零元、负元 |
| (ℤ, ·) 交换 monoid | 乘法交换、结合、单位元 1 |
| 分配律 | $a(b+c) = ab + ac$ |
| 整环 | 无零因子 |

## 八、乘 × · 单字「重」

### 算子：重（乘 / 倍 / 积）

| 项 | 内容 |
|---|------|
| 单字 | **重** |
| 别字 | 乘、倍、积、贯 |
| 类型 | $\text{ℤ} \times \text{ℤ} \to \text{ℤ}$ |
| 出处 | 系辞「因而重之」；G 体系之「八卦相重 = 64 卦」 |

### Peano 定义

$$a \cdot 0 = 0, \qquad a \cdot \text{生}(b) = a \cdot b + a$$

即「重 = 反复合」——乘法是加法之有限多次复合。

### 与四级生成之直接命中

**重卦公式**：

$$8 \times 8 = 64 \quad\text{即}\quad 2^3 \cdot 2^3 = 2^6$$

八卦相重得 64 卦，正是 H 之乘法在 2ⁿ 层之**具体物化**。

更一般：

$$|A^n| = |A|^n = \underbrace{|A| \cdot |A| \cdots |A|}_{n \text{ 次重}}$$

四级生成之**指数律**即 `重` 之 n 次复合。

### 道理

- 「积少成多」、「积水成渊」（荀子）——重为累
- 「贯而通之」——重为穿连同质量
- 易经之「重」既是物理叠（重卦），也是数学积——二者同构

### 性质（皆可由加法 Peano 归纳证）

| 律 | 表 |
|---|---|
| 交换 | $a \cdot b = b \cdot a$ |
| 结合 | $(ab)c = a(bc)$ |
| 单位元 | $a \cdot 1 = a$ |
| 分配 | $a(b+c) = ab + ac$ |
| 零元 | $a \cdot 0 = 0$ |

## 九、除 ÷ · 单字「除」（带余）

### 算子：除（分 / 半 / 折）

| 项 | 内容 |
|---|------|
| 单字 | **除** |
| 别字 | 分、半、折、份 |
| 类型 | $\text{ℤ} \times (\text{ℤ}\setminus\{0\}) \to \text{ℤ} \times \text{ℤ}$，即 $(a, b) \mapsto (q, r)$ 满足 $a = qb + r, \, 0 \leq r < |b|$ |
| 出处 | 「分而析之」；D-6 之半；I-6 之分 |

### 字符歧义之澄清

**分** 在 G 中专指 `A → A × Bool`（split），与算术除之 `Nat × Nat → Nat × Nat`（quotient/remainder）形式不同但精神同——皆是「一拆为多」。

为避免歧义，本文用：
- **分**：generic 拆（含 split 与 partition）
- **除**：算术专义（带余除法）
- **半**：专 ÷2

### 与重之对偶（Galois 伴随，ℕ 上）

在 ℕ 上 `b > 0` 时：

$$\text{重}_b \dashv \text{除}_b \quad\text{即}\quad a \cdot b \leq c \iff a \leq \lfloor c / b \rfloor$$

证（标准 Euclidean）：
- (⇒) 若 $ab \leq c$，则 $a \leq c/b$（实数除），故 $a \leq \lfloor c/b \rfloor$。
- (⇐) 若 $a \leq \lfloor c/b \rfloor$，则 $ab \leq \lfloor c/b \rfloor \cdot b \leq c$。

整除时：$\text{除}(\text{重}(a, b), b) = (a, 0)$。
非整除：余 r ≠ 0，是 ℤ 不闭合于除之记号——故引 ℚ。

### ℚ 之引入

要 ÷ 完全闭合（除以非零得唯一），须扩 ℤ 至 **ℚ**：

$$\text{ℚ} = (\text{ℤ} \times (\text{ℤ}\setminus\{0\})) / {\sim}, \qquad (a, b) \sim (c, d) \iff ad = bc$$

字元：分（用作分数之「分」）、份、率。
道理：「分而析之」之极致——任意整数皆可被任意非零整数细分。
(ℚ, +, ·) 升为**域**（无平凡之外的零除）。

## 十、实数 ℝ · 「连续」之入

### 何须实数

ℚ 虽闭合于四则，但**不完备**——
有 Cauchy 序列收敛点不在 ℚ 中（如 √2、π、e）。
要数轴「无旷」（v14 第 469 行字眼），须扩 ℚ 至 **ℝ**。

### 字元（v14 第 469 行原列）

> 连续 | 序、邻近、极限、完备性 | **序、近、趋、极、完、无旷** |

| 字 | 数学义 |
|---|-------|
| 序 | linear order $\leq$ |
| 近 | neighborhood / metric |
| 趋 | convergence $a_n \to L$ |
| 极 | limit |
| 完 | completeness |
| 无旷 | no-gap (Dedekind cut) |

### 两种构造

**构造一：Cauchy 序列商集**

$$\text{ℝ} = \text{Cauchy}(\text{ℚ}) / {\sim}$$

其中 Cauchy(ℚ) 为 ℚ 中所有 Cauchy 序列，~ 为「差为零序列」之等价。
道理：与「趋极」之字最贴——`趋` 即 Cauchy 收敛。

**构造二：Dedekind 分割**

$$\text{ℝ} = \{ A \subset \text{ℚ} : A \text{ 为非空真下集，无最大元} \}$$

每实数表为「ℚ 之上下分」。
道理：与「无旷」之字最贴——填入 ℚ 之间的「裂缝」。

**项目主用**：构造一（Cauchy）——`趋` 是「生」在连续侧之类比，`极` 是不动点之复刻，皆与项目核心算子同源。

### 完备性公设

> ℝ 之合法性须**显式声明完备性公设**（v14 第 469 行入「完备性公设」族）。

形式：

$$\forall S \subset \text{ℝ},\, S \neq \emptyset \wedge S \text{ 有上界} \;\Rightarrow\; \sup S \in \text{ℝ}$$

此公设**不可由 ℚ 之四则推出**——是 ℕ→ℤ→ℚ→ℝ 链中**第一处真正越层**。
契约 (v14 第 446 行之 Contract)：模型须提供完备格或对应闭包条件，否则 ℝ-命题入 $\mathsf{U}$（待校）。

### 道理

- 「无旷」之「旷」即 ℚ 之裂缝；填之即 ℝ
- 「穷则变，变则通，通则久」（系辞下 11）——`极` 之复刻（取上确界）即「穷而后通」
- 「曲则全」（老子 22）——曲线之连续即 ℝ 之表征

### (ℝ, +, ·) 是完备有序域

性质继承 ℚ 之域结构，加完备性 = ℝ 之**特征公理**。

### ℝ 之四则

加减乘除继承 ℚ：

- 合：$(a_n) + (b_n) := (a_n + b_n)$
- 损：$(a_n) - (b_n) := (a_n - b_n)$
- 重：$(a_n) \cdot (b_n) := (a_n \cdot b_n)$
- 除：在 $b_n \not\to 0$ 之前提下，$(a_n) / (b_n) := (a_n / b_n)$（最终项有定义）

每运算皆需**保 Cauchy 性**：若 $(a_n), (b_n)$ 是 Cauchy 序列，证 $(a_n + b_n)$、$(a_n \cdot b_n)$ 亦 Cauchy。
此为标准引理（参 Rudin §3）。

**保 Cauchy 性 + 商集等价 ~ 之相容性 = ℝ 上四则良定义**。

## 十一、扩域链总图

```
   生(succ)        反(negation)       份(fraction)     趋极(completion)
1 ──────► ℕ ──────────► ℤ ──────────► ℚ ──────────► ℝ
   Peano        Grothendieck     Field of           Cauchy 完备化
   归纳          群完成           Fractions         (或 Dedekind)

字元根:
ℕ: 元、一、续、再、归
ℤ: 反、负、亏、欠、异
ℚ: 分、份、率
ℝ: 序、近、趋、极、完、无旷
```

每层是上层之**结构完成**（不一定是「完备化」之专技义）：

| 扩域 | 解决之「不闭合」 | 完成方式 |
|------|---------------|---------|
| 1 → ℕ | 计数无后继 | Peano 归纳 |
| ℕ → ℤ | 减不闭合（损过 0）| Grothendieck 群完成 |
| ℤ → ℚ | 除不闭合（非整除）| 分式域 |
| ℚ → ℝ | Cauchy 序列不收敛 | Cauchy 完备化（拓扑）|
| ℝ → ℂ | 多项式不闭合（√-1 不存）| 代数闭包之最小步 |

**每次扩域 = 增一字元根 + 增一公设契约。**

### 16.1 加法之性质（Peano 归纳）

记 $S = \text{生}$。定义：
- $a + 0 = a$
- $a + S(b) = S(a + b)$

**右零元** $a + 0 = a$：定义直得。

**左零元** $0 + a = a$：依 $a$ 归纳。
- $a = 0$：$0 + 0 = 0$（定义）。
- $a \to S(a)$：设 $0 + a = a$，则 $0 + S(a) = S(0 + a) = S(a)$。 ∎

**辅引理（succ 通过加法）**：$S(a) + b = S(a + b)$。
- $b = 0$：$S(a) + 0 = S(a) = S(a + 0)$。
- $b \to S(b)$：$S(a) + S(b) = S(S(a) + b) \stackrel{\text{IH}}{=} S(S(a + b)) = S(a + S(b))$。 ∎

**结合律** $(a + b) + c = a + (b + c)$：依 $c$ 归纳。
- $c = 0$：$(a + b) + 0 = a + b = a + (b + 0)$。
- $c \to S(c)$：$(a+b) + S(c) = S((a+b)+c) \stackrel{\text{IH}}{=} S(a+(b+c)) = a + S(b+c) = a + (b + S(c))$。 ∎

**交换律** $a + b = b + a$：依 $b$ 归纳。
- $b = 0$：左零元已证。
- $b \to S(b)$：$a + S(b) = S(a+b) \stackrel{\text{IH}}{=} S(b+a) \stackrel{\text{辅引理}}{=} S(b) + a$。 ∎

### 16.2 乘法之性质

定义：
- $a \cdot 0 = 0$
- $a \cdot S(b) = a \cdot b + a$

**左零元** $0 \cdot a = 0$：依 $a$ 归纳。
- $a = 0$：$0 \cdot 0 = 0$。
- $a \to S(a)$：$0 \cdot S(a) = 0 \cdot a + 0 \stackrel{\text{IH}}{=} 0 + 0 = 0$。 ∎

**右单位元** $a \cdot 1 = a$：$a \cdot S(0) = a \cdot 0 + a = 0 + a = a$。 ∎

**左单位元** $1 \cdot a = a$：依 $a$ 归纳。
- $a = 0$：$1 \cdot 0 = 0$。
- $a \to S(a)$：$1 \cdot S(a) = 1 \cdot a + 1 \stackrel{\text{IH}}{=} a + 1 = S(a)$。 ∎

**左分配** $a \cdot (b + c) = a \cdot b + a \cdot c$：依 $c$ 归纳。
- $c = 0$：$a(b+0) = ab = ab + 0 = ab + a \cdot 0$。
- $c \to S(c)$：$a(b+S(c)) = a \cdot S(b+c) = a(b+c) + a \stackrel{\text{IH}}{=} (ab + ac) + a = ab + (ac + a) = ab + a \cdot S(c)$。 ∎

**结合律** $(ab)c = a(bc)$：依 $c$ 归纳，用左分配。 ∎（标准）

**交换律** $ab = ba$：依 $b$ 归纳，用辅引理 $S(a) \cdot b = a \cdot b + b$。 ∎（标准）

### 16.3 ℤ 之 Grothendieck 构造良定义

设 ℤ := ℕ × ℕ / ~，其中 $(a,b) \sim (c,d) :\iff a + d = b + c$。

**~ 是等价关系**：
- **自反**：$(a,b) \sim (a,b)$ ⟺ $a + b = b + a$ ✓ 由 §16.1 交换律。
- **对称**：$(a,b) \sim (c,d) \iff a + d = b + c \iff c + b = d + a \iff (c,d) \sim (a,b)$。
- **传递**：$(a,b) \sim (c,d)$ 且 $(c,d) \sim (e,f)$ ⟹ $a+d = b+c$, $c+f = d+e$。
  二式相加得 $a + d + c + f = b + c + d + e$，约去 $c + d$（ℕ 满足消去律），得 $a + f = b + e$，即 $(a,b) \sim (e,f)$。 ∎

**运算良定义**：
- $[a,b] + [c,d] := [a+c, b+d]$
- $-[a,b] := [b,a]$
- $[a,b] \cdot [c,d] := [ac + bd, ad + bc]$

须验证三式之 ~ 兼容性（即代表元独立）。直接展开等式可证；标准引理。

**(ℤ, +, ·) 是交换环且无零因子**：所有 ring axiom 由 §16.1–§16.2 之 ℕ 性质提升而来；无零因子由 ℕ 之**消去律**得。 ∎

### 16.4 ℚ 是域

设 ℚ := ℤ × (ℤ\{0}) / ~，其中 $(a,b) \sim (c,d) :\iff ad = bc$。

**~ 是等价关系**：自反、对称同上；传递须用 ℤ 之消去律（无零因子）。 ∎

**乘法可逆**：$[a,b]^{-1} := [b,a]$，对 $a \neq 0$ 良定义。
验证：$[a,b] \cdot [b,a] = [ab, ba] = [1, 1]$（因 $ab \cdot 1 = ba \cdot 1$）。 ∎

故 (ℚ, +, ·) 是域。

### 16.5 ℝ 之 Cauchy 完备化（草）

设 $C(\mathbb{Q}) := \{(a_n) \in \mathbb{Q}^{\mathbb{N}} : (a_n) \text{ Cauchy}\}$。
设 $N := \{(a_n) \in C(\mathbb{Q}) : a_n \to 0\}$。

**$C(\mathbb{Q})$ 是 commutative ring**：逐项加乘；保 Cauchy 性是标准引理（用 ε/2 与有界性）。
**$N$ 是 ideal**：null 序列加之仍 null；Cauchy × null = null（用 Cauchy 之有界性）。

**ℝ := $C(\mathbb{Q}) / N$**——商环；可证为有序域，且**完备**（每 Cauchy 之 ℝ-像收敛）。

**与 Dedekind 之同构**：每实数 $r \in \mathbb{R}$ 对应分割 $\{q \in \mathbb{Q} : q < r\}$；
反之每分割 $A \subset \mathbb{Q}$ 对应 $\sup A$（在 Cauchy 完备化中存在）。
此映射保序、保运算，是同构。 ∎（标准；参 Rudin §1）
