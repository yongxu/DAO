## 〇 · 边界声明

本文不主张：

| 不主张 | 原因 |
|---|---|
| 已证明一切宇宙实在都被三元组穷尽 | Lean 只证明本接口内的 typed skeleton |
| `事` 等同概率论里的 event | 这里的事更宽：发生、作用、变化、成行 |
| `间` 只等同物理时空 | 间也可读作关系、间隔、位置、先后、内外、可分辨场域 |
| `物` 是静止实体 | 物只是可指认、可保持某种同一性的承载面 |
| 本文完成量子引力或物理统一 | “时空 + event”只是接口同构，不是物理理论证明 |

本文主张：

| 主张 | 精确读法 |
|---|---|
| 三面完整 | 在 `WuJianShiExpression` 接口中，任一表达都可由 `thing / interval / event` 三字段重建 |
| 三面不塌缩 | `物`、`间`、`事` 是三个不同结构位置，不能把其中一个当成另一个 |
| 三面一体 | 它们不是三种孤立实体，而是同一表达的三个必要投影 |
| 时空事件可互译 | 当 `间` 专门读作 `SpaceTime` 时，物间事表达与 carrier + spacetime + event 往返等价 |

## 一 · 三个面向

| 面向 | 形式字段 | 义理读法 | 物理化读法 |
|---|---|---|---|
| 物 | `thing : Thing` | 可指认、可承载同一性的东西 | carrier、system、field excitation、record bearer |
| 间 | `interval : Interval` | 区分、定位、相邻、远近、先后、内外之场域 | spacetime、frame、relation、metric / topology |
| 事 | `event : Event` | 发生、作用、变化、生成 | event、interaction、transition、occurrence |

物、间、事不是三堆东西，而是一个表达的三个投影。

只说“物”，会丢失它如何被分辨与如何发生。
只说“间”，会丢失被定位者与所发生者。
只说“事”，会丢失承载者与发生场域。

所以最薄的完整式是：

```lean
structure WuJianShiExpression (Thing Interval Event : Type _) where
  thing : Thing
  interval : Interval
  event : Event
```

这里的 `Thing / Interval / Event` 都是参数。本文不预设它们具体是什么，只说：一旦某个领域要进入此接口，就要给出这三类承载。

## 二 · 形式化描述

Lean 中先定义三面：

```lean
inductive TriuneAspect : Type
  | wu
  | jian
  | shi
```

然后证明三面穷尽：

```lean
theorem TriuneAspect.exhaustive (a : TriuneAspect) :
    a = .wu ∨ a = .jian ∨ a = .shi
```

这对应“没有第四槽位”。

再证明三面互异：

```lean
theorem TriuneAspect.pairwise_distinct :
    TriuneAspect.wu ≠ TriuneAspect.jian
      ∧ TriuneAspect.wu ≠ TriuneAspect.shi
      ∧ TriuneAspect.jian ≠ TriuneAspect.shi
```

这对应“不是三者混成一团”。

完整表达本身定义为：

```lean
structure WuJianShiExpression (Thing Interval Event : Type _) where
  thing : Thing
  interval : Interval
  event : Event
```

对任意表达 `x`，可证明：

```lean
theorem every_wjs_expression_decomposes
    (x : WuJianShiExpression Thing Interval Event) :
    ExpressedByTriad x
```

也就是 `x` 必有 `thing / interval / event` 三个见证。

再证明：

```lean
theorem wjs_rebuild_identity (x : WuJianShiExpression Thing Interval Event) :
    rebuild x.thing x.interval x.event = x
```

这就是本文“完整”的精确定义：在此接口内，取三字段再重建，不损失原表达。

## 三 · 证明

### 3.1 三面穷尽

`TriuneAspect` 是一个三构造子的归纳类型：

```lean
| wu
| jian
| shi
```

证明 `exhaustive` 时，对 `a` 做 cases。若 `a = wu`，落入第一支；若 `a = jian`，落入第二支；若 `a = shi`，落入第三支。

所以：

> 任一面向若属于这个接口，只能是物、间、事之一。

这不是经验归纳，而是接口定义的穷尽性。

### 3.2 三面互异

归纳类型的三个构造子不相等：

```lean
wu ≠ jian
wu ≠ shi
jian ≠ shi
```

所以：

> 三位一体不是三者相同，而是三者在同一表达结构中各守其位。

这保留了边界：物不能替代间，间不能替代事，事不能替代物。

### 3.3 表达可分解

任取：

```lean
x : WuJianShiExpression Thing Interval Event
```

对 `x` 做结构拆解，它必然形如：

```lean
{ thing := t, interval := i, event := e }
```

于是得到见证：

```lean
∃ t, ∃ i, ∃ e, x = { thing := t, interval := i, event := e }
```

这就是 `every_wjs_expression_decomposes`。

### 3.4 三字段可重建

定义：

```lean
def rebuild (thing interval event) :=
  { thing := thing, interval := interval, event := event }
```

于是：

```lean
rebuild x.thing x.interval x.event = x
```

由结构 eta 原理直接成立。

所以本文的证明核心是：

> 表达项的完整数据正是三字段；取出三字段再合回，仍是同一表达项。

### 3.5 三字段决定表达

若两个表达 `x` 与 `y` 满足：

```lean
x.thing = y.thing
x.interval = y.interval
x.event = y.event
```

则可推出：

```lean
x = y
```

这说明“完整表达”不是只凭一个名字、一个位置或一个变化来判定，而是由三面共同决定。

## 四 · “时空 + event”的精确定义

如果把 `间` 专门收窄为 `SpaceTime`，则物间事表达变成：

```lean
WuJianShiExpression Thing SpaceTime Event
```

为了对应“时空 + event”的说法，Lean 中定义：

```lean
structure SpacetimeEventExpression
    (Thing SpaceTime Event : Type _) where
  carrier : Thing
  spacetime : SpaceTime
  occurrence : Event
```

注意这里保留 `carrier`。

原因是：若只说裸的 `spacetime + event`，容易把“物”吞掉；但在可审计表达中，事件总要有承载、参与者、记录者、系统或可指认的同一性面。这个面就是 `物`。

因此本文把“时空 + event”严格读作：

```text
carrier + spacetime + occurrence
```

它与：

```text
thing + interval + event
```

往返等价：

```lean
theorem spacetime_event_equivalence_summary
    (x : WuJianShiExpression Thing SpaceTime Event)
    (y : SpacetimeEventExpression Thing SpaceTime Event) :
    fromSpacetimeEvent (toSpacetimeEvent x) = x
      ∧ toSpacetimeEvent (fromSpacetimeEvent y) = y
```

所以：

> “物、间、事”是义理表述；“carrier + spacetime + event”是物理化接口。二者在本文 typed skeleton 中同构。

## 五 · 三位一体的义理读法

可以压成三句：

| 句 | 含义 |
|---|---|
| 物是事的凝定 | 若无某种可指认承载，发生不可归属 |
| 事是物的展开 | 若无发生变化，物只是死名 |
| 间是物与事得以分辨的条件 | 若无间，物无位，事无处，差异不可说 |

更形式地说：

```text
Expression ≃ Thing × Interval × Event
```

但不要误读为：

```text
Thing = Interval = Event
```

正确读法是：

```text
一个表达有三面；三面互不相等；三面合取才给出完整表达。
```

所以“三位一体”的重点不是神秘同一，而是结构一体：

| 错读 | 正读 |
|---|---|
| 三者是同一个东西 | 三者是同一表达的三个不可混同投影 |
| 物可单独穷尽世界 | 物必须在间中成事 |
| event 可单独穷尽世界 | event 要有承载与定位 |
| spacetime 可单独穷尽世界 | spacetime 要有被分辨者与发生者 |

## 六 · 主定理入口

```lean

    (x : WuJianShiExpression Thing Interval Event) :
    ExpressedByTriad x
      ∧ rebuild x.thing x.interval x.event = x
      ∧ (∀ a : TriuneAspect, a = .wu ∨ a = .jian ∨ a = .shi)
      ∧ AspectPresent x .wu
      ∧ AspectPresent x .jian
      ∧ AspectPresent x .shi
      ∧ TriuneAspect.wu ≠ TriuneAspect.jian
      ∧ TriuneAspect.wu ≠ TriuneAspect.shi
      ∧ TriuneAspect.jian ≠ TriuneAspect.shi
```

## 七 · 最短表述

> 一切进入表达接口者，皆可作物、间、事三面读；物为承载，间为分辨定位，事为发生变化。三者互不相等，却共同决定同一表达。

或更短：

> 物在间中成事；事依物而发生；间使物与事可分辨。
