# E · 古代不同源流的诸 mapping

四级生成模式在中国古代非易传一家专属。本文按源流并列各家变奏，
显其**共性**（皆为「一→多」之分级展开）与**异性**（每级所取之内容不同）。

## 各家详记

### 1. 易·系辞（主干）

> 易有太极，是生两仪，两仪生四象，四象生八卦。

严格 2ⁿ 二分。唯一算子 `分` 重复施加。详见 `A_经典易传.md`。

### 2. 老子 42 章

> 道生一，一生二，二生三，三生万物。

| 级 | 内容 | 元数律 |
|----|-----|-------|
| 道 | 太极位 | — |
| 一 | 太极之初分 | 1 |
| 二 | 阴阳 | 2 |
| 三 | 阴阳冲气以为和（中态） | 3 |
| 万物 | 末级 | 不定 |

老子之**三**是关键变异——非 2ⁿ，而是 2 + 1（阳/阴/冲气），
更接近本目录 B 体系之**三态**（阳极/阴极/中态）。

### 3. 周敦颐《太极图说》

> 无极而太极。太极动而生阳，动极而静；静而生阴。
> 静极复动。一动一静，互为其根。分阴分阳，两仪立焉。
> 阳变阴合，而生水火木金土，五气顺布，四时行焉。

| 级 | 内容 |
|----|-----|
| 无极 | 太极之先 |
| 太极 | 一 |
| 两仪 | 阳/阴（由动/静生） |
| 五行 | 水火木金土（4 + 1 而非 4） |
| 万物 | 末级 |

变异：四级被**五行**替代，元数为 5 而非 4。
五行非 2² 之四象，而是**循环图**（生克轮）。

### 4. 邵雍 先天图

| 级 | 内容 | 元数 |
|----|-----|-----|
| 太极 | 一 | 1 |
| 阴阳 | 两仪 | 2 |
| 四象 | 老阳/少阴/少阳/太阴 | 4 |
| 八卦 | 先天序：乾兑离震巽坎艮坤 | 8 |
| 六十四卦 | 方圆图 | 64 |

邵雍**严格继承易传**，但提出**先天图**（自然序）与**后天图**（文王序）之分。
先天图即纯 2ⁿ 二进编码（与现代二进制同构，莱布尼茨即由此受启）。

### 5. 京房 卦气

| 级 | 内容 |
|----|-----|
| 太极 | — |
| 乾坤 | 父母卦 |
| 四正卦 | 坎离震兑（主四时） |
| 八宫 | 乾兑离震巽坎艮坤为宫主 |
| 64 卦 | 八宫各 8 卦 |

变异：二级取**乾坤**而非阴阳，四级取**四正卦**而非四象，
是**具体卦**作分级而非抽象阴阳。

### 6. 汉象数 / 卦气说

| 级 | 内容 |
|----|-----|
| 太极 | — |
| 两仪 | 阴阳 |
| 四时 | 春夏秋冬 |
| 八节 | 立春/春分/立夏/夏至/立秋/秋分/立冬/冬至 |
| 24 节气 | 八节各分三 |

变异：分级映射至**时序**，元数 1, 2, 4, 8, 24（末级 8×3）。
本变异与本目录 C 体系（实虚史真）之**史轴投影**相通。

### 7. 黄帝内经

| 级 | 内容 |
|----|-----|
| 太一 | 太极之异名 |
| 阴阳 | 两仪 |
| 四时 | 春夏秋冬 |
| 八风 | 八方之风 |
| 五运六气 | 末级（30 配） |

医家变异，重在**节律**而非组合。

### 8. 扬雄《太玄经》

| 级 | 内容 | 元数律 |
|----|-----|-------|
| 玄 | 太极位 | 1 |
| 三方 | 天/地/人 | 3 |
| 九州 | 三方各分三 | 9 |
| 二十七部 | 九州各分三 | 27 |
| 八十一首 | 二十七部各分三 | 81 |

**最重要之变异**：用 **3ⁿ** 而非 **2ⁿ**！
扬雄三分而易传二分，故《太玄》81 首与《周易》64 卦取异路。

此即本目录 C 体系**三态化**（27 格 = 3³）之古代先驱——
表五（27 格）与扬雄《太玄》之九州/二十七部直接同构。

## 元数律对照

| 源流 | 元数序列 | 律 |
|------|---------|---|
| 易传 | 1, 2, 4, 8, 16, 32, 64 | $2^n$ |
| 老子 | 1, 2, 3, ∞ | 二（阴阳）+ 一（冲气）→ 多 |
| 周敦颐 | 1, 2, 5, ∞ | $2 \to 5$ 跳跃（五行非幂） |
| 邵雍 | 1, 2, 4, 8, 64 | $2^n$（同易传） |
| 太玄 | 1, 3, 9, 27, 81 | $3^n$ |
| 卦气 | 1, 2, 4, 8, 24 | $2^n$ 截于 8，末级 $8 \times 3$ |

## 主判

**2ⁿ 体系**是主流（易传、邵雍），结构最严，落字最齐。
**3ⁿ 体系**（太玄）次主流，对应中态承担之精微。
**5、24 等非幂体系**（周敦颐、卦气）反映节律/循环结构，需另作图论刻画，
不在「四级生成」本框架内。

项目主线（生生不息论）取 **2ⁿ 骨架（D 体系）+ 3ⁿ 三态（C 体系/表五）+ 六征三对（B 体系）**
之三层叠合。
