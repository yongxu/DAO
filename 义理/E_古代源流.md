# E · 古代不同源流的诸 mapping

四级生成模式在中国古代非易传一家专属。本文按源流并列各家变奏，
显其**共性**（皆为「一→多」之分级展开）与**异性**（每级所取之内容不同）。

## 各家详记

### 1. 易·系辞（主干）

> 易有太极，是生两仪，两仪生四象，四象生八卦。

严格 2ⁿ 二分。唯一算子 `分` 重复施加。详见 `A_经典易传.md`。

### 2. 老子 42 章

> 道生一，一生二，二生三，三生万物。

| 级 | 内容 | 元数律 |
|----|-----|-------|
| 道 | 太极位 | — |
| 一 | 太极之初分 | 1 |
| 二 | 阴阳 | 2 |
| 三 | 阴阳冲气以为和（中态） | 3 |
| 万物 | 末级 | 不定 |

老子之**三**是关键变异——非 2ⁿ，而是 2 + 1（阳/阴/冲气），
更接近本目录 B 体系之**三态**（阳极/阴极/中态）。

### 3. 周敦颐《太极图说》

> 无极而太极。太极动而生阳，动极而静；静而生阴。
> 静极复动。一动一静，互为其根。分阴分阳，两仪立焉。
> 阳变阴合，而生水火木金土，五气顺布，四时行焉。

| 级 | 内容 |
|----|-----|
| 无极 | 太极之先 |
| 太极 | 一 |
| 两仪 | 阳/阴（由动/静生） |
| 五行 | 水火木金土（4 + 1 而非 4） |
| 万物 | 末级 |

变异：四级被**五行**替代，元数为 5 而非 4。
五行非 2² 之四象，而是**循环图**（生克轮）。

### 4. 邵雍 先天图

| 级 | 内容 | 元数 |
|----|-----|-----|
| 太极 | 一 | 1 |
| 阴阳 | 两仪 | 2 |
| 四象 | 老阳/少阴/少阳/太阴 | 4 |
| 八卦 | 先天序：乾兑离震巽坎艮坤 | 8 |
| 六十四卦 | 方圆图 | 64 |

邵雍**严格继承易传**，但提出**先天图**（自然序）与**后天图**（文王序）之分。
先天图即纯 2ⁿ 二进编码（与现代二进制同构，莱布尼茨即由此受启）。

### 5. 京房 卦气

| 级 | 内容 |
|----|-----|
| 太极 | — |
| 乾坤 | 父母卦 |
| 四正卦 | 坎离震兑（主四时） |
| 八宫 | 乾兑离震巽坎艮坤为宫主 |
| 64 卦 | 八宫各 8 卦 |

变异：二级取**乾坤**而非阴阳，四级取**四正卦**而非四象，
是**具体卦**作分级而非抽象阴阳。

### 6. 汉象数 / 卦气说

| 级 | 内容 |
|----|-----|
| 太极 | — |
| 两仪 | 阴阳 |
| 四时 | 春夏秋冬 |
| 八节 | 立春/春分/立夏/夏至/立秋/秋分/立冬/冬至 |
| 24 节气 | 八节各分三 |

变异：分级映射至**时序**，元数 1, 2, 4, 8, 24（末级 8×3）。
本变异与本目录 C 体系（实虚史真）之**史轴投影**相通。

### 7. 黄帝内经

| 级 | 内容 |
|----|-----|
| 太一 | 太极之异名 |
| 阴阳 | 两仪 |
| 四时 | 春夏秋冬 |
| 八风 | 八方之风 |
| 五运六气 | 末级（30 配） |

医家变异，重在**节律**而非组合。

### 8. 扬雄《太玄经》

| 级 | 内容 | 元数律 |
|----|-----|-------|
| 玄 | 太极位 | 1 |
| 三方 | 天/地/人 | 3 |
| 九州 | 三方各分三 | 9 |
| 二十七部 | 九州各分三 | 27 |
| 八十一首 | 二十七部各分三 | 81 |

**最重要之变异**：用 **3ⁿ** 而非 **2ⁿ**！
扬雄三分而易传二分，故《太玄》81 首与《周易》64 卦取异路。

此即本目录 C 体系**三态化**（27 格 = 3³）之古代先驱——
表五（27 格）与扬雄《太玄》之九州/二十七部直接同构。

### 9. 《尚书·洪范》九畴

史料复核时发现：本卷原已列五行，却未列更早、更完整的「五行入政」框架。
《洪范》以九畴总摄天人治理：五行、五事、八政、五纪、皇极、三德、稽疑、庶征、五福六极。
其中「皇极」居中，不是单纯 $5$ 或 $9$ 的数表，而是把自然、身体、历法、政治、占验、福祸连为一张治理网。

| 畴 | 内容 | 与本体系之关系 |
|----|------|----------------|
| 初一 | 五行（水火木金土） | 可补足 E 中「五行非本目录涵盖」之缺口：五行不是 2ⁿ 生成树，而是五态过程图 |
| 次二 | 五事（貌言视听思） | 与心智 / 知觉 / 信诚层相接 |
| 次三 | 八政 | 与政治哲学、经济博弈层相接 |
| 次四 | 五纪（岁月日星辰历数） | 与史维、时序、物候层相接 |
| 次五 | 皇极 | 与「中 / 极 / 不偏」之 Kernel 判准相接 |
| 次六 | 三德（正直、刚克、柔克） | 与三态治理、开闭调节相接 |
| 次七至九 | 稽疑、庶征、五福六极 | 与判定协议、反馈、风险审计相接 |

**补入判定**：洪范九畴是本目录遗漏的一级古代矩阵。它不是四级生成，但非常适合作为
「五行循环 + 三德调节 + 皇极中枢 + 历法反馈」的 statecraft-DAG。

### 10. 《礼记·月令》/《吕氏春秋·十二纪》月令体系

本卷原列「卦气 24」与《黄帝内经》，但还缺少礼制—政令版本的时序矩阵。
月令体系按十二月展开，把四时、五方、五帝、五神、五虫、五音、五味、政令、禁令绑定为一套
calendar-to-governance 协议。

| 层 | 内容 | 元数律 |
|----|------|--------|
| 四时 | 春夏秋冬 | 4 |
| 十二月 | 孟 / 仲 / 季 × 四时 | 3 × 4 |
| 五方五行 | 东南中西北 / 木火土金水 | 5 |
| 政令 | 每月祭祀、农事、刑赏、禁令 | 12-cycle |

**补入判定**：月令不是卦爻组合，而是「时—物候—政治」投影；它能补本体系 C（史维三态）与 Z（经济/政令）之间的
古代制度史桥。

### 11. 《周易·说卦》八卦互关系图

本卷对八卦多作生成树处理，但《说卦》另有一套关系图：天地定位、山泽通气、雷风相薄、水火不相射。
此处重点不是「一分为八」，而是八卦成对以后形成互补、相感、不相害的 graph topology。

| 卦对 | 关系 | 形式读法 |
|------|------|----------|
| 乾 / 坤 | 天地定位 | 极点定位 / 坐标系建立 |
| 艮 / 兑 | 山泽通气 | 异类通道 / flow edge |
| 震 / 巽 | 雷风相薄 | 动力耦合 / perturbation |
| 坎 / 离 | 水火不相射 | 对立不毁 / safe separation |

**补入判定**：《说卦》可补 D/G 中「八卦互通」的史料根，不只证明八卦可生成，也证明八卦之间有原生网络结构。

### 12. 《周易·系辞》大衍筮法与七八九六

本卷已有「易传 2ⁿ」主干，但未把筮法数论单列。大衍筮法以「大衍之数五十，其用四十有九」起算，经四营十八变，
落到老阳九、少阴八、少阳七、老阴六，再由六爻成卦。

| 层 | 内容 | 形式读法 |
|----|------|----------|
| 50 / 49 | 全数与可用数 | total vs operational fuel |
| 四营 | 分、挂、揲、归奇 | procedural algorithm |
| 十八变 | 三变成一爻，六爻成卦 | bounded iteration |
| 6/7/8/9 | 老阴、少阳、少阴、老阳 | 四象作为状态值，而非仅二爻名 |

**补入判定**：大衍筮法是「生成树」之外的可运行算法史料；可接到补篇〇之 runnable model 与 Lean 中 fuel/iteration 叙述。

### 13. 京房纳甲、世应、飞伏（待专篇）

本卷已列京房卦气，但京房易还有更工程化的层：八宫、纳甲、世应、飞伏、六亲、六神等。
它把六十四卦变成占验网络与索引协议，不只是气候历法。

**补入判定**：此项需要单独整理，不宜在 E 卷粗略吞并。现先标为待专篇：
「京房易 = 八宫索引 + 干支映射 + 关系位（世应）+ 隐显位（飞伏）」。

## 主判

**2ⁿ 体系**是主流（易传、邵雍），结构最严，落字最齐。
**3ⁿ 体系**（太玄）次主流，对应中态承担之精微。
**5、9、12、24 等非幂体系**（洪范、周敦颐、月令、卦气）反映节律/循环/治理结构，需另作图论刻画；
它们不在「四级生成」窄框架内，但应纳入本项目的史维、政令、审计与可运行模型。

项目主线（生生不息论）取 **2ⁿ 骨架（D 体系）+ 3ⁿ 三态（C 体系/表五）+ 六征三对（B 体系）**
之三层叠合。


## 史料复核备忘（2026-05-07）

本次复核依据公开古籍文本重新扫过「生成 / 数术 / 时令 / 治理」四类材料，结论如下：

1. **确有遗漏**：最大遗漏不是另一个 2ⁿ 生成链，而是《洪范》九畴、《月令》十二月政令、《说卦》八卦互关系图与《系辞》大衍筮法。
2. **主线不需推翻**：易传 2ⁿ、太玄 3ⁿ、六征三对仍是主骨架；新增史料主要补「图结构、过程算法、制度时序」。
3. **应新开两类工作**：
   - 「五行循环图谱」：洪范 → 月令 / 十二纪 → 周敦颐 → 内经。
   - 「筮法可运行模型」：大衍筮法 → 七八九六 → 六爻成卦 → 京房八宫 / 纳甲。
