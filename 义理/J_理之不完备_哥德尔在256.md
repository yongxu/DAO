### 5.1 实际主定理 (已形式化, 主 HEAD 1c76a55)

```lean
/-- 停机谓词：Σ₁ over Cell256-based BaguaTuring. -/
def Halts (P : List YiInstr) (h : Hexagram) : Prop :=
  ∃ n : Nat, ((YiState.init h P).runFuel n).halted = true

/-- Kleene 递归性质 (V₄-Shi-aware, cuo-invariant restricted). -/
def KleeneInverter : Prop :=
  ∀ (decide : List YiInstr → Hexagram → Bool),
    CuoInvariantDecide decide →
    ∃ D : List YiInstr, ∀ h : Hexagram, Halts D h ↔ decide D h = false

/-- 标准对角线：在 Kleene 递归下，Halts 不可由任何 cuo-invariant Lean Bool 函数判定. -/
theorem halts_undecidable_under_kleene (h_kleene : KleeneInverter) :
    ¬ ∃ decide : List YiInstr → Hexagram → Bool,
        CuoInvariantDecide decide ∧
        ∀ P h, decide P h = true ↔ Halts P h := by
  intro ⟨decide, h_cuo, hd⟩
  obtain ⟨D, hD⟩ := h_kleene decide h_cuo
  -- 对角线矛盾 (与 v1 同样的标准证明)
  have h_iff_halts := hd D Hexagram.qian
  have h_iff_invert := hD Hexagram.qian
  cases hb : decide D Hexagram.qian with
  | true =>
    have h_halts := h_iff_halts.mp hb
    have h_false := h_iff_invert.mp h_halts
    rw [hb] at h_false; contradiction
  | false =>
    have h_halts := h_iff_invert.mpr hb
    have h_true := h_iff_halts.mpr h_halts
    rw [hb] at h_true; contradiction

/-- 唯一项目 axiom: Kleene 递归对 BaguaTuring on Cell256 之实例. -/
axiom kleene_recursion_axiom : KleeneInverter

/-- **理之不完备主定理 (无条件版)**. -/
theorem halts_undecidable_internally :
    ¬ ∃ decide : List YiInstr → Hexagram → Bool,
        CuoInvariantDecide decide ∧
        ∀ P h, decide P h = true ↔ Halts P h :=
  halts_undecidable_under_kleene kleene_recursion_axiom
```

**关键修正 (v3 vs v1)**: v3 加了 `CuoInvariantDecide` 约束 — 因为 `cuo` 是 P (parity) 之 algebraic anchor，`Halts` 是 cuo-invariant predicate (cuo 反 hex 全部 yao 不影响 Halting)，所以决策器必须也是 cuo-invariant 才有意义。这是 v3 比 v1 严格之处。
