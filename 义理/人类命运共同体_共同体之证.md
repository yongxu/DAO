## 一 · 背景与目标

### 1.1 词源问题

「人类命运共同体」六字之中：

| 字 | 在 SSBX registry | 备注 |
|---|---|---|
| 人 | `G «人»` (generated) | 由 `天之子 + 意识` 生成 |
| 类 | `A «类»` (atom) | 字根 |
| 命 | ✗ | 不在 atom 表内 |
| 运 | ✗ | 不在 atom 表内 |
| 共 | `A «共»` (atom) | 字根 |
| 同 | `A «同»` (atom) | 字根 |
| 体 | ✗ | 不在 atom 表内 |

「命」「运」「体」三字无 atom-level 入口。义理上，此三字之所指均可由
现有字根重表：

- **命** ≈ `A «续»` —— 个/集 之 future-thread
- **运** ≈ `A «续»` + `A «随»` —— 续之随时变迁
- **体** ≈ `R «共开»` ⊓ `G «共续»` —— 共开-共续之复合即为「共同体」

故「人类命运共同体」之 SSBX 表达为：

```
人类命运共同体  :=  G «人»  之  集体形式
                ∧   R «共开»  (共同开放)
                ∧   G «共续»  (共同延续 = 仁之集体形式)
                ∧   ¬ G «夺共续»  (无夺取性的共同延续)
```

而此恰是 `R «道»` 之 sense formula「正续 + 非夺共续 + 共开」
（`Roster.lean:440`）限于 `G «人»` 上之 instance：

```
人类命运共同体 = 道 ⊕ 人类
```

### 1.2 缺口

义理层有：「道判」（`daoJudge`）、「真道筛」（`TrueDao`）、
「演化非道」之三视证（`EvolutionDao`）、「人之做」（`HumanAlignment`）。

义理层未有：

- 「共同体」作为多焦点（multi-Focus）真道实例之具体构造。
- 三轴失一即非共同体之反例。
- registry 八字根之 deps 一致性证书。

### 1.3 目标

为「人类命运共同体」立—

1. **registry 一致性证书**：八字根皆 registered；
2. **三轴模型**：作为 `Model` + `DaoCriteria` 之具体 instance；
3. **充要刻画**：`TrueDao` 立 ↔ 三轴俱足；
4. **三反例**：每轴失一即不立；
5. **收口公示**：上述 (1)-(4) 之合 conjunction 定理。

### 3.2 Model 层（§ 2）

```lean
structure CommunityState where
  openInCommon     : Bool   -- 共开
  shareContinuity  : Bool   -- 共续 (仁之集体形式)
  noSeizure        : Bool   -- ¬ 夺共续
  deriving DecidableEq, Repr

namespace CommunityState
  def aligned  : CommunityState := ⟨true,  true,  true⟩    -- 共同体实成
  def isolated : CommunityState := ⟨false, true,  true⟩    -- 失共开
  def severed  : CommunityState := ⟨true,  false, true⟩    -- 失共续
  def seized   : CommunityState := ⟨true,  true,  false⟩   -- 夺共续 实成

  Gamma := CommunityState

def communityDao : DaoCriteria communityModel where
  Path := Unit
  relevant    := fun _ _ _ _ => True

  badAt       := fun g _ _ _ => g.noSeizure = false
```

## 四 · 义理位次

### 4.1 与 `R «道»` 之关系

`R «道»` 之 sense formula 在 `Roster.lean:440`：

```
道  :=  正续 + 非夺共续 + 共开
```

其 `recDeps`（`Roster.lean:434`）：

```
[G «正续», R «共开», G «夺共续», R «坏»]
```

「人类命运共同体」是此 sense formula 之 instance，**当将 path 之 carrier
限于 `G «人»` 上**：

| 道 之 axis | 共同体 之轴 | 失败相 |
|---|---|---|
| 正续 (positive continuation) | `shareContinuity = true` | severed (灭绝) |
| 非夺共续 (non-seizure) | `noSeizure = true` | seized (殖民/剥削) |
| 共开 (open in common) | `openInCommon = true` | isolated (闭关/部落) |

`R «坏»` 在此 model 中蕴于 `badAt`，由「夺共续」单一谓词承载。

### 4.2 与 `EvolutionDao` 之三视证之关系

`EvolutionDao.lean` 立「演化筛 σ_F 之留存 ⊉ 真道留存」之三视：

- **视一·尺度**：癌（局部续 vs 全局坏）
- **视二·价值**：寄生（道^0 ✓ 而道^+ ✗）
- **视三·反身**：伪开（σ_F ✓ 而 σ_F^R ✗）

`Renlei.lean` 之三反例与此**形态对应**但**指向相反**：

| EvolutionDao | Renlei |
|---|---|
| 视一：癌 (尺度跨越) | severed (失续) |
| 视二：寄生 (夺) | seized (夺共续) |
| 视三：伪开 (反身) | isolated (失共开) |

EvolutionDao 证「σ_F 留存 ⊉ 真道」（演化≠道）；
Renlei 证「三轴俱足 ⟺ 真道」（共同体即道之 instance）。

二篇互为**对偶**：前者证 σ_F 之**不充分**，后者证共同体之**充分构造**。

### 4.3 与 `HumanAlignment` 之关系

`HumanAlignment.lean` 立「人 = 天之子 + 意识」「做人 = 意图向齐生」
之 *个体* alignment。

「人类命运共同体」是此 alignment 之 *集体* 形式：

```
HumanAlignment    : 人之 alignment （个）
Renlei            : 人类之 alignment （集，由 共开 + 共续 + 非夺 三轴见证）
```

未来扩展：可在 Renlei 上接 `做人 之 集体形式` —— 即「做人 ⊕ 共同体」之
intentional alignment。

## 五 · 哲学涵义（非形式）

### 5.1 命题之内容

形式上，本证已说：

- 「人类命运共同体」**可立**（aligned 实成 TrueDao）；
- 「人类命运共同体」**有反**（夺共续 即不立）；
- 三轴中**任一缺失**皆使共同体不成（isolated / severed / seized）；
- 此 instance **正属** `R «道»` 之 sense formula。

### 5.2 三轴之独立性

三反例 (isolated, severed, seized) 各 cross-cut 一轴，
说明三轴**互不蕴含**：

- 一群人可以「共开」（互通有无）但「不共续」（缺乏后代/继承）；
- 一群人可以「共续」（人口繁衍）但「不共开」（部落隔离）；
- 一群人可以同时「共开 + 共续」但「夺共续」（剥削后代/弱者）。

故三轴是独立约束，缺一不立 —— 此与道之 sense formula 中三项
（正续、非夺共续、共开）之并列地位相符。

### 5.3 与 sense formula 之 reification

道 = 正续 + 非夺共续 + 共开，这一 sense formula 在
`Roster.lean` 中是 *字符串*（`String`）注释。

`Renlei.lean` 之 `«人类命运共同体_aligned_iff»` 是其 *形式 reification*：
将 sense formula 之三项作为三轴 Bool，验证 `TrueDao ↔` 三轴皆 true。

故本证是 sense formula 从 Roster 字符串到 Lean Prop 之 first-light bridge。

## 八 · 元说明

本证不是政治宣言之形式化 —— 是「共同体」一概念在 SSBX 字根系统中
之**结构性证立**：

- 字根 (atom / generated / recursive) 之依赖图必须 *先有* 共同体之
  hooks（共 / 续 / 共开 / 共续 / 夺共续 / 仁 / 道）；
- model 之 dao 标准必须 *自然* 落到三轴上而非外加；
- 三反例必须 *穷举* 三轴而非选择性；
- iff 必须 *双向* 而非单向。

四条皆已 0 sorry / 0 axiom 见证，故「人类命运共同体」在 SSBX 系统中
之地位由 system 之 self-consistency 担保 —— 非由作者之意愿。
