## 二、真值二态 · 阴阳之逻辑投影

### 基础映射

| 易经 | 真值 | 二进 | 字元 |
|------|-----|-----|-----|
| 阳 ⚊ | ⊤（真）| 1 | 是、有、然 |
| 阴 ⚋ | ⊥（假）| 0 | 非、无、否 |

形式：

$$\text{Bool} = \{\bot, \top\} = \{0, 1\} \cong \text{Yao} = \{\text{阴}, \text{阳}\}$$

**一爻 = 一真值之承担**——这是项目最核心之同构。

### 字元之精细分

| 字 | 偏向 | 强度 |
|---|------|-----|
| 是 | 一般肯定 | 标准 |
| 然 | 状态肯定（如此）| 中 |
| 有 | 存在肯定 | 弱 |
| 非 | 类型否定（≠）| 结构 |
| 不 | 命题否定（¬）| 标准 |
| 弗 | 强否定（古） | 强 |
| 无 | 存在否定 | 弱 |

「是 / 非」之对：墨经已用作真假二极。

项目宜将此二字之**职能分离**：

- **是 / 非** = 真值（值层）
- **不** = 命题否定算子（连接词）

## 三、三值之入 · 项目独有之特色

### 表四之三态

由 `表四_真假认识三态.md`，项目逻辑非二值，是**三值**：

| 真值 | 字元 | 释 | 物理对应 |
|------|-----|---|---------|
| ⊤ | 是 / 真 | 已断为真 | 已塌缩之实 |
| ⊥ | 非 / 假 | 已断为假 | 已塌缩之虚 |
| **U** | **疑** / 玄 / 几 | **未决之悬** | **superposition / 互含** |

### 三值之必要性（项目论证）

由 `生生不息论_v14_形式证明骨架版.md` 第 509 行：

$$\mathsf{U} \nRightarrow \top$$

**三值保守律**：未断之命题不得冒称为真。
此律在二值经典逻辑下不成立（二值之外无 U 位）；
项目特别引入 U 以承担**未决之态**——
对应表三之 modal 互含、表二之时态「今」、表四之认识「疑」三轴位。

### 三值真值表之五联结词

| P | Q | ¬P | P ∧ Q | P ∨ Q | P → Q | P ↔ Q |
|---|---|----|-------|-------|-------|-------|
| ⊤ | ⊤ | ⊥ | ⊤ | ⊤ | ⊤ | ⊤ |
| ⊤ | U | ⊥ | U | ⊤ | U | U |
| ⊤ | ⊥ | ⊥ | ⊥ | ⊤ | ⊥ | ⊥ |
| U | ⊤ | U | U | ⊤ | ⊤ | U |
| U | U | U | U | U | U | U |
| U | ⊥ | U | ⊥ | U | U | U |
| ⊥ | ⊤ | ⊤ | ⊥ | ⊤ | ⊤ | ⊥ |
| ⊥ | U | ⊤ | ⊥ | U | ⊤ | U |
| ⊥ | ⊥ | ⊤ | ⊥ | ⊥ | ⊤ | ⊤ |

此表即 **Kleene 强三值逻辑（K3）**。

**注**：K3 ≠ Łukasiewicz Ł3。
二者于多数联结词同，但 **`U → U` 取值不同**：
- K3 之蕴含按 $P \to Q := \neg P \vee Q$ 定义，故 $\text{U} \to \text{U} = \neg\text{U} \vee \text{U} = \text{U} \vee \text{U} = \text{U}$
- Ł3 之蕴含 $\text{U} \to \text{U} = \top$（自反性）

项目主用 **K3**（与三值保守律一致）：未决之蕴含未决，不可冒称为真。

**关键观察**：U 在 K3 中是 **infectious（传染元）**——
若一操作数为 U 而**另一操作数不能单独决定结果**，则结果为 U。
但若另一操作数足以决定（如 $\bot \wedge \text{U} = \bot$、$\top \vee \text{U} = \top$），则结果为该确定值。
此非"吸收元"——吸收元会让所有运算返回固定值。

### 三值与中态字之对应

| U 之子位 | 字 | 表 |
|---------|---|---|
| 本体侧 | 玄 | 表三 |
| 认识侧 | 妙 / 疑 | 表三、表四 |
| 动力侧 | 几 | 表三 |
| 时序侧 | 今 / 待 | 表二 |

U 在不同维度承担不同字——**三轴中态共显于「U」之逻辑位**，
但语义独立（参表三末段）。

## 四、命题 · 元逻辑之单元

### 形式

$$\text{Prop} \cong \text{Bool} \quad\text{（二值）}\quad \cong \text{Trit} \quad\text{（三值）}$$

命题 = 真值之承担者。最简：单爻 = 单命题。

### 字元封装：者 / 也

由 wenyan-operators §S-13–S-14：

| 字 | 类型 | 释 |
|---|------|---|
| 者 | Term/Prop → DefinitionBinder | 「X 者，P 也」≅ `X := P` 或 `X(x) ⇔ P(x)` |
| 也 | Prop → Assertion | 断定 / 命题完成 |

**「X 者，P 也」是文言文之命题断言式**——比西方「A is B」更明确「定义」与「断定」之分。

形式刻画：

```
者 : (Term, Prop) → Definition
也 : Prop → Asserted-Prop
```

### 谓词与命题之分

由 wenyan-operators §S-16：

| 字 | 类型 | 释 |
|---|------|---|
| 所 | Predicate / Action → Nominal | 「所 P」≅ `{x | P(x)}` |

**「所」是名物化算子**——把谓词降为名词集合。
形式：`所 P = {x | P(x)}`。

至此项目已含三大语法层算子：

| 层 | 字 | 类型 |
|---|---|------|
| 抽象/定义 | 者 | binding |
| 断定 | 也 | assertion |
| 名物化 | 所 | comprehension |

## 五、五个核心联结词（逻辑核心）

由 wenyan-operators §24（line 2165）项目已列**最小完备子集**：

> ### 逻辑核心 (5 个)
> - 不 (¬)
> - 並 (∧)
> - 或 (∨)
> - 凡 (∀)
> - 故 (⊢)

但项目同节注：「→ = ¬∨」可由前者推出。
本文统一为**五联结词**（值层）+ **二量词**（变元层）+ **一推理**（系层）：

### 5.1 不（¬）

| 项 | 内容 |
|---|------|
| 单字 | **不** |
| 别字 | 弗、莫（强否）|
| 类型 | $\text{Prop} \to \text{Prop}$ |
| 出处 | 通用；老子「不言之教」 |

性质：$\neg \neg P = P$（古典层）；三值层 $\neg \neg P \neq P$（U 守身）。

### 5.2 並 / 且（∧）

| 项 | 内容 |
|---|------|
| 单字 | **並** / **且** |
| 别字 | 与、共、兼 |
| 类型 | $\text{Prop} \times \text{Prop} \to \text{Prop}$ |
| 出处 | 墨经 P-3「兼」；通用「且」 |

注：墨经之**兼**严格指「整合诸部分为整」，与命题合取虽精神同，使用层不同。

### 5.3 或（∨）

| 项 | 内容 |
|---|------|
| 单字 | **或** |
| 别字 | 抑、若 |
| 类型 | $\text{Prop} \times \text{Prop} \to \text{Prop}$ |
| 出处 | 通用；论语「或谓孔子曰」 |

「或」在文言文中既作 ∨（择一），又作 ∃（某）——
**选择性消歧**靠语境，本文专用前者；后者归 §6.2「有」字。

### 5.4 若…则（→）

| 项 | 内容 |
|---|------|
| 单字 | **若** + **则** |
| 别字 | 苟、使（前件）；乃（后件）|
| 类型 | $\text{Prop} \times \text{Prop} \to \text{Prop}$ |
| 出处 | 大学「物格而后知至」；通用 |

由 wenyan-operators §S-3、§S-4 已立。
等价定义：$P \to Q := \neg P \vee Q$（古典）。

### 5.5 等同（↔）

| 项 | 内容 |
|---|------|
| 单字 | **则** + **则** 互（双向）；古无单字，常用「即」 |
| 别字 | 即、犹 |
| 类型 | $\text{Prop} \times \text{Prop} \to \text{Prop}$ |
| 出处 | 「即」自 wenyan §I-3 |

等价定义：$P \leftrightarrow Q := (P \to Q) \wedge (Q \to P)$。

### 联结词总表

| 算子 | 单字 | 类型 | 二值真值表 |
|------|-----|------|----------|
| 不 | 不 | Prop → Prop | T→F, F→T |
| 並 | 並/且 | Prop² → Prop | TT→T, 余→F |
| 或 | 或 | Prop² → Prop | FF→F, 余→T |
| 蕴 | 若…则 | Prop² → Prop | TF→F, 余→T |

## 六、联结词与 G 之同构 · 八卦即真值表

### Bool 之代数 = (Z/2)

二值 Bool = (Z/2) 群带乘法：

| 算子 | 形式 | 单字 |
|------|------|-----|
| ¬ | $1 - x$（即 1 ⊕ x）| 不 |
| ∧ | $x \cdot y$ | 重 / 並 |
| ⊕ (XOR) | $x + y \pmod 2$ | 合（在 Bool 上）/ 异 |
| ∨ | $x + y - xy$ | 或 |

**关键**：

$$\text{Bool 上之 ∧ = 算术之 ·} \qquad \text{Bool 上之 ⊕ = 算术之 +}$$

故由 `数与算术 · 从元到数.md` 之**重 / 合**降至 (Z/2) 即得**布尔代数**。
**逻辑联结词不是另立体系，而是数算之有限化**。

### 八卦 = 三命题之真值表

设三个命题变量 $P, Q, R \in \text{Bool}$，对应三爻：

$$P \leftrightarrow y_1 \text{（初爻 / 下）}, \quad Q \leftrightarrow y_2 \text{（中爻）}, \quad R \leftrightarrow y_3 \text{（上爻 / 顶）}$$

其真值赋值表共 $2^3 = 8$ 行——**正是八卦**（按本目录统一约定 $\langle y_1, y_2, y_3 \rangle = \langle 下, 中, 上 \rangle$）：

| 卦 | 编码 $(P, Q, R) = \langle y_1, y_2, y_3 \rangle$ | 解读 |
|----|---------------------------|------|
| 乾 ☰ | $\langle 1, 1, 1 \rangle$ | 三命题全真 |
| 兑 ☱ | $\langle 1, 1, 0 \rangle$ | R（上爻）假，P∧Q 真 |
| 离 ☲ | $\langle 1, 0, 1 \rangle$ | Q（中）假，P∧R 真 |
| 震 ☳ | $\langle 1, 0, 0 \rangle$ | 仅 P（下）真 |
| 巽 ☴ | $\langle 0, 1, 1 \rangle$ | P 假，Q∧R 真 |
| 坎 ☵ | $\langle 0, 1, 0 \rangle$ | 仅 Q 真 |
| 艮 ☶ | $\langle 0, 0, 1 \rangle$ | 仅 R 真 |
| 坤 ☷ | $\langle 0, 0, 0 \rangle$ | 三命题全假 |

**八卦之真值表性质**：

| 联结词 | 八卦操作 |
|--------|---------|
| ¬P（否定下爻） | 翻 $y_1$ = G 之「**动**」 |
| ¬Q（否定中爻） | 翻 $y_2$ = G 之「**化**」 |
| ¬R（否定上爻） | 翻 $y_3$ = G 之「**变**」 |
| ¬P ∧ ¬Q ∧ ¬R | G 之「**错**」（三反复合） |
| 「P ∧ Q ∧ R 真」之断言 | 仅在「乾 ⟨1,1,1⟩」处 |
| 「P ∨ Q ∨ R 假」之断言 | 仅在「坤 ⟨0,0,0⟩」处 |

**精确说明**：
- G 之 (Z/2)³ 作为**群**——只承担**否定**（爻反 = XOR by basis vector）。这部分群作用同构于命题逻辑之**否定子代数**。
- 完整命题逻辑（含 ∧, ∨, ¬）作为**布尔代数 (Bool³, ∧, ∨, ¬)**——比群结构丰富：群只有一个二元操作（+），布尔代数有两个（∧, ∨），加上一元 ¬。
- 故：**G 之群 ⊊ Bool³ 之联结词代数**——G 给的是后者之否定子部分。

故 G 体系之 (Z/2)³ 群作用 = **三命题真值表之否定算子族**（不含 ∧∨）；
完整三变元布尔代数须 G 之群 + 额外的 ∧∨ 算子。

### Boole（1854）与中国阴阳之同构

George Boole 之《Laws of Thought》（1854）建立 Boolean algebra，
莱布尼茨（1701）已注意到伏羲先天图之二进编码。
**项目立场**：

> 阴阳之 Bool 代数，先伏羲于先秦，先 Leibniz 于 17c，先 Boole 于 19c——
> 但**形式同构**直至 Boole 才完成代数公理化。
> 易经之**爻变**即 Boole 之**变量取反**，
> 易经之**重卦**即 Boole 之**联结词作用于多变量**。

### 64 卦 = 6 变元真值表 = 双 3-命题系统之关系

64 = 2⁶。一卦由六爻 = 六真值变元构成：

$$\text{64 卦} \cong \{0, 1\}^6 \cong \{$$ 6 变元真值表所有行 $$\}$$

**重卦 = 内卦（下三爻）+ 外卦（上三爻）**——
即「内三命题系统」与「外三命题系统」之**关系断言**。

例：「乾上乾下」= 内外皆真，即 $\bigwedge_{i=1..6} P_i$。

故 64 卦 = 二阶 3-命题系统之**关系全枚举**——
此即古人「八卦相重以为 64」之**形式逻辑解读**。

## 七、量词

### 7.1 全（∀）

| 项 | 内容 |
|---|------|
| 单字 | **凡** / **全** |
| 别字 | 皆、悉、咸 |
| 类型 | $(A \to \text{Prop}) \to \text{Prop}$ |
| 出处 | 通用；论语「凡四海之内皆兄弟也」|

形式：

$$\forall x \in A.\, P(x) := \bigwedge_{x \in A} P(x)$$

在有限论域下，∀ = 多重 ∧。
项目之 64 卦上之全称：「凡卦皆有一阴一阳」即 $\forall h \in \text{Hex}.\, \exists \text{阴爻} \wedge \exists \text{阳爻}$（除乾坤外）。

### 7.2 有（∃）

| 项 | 内容 |
|---|------|
| 单字 | **有** / **某** |
| 别字 | 或、莫不（双重否）|
| 类型 | $(A \to \text{Prop}) \to \text{Prop}$ |
| 出处 | 通用 |

形式：

$$\exists x \in A.\, P(x) := \bigvee_{x \in A} P(x)$$

等价：$\exists x.\, P(x) := \neg \forall x.\, \neg P(x)$。

### 7.3 无 / 莫（¬∃）

| 项 | 内容 |
|---|------|
| 单字 | **莫** / **无** |
| 类型 | $(A \to \text{Prop}) \to \text{Prop}$ |
| 出处 | 论语「莫之御也」|

形式：$\neg \exists x.\, P(x) = \forall x.\, \neg P(x)$。
**「莫」是 ¬∃ 之单字承担**——比西方「nobody」更精确。

### 量词与三值

在三值层：

| 经典 | 三值 |
|------|-----|
| ∀x. P(x) | $\bigwedge P(x)$（含 U → U 主导）|
| ∃x. P(x) | $\bigvee P(x)$（含 U → 视他元）|

三值量词依 Kleene 表逐元归并。

## 八、推理 · 故 之精细化（墨经版）

### K-1 之故（标准推论）

| 项 | 内容 |
|---|------|
| 单字 | **故** |
| 类型 | $\text{Prop list} \to \text{Prop}$，即 $P_1, \ldots, P_n \vdash Q$ |
| 出处 | 通用 |

**Modus Ponens**：

$$\text{若 } P \text{ 则 } Q;\;\; P;\;\; \text{故 } Q.$$

### P-1 墨经之故（必充之分）

由 wenyan-operators §P-1（line 855）：

> 「故，所得而后成也」  
> 「小故，有之不必然，无之必不然；大故，有之必然」

**墨家明确区分必要与充分条件**——这是中国本土形式逻辑核心。

| 故之类 | 形式 | 释 |
|--------|-----|---|
| **小故**（必要）| $\neg P \Rightarrow \neg Q$ | P 是 Q 之必要条件 |
| **大故**（充分）| $P \Rightarrow Q$ | P 是 Q 之充分条件 |
| **二者**（充分必要）| $P \Leftrightarrow Q$ | iff |

**墨家比西方早约两千年区分必要/充分**。
本文形式逻辑层之推理符按墨经四类区分：

| 单字 | 类 | 形式 |
|------|----|------|
| 故 | 通用推论 | ⊢ |
| 必（须）| 必要 | $\neg p \Rightarrow \neg q$ |
| 致 | 充分 | $p \Rightarrow q$ |
| 即 | 等价 | $p \Leftrightarrow q$ |

### 推理规则总集（自然演绎，含上下文 Γ）

记 $\Gamma$ 为上下文（已知命题集），$\vdash$ 为可推关系。

| 名 | 规则 | 字元 |
|---|------|-----|
| 假设 | $\Gamma, P \vdash P$ | 设 |
| 引入 → | $\Gamma, P \vdash Q \;\Rightarrow\; \Gamma \vdash P \to Q$ | 设…则 |
| 消去 → | $\Gamma \vdash P \to Q,\;\; \Gamma \vdash P \;\Rightarrow\; \Gamma \vdash Q$ | 故 |
| 引入 ∧ | $\Gamma \vdash P,\;\; \Gamma \vdash Q \;\Rightarrow\; \Gamma \vdash P \wedge Q$ | 兼 |
| 消去 ∧ | $\Gamma \vdash P \wedge Q \;\Rightarrow\; \Gamma \vdash P$（同理 $\Gamma \vdash Q$）| 取 |
| 引入 ∨ | $\Gamma \vdash P \;\Rightarrow\; \Gamma \vdash P \vee Q$ | 或 |
| 消去 ∨ | $\Gamma \vdash P \vee Q,\;\; \Gamma, P \vdash R,\;\; \Gamma, Q \vdash R \;\Rightarrow\; \Gamma \vdash R$ | 析 |
| 引入 ¬ | $\Gamma, P \vdash \bot \;\Rightarrow\; \Gamma \vdash \neg P$ | 反证 |
| 消去 ¬ (NK)  | $\Gamma \vdash \neg \neg P \;\Rightarrow\; \Gamma \vdash P$（仅古典）| 双反 |
| 引入 ∀ | $\Gamma \vdash P(x),\; x \notin \mathrm{FV}(\Gamma) \;\Rightarrow\; \Gamma \vdash \forall x.\, P(x)$ | 通 |
| 消去 ∀ | $\Gamma \vdash \forall x.\, P(x) \;\Rightarrow\; \Gamma \vdash P(t)$ | 取 |
| 引入 ∃ | $\Gamma \vdash P(t) \;\Rightarrow\; \Gamma \vdash \exists x.\, P(x)$ | 举 |
| 消去 ∃ | $\Gamma \vdash \exists x.\, P(x),\;\; \Gamma, P(y) \vdash R,\; y \notin \mathrm{FV}(\Gamma, R) \;\Rightarrow\; \Gamma \vdash R$ | 设 |

## 十一、古典 vs 直觉 · 项目之直觉倾向

### 排中律 LEM

## 十七、证明骨架

本节给关键命题之验证。前两证是 K3 真值表之派生与 K3 ≠ Ł3 之分歧；中段证 LEM/DNE 在 K3 失效；末段触 Curry–Howard 之核心同构。

### 17.1 K3 真值表之派生（§3 之表来由）

K3 取**强 Kleene 三值**：以 $\bot < \mathsf{U} < \top$ 序，定义
$$\neg(\bot) = \top, \quad \neg(\mathsf{U}) = \mathsf{U}, \quad \neg(\top) = \bot$$
$$P \wedge Q := \min(P, Q), \qquad P \vee Q := \max(P, Q), \qquad P \to Q := \neg P \vee Q$$

由此可派生 §3 全表。验证关键行：
- $\mathsf{U} \to \mathsf{U} = \neg \mathsf{U} \vee \mathsf{U} = \mathsf{U} \vee \mathsf{U} = \max(\mathsf{U}, \mathsf{U}) = \mathsf{U}$ ✓（与表同）
- $\bot \to \mathsf{U} = \top \vee \mathsf{U} = \max(\top, \mathsf{U}) = \top$ ✓
- $\top \wedge \mathsf{U} = \min(\top, \mathsf{U}) = \mathsf{U}$ ✓（"infectious"，非吸收）
- $\bot \wedge \mathsf{U} = \min(\bot, \mathsf{U}) = \bot$ ✓（U 不主导，因 ⊥ 已最小）

故 §3 表中之 9 行 × 5 列 = 45 单元皆由上述定义机械派生。 ∎

### 17.2 K3 ≠ Ł3 之关键分歧

Łukasiewicz Ł3 定义 $\to$ 不同：
$$P \to_{\text{Ł}} Q = \begin{cases} \top & P \leq Q \\ 1 - (P - Q) & P > Q \end{cases}$$

（其中 $\bot = 0, \mathsf{U} = 1/2, \top = 1$。）

**反例**：$\mathsf{U} \to_{\text{Ł}} \mathsf{U} = \top$（因 $\mathsf{U} \leq \mathsf{U}$）；K3 给 U。
故 K3 ≠ Ł3。

**项目选择 K3** 之理由：与三值保守律（$\mathsf{U} \nRightarrow \top$）一致——
若取 Ł3，$\mathsf{U} \to_{\text{Ł}} \mathsf{U} = \top$ 即"未决可推未决"，与保守律精神不符。 ∎

### 17.3 LEM 在 K3 失效之反例

**断言**：K3 中 $P \vee \neg P$ 不恒为 ⊤。

**证**：取 $P = \mathsf{U}$。则
$$P \vee \neg P = \mathsf{U} \vee \neg \mathsf{U} = \mathsf{U} \vee \mathsf{U} = \mathsf{U} \neq \top$$
故 LEM 在 K3 失效。 ∎

### 17.4 DNE 在 K3 失效之反例

**断言**：K3 中 $\neg \neg P \to P$ 不恒为 ⊤。

**证**：取 $P = \mathsf{U}$。则
- $\neg P = \mathsf{U}$，$\neg \neg P = \mathsf{U}$
- $\neg \neg P \to P = \mathsf{U} \to \mathsf{U} = \mathsf{U} \neq \top$（按 17.1 派生）

故 DNE 失效。 ∎

### 17.5 八卦 ≅ Bool³ ≅ 三变量真值表之三重同构

设：
- $\mathcal{T}_8$ := 八卦集（八宫卦象之集）
- $\mathbb{B}^3$ := $\{0,1\}^3$
- $\mathcal{F}_3$ := 三命题变元 $\{P, Q, R\}$ 之真值赋值集

**断言**：$\mathcal{T}_8 \cong \mathbb{B}^3 \cong \mathcal{F}_3$。

**证**：
- $\mathcal{T}_8 \cong \mathbb{B}^3$：由 G §2.1 之编码（每爻 ∈ {0,1}），三爻有序对 ↔ Bool³ 元，是双射。
- $\mathbb{B}^3 \cong \mathcal{F}_3$：每 (b₁,b₂,b₃) ↔ 赋值 P↦b₁, Q↦b₂, R↦b₃，是双射。
- 两双射皆保 cardinality（皆 8）；进一步保运算（爻反 ↔ 命题否定；笛卡尔积 ↔ 联合赋值）。 ∎

**推论**：64 卦 ≅ Bool⁶ ≅ 六变元真值表（同理 18 行）。 ∎

### 17.6 Curry–Howard 之核心样例

| 命题 | 类型 | 证明项 / 程序 |
|------|-----|-------------|
| $\top$ | Unit | $* : $ Unit |
| $P \wedge Q$ | $P \times Q$ | $(p, q) : P \times Q$ |
| $P \vee Q$ | $P + Q$ | $\text{inl}(p)$ 或 $\text{inr}(q)$ |
| $P \to Q$ | $P \to Q$（函数）| $\lambda p.\, q : P \to Q$ |
| $\neg P$ | $P \to \bot$ | $\lambda p.\, \text{absurd} : P \to \bot$ |
| $\forall x. P(x)$ | $\Pi x. P(x)$ | $\lambda x.\, p(x)$ |
| $\exists x. P(x)$ | $\Sigma x. P(x)$ | $(t, p(t)) : \Sigma x. P(x)$ |

**断言**：「X 者，P(X) 也」≅ $\lambda X.\, P(X)$。

**证**：文言之**「者…也」**结构是**带名定义**：
- 「X 者」绑定变元 X（abstraction site）
- 「P(X) 也」陈述 X 之性质（body）

正合 lambda 之 binder–body 二元结构。
将「者」译为 $\lambda$，「也」译为 body 之 evaluation context（Curry-Howard 中 ".." closure）即得同构。 ∎

### 17.7 墨经四同作为等价关系之精细化

设 $\sim_{\text{重}}, \sim_{\text{体}}, \sim_{\text{合}}, \sim_{\text{类}}$ 为四种 sameness 关系。

**断言**：四关系之 conjunction $\sim_{\text{全}} := \sim_{\text{重}} \cap \sim_{\text{体}} \cap \sim_{\text{合}} \cap \sim_{\text{类}}$ 是 strictest equivalence；任一 $\sim_i$ 单独亦为 equivalence。

**证**：每 $\sim_i$ 须验证自反、对称、传递：
- **重同**（identity）：trivially equivalence。
- **体同**（mereological）：自反（$x \subseteq x$ 之 part-of）；对称（须改用同体之关系，而非真子部分）；传递（同属同一 whole）。
- **合同**（co-location）：自反、对称（共处一处 ↔ 共处一处）、传递（须 transitive context；近似 spatial overlap，传递性可能失败——故合同非严格 equivalence，**应作 tolerance relation**）。
- **类同**（type-equivalence）：自反、对称、传递皆显然。

**注**：「合同」严格说**非 equivalence**（传递性可能失败：A 与 B 共处、B 与 C 共处，A 与 C 不必共处）。墨家未严格分等价/容错关系，本文从之。**严格化时应将合同标为 tolerance relation 而非 equivalence**。 ∎
