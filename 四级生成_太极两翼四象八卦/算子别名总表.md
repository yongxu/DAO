# 算子别名总表 · v0

> **用途**：解决「一算子，多符号」问题——同一算子在易家、道家、儒家、释家、现代汉语、数学符号下各有不同落字，本表统一登记。
>
> **锚点**：每个算子有唯一 ASCII canonical（与 Lean 标识一致），中文落字皆视为锚点之**别名**。
>
> **法则依据**：道理 v12 法则 51——*truth invariant, vocabulary cultural*（真理不变，字汇随文化）。多名非冗，是结构在多语境之**显**。
>
> **差等记号**（"差"列）：
> - **=** 等同：可互换，签名同
> - **≈** 几近：核心同，语气/侧重微异
> - **⊏** 弱化：含义子集（覆盖更窄）
> - **⊐** 强化：含义超集
> - **⊥** 异轴：表面似而轴向不同（不可换）

## 一、命名优先级（落字时的取舍规则）

当一算子在四家皆有字时，按下表优先：

| 算子层 | 首选家 | 理由 |
|---|---|---|
| 结构层（变换、组合、投影） | **易家** | 易经-系辞已给出最精确之结构语义 |
| 自指层（不动点、归一、不可道） | **道家** | 老子-庄子之自指传统最深 |
| 规范层（正、序、和） | **儒家** | 礼-乐之规范化语言最完整 |
| 缘起层（空、依他、相互依赖） | **释家** | 中观-华严之关系本体最精 |
| 量化/构造层 | **数学** | 现代符号最无歧义 |

> **不冲突的理由**：四家本来分工不同。易讲结构、道讲自指、儒讲规范、释讲缘起——若都在描述同一算子而落字不同，多半是侧重不同（差为 ≈ 而非 =）。

## 四、Lean 形式层映射

所有 canonical（ASCII）即 Lean 标识符。中文落字仅出现在文档与注释。

```lean
-- 在 formal/SSBX/Foundation/Bagua/BaguaAlgebra.lean
def dong   : Trigram → Trigram := ...   -- 动 / 反_下
def hua    : Trigram → Trigram := ...   -- 化 / 反_中
def bian   : Trigram → Trigram := ...   -- 变 / 反_上
def heShang : Trigram → SiXiang := ...  -- 合_上 / 省 / 损（道）
def fenToTrigram : SiXiang → Yao → Trigram := ...  -- 分 / 生
def chong : Trigram → Trigram → Hexagram := ...    -- 重
def guiyi : Trigram → Unit := ...                  -- 归一 / 抱 / 一以贯之
inductive Sheng : Nat → Type ...                   -- 生生 / Y
```

对应 `BaguaAlgebra.lean` `bagua_algebra_complete` 主定理（5 条断言，零 sorry）。

## 五、跨家"同字异义"警示表

下列字在多家皆出现但**不可视为别名**，须按上下文消歧：

| 字 | 易家义 | 道家义 | 儒家义 | 释家义 | 处理 |
|---|---|---|---|---|---|
| **化** | 反爻（中爻翻转） | 本体流转 | 风化、教化 | 转化（业之转） | 项目锚定**易家义**为 canonical `hua`；他义另字 |
| **反** | 爻反（单爻） | 反者道之动（普反） | 自反求己 | — | canonical `fan` = 道家义；易家"爻反"用 `bian/hua/dong` |
| **生** | 太极生两仪（升维） | 道生万物（本体生） | — | 生灭（缘起） | canonical `fen` 承担"升维"；本体生留作哲学层 |
| **一** | — | 抱一（归一） | 一以贯之（贯通） | 一真法界（华严） | canonical `guiyi` 锚定"归一"算子；他义为派生 |
| **空** | — | 虚（≈无） | — | 空性（中观核心） | 不入算子集；属本体论概念，非操作 |

**规则**：当一字在多家有不同算子义时，**项目锚 canonical 于该算子最精确的家**——其他家的同字归为"别义"，需另外 canonical 字承担。

## 六、添加新算子的工作流

发现新算子（如读到一段经典中算子用法）时：

1. **判定层**：结构 / 自指 / 规范 / 缘起 / 量化？→ 决定首选家
2. **查现表**：是否已有 canonical 锚（同义合并 / 几近补 alias / 异轴另立）？
3. **取首选家本字**作正字；他家有则补别名列
4. **判差等**：=, ≈, ⊏, ⊐, ⊥
5. **写签名**：必须能落到类型论 / 范畴 / 群论之一
6. **登 Lean**（如属理层）：在 `BaguaAlgebra.lean` 或新文件加 def + 至少一条性质定理
