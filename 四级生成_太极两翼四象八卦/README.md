## 全局约定

- **爻位**：$\langle y_1, y_2, y_3 \rangle = \langle 下, 中, 上 \rangle$，$y_1$ 为初爻（最下）
- **算子**：动 / 化 / 变 = 反 $y_1$ / $y_2$ / $y_3$（即「动在下、化在中、变在上」）
- **错** = 阴阳全反（在 (Z/2)ⁿ 内）；**综** = 位置反序（不在 flip 群内）
- **三值保守律**：U ⇏ ⊤（未决态不可任意上升至真）；详见 v14 骨架与形式逻辑衍 §3 / 统计衍 §13 / K 之 §四
