## 卷一 · 字元义位表

本表为正式系统之根。下列每一字，默认指 `字1`；凡同字多义已编号者，以卷〇义位表为准。不等于自然语言中该字之一切义。

### 一、显生类

| 字 | 本论定用 | 可生成 |
|---|---|---|
| 显 | 有所呈现，可被感得或指认 | 显现 |
| 未 | 非已定、非已尽 | 未尽、未定 |
| 尽 | 被已然结构穷尽，无余可生 | 尽、绝尽 |
| 间 | 显而未尽之可生隙 | 间、间域 |
| 可 | 有实入之可能 | 可达、可生、可识 |
| 生 | 入场而成新场 | 生、再生 |
| 续 | 前后相承，且仍有可生之路 | 续、己续、共续 |
| 开 | 未尽、有可、能应、能转之态 | 开、护开 |
| 闭 | 可生之隙被断、冻、夺、蔽、伪 | 闭、成闭 |
| 绝 | 路断、域空、无可入 | 绝闭 |
| 断 | 连续被截，路不可承 | 断续、断开 |
| 达 | 有路可至，且至后可验 | 可达、实达 |
| 候 | 待入而未定，可转为间 | 候间 |
| 新 | 未入史法而可受限试 | 新、暂法 |

### 二、场物类

| 字 | 本论定用 | 可生成 |
|---|---|---|
| 场 | 诸间共在、共限、共变之整体 | 场态 |
| 焦 | 场中暂可聚指者 | 聚焦、稳焦 |
| 物 | 稳焦之成形者 | 成物 |
| 境 | 场之经验相 | 境相 |
| 心 | 能感、识、择、校之焦 | 心、感心 |
| 身 | 焦之承受与行动条件 | 身限、身势 |
| 态 | 某时某度下之成形状态 | 场态、状态 |
| 状 | 可描述、可指认之形貌 | 状、形状 |
| 维 | 场可展开之新度向 | 开维、维度 |

### 三、形迹类

| 字 | 本论定用 | 可生成 |
|---|---|---|
| 形 | 可识别之模式 | 形据 |
| 相 | 可显之貌，未必有其实 | 相、似相 |
| 因 | 可考之生成前因 | 因据 |
| 结 | 行后所成之后果与约束 | 结据 |
| 据 | 可查之证与迹 | 证据 |
| 证 | 可支持某判之据 | 证、互证 |
| 迹 | 过程所留之后效痕 | 迹查 |
| 史 | 迹在时中之积 | 史法 |
| 积 | 行与感沉淀为后续条件 | 积、性 |
| 精 | 形中微密而可凝之要 | 精义、精微 |
| 气 | 场中权、势、流、扰、耦之动相 | 气、气势 |
| 神 | 精气通显而能应之整体活相 | 神、精神 |

> **精为微密之要，气为流动之势，神为精气通显而能应。**

### 十四、辅助字元校正

下列字元不改变主链，只补足 v13.1 中已入式而未显列之项：

$$
\mathrm{达},\mathrm{候},\mathrm{新},\mathrm{维},\mathrm{自},\mathrm{荣},\mathrm{感},\mathrm{择},\mathrm{情},\mathrm{礼},\mathrm{信},\mathrm{性},\mathrm{待},\mathrm{能},\mathrm{归},\mathrm{轨},\mathrm{息},\mathrm{迫},\mathrm{替},\mathrm{基},\mathrm{线},\mathrm{域},\mathrm{试},\mathrm{定},\mathrm{再},\mathrm{关},\mathrm{格}
$$

凡由此生成之复词，仍须入总籍，不得因补字而偷义。
