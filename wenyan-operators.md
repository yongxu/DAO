# 文言文算子集 — 结构与变形的字

*目的*：从文言文中识别出**本身即作算子用的字** — 不是描述对象的名词，而是作用于结构、产生新结构的操作。这些字在经典中早已构成一套有内在算法结构的语言；任务是把它分类、签名化、暴露其代数。

每条记录格式：
- **字** — 简注
- **签名** — 类型 (域 → 陪域)
- **源** — 经典出处
- **读** — 算子如何作用
- **注** — 形式观察

# 第一部 — 算子目录 (Catalogue)

## 一、关系算子 — 结构内之静态关系

### R-1: 在
- **签名**: A × Loc → Prop
- **源**: 易·系辞「在天成象，在地成形」
- **读**: 定位算子 — 元素在某结构中的位置
- **注**: 等价于 ∈ 或 located-at；最基础的关系算子

### R-2: 屬 / 属
- **签名**: A × Class → Prop
- **源**: 礼记「凡冠者，屬其次」；荀子「屬之以礼义」
- **读**: 类属关系 — 元素属于某范畴
- **注**: ∈ ; 比 在 更抽象，强调类成员

### R-3: 際 / 际
- **签名**: A × A → Boundary
- **源**: 易·泰「无平不陂，无往不复」释为「天地际也」
- **读**: 接界算子 — 两结构相接处
- **注**: 这是 间 的近邻，但更强调"接缝"

### R-4: 間 / 间
- **签名**: A × A → Region
- **源**: 庄子「彼节者有间」
- **读**: 间隔算子 — 两元素之间的空间/区域
- **注**: 不只是减法，是**容纳第三者**的开放区域。Jian 框架的核心字

### R-5: 中
- **签名**: Region → Point
- **源**: 中庸「不偏之谓中」；易「黄裳元吉，文在中也」
- **读**: 取中算子 — 取结构的中位/重心
- **注**: 既是位置（noun），也是动作（取中 = center）。中庸本身是一个把"中"作算子使用的传统

### R-6: 正
- **签名**: A → A
- **源**: 易·彖「位正中也」；论语「政者，正也」
- **读**: 端正算子 — 把结构调到正确位置
- **注**: 与 失位 对立。在易卦中，正 = 阳爻在奇位、阴爻在偶位

### R-7: 應 / 应
- **签名**: A × A → Prop
- **源**: 易·系辞「同声相应，同气相求」；卦中初↔四、二↔五、三↔上
- **读**: 共振算子 — 两位置的对称对应
- **注**: 是 易 给出的非局部关系算子 — 跨距离的结构关联

### R-8: 比
- **签名**: A × A → Prop
- **源**: 易·彖「比，辅也，下顺从也」；爻位中相邻者
- **读**: 邻接算子 — 两位置相邻
- **注**: 与 應 互补：比 是局部，應 是远程

### R-9: 承
- **签名**: A × A → Prop
- **源**: 易·象「君子以承天而时行」
- **读**: 承载算子 — 下位支持上位
- **注**: 不对称的二元关系；A 承 B = A 在下、B 在上、B 由 A 支撑

### R-10: 乘
- **签名**: A × A → Prop
- **源**: 易·彖「柔乘刚也」
- **读**: 凌乘算子 — 上位压在下位
- **注**: 与 承 反向；常用作不当之配置

### R-11: 對 / 对
- **签名**: A → A
- **源**: 易·杂卦传以"对"组配 64 卦；老子「贵以贱为本」
- **读**: 配对算子 — 给一个结构其对偶
- **注**: 对偶的基本构造；64 卦中每一卦都有 對 卦

### R-12: 偶 / 耦
- **签名**: A × B → A × B
- **源**: 墨经「偶，二也」
- **读**: 配偶 — 二者并立成对
- **注**: 比 對 更弱，只是双重无序

### R-13: 並 / 并
- **签名**: A × A → A
- **源**: 论语「并行不相悖」
- **读**: 并列 — 同位置同时存在
- **注**: 相当于 ∧ 或 monoidal product

### R-14: 與 / 与
- **签名**: A × A → A 同上
- **源**: 论语「与点也」
- **读**: 协同 — 与并并立
- **注**: 比 並 更弱，常带"并且参与"语气

### R-15: 偕
- **签名**: A × A → A
- **源**: 易·豫「介于石，不终日，贞吉」象「不终日贞吉，以中正也」；诗「执子之手，与子偕老」
- **读**: 同行 — 两者一同变化
- **注**: 是带时间的 与 — 不是静态的并列，是同步进化

## 二、含包算子 — 空间/嵌套关系

### C-1: 含
- **签名**: A × A → Prop
- **源**: 易·坤「含弘光大」
- **读**: 包含 — 一结构包另一结构
- **注**: 等价于 ⊇

### C-2: 包
- **签名**: A × A → A
- **源**: 易·泰「包荒，用冯河」
- **读**: 包裹 — 把一物完全包入
- **注**: 空间动作；与 含 的区别在于 包 强调动作

### C-3: 容
- **签名**: A × A → Prop / A × A → A
- **源**: 易·益「圣人感人心而天下和平」释「君子以见善则迁，有过则改」；尚书「有容德乃大」
- **读**: 容纳 — 给空间盛纳
- **注**: 关键：容 强调**预留空间**；可容是结构的属性

### C-4: 中 (用作 inside)
- **签名**: A → Region
- **源**: 论语「居简而行简，无乃太简乎」释「夫子之道，忠恕而已矣」
- **读**: 内部算子 — 结构的内部
- **注**: 与 R-5 的"取中"不同 — 此处 中 = interior 集合

### C-5: 外
- **签名**: A → Region
- **源**: 大学「内圣外王」
- **读**: 外部算子 — 结构的外部
- **注**: 与 C-4 互补；构成 内/外 二元划分

### C-6: 內 / 内
- **签名**: A → Region (强)
- **源**: 同上；内圣
- **读**: 内 — 比 中 更明确指内/外二分
- **注**: 中 偏重几何中心；内 偏重内部域

### C-7: 表
- **签名**: A → Surface
- **源**: 礼记「表里如一」
- **读**: 表面 — 取结构的边界面
- **注**: 是 边界 算子的一种；与 里 配对

### C-8: 裡 / 里
- **签名**: A → Interior
- **源**: 同上
- **读**: 里面 — 取结构内部
- **注**: 与 表 配对，构成 surface/interior 区分

## 三、变换算子 — 动态结构变化

### T-1: 化
- **签名**: A → A' (常含类型变化)
- **源**: 易·系辞「精气为物，游魂为变，是故知鬼神之情状」；庄子「化而为鸟」；中庸「能尽其性，则能尽人之性…可以赞天地之化育」
- **读**: 渐变 — 连续的、可能改变类型的变化
- **注**: 文言文中**最强的变换算子**。区别于 變：化 是 continuous，变 是 discrete

### T-2: 變 / 变
- **签名**: A → A' (离散)
- **源**: 易·系辞「变化者，进退之象也」
- **读**: 突变 — 离散的状态切换
- **注**: 在易中：阳爻↔阴爻的离散翻转。化是渐变积累，变是质变跳跃

### T-3: 易
- **签名**: A × A → A × A (交换)
- **源**: 易经书名；「日新之谓盛德，生生之谓易」
- **读**: 交换 — 二者位置互换
- **注**: 三义俱全：变易、不易、简易。作为算子，最直接的读法是 swap

### T-4: 革
- **签名**: A → A' (彻底)
- **源**: 易·革卦「天地革而四时成」
- **读**: 革命 — 全面、彻底的变换
- **注**: 比 化 更剧烈；对应 overhaul / revolution

### T-5: 改
- **签名**: A → A (局部修改)
- **源**: 论语「过则勿惮改」
- **读**: 修改 — 局部修正
- **注**: 比 革 局部，比 化 主动

### T-6: 反
- **签名**: A → A⁻¹
- **源**: 老子「反者道之动」；易「无往不复」
- **读**: 反转 — 取逆向
- **注**: 老子的核心算子。注意有三义：(a) 返回 (b) 相反 (c) 反思。三者都是 involution

### T-7: 復 / 复
- **签名**: A → A (返回原态)
- **源**: 易·复卦；老子「夫物芸芸，各复归其根」
- **读**: 复归 — 回到起点
- **注**: 与 反 的区别：反是方向倒转，复是回到原点

### T-8: 還 / 还
- **签名**: A → A
- **源**: 老子「以道佐人主者，不以兵强天下，其事好还」
- **读**: 返回 — 必然性的回归
- **注**: 含因果回报色彩 — 不只是几何返回，是 karmic 返回

### T-9: 轉 / 转
- **签名**: A → A (旋转)
- **源**: 易「日往则月来，月往则日来，日月相推而明生焉」
- **读**: 旋转 — 沿轴变换
- **注**: 与 反 不同；反 是镜像反射，转 是旋转

### T-10: 推
- **签名**: A → A (沿轴推进)
- **源**: 易·系辞「刚柔相推而生变化」；孟子「推恩足以保四海」
- **读**: 推动 — 沿一个方向延伸
- **注**: 孟子的"推"是把一个原则推及更大范围 — 是推广算子

### T-11: 致
- **签名**: A × Goal → A
- **源**: 大学「致知在格物」；中庸「致中和」
- **读**: 致使到达 — 把状态推到目标
- **注**: 是 推 的有目的版本；含目的 telos

### T-12: 損 / 损
- **签名**: A → A⁻
- **源**: 易·损卦；老子「为道日损」
- **读**: 减损 — 减少元素或强度
- **注**: 是 易 的两个数量算子之一

### T-13: 益
- **签名**: A → A⁺
- **源**: 易·益卦
- **读**: 增益 — 增加元素或强度
- **注**: 与 損 配对；損益是经典的对偶变换

### T-14: 屈
- **签名**: A → A (收缩)
- **源**: 易·系辞「尺蠖之屈，以求信也」；老子「曲则全」
- **读**: 屈折 — 收缩、弯曲、退让
- **注**: 与 伸 配对。表征收缩态

### T-15: 伸
- **签名**: A → A (展开)
- **源**: 同上
- **读**: 伸展 — 展开、延伸
- **注**: 与 屈 配对。屈伸 是动态学的基本对偶

## 四、流动算子 — 有向运动

### F-1: 往
- **签名**: A × Dir → A
- **源**: 易「无往不复」
- **读**: 前往 — 朝一方向的移动
- **注**: 与 来 配对，构成方向的二元

### F-2: 來 / 来
- **签名**: A × Dir → A
- **源**: 同上
- **读**: 到来 — 反向移动
- **注**: 与 往 配对

### F-3: 進 / 进
- **签名**: A → A (前进)
- **源**: 易「君子以独立不惧，遁世无闷」彖「进退存亡而不失其正者」
- **读**: 前进 — 沿正向推进一步
- **注**: 与 退 配对；离散步进

### F-4: 退
- **签名**: A → A (后退)
- **源**: 同上
- **读**: 后退 — 反向一步
- **注**: 与 进 配对；离散

### F-5: 升
- **签名**: A → A (上升)
- **源**: 易·升卦「君子以顺德，积小以高大」
- **读**: 上升 — 沿垂直正向
- **注**: 隐含纵向梯度

### F-6: 降
- **签名**: A → A (下降)
- **源**: 易·泰「天地交而万物通也，上下交而其志同也」
- **读**: 下降 — 沿垂直反向
- **注**: 与 升 配对

### F-7: 出
- **签名**: A × Boundary → A
- **源**: 老子「出生入死」
- **读**: 出 — 从某域进入外部
- **注**: 与 入 配对；穿过边界算子

### F-8: 入
- **签名**: A × Boundary → A
- **源**: 同上
- **读**: 入 — 从外部进入某域
- **注**: 与 出 配对

### F-9: 行
- **签名**: A → A (持续移动)
- **源**: 易「天行健」；「日月运行」
- **读**: 行进 — 持续的运动过程
- **注**: 比 进 更强调 ongoing process；持续算子

### F-10: 動 / 动
- **签名**: A → A
- **源**: 易·系辞「吉凶悔吝者，生乎动者也」
- **读**: 启动 — 从静态进入动态
- **注**: 与 静 对立；表征 process initiation

### F-11: 靜 / 静
- **签名**: A → A
- **源**: 老子「致虚极，守静笃」
- **读**: 归静 — 进入静态
- **注**: 与 动 互为对偶

### F-12: 通
- **签名**: A × A → Prop / A → A (穿透)
- **源**: 易·系辞「往来不穷谓之通」；庄子「道通为一」
- **读**: 通达 — 在结构间穿行无碍
- **注**: 既是关系算子（A与B相通）也是变换算子（使其相通）

## 五、边界算子 — 端点、起止

### B-1: 始
- **签名**: A → Point
- **源**: 大学「物有本末，事有终始」；老子「无名天地之始」
- **读**: 起点算子 — 取过程的开端
- **注**: 与 终 配对

### B-2: 終 / 终
- **签名**: A → Point
- **源**: 同上
- **读**: 终点算子 — 取过程的终结
- **注**: 与 始 配对

### B-3: 起
- **签名**: A → A (启动)
- **源**: 易·震「震来虩虩，笑言哑哑」
- **读**: 兴起 — 从无到有
- **注**: 与 止 配对；起 是 0→1 的初始化

### B-4: 止
- **签名**: A → A (终止)
- **源**: 易·艮卦「艮其止，止其所也」；大学「在止于至善」
- **读**: 停止 — 终结过程
- **注**: 大学的 止 是高阶算子：止于至善 = 把目标固定在某点

### B-5: 立
- **签名**: A → A (确立)
- **源**: 论语「三十而立」；中庸「中立而不倚」
- **读**: 建立 — 让结构稳定存在
- **注**: 是带 stability 的 起；结构存在性算子

### B-6: 成
- **签名**: A → A (完成)
- **源**: 中庸「成己仁也，成物知也」
- **读**: 完成 — 让结构达到完整态
- **注**: 比 立 更进 — 立是建起，成是完工

### B-7: 極 / 极
- **签名**: A → Point (极值)
- **源**: 易·系辞「易有太极」；「物极必反」
- **读**: 取极算子 — 推到边界
- **注**: 与 反 联用：极→反 是反复律的核心

### B-8: 節 / 节
- **签名**: A × A → Point / A → [Point]
- **源**: 易·节卦「节以制度」；庄子「彼节者有间」
- **读**: 节点 — 在连续过程上插入离散标记
- **注**: 让连续物可被分段；是离散化算子

## 六、量化算子

### Q-1: 凡
- **签名**: (A → Prop) → Prop
- **源**: 礼记「凡为人子之礼」；荀子「凡论者，贵其有辨合」
- **读**: 全称量化 — ∀
- **注**: 文言文标准的 ∀

### Q-2: 皆
- **签名**: 同 ∀ 但更强调实例
- **源**: 论语「四海之内皆兄弟也」
- **读**: 皆然 — 所有实例
- **注**: 与 凡 微差：凡 是抽象普遍，皆 是具体分布

### Q-3: 各
- **签名**: A → (A → P) → ∏ P
- **源**: 易·系辞「方以类聚，物以群分」释「各从其类也」
- **读**: 各自 — 分布化的全称
- **注**: 是 dependent quantification — 每一个有自己的某属性

### Q-4: 莫
- **签名**: (A → Prop) → Prop
- **源**: 论语「过则莫之改」；老子「天下莫柔弱于水」
- **读**: 否定全称 — ¬∃ 或 ∀¬
- **注**: 既可读为"无人/无物"也可读为"没有比"

### Q-5: 或
- **签名**: (A → Prop) → Prop
- **源**: 易「或跃在渊」；论语「或问禘之说」
- **读**: 存在量化 — ∃
- **注**: 文言文的 ∃；常带不定意味

### Q-6: 有
- **签名**: A → Prop / Type → Type
- **源**: 老子「有物混成，先天地生」
- **读**: 存在 — 比 或 更一般
- **注**: 既存在量化，又类型构造（"有X者"）

### Q-7: 無 / 无
- **签名**: A → Prop
- **源**: 老子「无名天地之始」
- **读**: 不存在 — ¬有
- **注**: 与 有 对偶；老子哲学的核心算子

### Q-8: 唯
- **签名**: A → ∃! Prop
- **源**: 论语「唯仁者能好人，能恶人」
- **读**: 唯一 — ∃!
- **注**: 量化加唯一性；强约束

## 十一、序贯算子 — 文言文虚词层

这一类是经典所谓"虚词" — 不指物，专门组合命题/动作。它们在形式系统中相当于 connectives 和 modifiers。

### S-1: 之
- **签名**: A × B → A·B (附属 / 类似 genitive)
- **源**: 通用；论语「学而时习之」（代词用法）；「孔子之学」（属格用法）
- **读**: (a) 代指 (b) 属格 (c) 趋向 "去往"
- **注**: 多义虚词；作为算子主要是 generic binding

### S-2: 而
- **签名**: Action × Action → Action / Prop × Prop → Prop
- **源**: 通用；论语「学而时习之」
- **读**: 序贯连接 — and-then / and / but
- **注**: 文言文最通用的连接算子；可表 conjunction 也可表 sequencing

### S-3: 則 / 则
- **签名**: Prop × Prop → Prop
- **源**: 通用；大学「物格而后知至」
- **读**: 蕴含 — →
- **注**: 标准蕴含；常 "若...则..."

### S-4: 若
- **签名**: Prop → Prop (前提)
- **源**: 通用
- **读**: 假设 — if
- **注**: 与 则 配对成 if-then

### S-5: 雖 / 虽
- **签名**: Prop → Prop (虽然)
- **源**: 通用
- **读**: 让步 — although
- **注**: 与 而 / 然 联合表 although-yet

### S-6: 然
- **签名**: Prop → Prop / Anaphora
- **源**: 通用
- **读**: 如此 / 然而 — thus / however
- **注**: 既是指代算子（thus, this-state），又是连接算子（however）

### S-7: 故 (复用 K-1)
- 在序贯层：therefore connector

### S-8: 乃
- **签名**: Prop → Prop
- **源**: 通用；尚书「乃命羲和」
- **读**: 于是 — thereupon
- **注**: 序贯算子，比 则 更强调时间 then

### S-9: 既
- **签名**: Action → Action (完成)
- **源**: 易「既济」；论语「既往不咎」
- **读**: 已经 — perfective aspect
- **注**: 是 aspect operator — 把动作标记为完成态

### S-10: 將 / 将
- **签名**: Action → Action (将来)
- **源**: 老子「将欲歙之，必固张之」
- **读**: 将要 — future / prospective aspect
- **注**: 与 既 对偶；构成 perfective/prospective 二元

### S-11: 方
- **签名**: Action → Action (正在)
- **源**: 论语「方丧三年之久」
- **读**: 正在 — progressive aspect
- **注**: 与 既/将 构成三体 aspect 系统：既/方/将 = past/present/future progressive

### S-12: 嘗 / 尝
- **签名**: Action → Action (曾经)
- **源**: 论语「吾尝终日不食」
- **读**: 曾经 — experiential past
- **注**: 经验式过去 — 与 既 (perfective) 微差

### S-13: 者
- **义位**: 者1
- **签名**: Term / Prop → DefinitionBinder
- **读**: 定义、抽象、谓词封装
- **注**: 「X者，P也」对应 `X := P` 或 `X(x) ⇔ P(x)`。

### S-14: 也
- **义位**: 也1
- **签名**: Prop → Assertion
- **读**: 断定 / 类型闭合 / 命题完成
- **注**: 常与 者 联合，闭合定义句或判断句。

### S-15: 于 / 於
- **义位**: 于1 / 於1
- **签名**: Prop × Context → Prop
- **读**: 境域绑定 — holds-in
- **注**: 「P于Γ」对应 `Γ ⊨ P` 或 `P` 在境 `Γ` 中成立。

### S-16: 所
- **义位**: 所1
- **签名**: Predicate / Action → Nominal
- **读**: 名物化 / 集合化 / 所指化
- **注**: 「所是(Γ)」即 `what Γ is`；「所P」可读作 `{x | P(x)}` 或动作结果名物。

> **Isness 单字簇（补）**：`是 / 系 / 之 / 的 / 地` 与伴随算子 `有 / 无 / 所 / 者 / 也 / 为 / 然 / 于`
> 共同承担英文 *is / isness* 的分解。`是` 为等同，`系` 为关系连接，`之/的` 为属格或投影绑定，
> `地/于` 为境域成立；`有/无` 给存在边界，`所` 名物化，`者/也` 定义闭合，`为/然` 给谓述与断定。
> Lean 侧见 `Foundation/Wen/Operators.lean` 的 `isness_with_companion_operators` 与
> `Text/Completeness.lean` 的 `isness_single_glyph_operators_registered`。

### S-17: 未
- **义位**: 未1
- **签名**: Prop → Prop
- **读**: 尚未 — not-yet
- **注**: 非简单否定；保留未来开口，可写作 `¬P_now ∧ ◇P`。

### S-18: 已
- **义位**: 已1
- **签名**: Action / Prop → Completed
- **读**: 已经 / 完成 / 稳定
- **注**: 与 未 对偶，标记动作或判态已实际化、已闭合。

## 十二、复合算子 — 高阶 / 算子之算子

### H-1: 道
- **签名**: A → Path
- **源**: 老子「道可道，非常道」；论语「朝闻道」
- **读**: 路径算子 — 给一物以其内在路径/法则
- **注**: 最高阶算子之一；既指 path object 又指 path-generating function

### H-2: 理
- **签名**: A → Pattern
- **源**: 韩非子「万物各异理」；宋儒「理一分殊」
- **读**: 抽出模式 — extract structure / pattern
- **注**: 是从具体物中抽出 universal 的算子

### H-3: 勢 / 势
- **签名**: Configuration → Tendency
- **源**: 韩非「乘势」；孙子「形者，胜之所以成；势者，胜之所以乘」
- **读**: 势 — 由结构生成的趋势
- **注**: 是从静态构型读出动态的算子；近似 gradient field

### H-4: 機 / 机
- **签名**: Process → CriticalPoint
- **源**: 庄子「万物皆出于机，皆入于机」；易「知几其神乎」
- **读**: 关键点 — extract critical point of a process
- **注**: 与 势 配对：势是趋势，机是关节点

### H-5: 法
- **签名**: A × A → Action
- **源**: 易「法象莫大乎天地」；韩非「法者，宪令著于官府」
- **读**: 法度 — generate action from pattern
- **注**: 双义：(a) 效法 imitate (b) 法度 norm

### H-6: 象
- **签名**: A → Glyph
- **源**: 易·系辞「在天成象，在地成形」；「圣人立象以尽意」
- **读**: 取象 — produce iconic representation
- **注**: 高阶 — 是从抽象生成可感知形象的算子；易 学的核心

### H-7: 名
- **签名**: A → Symbol
- **源**: 老子「无名天地之始，有名万物之母」；论语「必也正名乎」
- **读**: 命名 — assign symbol to entity
- **注**: 与 象 互补：象是图，名是号；正名 = 校准 名↔实

### H-8: 體 / 体 与 用
- **签名**: 一对 — 体: A → Essence；用: A → Function
- **源**: 宋儒；先秦实例如 "君子之于天下也，无适也，无莫也，义之与比"
- **读**: 体 = 静态本质算子；用 = 动态功能算子
- **注**: 体用是中国哲学最重要的算子配对；类似 type/term 或 specification/implementation

## 十三、墨经定义算子 — 中国古代最严格的定义代数

墨经 (经上、经说上、经下、经说下) 是先秦最接近形式系统的文本。它的格式本身是一种 **operator-as-definition** 的代数 — "经" 给字的简定义，"说" 给细节。每条都是一行 `字 = 解释`。这是**用字定义字**的递归代数。

### P-1: 故 (墨经版)
- **签名**: Cause × Effect → Prop
- **源**: 经上「故，所得而后成也」；说上：「故：小故，有之不必然，无之必不然；大故，有之必然」
- **读**: 充分/必要条件区分 — 小故 = 必要 (necessary), 大故 = 充分 (sufficient)
- **注**: 墨家**第一个明确区分必要与充分条件**。这是中国本土的形式逻辑核心。比 K-1 的简单"故"严格得多

### P-2: 體 / 体
- **签名**: Whole → Part
- **源**: 经上「体，分于兼也」
- **读**: 取部分 — extract a part from the whole
- **注**: 与 兼 严格对偶。这是**部分-整体关系**的明确公理化

### P-3: 兼
- **签名**: [Part] → Whole
- **源**: 经上「兼，体俱于之一也」(implicit)
- **读**: 整合 — collect parts into whole
- **注**: 与 体 互逆；构成 part/whole monad

### P-4: 同 (墨经四类)
- **签名**: A × A → Prop (但有四种)
- **源**: 经上「同：异而俱于之一也」；说上「同，二人而俱见是楹也，若事君」；分四类：重同 / 体同 / 合同 / 类同
- **读**: 四种相等 — (a) 重同: 一物两名 (b) 体同: 同属一体 (c) 合同: 共处一处 (d) 类同: 同类
- **注**: 墨家**区分四种 sameness**。对应到现代：identity, mereological, spatial co-location, type-equivalence。这是惊人精细的等价关系分类

### P-5: 異 / 异 (墨经四类)
- **签名**: A × A → Prop (四类)
- **源**: 经上「异，二必异，二也」；说上分四类：二 / 不体 / 不合 / 不类
- **读**: 与 同 严格对偶 — 四类不同
- **注**: 与 P-4 共同构成完整的等价/差异分类系统

### P-6: 久
- **签名**: Action → Duration
- **源**: 经上「久，弥异时也」
- **读**: 时间延展 — covers different times
- **注**: 时间作为可量化对象的算子化定义。"弥" = pervading

### P-7: 宇
- **签名**: Object → Region
- **源**: 经上「宇，弥异所也」
- **读**: 空间延展 — covers different places
- **注**: 与 久 平行 — 时间/空间双胞胎算子。已经是相对论式的时空概念

### P-8: 動 (墨经版)
- **签名**: A × Time → A × Region
- **源**: 经上「动，或徙也」；说上「动，偏际徙也」
- **读**: 移动 — translation in 宇
- **注**: 与 F-10 不同 — 墨经定义更严格：动是"边界的位移"

### P-9: 止 (墨经版)
- **签名**: A → A
- **源**: 经上「止，以久也」；说上「止，无久之止，当牛非马，若矢过楹」(瞬时停)；「有久之止，当马非马，若人过梁」(持续停)
- **读**: 停止 — 区分瞬时 vs 持续两种
- **注**: 比 B-4 更精细 — **区分瞬时停 (instantaneous) 与持续停 (duration)**，类似时序逻辑里的 punctual / persistent

### P-10: 端
- **签名**: Line → Point
- **源**: 经上「端，体之无序而最前者也」；说上「端，是无同也」
- **读**: 取端点 — extract endpoint of geometric object
- **注**: **零维点的代数化定义**。墨家几何学的基础

### P-11: 尺
- **签名**: 2 × Point → Line
- **源**: 经上「尺，前于区穴，而后于端」(implicit from context)
- **读**: 由两点构造线段
- **注**: 一维段的算子构造

### P-12: 中 (墨经版)
- **签名**: Region → Point (where 同长 holds)
- **源**: 经上「中，同长也」
- **读**: 中心 = 等距点
- **注**: **中心的距离公理化定义**。比 R-5 严格 — 不是模糊"取中"，是"到边界等距"

### P-13: 圜 / 圆
- **签名**: Center × Length → Region
- **源**: 经上「圜，一中同长也」
- **读**: 圆 = 一中心 + 等长 (i.e. 一切点到中等距)
- **注**: **圆的现代式定义** — 一个中心，所有点到中等距。比欧几里得更紧凑

### P-14: 平
- **签名**: A × A → Prop
- **源**: 经上「平，同高也」
- **读**: 平 = 等高
- **注**: 平行/水平的算子化

### P-15: 攖 / 撄
- **签名**: A × A → Prop
- **源**: 经上「攖，相得也」；说上「攖，尺与尺俱不尽，端与端俱尽，尺与端或尽或不尽」
- **读**: 接触 — touch/intersect
- **注**: **接触关系的完整公理化** — 区分线-线接触、点-点接触、线-点接触三种几何情形

### P-16: 仳
- **签名**: A × A → Prop
- **源**: 经上「仳，有以相撄，有不相撄也」
- **读**: 比邻 — adjacency without necessary contact
- **注**: 比 R-8 (比) 更精细 — 区分"接触的相邻"与"不接触的相邻"

### P-17: 次
- **签名**: A × A → Prop
- **源**: 经上「次，无间而不相撄也」
- **读**: 紧次 — adjacent without gap, but not touching
- **注**: 接触/邻接/紧次的三层精细区分。这种**层级化的拓扑算子**是先秦独有

### P-18: 法
- **签名**: A → Action
- **源**: 经上「法，所若而然也」；说上「法，意、规、员，三也，俱可以为法」
- **读**: 法则 — pattern that prescribes action
- **注**: 与 H-5 联系。墨家定义更严：法 是 "若X则然" 的模式抽取算子。三种实现：(a) 意 = mental model (b) 规 = compass/template (c) 员 = circle/instance。这预演了 model/specification/instance 的现代区分

### P-19: 名 (墨经三类)
- **签名**: Object → Symbol (三类)
- **源**: 经上「名：达、类、私」；说上「名，物，达也，有实必待之名也命之；马，类也，若实也者必以是名也命之；臧，私也，是名也，止于是实也」
- **读**: 三种命名 — (a) 达名: 通用 (e.g. "物") (b) 类名: 类指 (e.g. "马") (c) 私名: 专指 (e.g. "臧")
- **注**: **三层命名层级** = 现代的 universal / sortal / proper name。形式语义学的鼻祖

### P-20: 知 (墨经三类)
- **签名**: Subject × Object → KnowledgeRelation (三类)
- **源**: 经上「知：闻、说、亲」
- **读**: 三种知识来源 — (a) 闻知: 听说的 (b) 说知: 推论的 (c) 亲知: 亲身的
- **注**: 知识的来源代数 — 信息源算子。也是认知建模的早期分类

## 十四、名家关系算子 — 公孙龙的指物代数

名家 (公孙龙、惠施、邓析) 关心的是 **名/实 关系** — 符号与对象的连接。他们的算子是**符号学层面**的，比墨经更上一层。

### G-1: 指
- **签名**: Symbol × Object → Designation
- **源**: 公孙龙·指物论「物莫非指，而指非指」
- **读**: 指称 — designation/reference
- **注**: 公孙龙的核心算子。"物莫非指" = 一切物都是被指称物；"而指非指" = 但指称行为本身不是被指物。这是**指称与被指称的分离**

### G-2: 物
- **签名**: Object (基本类型)
- **源**: 同上
- **读**: 对象 — pure referent
- **注**: 与 指 严格区分

### G-3: 名 (名家版)
- **签名**: Symbol → Object
- **源**: 公孙龙·名实论「天地与其所产焉，物也。物以物其所物而不过焉，实也。实以实其所实而不旷焉，位也」
- **读**: 名命 — name's reference function
- **注**: 比 P-19 更哲学化 — 名 是 实 的语义函数

### G-4: 實 / 实
- **签名**: Object (实质)
- **源**: 同上
- **读**: 实在 — actuality of an object
- **注**: 与 名 配对。名实之辨是先秦哲学核心问题

### G-5: 位
- **签名**: Object → Region
- **源**: 同上「实以实其所实而不旷焉，位也」
- **读**: 位 — proper place of an actuality
- **注**: 实的容器；不旷 = 无空缺。每实有其位

### G-6: 正 (名家版)
- **签名**: (Name × Object) → Prop
- **源**: 公孙龙「正名者，所以正其实」；论语「必也正名乎」
- **读**: 校准 — calibrate name with reality
- **注**: 与 R-6 不同 — 此 正 是符号-对象配对的校正

### G-7: 離 / 离 (坚白离)
- **签名**: A × A → A + A
- **源**: 公孙龙·坚白论「视不得其所坚而得其所白者，无坚也；拊不得其所白而得其所坚者，无白也」
- **读**: 分离属性 — 把两个属性视为独立维度
- **注**: 公孙龙的"坚白离"是**属性独立性的论证**。把通常视为不可分的属性 (坚 + 白 in 石) 视为可分别感知 → 因此可分

### G-8: 兼 / 合 (名家版)
- **签名**: A × A → A
- **源**: 公孙龙·白马论「白马者，马与白也。马者，所以命形也；白者，所以命色也」
- **读**: 复合 — compound name
- **注**: 与 P-3 不同 — 此处是**符号合成**而非物理合成。"白马" = 合 (白, 马)

### G-9: 別 / 别 (名家版)
- **签名**: A → A × A
- **源**: 公孙龙「白马非马」
- **读**: 区分 — distinguish/diverge
- **注**: 是 G-8 的逆。"白马非马" 论证：复合名 ≠ 简单名

### G-10: 然
- **签名**: Subject × Predicate → Prop
- **源**: 通用，但名家精化；墨经也用「以然」
- **读**: 谓词系动 — predication
- **注**: 是判断算子。"X然" = X is so / X obtains. 不同于 S-6 (然 as anaphora)

## 十五、副词修饰算子 — Aspect / Manner / Frequency

这一类是 **修饰其他算子的算子**。它们不引入新结构，但调整其他算子的时态、强度、频率、方式。在形式语言里相当于 modal modifiers / aspect markers / frequency adverbs。

### A-1: 漸 / 渐
- **签名**: Action → Action (渐进)
- **源**: 易·渐卦「渐之进也」
- **读**: 渐变化 — gradually
- **注**: 是 化 (T-1) 的天然伴侣 — 强调连续性

### A-2: 驟 / 骤
- **签名**: Action → Action (突变)
- **源**: 论语「色斯举矣，翔而后集」释「不时，不驟」
- **读**: 突然 — suddenly
- **注**: 与 漸 对偶 — 离散跳跃

### A-3: 忽
- **签名**: Action → Action (瞬时)
- **源**: 通用；楚辞「忽魂悸以魄动」
- **读**: 倏忽 — abruptly, without warning
- **注**: 比 驟 更强调瞬间性

### A-4: 遂
- **签名**: Action × Action → Action (因果连接)
- **源**: 通用；荀子「遂以及之」
- **读**: 于是就 — consequently/thereby
- **注**: 比 S-8 (乃) 更强因果

### A-5: 復 / 复 (副词层)
- **签名**: Action → Action (重复)
- **源**: 论语「不再问」反义
- **读**: 再次 — again
- **注**: 复 既是变换算子 (T-7) 也是副词。语境决定签名

### A-6: 又
- **签名**: Action → Action (重复)
- **源**: 通用
- **读**: 又一次 — also/again
- **注**: 与 復 微差 — 又 强调累加，復 强调返回

### A-7: 屢 / 屡
- **签名**: Action → Action (频次)
- **源**: 论语「屡空」
- **读**: 屡次 — frequently
- **注**: 频率高于 又 / 復

### A-8: 愈
- **签名**: Property → Property (递增程度)
- **源**: 老子「为道日损，损之又损，以至于无为」释「愈贱愈贵」(implicit)
- **读**: 越发 — the more / increasingly
- **注**: 是 强度递增算子 — 修饰程度

### A-9: 益 (副词层)
- **签名**: Property → Property (增加)
- **源**: 老子「为学日益」
- **读**: 更加 — increasingly
- **注**: 与 T-13 不同 — 此处是修饰，不是变换

### A-10: 寖 / 浸
- **签名**: Action → Action (缓慢渐进)
- **源**: 史记，以及战国诸子；「寖以衰落」
- **读**: 渐渐 — gradually (slower than 漸)
- **注**: 比 漸 更慢；强调潜移默化

### A-11: 既...又... / 一...且... (双重 marker)
- **签名**: Prop × Prop → Prop
- **源**: 通用
- **读**: 同时具备 — both...and...
- **注**: 多算子标记语法。已经是合取的 sugar，但通过位置而不是 connective 实现

### A-12: 且
- **签名**: Prop × Prop → Prop
- **源**: 通用
- **读**: 而且 / 暂且 — moreover / for now
- **注**: 兼有 conjunction 与 hedging 两义

### A-13: 但
- **签名**: Prop × Prop → Prop
- **源**: 通用 (晚先秦/汉)
- **读**: 仅 / 却 — only / however
- **注**: 既是限定 (仅) 又是转折 (却)

### A-14: 唯 (副词层)
- **签名**: Prop → Prop
- **源**: 通用
- **读**: 唯独 — exclusively
- **注**: 与 Q-8 联系；副词层是 only/exclusively

### A-15: 亦
- **签名**: Prop → Prop
- **源**: 通用；论语「不亦乐乎」
- **读**: 也 — also
- **注**: 平行对应 marker — 标记 "X 也是 P"

## 十六、数词作算子 — 数字的非数字用法

文言文中数字常作算子使用，不仅是计数。

### D-1: 一 (作算子)
- **签名**: A → A (统一)
- **源**: 老子「道生一」；庄子「道通为一」
- **读**: 归一 / 统一
- **注**: 同 I-2。最重要的数字-算子

### D-2: 再
- **签名**: Action → Action (第二次)
- **源**: 论语「再，斯可矣」；「不贰过」(变体)
- **读**: 重复一次 — second time
- **注**: 离散重复算子；与 復 微差 — 再 是计数性的"二次"

### D-3: 三 (作算子)
- **签名**: 多种用法
- **源**: 论语「三人行，必有我师」；「日三省吾身」；「三思而后行」
- **读**: (a) 多数 (multitude) (b) 三次重复 (c) 完整集合 (e.g. 三才, 三纲)
- **注**: 三 在文言中常表示"完整"或"多次"，不一定确切是 3。是**模糊量化算子**

### D-4: 兩 / 两
- **签名**: A → A × A
- **源**: 通用；庄子「两行」
- **读**: 二元化 — make dual
- **注**: 与 一 反向；从合一到分二

### D-5: 倍
- **签名**: A → 2A
- **源**: 通用
- **读**: 加倍 — double
- **注**: 量级算子

### D-6: 半
- **签名**: A → A/2
- **源**: 通用；老子「揣而锐之，不可长保」释暗示节制即半
- **读**: 半 — halve
- **注**: 与 倍 互逆

### D-7: 全
- **签名**: A → A (全部)
- **源**: 老子「曲则全」；庄子「全人」
- **读**: 全部 / 完整 — whole
- **注**: 与 部分 (如 体) 对偶

### D-8: 餘 / 余
- **签名**: A × A → A
- **源**: 通用；易「无以教，则有余师」
- **读**: 剩余 — remainder
- **注**: 减法的余项；是部分-减法的 dual

### D-9: 終 / 终 (作量词层)
- **签名**: Range → Element
- **源**: 论语「终日不食」；「终身不忘」
- **读**: 终至 — through-out / for the entirety of
- **注**: 是普遍量化的时间版本 — "整个时段"

### D-10: 半 / 過半 / 大半
- **签名**: Set → Subset
- **源**: 史记 / 战国策
- **读**: 半数以上 — majority
- **注**: 比例量化算子

## 十七、史官元算子 — 春秋的文本元操作

春秋 / 史书使用一类**对文本本身的操作算子** — 它们不改变事件，但改变事件如何被记录。这是**元层 (meta-level) 算子**。 

孔子作春秋 "笔则笔，削则削" 即是用这类算子做出判断。

### E-1: 書 / 书
- **签名**: Event → Record / ¬Record
- **源**: 春秋经；公羊传「春秋书曰」
- **读**: 记载与否 — whether to record
- **注**: 春秋的"不书" = 删除算子，是判断的实施

### E-2: 曰
- **签名**: Speaker × Content → Quotation
- **源**: 通用
- **读**: 引用 — quote
- **注**: 是 quotation operator — 把内容嵌入"被某人说"的元层

### E-3: 稱 / 称
- **签名**: Object × Name → Designation
- **源**: 春秋；公羊「春秋称之曰子」
- **读**: 称呼为 — designate as
- **注**: 比 G-3 更主动 — 是 act of naming

### E-4: 諱 / 讳
- **签名**: Event → Concealed Event
- **源**: 春秋三传；「为尊者讳，为亲者讳，为贤者讳」
- **读**: 讳避 — taboo / euphemize
- **注**: **隐藏算子** — 不删除但模糊化。是中国史学独有的算子

### E-5: 譏 / 讥
- **签名**: Event → Critical Record
- **源**: 公羊传 / 谷梁传 "春秋讥..."
- **读**: 隐含批评 — implicit critique through recording choice
- **注**: 通过**用词选择**实现的批评。这是**用元数据传递元意义**的算子

### E-6: 與 / 与 (春秋层) 
- **签名**: Event → Endorsed Event
- **源**: 公羊「春秋与齐桓公伐戎也」
- **读**: 认可 — endorse through recording
- **注**: 与 譏 对偶 — 与是隐性肯定，譏是隐性否定

### E-7: 削
- **签名**: Record → ¬Record
- **源**: 史记「孔子作春秋，笔则笔，削则削」
- **读**: 删除 — strike from record
- **注**: 与 笔 (write) 对偶。是**编辑算子**的明确化

## 十八、法家算子 — 韩非的政治拓扑 (Legalist / Agent Governance)

韩非综合 商鞅 (法) + 申不害 (术) + 慎到 (势) 给出**三元政治算子代数**。这是先秦最接近现代制度设计/multi-agent governance 的算子集。

核心洞察：法家的算子明确假设 **agents 有自利倾向**、**信息不对称**、**结构位置决定权力**。这是 mechanism design 的早期形式。

### L-1: 法 (法家版)
- **签名**: System × Action → {允许, 禁止, 奖, 罚}
- **源**: 韩非·定法「法者，宪令著于官府，赏罚必于民心」；难三「法者，编著之图籍，设之于官府，而布之于百姓者也」
- **读**: 公开、成文、普遍、可预测的规则函数
- **注**: 比 H-5 (一般 法) 严格 — 必须满足四属性 (公开/成文/普遍/可预测)。**这是合同/类型系统的政治版本**：所有 agent 知道签名，行为有可计算后果

### L-2: 術 / 术
- **签名**: Ruler × State → Ruler' (隐藏)
- **源**: 韩非·定法「术者，藏之于胸中，以偶众端而潜御群臣者也」
- **读**: 君主隐藏的控制技术 — 不公开的 meta-operator
- **注**: 与 法 严格对偶 — **法是公开的，术是私密的**。这是 information asymmetry 的算子化。术是 ruler 操控 法 的算子 — 即 **算子之算子 (higher-order)**

### L-3: 勢 / 势 (法家版)
- **签名**: Position → Power
- **源**: 韩非·难势「飞龙乘云，腾蛇游雾」；引慎到「势者，胜众之资也」
- **读**: 由位置生成的非个人权力
- **注**: 比 H-3 (一般势) 政治化 — 关键特征是**与个人能力解耦**。"中人之资乘势" 即可治。这是 institutional power 的 早期论述

### L-4: 形名 (paired)
- **签名**: Form × Name → Prop (匹配关系)
- **源**: 韩非·主道「有言者自为名，有事者自为形，形名参同，君乃无事焉」
- **读**: 实际表现 (形) 与所宣称之名 (名) 是否相符
- **注**: **这是验证算子 (verification operator)** — 检查 output 是否符合 contract。等价于现代的 type-checking, signature matching, contract verification

### L-5: 參同 / 参同 (参验)
- **签名**: [Testimony] → Truth
- **源**: 韩非·内储说上「众端参观」；八经「参之以比物」
- **读**: 多源交叉验证
- **注**: 通过 N 个独立来源的证词比对来确定真相。是**多 agent 报告的拜占庭一致性算法**的早期版本

### L-6: 二柄 (刑/德)
- **签名**: 行为 → 反馈 (negative/positive)
- **源**: 韩非·二柄「明主之所导制其臣者，二柄而已矣。二柄者，刑、德也」
- **读**: **刑 = 罚函数；德 = 赏函数**。两个 control levers
- **注**: 是 reinforcement learning 的奖罚信号。韩非强调：**两柄必须由君主独占**，不可下放，否则 agent 会倒向赏罚的实际发出者。这是 incentive alignment 的强制约束

### L-7: 因 (法家版)
- **签名**: Tendency × Action → Effective Action
- **源**: 韩非·外储说右上「因人之情」；老子「以辅万物之自然而不敢为」(韩非引)
- **读**: 借助/利用既有倾向，不与之对抗
- **注**: **杠杆算子** — 把 action 嵌入到现有的 force field 中以增效。法家版的 因 是 **设计与人性自利相符的激励** 而不是 fight against it

### L-8: 任
- **签名**: Role × Person → Assignment
- **源**: 韩非·主道「明君之道，使智者尽其虑」；二柄「明主之所道制其臣」
- **读**: 把 function 委派给 specific role/person
- **注**: 是 delegation operator。结合 L-4 形名，构成完整的: **任(role) → 形名参同(verify) → 二柄(reinforce)** 流水线 = 现代 agent 系统的 task assignment + monitoring + reward

### L-9: 守
- **签名**: Role → Role (保持)
- **源**: 韩非·有度「明主使法择人，不自举也；使法量功，不自度也」；解老「圣人守一」
- **读**: 守住分内 — agent stays within assigned role
- **注**: 与 L-8 配对。任 是分配，守 是保持。**违守即越权**

### L-10: 靜 / 静 (法家版)
- **签名**: Ruler → Ruler (无主动)
- **源**: 韩非·主道「君无见其所欲」；「故曰：寂乎其无位而处，漻乎莫得其所」
- **读**: 君主不显露偏好，使 agents 无可揣摩
- **注**: 是 information hiding 算子 — 让 agents 无法 reverse-engineer ruler's utility function。**这是 mechanism design 中 the principal hides preferences 的策略**

### L-11: 無為 / 无为 (法家版)
- **签名**: Ruler → System (制度自动运行)
- **源**: 韩非·主道「明君无为于上，群臣竦惧乎下」；「上无为而下无不为」
- **读**: 君主不亲自介入，由 法 + 术 + 势 自动驱动系统
- **注**: **不同于 老子无为**。法家无为是**已设计好的制度自动运行**；老子无为是**不去设计**。法家版接近 "configure once, run automatically"

### L-12: 利
- **签名**: Action → Real (utility)
- **源**: 韩非·六反「夫安利者就之，危害者去之，此人之情也」
- **读**: agent 的效用函数
- **注**: 把 motivation 还原为可计算的 utility。是 **rational agent assumption** 的明确化

### L-13: 一 (法家版: 法令统一)
- **签名**: [Local Rules] → Global Rule
- **源**: 韩非·有度「故矫上之失，诘下之邪，治乱决缪，绌羡齐非，一民之轨，莫如法」
- **读**: 把多重/局部规则规范为单一规则
- **注**: 不同于 老子归一 (I-2) — 法家一是**法令统一执行**，去除 ad hoc 判决

### L-14: 制
- **签名**: Pattern → Institution
- **源**: 韩非·五蠹「夫圣人之治国，不恃人之为吾善也，而用其不得为非也」隐含；「制」频出
- **读**: 把模式凝固为正式制度
- **注**: 与 H-2 (理) 互补 — 理是**抽出**模式，制是**固化**模式

### L-15: 度 (法家版)
- **签名**: Action → Measure
- **源**: 韩非·有度「故以法治国，举措而已矣」
- **读**: 标准化测量 — 评估 action 用统一标准
- **注**: 不同于 G-5 (位) 的**位**概念。法家**度**是**度量衡的政治泛化** — 任何 action 都有可比较的标准

### L-16: 罰 / 赏
- **签名**: Action × Outcome → Feedback (二柄的实施)
- **源**: 韩非·六反 / 二柄
- **读**: 实施 reward/punishment
- **注**: L-6 二柄 的具体动作算子。罰/赏 是 application；二柄 是 design

### 法家算子的代数结构

法家三元 **法/术/势** 的关系：

```
势 (位置带权)        ─────  enables  ────→  agent operations
 ↓                                              ↓
法 (公开规则)       ─────  constrains  ────→  agent actions
 ↑                                              ↑
术 (隐藏控制)       ─────  modulates  ────→  rule enforcement
```

形式上：
- **势** = 静态 enabling structure (像 Hilbert space)
- **法** = 公共动力学 (state transition rules)
- **术** = 隐藏 modulation (像 hidden Hamiltonian)

**法家算子等式**:

$$\text{治国} = \text{无为}_\text{法家} = \text{术}\Big(\text{法}\big(\text{势}(系统)\big)\Big)$$

读法：在 势 之上施加 法，在 法 之上施加 术，则系统**自治**（君主静）。

### 与 multi-agent system / governance 的对应

| 法家算子 | Agent System 对应 |
|---|---|
| 法 | 公开协议 / contract / type spec |
| 术 | meta-supervisor 不公开的控制策略 |
| 势 | agent 在 architecture 中的 structural position |
| 形名参同 | output validation / type checking / signature match |
| 参验 | byzantine consensus / multi-source verification |
| 二柄 | RL reward signal / penalty |
| 任 | task assignment / role binding |
| 守 | scope/permission constraint |
| 因 | leverage existing dynamics (incentive design) |
| 利 | utility function |
| 静 (君静) | principal-side privacy / preference hiding |
| 制 | institutional configuration |
| 度 | metric / scoring function |

法家提供的不是哲学，是**完整的 multi-agent governance algebra** — 公开规则 + 隐藏策略 + 位置权力的三层嵌套设计。

## 十九、医家算子 — 黄帝内经的系统动力学

黄帝内经 (素问 + 灵枢) 是先秦/汉初**最完整的动力学系统语言**。它把 阴阳/五行/气/血/精/神 作为状态变量，给出：状态空间、流动方程、平衡条件、扰动响应、干预算子。

这套语言**不是医学专用** — 是普适动力系统语言，医学只是应用域。同样的算子被用于政治 (淮南子) / 农学 / 兵学 (孙子)。

### Y-1: 陰 / 阴 与 陽 / 阳
- **签名**: A → A_yin × A_yang (二元投影)
- **源**: 素问·阴阳应象大论「阴阳者，天地之道也，万物之纲纪」；阴阳离合论
- **读**: **不是简单二元** — 是把任意状态投影到对偶轴的算子对
- **注**: 阴阳的**对偶不完全**：每一阴中有阳之核，每一阳中有阴之核。形式：
  $$A = \alpha \cdot A_\text{阳} + \bar\alpha \cdot A_\text{阴}, \quad \alpha + \bar\alpha = 1$$
  这是**带种子的二元投影**，不是 Z/2 的纯二元

### Y-2: 五行 (木火土金水)
- **签名**: System → Z/5 上的循环结构
- **源**: 素问·阴阳应象大论；金匮真言论；尚书·洪范
- **读**: 五元状态空间，载二个循环算子
- **注**: 五行是 **finite group with two operations** — 不是单纯五元素，是 (Z/5, 相生, 相克) 的代数

### Y-3: 相生
- **签名**: 五行 → 五行 (周期)
- **源**: 素问·宝命全形论
- **读**: 木→火→土→金→水→木 的生成循环
- **注**: 在 Z/5 上是 +1 操作 (mod 5)。是 **生成性 / 喂养** 关系

### Y-4: 相克 (相剋)
- **签名**: 五行 → 五行 (跳跃)
- **源**: 同上
- **读**: 木→土→水→火→金→木 的克制循环
- **注**: 在 Z/5 上是 +2 操作 (mod 5)。是 **抑制 / 调节** 关系。
  
  **关键发现**: gcd(2, 5) = 1，所以 +2 也是 Z/5 的生成元 — 相生和相克都是循环群的两个不同生成元。两者构成 Z/5 的**自同构群** Z/5\* = {1, 2, 3, 4} 的两个元素

### Y-5: 相侮 / 反侮
- **签名**: 五行 → 五行 (反向克)
- **源**: 素问·五运行大论「气有余，则制己所胜而侮所不胜」
- **读**: 反方向的克 — 应该被克的反过来克
- **注**: 在 Z/5 上是 -2 操作 = +3。是病理学 — 当**强度越界**时正常关系颠倒

### Y-6: 氣 / 气
- **签名**: System → Flow (基础流动量)
- **源**: 素问·六微旨大论「气之升降，天地之更用也」；灵枢·邪客「五谷入于胃也，其糟粕、津液、宗气，分为三隧」
- **读**: 流动着的功能性物质
- **注**: **气 = state variable that flows**。不是物质而是 **substantive flow**。其守恒/转化是医家动力学的核心

### Y-7: 血
- **签名**: System → Substance (物质载体)
- **源**: 素问·调经论；灵枢·营卫生会
- **读**: 与 气 配对的物质基础
- **注**: **气血 = (function, substance) pair**。气是动 (verb-like)，血是体 (noun-like)。形式：气血是 (operation, operand) 对偶

### Y-8: 精
- **签名**: System → Reservoir
- **源**: 素问·上古天真论「肾者主水，受五脏六腑之精而藏之」
- **读**: 储藏态的根本能量
- **注**: 精 是 气血 的**积累形式**。三者关系: **精 → 气 → 神** = 物质积累 → 流动功能 → 整体涌现

### Y-9: 神
- **签名**: [State] → Coherence
- **源**: 素问·灵兰秘典论「心者，君主之官也，神明出焉」；灵枢·本神「故生之来谓之精，两精相搏谓之神」
- **读**: 整体协调的涌现态
- **注**: **神 = emergent coherence**。是系统层面的算子，不是局部算子。神 旺则系统自调；神 衰则需外部干预

### Y-10: 升 / 降 (医家专版)
- **签名**: Flow → Direction
- **源**: 素问·六微旨大论「升降出入，无器不有」
- **读**: 气的垂直流动 — 与 F-5/F-6 同名但有专门动力学含义
- **注**: 升降是**生理状态的关键诊断**。"清阳出上窍，浊阴出下窍" — 升降失常即病

### Y-11: 開 / 闔 / 樞 (开/合/枢 三才)
- **签名**: Gate → {open, closed, pivoting}
- **源**: 素问·阴阳离合论「太阳为开，阳明为阖，少阳为枢」；「太阴为开，厥阴为阖，少阴为枢」
- **读**: 三态门控算子 — 不是二元开/关，是 open/close/pivot 三元
- **注**: **这是医家独有的算子结构** — 三元门 (ternary gate)。不同于 Boolean 二元 {0,1}。
  - 开: 允许通过
  - 阖: 阻止通过
  - 枢: 调控开合的 meta-state
  
  形式上是 **finite automaton with 3 states**，且有 6 channels (3 阴 + 3 阳)，每 channel 实现一种门控功能

### Y-12: 經 / 经 与 絡 / 络
- **签名**: System → Network (Channel Topology)
- **源**: 灵枢·脉度；素问·调经论
- **读**: 经 = 主干通道；络 = 分支网络
- **注**: 经络系统是**有向图 + 流量动力学**。气血在其上流动，调控通过经络的 通/阻

### Y-13: 表 / 裡 (医家专版)
- **签名**: Body → Layer
- **源**: 素问·至真要大论「夫五脏六腑寒热相移者...病在阳谓之表，病在阴谓之里」
- **读**: 病位的内/外定位
- **注**: 与 C-7/C-8 (表里) 同形。但医家**表里**是与**气流方向**绑定的：邪从表入里，治法逆之

### Y-14: 寒 / 熱 (热)
- **签名**: System → Thermal State
- **源**: 通用；素问·阴阳应象大论
- **读**: 系统的热态轴
- **注**: **热 = 阳过盛 / 阴不足**；**寒 = 阴过盛 / 阳不足**。注意寒热不独立于阴阳，是 **阴阳的 manifestation in thermal axis**

### Y-15: 虛 / 實 (虚/实)
- **签名**: System → Capacity State
- **源**: 素问·通评虚实论「邪气盛则实，精气夺则虚」
- **读**: 系统的丰度态
- **注**: **虚 = 正气不足；实 = 邪气有余**。注意：实不一定坏，虚不一定好 — 视邪正而定

#### **四诊轴 = 8-orthant phase space**

阴/阳 + 表/里 + 寒/热 + 虚/实 = 4 binary axes = $2^4 = 16$ 状态。但实际医家用 **8 大类**因为 阴/阳 与其余三轴是**整体框架而非独立轴**：

```
                 阳
                ╱╲
              热    实
            ╱        ╲
         表           里
            ╲        ╱
              寒    虚
                ╲╱
                 阴
```

每病在此 8-orthant phase space 上有一个定位。治法按定位反向干预 (虚补、实泻、寒温、热凉)。**这是早期的 phase-space-guided control system**

### Y-16: 補 / 補 (补) 与 瀉 / 泻
- **签名**: State × Direction → State'
- **源**: 灵枢·终始；素问·针解
- **读**: 干预算子 — 补加 / 泻减
- **注**: 是 **PID controller** 的早期版本：测量偏差 (虚/实) → 反向干预 (补/泻) → 趋向平衡。形式: 
  $$\text{补}: \text{虚}(A) \to A^+, \quad \text{泻}: \text{实}(A) \to A^-$$

### Y-17: 平
- **签名**: System → System (target state)
- **源**: 素问·三部九候论「无问其病，以平为期」
- **读**: 治疗目标 — 把系统调到 balanced state
- **注**: **不是数学的零或最小，是 dynamic equilibrium**。平 是 attractor，不是 fixed value

### Y-18: 和
- **签名**: System → System (动态调和)
- **源**: 素问·上古天真论「外不劳形于事，内无思想之患，以恬愉为务，以自得为功，形体不敝，精神不散」释「和」
- **读**: 动态协调 — 让多个流动组件协同
- **注**: 与 平 的区别：**平是状态目标，和是过程**。和 是 trajectory through phase space; 平 是 destination

### Y-19: 調 / 调
- **签名**: Parameter × State → State'
- **源**: 灵枢·调经论
- **读**: 调节 — fine-tune parameters
- **注**: 比 补/泻 更精细 — 是连续调节而非二元增减。是**比例控制**

### Y-20: 順 / 逆 (顺/逆)
- **签名**: Action × Flow → {同向, 反向}
- **源**: 素问·四气调神大论「逆春气则少阳不生，肝气内变」；「顺四时而适寒暑」
- **读**: 与流动方向的关系
- **注**: 顺 = 顺势 (与势同向)；逆 = 逆势 (与势反向)。**治法常需逆症 (反symptom方向) 但顺势 (与人体自愈方向同向)** — 这是细微但重要的区分

### Y-21: 標 / 標 (标) 与 本
- **签名**: Phenomenon → (Symptom, Cause)
- **源**: 素问·标本病传论
- **读**: 表征 (标) vs 根因 (本)
- **注**: **诊断算子** — 区分症状 (标) 与病因 (本)。"治病必求于本" 是医家的 root-cause-analysis 律

### Y-22: 營 / 营 与 衛 / 卫
- **签名**: 气 → 气_internal × 气_external
- **源**: 灵枢·营卫生会
- **读**: 气的内向 (营/营养) vs 外向 (卫/防御) 分化
- **注**: 营卫是 **资源分配** — 系统总能量在内务和外防之间分配。这是**内部稳态 vs 外部应激响应**的早期建模

### Y-23: 通 / 滯 (通/滞)
- **签名**: Flow → {通畅, 阻滞}
- **源**: 素问·六微旨大论；金匮要略「不通则痛」
- **读**: 流动性诊断
- **注**: 通是健康；滞是病理。治疗常常是 **去除滞** = 重启动 flow。等价于 deadlock detection + recovery

### Y-24: 五运六气
- **签名**: Time → State (天文驱动的状态序列)
- **源**: 素问·天元纪大论 (七篇大论)
- **读**: 把时间映射到 5×6 = 30 个状态周期
- **注**: 是**外部驱动的 phase shift**。系统不仅有内部动力学，还由 时间-天文 周期驱动

### Y-25: 上工 / 中工 / 下工
- **签名**: 介入时机 → 干预层级
- **源**: 灵枢·逆顺「上工治未病，中工治欲病，下工治已病」；素问·四气调神大论
- **读**: 三层干预时序
- **注**: 时序算子的优先级 — **早干预 > 中干预 > 晚干预**。形式: 干预效用 ∝ 1/disease_progress。这是预防医学的算子化

### 医家算子的代数结构

整个内经系统是一个**闭合的状态空间动力学**：

```
  外 (天:五运六气)
    ↓ 驱动
  系统状态: (阴/阳, 表/里, 寒/热, 虚/实)
    ↓ 流动: 升降出入 + 经络
  气/血/精/神 (substance + flow + reservoir + coherence)
    ↓ 失衡时
  病: 标 (manifestation) ← 本 (root cause)
    ↓ 干预: 顺/逆 + 补/泻 + 调
  目标: 平 / 和 (balanced trajectory)
```

**核心方程** (隐含):
- **守恒**: 气血总量近守恒 (排泄 + 摄入)
- **流动**: 升降出入 = 4-direction flow (gradient field)
- **平衡**: 阴 + 阳 = constant (zero-sum on each axis)
- **耦合**: 五行通过相生相克实现 5×5 coupling matrix

| 气 | message flow / data stream |

医家提供的：**完整的系统健康监测与干预语言** — 状态空间、流动方程、平衡条件、扰动响应、干预算子俱全。

## 二十、社会算子 — 荀子的 化性 / 伪 / 群 / 分

荀子是先秦思想里**最直接讨论"社会怎么从个体涌现"的**。他的算子集是 **emergent collective formation algebra**。

### X-1: 性
- **签名**: Person → Initial-State
- **源**: 荀子·性恶「人之性恶，其善者伪也」；正名「生之所以然者谓之性」
- **读**: agent 的 initial / unmodified state
- **注**: 与儒家其他 (孟子) 不同 — 荀子的 性 是 **未受 socialization 的 raw state**。这是 **boot-time configuration** 的早期概念

### X-2: 偽 / 伪
- **签名**: Action × Person → Person'
- **源**: 荀子·性恶「不可学、不可事而在人者，谓之性；可学而能、可事而成之在人者，谓之伪」；正名「心虑而能为之动，谓之伪」
- **读**: 经心虑而后做的、经训练而成的、人为的构造
- **注**: **关键算子** — "伪" 不是欺骗，是 *artificial* (与 *natural* 对偶)。是 **constructed / engineered behavior**。形式上：

  $$\text{伪}: \text{Person}_\text{raw} \times \text{Training} \to \text{Person}_\text{shaped}$$
  
  这是 **education / programming** 的算子。性恶论的核心: 性 (initial) 不可恃，必赖 伪 (shaped)

### X-3: 化性 (起伪)
- **签名**: 性 → 性' (transformed by 伪)
- **源**: 荀子·性恶「故圣人化性而起伪」
- **读**: 通过 伪 来变换 性 — 复合算子: 性 × 伪 → Person_shaped
- **注**: **形式上**: 化性 = 化 (T-1) 应用于 性 用 伪 作为 catalyst。
  
  $$\text{化性起伪} = \text{化}_\text{伪}(\text{性})$$
  
  这是社会化整个过程的 **single-operator description**

### X-4: 群
- **签名**: [Person] → Collective
- **源**: 荀子·王制「人能群，彼不能群也」；「故人生不能无群」
- **读**: 把多个 agent 凝聚成集体的算子
- **注**: 是 **collective formation operator**。荀子: 人之异于禽兽在能群。即 **social aggregation function**

### X-5: 分 (荀子社会版)
- **签名**: Collective → Differentiated Roles
- **源**: 荀子·王制「群而无分则争，争则乱」；富国「明分使群」
- **读**: **群** 内部的差异化算子 — 把同质集合分成 hierarchical roles
- **注**: 与 I-6 (一般 分) 不同 — **此分是社会分工/角色分化**。荀子的关键洞察：
  
  $$\text{群} \text{ requires } \text{分} \text{ to avoid } \text{争}$$
  
  形式: collective without role differentiation → conflict. 必须 type-stratify

### X-6: 礼
- **签名**: 分 → Protocol
- **源**: 荀子·礼论「礼者，养也」；「礼起于何也？曰：人生而有欲...故制礼义以分之」
- **读**: 把 分 (角色差异) 编码为可执行的协议
- **注**: 礼 = **social protocol**。是 分 的形式化输出。形式上：
  
  $$\text{礼}: \text{分}(\text{群}) \to \text{Protocol}$$

### X-7: 義 / 义
- **签名**: Context × Action → Appropriate Action
- **源**: 荀子·王制「分均则不偏，势齐则不壹，众齐则不使。有天有地而上下有差，明王始立而处国有制。夫两贵之不能相事，两贱之不能相使，是天数也。势位齐而欲恶同，物不能澹则必争」
- **读**: 在具体情境下的 适当行动
- **注**: 义 是 礼 的 contextual application — 在特定 situation 下的 right action selection. 类似 **case-specific dispatch**

### X-8: 別 / 别 (荀子版)
- **签名**: Population → Stratified Population
- **源**: 礼论「贵贱有等，长幼有差，贫富轻重皆有称者也」
- **读**: 给定标准上的等级化划分
- **注**: 与 N-8 / G-9 (一般 別) 联系。荀子 别 是社会分层的**严格版** — 多维同时分层 (贵贱、长幼、贫富)

### X-9: 制 (荀子社会版)
- **签名**: Need → Institution
- **源**: 礼论「先王恶其乱也，故制礼义以分之」
- **读**: **设计 / 设立** 制度以满足需要
- **注**: 与 L-14 (法家 制) 联系，但荀子 制 强调**主动设计**而非凝固模式。是 **institutional design** 算子

### X-10: 隆 / 殺
- **签名**: Practice × Importance → Adjusted Practice
- **源**: 荀子·礼论「故文饰、麄恶，声乐、哭泣，恬愉、忧戚，是反也。然而礼兼而用之，时举而代御，故文饰、麄恶，声乐、哭泣，恬愉、忧戚...」
- **读**: 隆 = 提高、加重；殺 (shai) = 降低、减杀
- **注**: 是 **rate / weight 调整算子**。同一礼 在不同情境隆 vs 殺。这是 **continuous parameter** in social protocols

### X-11: 化 (荀子版)
- **签名**: Person × Environment × Time → Person'
- **源**: 荀子·劝学「蓬生麻中，不扶而直；白沙在涅，与之俱黑」；性恶
- **读**: 持续环境作用下的渐变
- **注**: 与 T-1 (一般化) 联系，荀子 化 强调**环境驱动的不可避免变化**。这是 **environmental conditioning operator** — 与现代行为主义/操作性条件作用相通

### X-12: 名分
- **签名**: Person → (Name × Role × Privileges)
- **源**: 荀子·正名；春秋公羊学「名分」
- **读**: 名 (G-3) + 分 (X-5) 的复合 — 给 agent 一个 (identity, role, scope) 三元组
- **注**: 是**完整的 agent capability descriptor**。不只是名字，包含 role 和 permissions

### X-13: 解 (蔽)
- **签名**: Belief × Bias → Belief'
- **源**: 荀子·解蔽「凡人之患，蔽于一曲而暗于大理」
- **读**: 解除 cognitive bias / partial view
- **注**: 是 **debiasing operator**。荀子认为人易**蔽** (被局部观点束缚)，需 解 来恢复 holistic view。是 **epistemic operator** — 调整认知状态

### X-14: 積 / 积
- **签名**: [Action] × Time → Capacity
- **源**: 荀子·劝学「积土成山，风雨兴焉；积水成渊，蛟龙生焉；积善成德，而神明自得，圣心备焉」
- **读**: 持续累积产生质变
- **注**: **小变积累 → 大质变** 算子。是 化 (T-1) 的离散版本: 多次小 step → 涌现新性质. 类似 phase transition

### 荀子算子的代数结构

**化性起伪 → 群分礼义** 是一条完整 social emergence chain:

```
个体 (性 raw)
    ↓ 化 by 伪 (training/socialization)
个体' (shaped)
    ↓ 聚 (aggregate)
群 (collective)
    ↓ 分 (differentiate roles)  避免 争
分化群 (stratified)
    ↓ 礼 (protocolize)
社会秩序 (with explicit protocols)
    ↓ 义 (situational application)
具体行为 (appropriate per context)
```

**荀子等式**:

$$\text{社会} = \text{礼}\big(\text{分}(\text{群}(\text{化性起伪}(\text{性})))\big)$$

读法：原始性 → 经训练成为社会化个体 → 聚成群 → 分工分级 → 礼制规范。

### 与 multi-agent 系统设计

| 荀子算子 | Multi-Agent 对应 |
|---|---|
| 性 | initial agent state / boot config |
| 伪 | training / fine-tuning / programming |
| 化性 | full agent socialization |
| 群 | aggregate / spawn collective |
| 分 | role assignment / specialization |
| 礼 | protocol / API contract |
| 义 | context-aware dispatch |
| 别 | stratification / privilege levels |
| 制 | system design / mechanism creation |
| 隆/殺 | weight tuning / priority adjustment |
| 化 (env) | environmental conditioning / continual learning |
| 名分 | capability descriptor (id + role + scope) |
| 解蔽 | bias correction / view broadening |
| 積 | continual accumulation → emergent capacity |

荀子提供的：**individual agent → collective formation 的完整算子链** — 从原始个体经过训练、聚集、分工、协议、情境化的全流程算子序列。

## 二十一、补遗算子 — 此前遗漏的核心算子

继续找算子。这一节是从前面文本 / 常见用法识别的、之前类目里没明确列出的算子。

### Z-1: 因 (一般)
- **签名**: Cause × Event → Effect
- **源**: 通用；老子「夫物芸芸，各复归其根」释暗含因；庄子·养生主「因其固然」
- **读**: (a) 因为 (b) 依凭 (c) 顺应
- **注**: **三义** — 一般认为是因果链的 cause；实际古文中常作 "依凭/顺应" 解。是 leverage operator

### Z-2: 相 (mutuality prefix)
- **签名**: Op → Op_mutual
- **源**: 通用；论语「相人偶」；相生、相克
- **读**: 把单向算子转为双向 — 是**算子上的 modifier**
- **注**: 非常重要的 **higher-order operator**。改变算子的方向性:
  - 见 → 相见 (mutual seeing)
  - 助 → 相助 (mutual help)
  - 生 → 相生 (mutual generation)
  
  形式: 相(f: A → B) := f × f^op : A × B → A × B 同时运行

### Z-3: 互
- **签名**: Op → Op_reciprocal
- **源**: 通用；春秋三传「互见」
- **读**: 类似 相 — 双向化
- **注**: 比 相 更强调**信息/属性的交换**而非动作的对称

### Z-4: 交
- **签名**: A × B → A↔B (交换关系)
- **源**: 易·泰「天地交而万物通」；论语「与朋友交」
- **读**: 交叉、交换、交往
- **注**: 是**信道建立**算子 — 在两个对象间建立双向通道

### Z-5: 錯 / 错
- **签名**: A × A → Interleaved
- **源**: 易·错卦理论；尚书「五行错落」
- **读**: 交错排列
- **注**: 比 交 更细 — 不只通道，是 **interleaving**。在易卦中，错 = 全爻反转 (Z/2 negation on 64 卦)

### Z-6: 綜 / 综
- **签名**: A → A^reverse
- **源**: 易·杂卦传隐含；通用
- **读**: 综合、倒置
- **注**: 在易卦中，综 = 卦序倒读。所以 错综 是 (Z/2 × Z/2) 作用于 64 卦上 — 形成 4 个对称类

### Z-7: 雜 / 杂
- **签名**: [A] → Mixed
- **源**: 易·杂卦传；荀子「群居和一」释「杂」
- **读**: 混合
- **注**: 是 mixing operator — 把多个不同元素同地放置

### Z-8: 隱 / 隐
- **签名**: A → ∅ (但仍存在)
- **源**: 通用；老子「大方无隅，大器晚成，大音希声，大象无形，道隐无名」
- **读**: 隐藏 — 存在但不可见
- **注**: 与 显 对偶。**关键**: 隐 ≠ 不存在 — 是 *invisible existence*

### Z-9: 顯 / 显 (見/见 - xiàn 读)
- **签名**: ∅_hidden → A (manifest)
- **源**: 通用；中庸「莫见乎隐，莫显乎微」
- **读**: 显现 — 从隐到现
- **注**: 与 隐 对偶。形式: 显 = 隐⁻¹

### Z-10: 藏
- **签名**: A → Reservoir
- **源**: 通用；庄子「藏天下于天下」；周易系辞「形而上者谓之道，形而下者谓之器」释「藏」
- **读**: 储藏 — 把对象放入 reservoir
- **注**: 是 storage operator. 与 Y-8 (精) 联系: 精 = 已 藏 之 气

### Z-11: 露
- **签名**: A_hidden → A_visible
- **源**: 通用
- **读**: 暴露
- **注**: 与 藏 对偶；与 显 微差 — 露 是被动显现，显 是主动彰显

### Z-12: 守
- **签名**: A → A (over time)
- **源**: 通用；老子「致虚极，守静笃」；论语「守死善道」
- **读**: 持守 — 保持当前状态不变
- **注**: 是**身份保持算子** (identity-preserving over time). 时间维度上的稳定性

### Z-13: 持
- **签名**: A → A (active hold)
- **源**: 通用；论语「中庸之为德也，其至矣乎」释「持」
- **读**: 把持 — 主动保持
- **注**: 比 守 更主动 — 守是不变，持是积极维护

### Z-14: 居
- **签名**: A × Place → State
- **源**: 通用；论语「居处恭」；老子「水善利万物而不争，处众人之所恶」
- **读**: 居于 — 在某 state/place 中长期 dwell
- **注**: 是**长时间 located-at**。比 在 (R-1) 时间维度更长

### Z-15: 處 / 处
- **签名**: A × Situation → State
- **源**: 通用；老子；论语
- **读**: 处于 — 在某 situation 中应对
- **注**: 与 居 微差: 居 是 dwelling，处 是 dealing-with

### Z-16: 聚
- **签名**: [A] → A_aggregated
- **源**: 易·萃卦「萃，聚也」；通用
- **读**: 聚集
- **注**: 与 散 对偶。是 X-4 (群) 的更基础形式 — 群是 聚 of agents

### Z-17: 集
- **签名**: 同 聚
- **源**: 通用；论语「集大成」
- **读**: 收集
- **注**: 与 聚 几近同义；集 偏 集合论 collection

### Z-18: 散
- **签名**: A_aggregated → [A]
- **源**: 通用；老子「朴散则为器」
- **读**: 分散
- **注**: 与 聚 对偶。**老子的"朴散"即 monoid → 多元素**

### Z-19: 積 / 积
- **签名**: [Action] × Time → Capacity
- **源**: 已有 X-14 (荀子) — 但更广泛存在
- **读**: 积累
- **注**: 是 monoid 的 fold operation: foldr(+, 0, [a₁, ..., aₙ])

### Z-20: 引
- **签名**: A × Direction → A_pulled
- **源**: 通用；孟子「引而不发」
- **读**: 拉、引导
- **注**: 与 推 对偶 — 推 (T-10) 是 push，引是 pull

### Z-21: 攝 / 摄
- **签名**: [A] → A_unified
- **源**: 通用；中庸「夫君子之道，闇然而日章」释「摄」
- **读**: 摄取、统摄、归摄
- **注**: 比 聚 更主动 — 摄是 active unification under one principle

### Z-22: 蘊 / 蕴
- **签名**: A → A (含未发)
- **源**: 易·系辞「成象之谓乾，效法之谓坤」释「蕴」；尔雅「蕴，藏也」
- **读**: 蕴含 — 内有但未现
- **注**: 与 隐 微差 — 蕴是 *latent contained*; 隐是 hidden present

### Z-23: 萌
- **签名**: ∅ → Seed
- **源**: 通用；易·复卦释「萌」
- **读**: 萌发 — 从无到微有
- **注**: 是 emergence operator at boundary 0→ε

### Z-24: 兆
- **签名**: A → Sign
- **源**: 通用；尚书「卜兆」；中庸「征兆」
- **读**: 兆现 — 出现征象
- **注**: 是 indicator generator — 系统将变之前出现的 sign

### Z-25: 苗
- **签名**: Seed × Time → Sprouting
- **源**: 论语「苗而不秀者有矣夫」
- **读**: 苗 = 中间发展态
- **注**: 萌→苗→秀→实 是发展四阶段算子链

### Z-26: 占
- **签名**: System → Information (query)
- **源**: 易·系辞「以卜筮者尚其占」
- **读**: 占卜 — 查询当前/未来状态
- **注**: 是 **read-only query operator**. 不改变系统，只获取信息

### Z-27: 卜
- **签名**: 同 占
- **源**: 通用
- **读**: 卜 — 用龟甲查询
- **注**: 与 占 共同构成查询家族

### Z-28: 演
- **签名**: A × Time → A_evolved
- **源**: 易·系辞「极其数遂定天下之象」释「演」；汉「演天地之数」
- **读**: 演化 — 沿系统内部规律展开
- **注**: 与 化 (T-1) 微差 — 演 强调**遵循规则的展开**，化 强调**质态改变**。演接近 deterministic time evolution

### Z-29: 推 (复用 T-10) — 算子上的推
- 算子 推 可以接受**算子作参数** — 把一个算子从局部域扩展到全局
- 例: 孟子 "推恩" — 推(恩, 家) → 推(恩, 邻) → 推(恩, 天下)

### Z-30: 譬 / 喻
- **签名**: A → B (类比映射)
- **源**: 通用；荀子·解蔽；公孙龙「白马」实是 譬
- **读**: 类比 — 通过相似性建立映射
- **注**: 是 **analogy operator**. 形式: 譬(A, X) := find B such that A: X :: B: ?

### Z-31: 反 (作算子上的算子)
- **签名**: Op → Op^(-1)
- **源**: 论语「反求诸己」；老子「反者道之动」
- **读**: 把任何算子取其反向
- **注**: 反 不只是状态算子 (T-6)，也是 **算子上的 functor** — 任何算子 f, 反f = f⁻¹ 在结构上对偶

### Z-32: 自
- **签名**: Op × A → Op(A, A)
- **源**: 通用；论语「君子求诸己」；大学「自天子以至于庶人，壹是皆以修身为本」
- **读**: 自身、自动、由自己
- **注**: 是 **reflexive prefix** — 把 binary 算子的两个 argument 同时设为 self。形式: 自f(A) := f(A, A)

### Z-33: 自反 / 反自 (合成)
- **签名**: A → reflective Op on A
- **源**: 大学「君子有诸己而后求诸人，无诸己而后非诸人」；中庸「反求诸身」
- **读**: 反过来作用于自身
- **注**: 是 **reflection / metacognition operator**. self ∘ reverse(op)

### Z-34: 觀 / 观
- **签名**: A → Information about A
- **源**: 易·观卦；论语「观其所由」
- **读**: 观察 — 信息提取
- **注**: 与 占/卜 联系；观 是 active observation, 占 是 ritual query

### Z-35: 察
- **签名**: 同观但更细致
- **源**: 通用；大学「察其义」
- **读**: 详察
- **注**: 比 观 精细 — 察是 fine-grained inspection

### Z-36: 明
- **签名**: A → A_understood
- **源**: 通用；大学「在明明德」
- **读**: 明白 — 把模糊变清晰
- **注**: 不只光学的"明"，是 **clarification operator**. 双重 明 (明明德) 是 self-application: clarify the (already-clarified)

### Z-37: 蔽
- **签名**: A_clear → A_obscured
- **源**: 荀子·解蔽
- **读**: 遮蔽 — 与 明 对偶
- **注**: 与 X-13 (解) 对偶

### Z-38: 悟
- **签名**: A_unintegrated → A_integrated
- **源**: 通用；佛家文献频繁
- **读**: 领悟 — 突然性的 insight
- **注**: 是 **non-continuous understanding operator** — 不是渐进，是 phase transition. 与 化 (continuous) 对偶

### Z-39: 識 / 识
- **签名**: A → Recognized A
- **源**: 通用；论语「君子识大」
- **读**: 识别
- **注**: 是**模式识别**算子 — pattern matching against known categories

### Z-40: 知 (作算子)
- **签名**: Subject × Object → Knowledge
- **源**: 论语；墨经 P-20 已部分覆盖
- **读**: 一般知识获取
- **注**: 与 P-20 互补 — P-20 是分类，此处是基础算子

# 第二部 — 代数分析 (Analysis)

## 二十二、算子组合规律 (基础)

### 1. 对偶配对
许多算子天然成对，且常组合使用：

| 算子对 | 关系 |
|---|---|
| 動 / 靜 | 状态对偶 |
| 屈 / 伸 | 收缩/展开 |
| 損 / 益 | 减/增 |
| 進 / 退 | 步进对偶 |
| 升 / 降 | 垂直对偶 |
| 出 / 入 | 边界对偶 |
| 往 / 來 | 方向对偶 |
| 始 / 終 | 时间对偶 |
| 本 / 末 | 因果根叶 |
| 中 / 邊 | 中心/周边 |
| 內 / 外 | 内/外 |
| 表 / 裡 | 表/里 |
| 體 / 用 | 静/动 |
| 形 / 神 | 形/神 |
| 勢 / 機 | 趋势/节点 |
| 同 / 異 | 一致/差异 |
| 合 / 分 | 聚/散 |
| 有 / 無 | 存在/不存 |
| 為 / 無為 | 行/不行 |
| 名 / 實 | 符号/对象 |

这构成 **天然的 involution 群**：每对都是 σ² = id 的二元对偶。

### 2. 序列组合
经典文本反复使用算子链：

- 大学：**格 → 致 → 誠 → 正 → 修 → 齊 → 治 → 平**（八条目本身是算子序列）
- 老子：**反 → 復 → 歸**（return chain）
- 易：**極 → 反**（极必反 — 极值算子触发反算子）
- 易：**動 → 變 → 化 → 通**（动力学链）

### 3. 高阶应用
有些算子接收**算子作参数**：

- 使 X (X = action) — apply action operator
- 為 X (X = transformation) — apply transformation
- 推 X — extend X to broader domain
- 致 X — push X to limit

这些是 higher-order combinators。

### 4. 极/反律 (老子算子律)
**T-6 (反) ∘ B-7 (极) = id**
"物极必反" 在算子层是：把状态推到极值点 (极) 然后取反 (反)，回到对偶。这是 老子 哲学的核心算子等式。

### 5. 损益对偶律
**T-12 (損) ∘ T-13 (益) ≠ id 一般情况**
但在某些守恒律下（"损上益下"等）成立。这是中国哲学版的 守恒/对称性 思考。

### 6. 体用律
**对任意 A，存在 体(A) 与 用(A)，且 用 = 体 的某种 elaboration**
这是 specification/implementation 配对的中国版本。

## 二十三、进一步算子组合规律 (流派链)

### 7. 大学八条目 = 算子流水线 (operator pipeline)

```
格 → 致 → 誠 → 正 → 修 → 齊 → 治 → 平
↓     ↓     ↓     ↓     ↓     ↓     ↓     ↓
物    知    意    心    身    家    國    天下
```

每个算子提升一个**作用域 (scope)** 层级。形式上：

$$\text{平}(\text{治}(\text{齊}(\text{修}(\text{正}(\text{誠}(\text{致}(\text{格}(\text{物}))))))))$$

这是**作用域跨越的递归 functor 链**。每层算子在自己 scope 上是相同的"端正"动作，但 scope 变了。

DSL 启示：这是 monadic chain，每个算子是 `M a → M a` for a series of expanding monads M_物 ⊆ M_家 ⊆ M_天下。

### 8. 易传动力学链

```
動 ─→ 變 ─→ 化 ─→ 通
discrete onset → discrete change → continuous transformation → connection
```

这是 **discrete → continuous → topological** 的算子升级链。

### 9. 老子的回归链

```
極 ─→ 反 ─→ 復 ─→ 歸 ─→ 根
extreme → reverse → return → return-to → root
```

终态吸引子的算子链。每一个增强了"返回"的语义负载。

### 10. 名家的指称链

```
物 ─→ 名 ─→ 指 ─→ 實 ─→ 位
object → name → designate → actualize → locate
```

是符号到对象到位置的 **semantic deployment chain**。

### 11. 墨经的几何构造链

```
端 ─→ 尺 ─→ 平面 ─→ 圜
0-dim → 1-dim → 2-dim → 圆/曲面
```

这是从 0 维点到 1 维线到 2 维面的**维度提升算子链**。墨经的几何代数已经隐含了"维度"的概念。

### 12. 时空对偶 (墨经)

**久** (P-6) 与 **宇** (P-7) 是严格对偶 — 时间与空间的算子化。它们的乘积构成 **时空** (久宇)。这是相对论之前两千年的时空本体论。

## 二十四、最小完备子集 (Minimal Complete Subset)

在所有 ~150 个算子中，哪些是**不可还原的原子算子**？哪些是其他算子的**组合**？

**提议的最小完备集**（约 18-24 个）:

### 逻辑核心 (5 个)
- 不 (¬)
- 並 (∧)
- 或 (∨)
- 凡 (∀)
- 故 (⊢)

(其他逻辑算子皆可由此 5 个定义：→ = ¬∨, ↔ = ∧[→,←], ∃ = ¬∀¬, ¬∃ = 莫, etc.)

### 关系核心 (4 个)
- 在 (locative ∈)
- 同 (=)
- 含 (⊇)
- 際 (boundary)

(其他关系算子可由此组合：屬 = 在·类, 比 = 際 + 同位, 應 = 距离 + 同极性, etc.)

### 变换核心 (3 个)
- 化 (continuous transform)
- 反 (involution)
- 化⁻¹ = 復 (return)

(变 = 化 of small step; 易 = 反 + swap; 革 = 化 of large step)

### 流动核心 (3 个)
- 動 (initiate motion)
- 行 (continue)
- 止 (halt)

(其他运动算子是这三的方向化版本：进 = 行 + 正向, 升 = 行 + 上向, 出 = 行 + 越界)

### 边界核心 (2 个)
- 始 (begin)
- 終 (end)

(立 = 始 + 稳, 成 = 終 + 完, 极 = 終 + 极值)

### 度量核心 (2 个)
- 一 (unify, also unit)
- 分 (split)

(合 = 一⁻¹ ∘ 一 of separate parts, 体/兼 由此衍生, 数字皆可由此构造)

### 元层核心 (2 个)
- 名 (designate)
- 体/用 (specification/implementation pair, 算作 1 因为对偶)

(指 = 名 + reference, 法 = 用 + 抽象)

### 模态核心 (1 个)
- 必 (□)

(可 = ¬□¬, 或 = ◇ = ¬□¬)

### 序贯核心 (2 个)
- 而 (sequence)
- 則 (implication, can be defined as ¬X ∨ Y but kept for ergonomics)

**总计: ~22 个原子算子**

这就是文言文操作系统的 **kernel**。其余 ~125 个是这些的**应用、方向化、强度化、域特化**的衍生形式。

> 22 个核心算子 + 124 个衍生算子 ≈ 146 — 几乎正好对应 间 框架的 146 字 vocabulary 目标。

## 二十五、参数化与降维 — 形式归约 (Parameterization)

很多算子表面独立，实际是某个**核心算子 + 参数**。承认参数化后，原子算子集大幅收缩。这一节系统识别可归约的算子簇。

### 18.1 方向化簇 — 都是 行 (F-9) + Direction

| 算子 | 等价于 | 参数 |
|---|---|---|
| 升 | 行 + Vec(↑) | 垂直正向 |
| 降 | 行 + Vec(↓) | 垂直反向 |
| 進 | 行 + Vec(forward) | 路径正向 |
| 退 | 行 + Vec(back) | 路径反向 |
| 出 | 行 + cross(boundary, out) | 跨界外向 |
| 入 | 行 + cross(boundary, in) | 跨界内向 |
| 往 | 行 + Vec(away-from-self) | 离心 |
| 來 | 行 + Vec(toward-self) | 向心 |
| 趨 | 行 + Vec(toward-target) | 趋向 |
| 避 | 行 + Vec(away-from-target) | 远离 |

**收缩**: 10 个算子 → **1 个算子 (行) + 1 个 Vec 参数空间**。

形式：

$$\text{行} : A \times \text{Vec} \to A$$

### 18.2 强度化簇 — 都是 化 (T-1) + Rate

| 算子 | 等价于 | 参数 |
|---|---|---|
| 漸 | 化 + Rate(slow) | 缓 |
| 寖 | 化 + Rate(very-slow) | 极缓 |
| 驟 | 化 + Rate(fast) | 快 |
| 忽 | 化 + Rate(instantaneous) | 瞬 |
| 革 | 化 + Rate(maximal) + scope(global) | 极速且全 |
| 變 | 化 + discrete-step | 离散一步 |

**收缩**: 6 个算子 → **1 个算子 (化) + Rate 参数 + 离散标记**。

$$\text{化} : A \times \text{Rate} \to A'$$

### 18.3 重复化簇 — 都是 復 (T-7) + Count

| 算子 | 等价于 | 参数 |
|---|---|---|
| 又 | 復 × 1 | 一次重复 |
| 再 | 復 × 1 (有序版) | 第二次 |
| 三 (作算子) | 復 × 3 (or 多) | 三/多次 |
| 屢 | 復 × N (N>3) | 多次 |
| 復 之又 復 | 復 × ω | 无限次 (→ 不动点, 见 §19) |

**收缩**: 5 → **1 个算子 (復) + Count**。

$$\text{復} : (A \to A) \times \mathbb{N} \to (A \to A)$$

### 18.4 量级化簇 — 都是 度量算子 + Magnitude

| 簇 | 核心 | 大版 / 小版 |
|---|---|---|
| 過 | 過 (excess) | 大過 / 小過 (易卦 28, 62) |
| 畜 | 畜 (accumulate) | 大畜 / 小畜 (易卦 26, 9) |
| 損益 | 損 / 益 | 损之又损 / 损少 (degree) |

**收缩**: 量级是参数，不是新算子。

### 18.5 范围化簇 — 都是 名 (H-7) + Scope

| 算子 | 等价于 | 参数 |
|---|---|---|
| 達名 | 名 + scope(universal) | 通名 |
| 類名 | 名 + scope(sortal) | 类名 |
| 私名 | 名 + scope(individual) | 专名 |

(墨经 P-19 已经做了这个分解 — 我们只是承认它就是参数化。)

### 18.6 视角化簇 — 都是 知 (P-20) + Source

类似的，知 = 闻知 / 说知 / 亲知 = 知 + Source 参数。

### 18.7 约简后的真正核心

参数化把 ~150 算子降到约 **35-40 个真正不可约的核心算子** + 一组参数空间 (Direction, Rate, Count, Scope, Magnitude, Source)。

修订后的最小核 (草拟)：

**逻辑** (5): 不、並、或、凡、故  
**关系** (4): 在、同、含、際  
**变换** (3): 化、反、易  
**流动** (2): 行、止 (升降进退出入往来皆 行 + Direction)  
**边界** (2): 始、終  
**度量** (2): 一、分  
**重复** (1): 復 (含 Y-iteration)  
**元层** (3): 名、體、用  
**模态** (2): 必、可  
**序贯** (2): 而、則  
**史官** (1): 書 (含 削 / 諱 / 譏 / 與 都是 書 + Mode)

**总核**: ~27 算子 + 6 参数空间。

这是文言文操作系统的最小内核。

### 18.8 参数空间的总览

```
Vec       :: 方向向量 — 6+ 基方向，可叠加

```

## 二十六、不动点 / 递归算子 (Fixed-points and Recursion)

文言文有显式的**不动点表达模式**和**原始递归**结构。这一节抽出这些模式并形式化。

### 19.1 "X 之又 X" 模式 = 迭代算子

经典实例：

- 老子: 「**损之又损**，以至于无为」 — iterate(损) → 无为
- 大学: 「**日日新，又日新**」 — iterate(新) (持续无止)
- 老子: 「**玄之又玄**，众妙之门」 — iterate(玄) → 道
- 大学: 「**苟日新，日日新，又日新**」 — iterate(新) ω-many times
- 庄子: 「**化而又化**，何为以异？」 — iterate(化)
- 易传: 「**生生**之谓易」 — iterate(生)，self-applicative

**形式化**: "X 之又 X" 是 fixed-point combinator 的 surface syntax:

$$\text{X} \text{ 之又 } \text{X} \;\equiv\; \mathsf{iter}(X) = \mu f. X \circ f$$

或更精确，含终止条件:

$$\text{X 之又 X 以至于 Y} \;\equiv\; \mathsf{iterUntil}(X, Y) = \mu s.\, \text{if } s = Y \text{ then } s \text{ else } X(s)$$

### 19.2 "以至于" — 收敛标记 / 终止条件

老子的 "**损之又损以至于无为**" 包含三个算子:
1. **损** — the operation
2. **之又** — iteration combinator
3. **以至于 Y** — convergence target (Y as fixed point)

`以至于` 是文言文的 **stopping condition operator**。其签名:

$$\text{以至于} : \text{State} \to \text{Predicate} \to \text{TerminationCriterion}$$

### 19.3 "生生" — 自指 / 自应用

"生生之谓易" — 生 self-applies。

$$\text{生生} \equiv \text{生}(\text{生}) \equiv \lambda x. \text{生}(\text{生}(x))$$

这是**自指算子** (self-application) — 不是简单 iteration，是 *creation that creates more creation*。计算上：

$$\text{生生} = \text{fix}(\text{生}) = \text{Y}(\text{生})$$

如果 生 是创造算子，则 生生 是 Y 子的实例。"生生不息" 直接对应非终止的 Y(生) 求值。

### 19.4 "反" 自映射 = 标准 involution

"反者道之动" 中的反作算子有 σ² = id：

$$\text{反} \circ \text{反} = \text{id}$$

但 "反复其道" (易·复卦) 是不一样的 — **反复** = 反 ∘ 復 ≠ 反 ∘ 反。

$$\text{反復} : \text{走出去再回来} \neq \text{反 之 反}$$

文言文区分: 反復 = round-trip; 反 之 反 = double-negation (回到原点)。这两者在文言文里是分开标记的。

### 19.5 "归根" — 终态吸引子

老子: 「夫物芸芸，各复归其根」

"**根**" = 不动点 (fixed point of 復)。"**归根**" = ascend to fixed point。

$$\text{归根}(x) = \lim_{n \to \infty} \text{復}^n(x)$$

每个轨迹都向其根收敛 — 这是文言文的 **ω-limit / attractor** 概念。

### 19.6 大学八条目作为 fold

回顾 §7：格→致→誠→正→修→齊→治→平 是 monadic chain。

但更精确的是 **fold/scan** 模式:

$$\text{平}(\text{天下}) = \text{scan}(F, \text{物}, [\text{格,致,誠,正,修,齊,治,平}])$$

每个算子都是 `M_n A → M_{n+1} A` 的提升 — 把作用域往外扩一层。形式：

$$\text{大学之道} = \text{fold}_\text{scope-up}([\text{格},\text{致},\text{誠},\text{正},\text{修},\text{齊},\text{治},\text{平}])(物)$$

### 19.7 易传的 64 卦展开 = 显式枚举的 fold

易传的 "易有太极，是生两仪，两仪生四象，四象生八卦" 是显式的递归枚举：

$$\text{易} = \text{太极}$$
$$\text{两仪} = \text{分}(\text{太极}) = \{\yang, \yin\}$$
$$\text{四象} = \text{分}(\text{两仪}) = \{\yang\yang, \yang\yin, \yin\yang, \yin\yin\}$$
$$\text{八卦} = \text{分}^3(\text{太极})$$
$$\text{六十四卦} = \text{分}^6(\text{太极})$$

这是 `iterate(分, 太极, 6)` — 一个固定深度的递归生成。从 1 个原子用 6 次分裂得到 64 个。

### 19.8 文言文里的 Y combinator?

严格 Y combinator 是 `Y = λf.(λx.f (x x)) (λx.f (x x))`。

文言文没有显式 lambda，但**有 self-application 通过 "之又" 模式实现的同等效果**。最接近的表达：

> 「损之又损」(λ-语义) ≅ 「让损反复施于自身的结果」 ≅ `(λf. f (...)) (损)` = `Y(损)`

实际上文言文的 **iter** 模式不依赖 self-application — 它通过**反复**这个语言级别的算子直接表达。比 Y 子更原始也更优雅。

不动点的核心算子:

- **iter** : (A → A) × A → A — 迭代 (反复施加)
- **iterUntil** : (A → A) × Predicate × A → A — 直至满足条件 ("以至于")
- **fix** : (A → A) → A — 不动点 ("归根")
- **生生** : 自指 — Y combinator 的实例

## 二十七、非交换性分析 (Commutativity)

哪些算子组合 A∘B ≠ B∘A？这关乎计算的 **顺序敏感性** 和 **confluence**。

### 20.1 交换的算子对

| 对 | 公理 | 出处 |
|---|---|---|
| 並 / 與 | A 並 B = B 並 A | 通用 (∧ 对称) |
| 同 (=) | A 同 B ↔ B 同 A | 墨经 (P-4) |
| 際 | A 際 B = B 際 A | 易 (R-3) |
| 易 (swap) | 易²= id | 易传 |
| 反 (involution) | 反² = id | 老子 (T-6) |

这些是显然交换 — 二元对称或自逆。

### 20.2 不交换 — 路径敏感

| 对 | 不交换的方向 | 出处/示例 |
|---|---|---|
| 損 / 益 | 损益 ≠ 益损 (在有界资源中) | 易·损益卦 |
| 進 / 退 | 进 ∘ 退 = id 但 不在所有 path 都成立 | 易 |
| 化 / 反 | 化 ∘ 反 ≠ 反 ∘ 化 | 通用 |
| 名 / 实 | 名(实(X)) ≠ 实(名(X)) | 公孙龙 |
| 含 / 含⁻¹ | A 含 B ≠ B 含 A | 自然 (asymmetric) |
| 承 / 乘 | 承 与 乘 倒置 | 易·爻位 |

### 20.3 文言文标记非交换的方式

文言文用**词序**标记非交换性。"损益" 与 "益损" 不仅意义不同，**习惯说法只有一个** — 经典固定 "损益" (易卦 41-42 的顺序)，因为这是结构上的"先减后加"或"先损己后益人"。

所以**经典选择**了非交换组合中的一个具体顺序作为规范。这是 *normalizing convention by canonization* — 与软件工程里的 "canonical order" 相同的策略。

### 20.4 损益的精确性质

老子 「为学日益，为道日损」给出两个独立 trajectory:
- 学: 益 ∘ 益 ∘ 益 ... — fixed point at 学问无穷 (no terminus)
- 道: 损 ∘ 损 ∘ 损 ... — fixed point at 无为

这两个 trajectory 不相交。`损 之又 损 → 无为`，但 `益 之又 益` 不收敛。

这意味着:
- **損** 是 contractive (向 fixed point 收缩)
- **益** 是 expansive (无 fixed point)

形式上: 损 ∈ Banach contraction; 益 ∉。

### 20.5 化与反的非交换性 (重要)

化 ∘ 反 ≠ 反 ∘ 化:
- **化 然后 反**: 化(A) → A', 然后取 A' 的反 → 反(A')
- **反 然后 化**: 反(A) → ¬A, 然后变化 → 化(¬A)

这两者一般不相等。例：
- 化(贫) = 富; 反(化(贫)) = 反(富) = 不富 = 贫.
- 反(贫) = 不贫; 化(反(贫)) = 化(不贫) = 不贫之化 (e.g. 中产)
不一样。

### 20.6 易卦的 错综 = 双重非交换性的范畴学结构

每卦有两个变换:
- **错** (complement): 每爻翻转 (yin↔yang) — 是 ¬ (NOT)
- **综** (reverse): 上下颠倒 — 是 reverse

64 卦在 (错, 综) 群作用下分组:
- 错² = id
- 综² = id
- 错 ∘ 综 = 综 ∘ 错 (一般不成立！)

实际：错与综**不交换**。错综同时作用 ≠ 综错同时作用。

这给了 64 卦一个 **D₂ × D₂ 群作用，但带 obstruction** — 不是干净的 Klein 四群。

### 20.7 confluence 含义

非交换性意味着原始的 rewrite 系统 **不confluent**。需要规范化策略：

1. **标准顺序**: 所有 (损, 益) 对都展开为 "先损后益"。
2. **优先级**: 给每个算子一个优先级，按优先级 reduce。
3. **范畴化**: 接受非交换性，使用 PROP 而不是对称 monoidal。

间 DSL 实现的话，**第三条最干净**。

### 20.8 非交换组合的 monoid 结构

设 𝒲 = 文言文算子的自由 monoid。则:

- **可交换的子幺半群**: ⟨並, 同, 際, 易, 反, ...⟩ — 在此子结构中可任意 reorder
- **核心非交换部分**: ⟨损, 益, 化, 反, 进, 退, 名, 实⟩ — 必须保持顺序
- **混合**: 大多数算子组合属于这一部分

这暗示文言文的"语序自由度"是分层的：某些类型的算子可以重排，某些必须保持。

## 二十九、新发现的算子代数律

### 律 1: 法家三元的范畴等式
$$\text{治国} = \text{术}_\text{op-on-ops}\Big( \text{法}_\text{public-rules}\big( \text{势}_\text{positional}( \text{system} ) \big) \Big)$$

是 **三层嵌套** — 势是 0-cell (位置), 法是 1-cell (规则), 术是 2-cell (规则上的操作). 这是 **2-categorical structure**.

### 律 2: 五行的群论身份
五行 (Z/5, 相生=+1, 相克=+2):
- (Z/5, +) 是阿贝尔群
- {+1, +2, +3, +4} = (Z/5)* 是其自同构群
- 相生(+1) 与 相克(+2) 是 (Z/5)* 的两个元素
- 注意: 4 = -1 ≡ 反生 (reverse generation), 3 = -2 ≡ 相侮

所以五行的**完整动力学** = (Z/5, +) ⋊ (Z/5)*, semidirect product = **affine group of 5-element line**.

### 律 3: 阴阳是带种子的二元投影
不是 Z/2 (二元否定) — 而是:
$$A = \alpha A_\text{阳} + (1-\alpha) A_\text{阴}, \quad A_\text{阳} \supset \epsilon_\text{阴}, \quad A_\text{阴} \supset \epsilon_\text{阳}$$

每极包含对极的 *seed*. 这等价于 **complex involution with non-trivial fixed-point structure**.

### 律 4: 三元门 (开/合/枢) 的超 Boolean 代数
医家 开/合/枢 不是 Boolean (2-valued) 而是 ternary. 形式:
- 开: input → output (allow)
- 合: input ↛ output (block)  
- 枢: switching state — controls 开/合 transition

这是 **automaton with 3 states + transition function**. 比 Boolean 网络更表达力强. 适合建模 6-channel (3 阴 + 3 阳) 系统.

### 律 5: 化性起伪 = 高阶函数应用
$$\text{化性起伪}: \text{Person} \xrightarrow{\text{伪} \in \text{Trainings}} \text{Person}'$$

读: 化 是高阶 functor (T-1), 伪 是 functor 的参数 (training schedule), 性 是被作用对象. 这是 **higher-order operator with environmental parameterization**.

### 律 6: 群-分的反直觉单调性
$$\text{群} \text{ alone} \to \text{conflict} \quad \text{but} \quad \text{群} + \text{分} \to \text{order}$$

加入 differentiation (分) **降低** total entropy 而不是增加. 这是反直觉的: usually adding constraints reduces ordered states. 这里 分 is *role-stratifying* — 它消除 同位竞争 (positional conflict).

形式: \|order(群)\| < \|order(分(群))\|.

### 律 7: 标本 = 时序因果分析
治标 ≠ 治本. 是 **causal hierarchy**:
$$\text{cause} \to \text{symptom}, \quad \text{治本} \implies \text{治标}, \quad \text{治标} \not\implies \text{治本}$$

是 **DAG over time** with intervention preference at root.

### 律 8: 反 是 functor on the operator category
对每个 operator f: A → B, 反f: B → A 存在 (在某条件下).

是 **dagger structure** on operator category. 给 operator category C 一个 dagger †: C → C^op.

法家 / 老子 / 庄子 / 内经 都隐含使用 dagger structure (反求诸己, 反者道之动, 反损为益).

## 三十、Categorical 形式化

### 候选 1: 算子作为某 monoidal category 的 morphisms

设 𝒞 = 结构的范畴：
- **对象**: 各种结构 (字、命题、行动、过程、域)  
- **态射**: 算子，作用于结构
- **⊗**: 結构合并 (会意 / 並 / 合 这一类)
- **I**: 单位结构 (无 / 空)

各类算子的范畴学位置：
- **关系算子 R/C**: 范畴的 hom-relations (relations between objects)
- **变换算子 T**: 内部 endomorphisms, T : A → A 或 A → A'
- **流动算子 F**: 类 mono / epi 的方向性态射
- **边界算子 B**: limit / colimit 算子
- **量化算子 Q**: 在 fibered category 上的 ∀/∃ 取极限/colim
- **因果算子 K**: 蕴含态射；composition rules
- **模态算子 M**: 在某 modal monad 中
- **对偶 N**: 自然反对偶 (involution)
- **同一 I**: identity / iso
- **序贯 S**: 范畴的 composition itself
- **复合 H**: higher-order — endofunctors / monads / adjunctions

### 候选 2: 算子作为 operad

更细致：算子集 = 一个 **operad 𝒪**：
- **𝒪(n)**: 接受 n 个结构、产生 1 个结构的算子集
- **complex composition**: 算子的嵌套使用

例：
- 化 ∈ 𝒪(1) — unary
- 並, 合, 配 ∈ 𝒪(2) — binary
- 分 ∈ 𝒪(1) → 𝒪(2) — splits
- 致 (大学版) ∈ 𝒪(2) — binary, with telos

operad 公理：composition associative + unit + equivariant under permutation。文言文算子大致满足，例外即是非交换性 (順序敏感 — e.g. 損益 ≠ 益損 in some contexts)。

### 候选 3: 算子作为 PROP (product and permutation category)

PROP 比 operad 更松，允许多输入多输出：
- 分 ∈ PROP(1, 2) — 1 input, 2 outputs
- 合 ∈ PROP(2, 1)
- 易 ∈ PROP(2, 2) — swap
- 反 ∈ PROP(1, 1) — involution

PROP 框架可能最合适 — 文言文确实有 splitter 和 joiner 算子，operad 容不下。

### 启示
**算子代数 ≈ symmetric monoidal PROP，带额外的 modal layer 和 reflection operators (体用、名实)。**

实现路径（间 DSL）：
1. 内核：PROP 上的术语重写
2. 模态层：层叠 modal monads (必、可、或)
3. 反思层：体/用 对偶作为 adjunction

其中 ⊗ 是 monoidal product (parallel composition)。

### 22.2 大学八条目

> 「**古之欲明明德于天下者，先治其国；欲治其国者，先齐其家；欲齐其家者，先修其身；欲修其身者，先正其心；欲正其心者，先诚其意；欲诚其意者，先致其知。致知在格物。**」

解析为反向构造序列:

```
明(明德, 天下) 
  ⇐ 治(国)
    ⇐ 齐(家)  
      ⇐ 修(身)
        ⇐ 正(心)
          ⇐ 诚(意)
            ⇐ 致(知)
              ⇐ 格(物)
```

其中 ⇐ 是 *prerequisite arrow* (先...而后...)。

形式化为 monadic chain:

$$\text{明明德}_{天下} = \text{平} \circ \text{治} \circ \text{齐} \circ \text{修} \circ \text{正} \circ \text{诚} \circ \text{致} \circ \text{格}$$

每个算子升一级 scope。

**算子集解析完成**: 8 个算子全部在我们的核心集 (P, T, T, T, R-6, T, T-11, K) 中。✓

### 22.3 老子第 48 章

> 「**为学日益，为道日损。损之又损，以至于无为。无为而无不为**。」

解析:

```
为(学) ⟿ iter(益, 日)         # 益 in time-iterate
为(道) ⟿ iter(损, 日)         # 损 in time-iterate

iterUntil(损, 状态=无为)        # iteration with stopping
  → 无为                         # fixed point reached

无为 ⟿ 无 不为                  # 无 ∘ 不 ∘ 为 = 全 为 (universal action via non-action)
```

最后一句的 **`无 ∘ 不` = 全** 是文言文的 De Morgan: 无(不为) = ∀X. 为(X)。

形式:

$$\text{无为} = \text{fix}(\text{损}) = \mu s. \text{损}(s)$$
$$\text{无不为} = \neg \exists X. \neg\text{为}(X) = \forall X. \text{为}(X)$$
$$\text{无为而无不为} = \text{fix}(\text{损}) \implies \forall X. \text{为}(X)$$

这是老子的核心 **theorem statement**。算子集完整覆盖。✓

### 22.4 易传 系辞

> 「**一阴一阳之谓道。**」

```
道 = alternate(阴, 阳)
   = iter([阴, 阳])
   = (阴 ⊕ 阳)^ω
```

道 = 两元素的无穷交替序列。一个 **infinite stream of alternations**。

形式:

$$\text{道} = \overline{\text{阴} \cdot \text{阳}} \in \{\text{阴}, \text{阳}\}^\omega$$

是一个 ω-regular language 中的 fairness 约束 — 阴阳必须无穷地各自出现。

### 22.5 庄子 齐物论

> 「**彼出于是，是亦因彼。彼是方生之说也。虽然，方生方死，方死方生；方可方不可，方不可方可。**」

解析:

```
彼 出于 是                     # 彼 = 出(是) — 来源关系
是 亦 因 彼                    # 是 = 因(彼) — 互相依赖
∴ 彼 ↔ 是 (互生)              # 二者同生
方(生) ⊕ 方(死) ⊕ ...        # progressive aspect 同时
```

`方` 是 progressive aspect 算子 (S-11)。"方生方死" = "正生正死" 同时 = 不分阶段。

庄子在这里**用 aspect 算子做哲学论证** — 通过让两个对立操作同时进行 (progressive overlay)，消解了它们的对立性。这是 **aspect-based dialectic**。

形式:

$$\forall t.\, \text{方}(\text{生})(t) \wedge \text{方}(\text{死})(t)$$

即: 每个时刻同时在生和死中。这等价于说生死不是离散事件而是连续过程的同一动作。

### 22.6 总结观察

测试结果:

- 4/5 段落用核心算子集 100% 覆盖
- 庄子段落需要更精细的 aspect 处理 (方 算子的并列叠加)
- 没有需要超出现有算子集的语义

这表明 **~30 个核心算子 + 参数空间 + 序贯/不动点 combinator** 就足够覆盖经典文本的核心结构。
