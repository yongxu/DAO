## N：儒家从元到圣

N 将儒家读成从个体修身到共同体秩序的生成链。核心不是“引用儒家证明系统”，而是看仁、义、礼、智、信怎样分别承担关系、判断、规范、认知和可信实践。

| 概念 | 在本框架中的位置 |
|---|---|
| 仁 | 同根共生的近身实践 |
| 义 | 在关系中作合宜判断 |
| 礼 | 让关系可持续的外化秩序 |
| 智 | 对情境、边界、后果的辨识 |
| 信 | 可观察的一致性与承诺延续 |
| 圣 | 多层实践的高阶一致，不是任意权威 |

若旧文标注 Kernel 层，当前 docs-next 只作入口提示；实际 Lean 状态仍以生成索引为准。

## O：进化非道者生存

O 的关键是反对把“活下来”直接等同于“合道”。演化是选择过程，但选择压力可能奖励短期掠夺、伪装、寄生和不可持续扩张。生生不息要求加入尺度、反身性和共续条件。

三层区分：

| 层 | 问题 |
|---|---|
| 生物事实 | 什么策略在某环境中复制成功 |
| 价值判断 | 该策略是否保生、共开、可续 |
| 反身制度 | 系统是否能识别并修正破坏性成功 |

涉及现代形式层时，可查 `EvolutionDao`、`Alignment`、`HumanAlignment` 等模块在 [../_generated/lean-index.md](../_generated/lean-index.md) 中的位置。

## P：道家从无到化

P 强调无、虚、静、自然、反复与化成。它与 A-G 的生成骨架相接，但更关注不强夺、不僵化、不把局部控制当作整体道。

| 读法 | 说明 |
|---|---|
| 无 | 给生成留下余地，不是空无所有 |
| 虚 | 可容、可受、可变，不是虚假 |
| 自然 | 顺其结构与势，不等于放弃判断 |
| 反 | 反向、复归、对偶，不等于破坏 |
| 化 | 变而不失其可续条件 |

## Q：佛家从苦到觉

Q 可读成对执著结构的解除。缘起说明对象依关系而起；空说明没有孤立自足的实体；无我松动固定自我中心；中道避免落入实有与断灭两边。

本区只保留结构映射：

| 佛家语汇 | 结构读法 |
|---|---|
| 苦 | 关系系统中的困滞与反复损害 |
| 集 | 执著、误识、贪取形成的生成链 |
| 灭 | 损害链条可被止息 |
| 道 | 可实践、可校正的解除路径 |

不要把佛学史料在此展开；需要出处时回旧文和归档指针。

## R：对齐必然

R 把 alignment 读成过程对齐：系统的学习、行动、反馈和修正，应与生生不息的共续条件一致。它不是要求固定目标永不变，而是要求目标更新不能破坏根本的保生、共开、守中、反身审校。

与 [map-u-z.md](map-u-z.md) 的 Y 相接：R 给正面框架，Y 给失败模式。

## S：先秦百家

S 适合做制度面映射。墨家可读为兼爱、功利、名实与技术理性；法家可读为规则、权力、赏罚和控制；名家处理名实与语言边界；兵家处理冲突、势、风险和成本。

建议写法：

| 维度 | 可映射问题 |
|---|---|
| 名实 | 名称是否贴合对象和作用 |
| 制度 | 规则是否保生或只保控制 |
| 行动 | 策略是否短期有效且长期可续 |
| 技术 | 工具是否服务共同体而非反噬 |

## T：西哲与亚伯拉罕

T 是跨文明对照页。它可以比较 logos、creation、covenant、law、grace、person、community 等概念与本框架的关系，但必须区分类比和同构。

三类标注：

| 标注 | 含义 |
|---|---|
| 类比 | 帮助理解，但结构不完全相同 |
| 同构 | 关键关系、方向、约束可互译 |
| 形式化 | 已有 Lean 或 claim index 支撑 |

## 共同体入口

`义理/人类命运共同体_共同体之证.md` 可作为 N-T 与 U-Z 的桥。它把传统修身、跨文明共识和现代制度问题汇入共同体条件：同根、共开、互不毁根、可审校、可续。
