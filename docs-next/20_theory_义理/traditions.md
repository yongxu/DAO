# 传统义理

本类文档把传统思想作为同一框架下的内容投影，而不是把传统文本搬入新版目录。

## 儒道佛

| 传统 | 旧入口 | 读法 |
|---|---|---|
| 儒家 | `义理/N_儒家从元到圣.md` | 仁义礼智信、诚、恕、中庸、大同等作为同根、共开、守中之展开 |
| 道家 | `义理/P_道家从无到化.md` | 无为、自然、反者道之动、虚静等作为不夺其生与顺势化成 |
| 佛家 | `义理/Q_佛家从苦到觉.md` | 缘起、空、无我、中道、八正道等作为执著松动与苦的结构性解除 |

三家同读的重点不是强行混同，而是看它们是否能在“元、间、化、共续”的层级上给出可互译的结构。
