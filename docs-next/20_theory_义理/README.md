# 义理索引

本区重排旧 `义理/` 的作者论证文档，只做导航与摘要，不搬运长文。旧文仍保留在仓库根部 `义理/`；机器生成索引若已生成，优先对照 `../_generated/` 下的文件清单、术语表与 Lean 声明索引。

## 阅读顺序

1. 先按 [阅读次第](reading-order.md) 选路径：快速、形式、传统、现代、专题。
2. 读 [核心框架](core-framework.md)：四级生成、六征、实虚史真、算子代数。
3. 查分段映射：[A-G](map-a-g.md)、[H-M](map-h-m.md)、[N-T](map-n-t.md)、[U-Z](map-u-z.md)。
4. 再读 [形式与证明](formal-and-proof.md) 与 [义理到 Lean](bridge-to-lean.md)：确认哪些论断可落到形式层。
5. 按需读 [传统义理](traditions.md)、[现代与对齐](modern-and-alignment.md)、[八衍总览](eight-expansions.md) 与 [八衍与专题](extensions.md)。
