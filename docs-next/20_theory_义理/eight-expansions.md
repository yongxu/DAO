# 八衍总览

八衍把八卦层从共同骨架展开到八个内容域：数、推、测、形、映、行、识、象。它们共享“从元出发、经差别生成、再由算子组织关系”的读法，但各自的严格度不同。

## 共同读法

每一衍都可按四问阅读：

| 问 | 说明 |
|---|---|
| 从何为元 | 本域的最小起点是什么 |
| 如何分化 | 何种差别生成对象、关系或状态 |
| 用何算子 | 何种变换、判断、组合组织本域 |
| 如何归根 | 何种等价、守恒、规范形或审校使其不散 |

这个四问只是导览，不替代各域本身的数学、逻辑、统计、物理或心智理论。

## 数：从元到数

数衍处理可数、可序、可加、可乘和可完备化的结构。义理上，它说明“多”不是杂乱堆积，而是可由起点、后继、组合和等价关系组织。

边界：自然数、整数、有理数、实数等层级若声称形式化，应查 Lean index；旧专题中的解释性段落不能自动升级为 theorem。

## 推：从元到推

推衍处理判断如何从前提到结论。它连接布尔、三值、自然演绎和 Curry-Howard 读法。义理重点是：推理要保留规则、上下文和可检验步骤。

边界：三值保守性、自然演绎等若有 machineChecked 项，应以 [../_generated/claim-index.md](../_generated/claim-index.md) 和 Lean index 为准。

## 测：从元到测

测衍处理不确定性。概率、Bayes、熵、Markov 过程可读成从未知到更新、从样本到分布、从证据到信念调整的结构。

边界：占筮或象数解释只可作为历史或义理接口；概率定理和统计模型要查形式模块与具体假设。

## 形：从元到形

形衍处理位置、距离、邻接、边界与连通。它使“位”不只是卦位，也可读成结构中的相对处所。

边界：几何和拓扑直觉需要分开。若进入“障、连通、边界”的专题，可参考旧 `拓扑 · 从元到障`，但 docs-next 不复制其长文。

## 映：从元到映

映衍处理对象之间的保持结构之变换。范畴、函子、自然变换、伴随和 Monad 可作为“关系比对象更稳”的现代形式读法。

边界：范畴类比很容易过宽。只有当对象、态射、复合、恒等和相应公理明确时，才写成结构映射。

## 行：从元到行

行衍处理时间中的状态变化。相空间、轨道、不动点、周期、稳定性和微分方程都可进入此域。

边界：动力学语言不等于现实预测。若无模型、参数、初值和误差界，只能写为结构启发。

## 识：从元到识

识衍连接注意、心智、神经、唯识与反身意识。它关心系统如何选择对象、维持记忆、形成自我模型并修正误识。

边界：心智解释尤其要避免把类比写成科学结论。形式层可查 `Attention`、`XinZhi`、`Neuro`，但自然语言解释仍需保留条件。

## 象：从元到象

象衍处理物理与现象的显现方式。qubit、对偶、守恒、自旋、测量等可作为形式或类比资源。

边界：物理术语必须说明是严格模型、数学同构还是启发性旁证。与 F 的关系见 [map-a-g.md](map-a-g.md)。
