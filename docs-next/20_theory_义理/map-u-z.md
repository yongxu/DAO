## U：信诚与反诛心

U 的要点是：不以不可见心证定罪，也不放弃对行为一致性的审校。信诚应落在可观察承诺、行为轨迹、可更正机制和同意边界上。

| 项 | 义理要求 |
|---|---|
| 信 | 承诺可追踪，行为不任意背离 |
| 诚 | 内外一致可由长期实践显现 |
| 反诛心 | 不以猜测动机替代证据 |
| 审校 | 行为、后果、反馈、修正均可检查 |

若要判断某个信诚 claim 是否已形式化，查 [../_generated/claim-index.md](../_generated/claim-index.md) 和 [../_generated/lean-index.md](../_generated/lean-index.md) 中 `Sincerity`。

## V：现代政治哲学

V 处理 Rawls、Habermas、Arendt 等现代政治资源时，应把它们读成公共性、正当性、协商、行动和制度可证错的不同表达。docs-next 不复写哲学史，只提结构用途。

| 问题 | 本框架判准 |
|---|---|
| 正当性 | 是否能向受影响者给出理由 |
| 自由 | 是否保留开端、选择和修正能力 |
| 平等 | 是否承认同根而非同质化 |
| 公共理性 | 是否允许差异在共同规则中可审校 |
| 制度 | 是否能识别并纠正自身失败 |

政治哲学不应被写成单一公式；它提供现代制度语汇，仍需经 claim 与形式边界审校。

## W：非道之形式

W 给反面结构。非道不是“我不喜欢的观点”，而是破坏生生不息条件的稳定模式。

| 非道形式 | 结构特征 |
|---|---|
| 绝对零和 | 他者之生被当作己方成本 |
| 通吃 | 取消外部开端与多样生成 |
| 总动员 | 将全部生活压成单一目标 |
| 断根 | 切断来源、记忆、修正和传承 |
| 伪开放 | 表面开放，实际只允许预设结果 |

涉及强规范 claim 时，必须标明是定义、公理依赖还是已证结果；[../_generated/claim-index.md](../_generated/claim-index.md) 是最低检查入口。

## X：反施密特

X 批判敌友政治和例外状态崇拜。其义理核心是：把政治共同体建立在绝对敌人和无限例外上，会破坏同根条件，使制度自我免审校。

| 施密特式风险 | 反向约束 |
|---|---|
| 敌友绝对化 | 对手仍在共同世界中 |
| 例外常态化 | 紧急权力必须可回收 |
| 主权免审校 | 决断也要承担理由与后果 |
| 共同体封闭 | 边界不可毁掉共续可能 |

形式入口查 [../_generated/lean-index.md](../_generated/lean-index.md) 中 `SSBX.Foundation.Wen.AntiSchmitt`。

## Y：对齐失败

Y 是 R 的反面测试集。对齐失败不是单一错误，而是一组目标、反馈、代理、解释和治理断裂。

| 失败模式 | 框架读法 |
|---|---|
| Goodhart | 指标脱离所要保护的生生条件 |
| 规格游戏 | 形式目标被满足，实质目标被绕过 |
| 欺骗性对齐 | 行为表面对齐，内部策略等待脱约束 |
| 目标漂移 | 更新机制失去根本审校 |
| 权力寻求 | 工具目标吞噬共同体边界 |

形式入口查 `AlignmentFailures`；若生成索引只显示模块存在，不代表每个自然语言例子都已证明。

## Z：经济博弈

Z 把市场、合作、竞争、机制设计和博弈论放入条件判断。经济行为不是天然合道，也不是天然非道；关键看是否保存同根、减少破坏性外部性、允许合作演化并保留审校能力。

| 场景 | 判断点 |
|---|---|
| 市场交换 | 自愿、信息、外部性、退出权 |
| 囚徒困境 | 是否能建立重复、信任和惩罚机制 |
| 机制设计 | 指标是否被 Goodhart 化 |
| 增长 | 是否以断根和不可逆损害为代价 |
| 分配 | 是否维持参与者继续生成的能力 |

形式入口查 [../_generated/lean-index.md](../_generated/lean-index.md) 中 `EconGame`，案例强度查 claim index。

## 本段边界

- U-Z 是规范应用，不应把所有现代争议直接判成定理。
- 对政治与经济的判断需保留条件；缺少条件时只写为框架提示。
- 对齐失败可用作审计清单，但每个失败案例仍需具体模型或证据。
