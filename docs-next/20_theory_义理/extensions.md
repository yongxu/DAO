# 八衍与专题

八衍文档说明八卦层可以承载哪些内容域。它们不是核心框架的替代品，而是从同一生成骨架向不同学科的展开。

## 主线四衍

| 方向 | 旧入口 | 内容 |
|---|---|---|
| 数 | `义理/数与算术 · 从元到数.md` | 自然数、整数、有理数、实数与四则结构 |
| 推 | `义理/形式逻辑 · 从元到推.md` | 布尔、三值、自然演绎、Curry-Howard |
| 测 | `义理/统计 · 从元到测.md` | 概率空间、Bayes、熵、Markov 与占筮读法 |
| 形 | `义理/几何位 · 从元到形.md` | 点、位、距、邻、连、拓扑与形位 |
