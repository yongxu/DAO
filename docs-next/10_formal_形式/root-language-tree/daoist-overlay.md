# 道家核心词 Overlay Mapping

> 状态：draft v0.1 (2026-05-13)
> 范围：把道家核心字/词映射到现有 `R0..R8` root language tree 条目，作为审名、教学和引用索引。
> 口径：本页不改 canonical root 名，不新增坐标，也不把老庄术语声称为形式系统 primitive。形式真值仍以 bitstring、cell / operator role 和 XOR mask 为准。

## 读法原则

1. `道` 有三层锚点：`ROOT-undivided` 作未分之根，`R0-ε-cell` 作太极/一元根，`OX["oooooooo"]` 与 R8 `道·*` family 作 Shi identity anchor。不要把任一局部卦名直接等同“道体”。
2. `德` 不只读作伦理 virtue，更接近 `得 / potency / efficacy`：优先映到 `力 / 中孚 / 无妄 / 益`，再按语境读作内在效能或成物之得。
3. `無 / 有` 在本 tree 中已有 `無跡 / 有跡` 的 R7 因位读法；那是 trace / no-trace，不等于老庄的 metaphysical 無有。老庄 `無有` 作为 overlay 要同时参考 `道 anchor`、`陰陽 / 虛實`、R7 trace state。
4. `無為` 不是 no-op 的简单同义词。`恒` 可作“无强作”的 operator 参照，但语义还要连到 `自然 / 复 / 损 / 遁 / 艮`。
5. `玄 / 妙 / 幾 / 機` 走 subtle threshold 读法；`樸 / 素 / 真` 走 trace-free、质实、无妄读法。不要为这些词强造一个新 root。
