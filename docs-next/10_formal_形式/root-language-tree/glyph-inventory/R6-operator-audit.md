# R6 Operator Audit

> 状态：审名 v0.1 (2026-05-13)。
> 范围：R6 重的 64 个 operator reader names；R4 见 [R4-operator-audit.md](R4-operator-audit.md)，R5 见 [R5-operator-audit.md](R5-operator-audit.md)。
> 形式口径：每个 bitstring `m` 表示 `(Z/2)^6` 上的平移 `T_m(s)=m xor s`。若 `e_i` 是第 `i` 位 basis，则 `τ_i = T_{e_i}`，所有 `τ_i` 交换。

## Naming Rule

- 精确名永远是 `T_m` 或 `τ_i` 的和成；中文名只是 reader alias。
- 能自然成词且不混淆 mask 的组合可升为公开名：`變化 = 化+變`，`改變 = 改+變`。
- 若 `化+變` 与 `改` 同时出现，优先写 `改+變化`，避免把 y2 吸进 `改變`。
- 其余多原子组合用 `+` 分隔，不写成 `改化變臨易內易外` 这种串。
- `儒道法佛 / 形式逻辑数学 / 哲学 / 科学` 四列是跨语境表达候选；不是 parser 名，也不是声称各传统已有这些 operator 的严格定名。
