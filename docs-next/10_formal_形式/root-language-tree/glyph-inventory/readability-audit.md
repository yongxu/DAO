# R5-R8 Glyph Readability Audit

> 状态：philological review v0.2 (2026-05-13)。
> 范围：审计 `glyph-inventory/R5.md`..`R8.md` 与对应 `layers/R5.md`..`R8.md` 的默认公开字词。
> 原则：优先选择古文字和古文语法中能读通的名字；其次取当代中文或形式逻辑透明读法；仍无合适名字时才退到组合式结构名。

## 0. 审计结论

| layer | 原问题 | 处理 |
|---|---|---|
| R5 | 前一轮过度结构化，把 `力/作/徑/契/...` 误降级。 | 恢复 32 个单字作默认入口；`內本/內征·R4` 只作消歧 fallback。 |
| R6 | 六十四卦名可保留，但 operator 串写如 `化變臨易內易外` 不可读。 | cell 保留卦名；多原子 operator 改为 `化+變+臨+易內+易外`。 |
| R7 | `素/跡` 中 `素` 太泛；`無因/有因` 容易被读成“没有原因/有原因”。 | 默认改为 `無跡/有跡·卦名`；`無因/有因` 仅作 formal fallback。 |
| R8 | `乾未/乾已/乾今` 像内部编码，不像公开读法。 | 默认改为 `Shi·卦名`，如 `未·乾`；紧凑写法 `乾未` 只作短 alias。 |

## 1. 默认入口与 alias 分层

| 类别 | 可作默认 | 只作 alias |
|---|---|---|
| 结构组合 | `無跡·乾`、`未·乾`；R5 中仅作 fallback | `內本·動` 不替代 `力` |
| 传统强名 | 六十四卦名，如 `乾`、`夬`、`大有` | 无 |
| 单字雅训 | R5 32 单字，如 `力`、`作`、`徑`、`契` | 无 |
| 紧凑组合 | 仅在表格或熟悉上下文中可短写，如 `乾未` | R8 `卦名+Shi` |
| operator 组合 | `改+化+變` | `改化變` 串写 |

## 2. 仍需人工判断的风险

- R5 单字不是“越现代越好”；只要古文字义和语法位置成立，就优先保留为推荐字。
- R6 卦名对易学读者是强名，对普通中文学者仍需结构读法辅助；所以表中必须继续保留 `本本/本征/征本/征征·R4`。
- `易內/易外` 是技术性透明词，不是古已有定名；优点是不会误导。
- `無跡/有跡` 比 `素/跡` 可读，但它把 R7 的公开语义更靠近 trace，而不是 cause；formal fallback 仍保留 `無因/有因` 来接 Lean 名称。
- R8 的 `道/未/已/今` 是 Shi V4 既定位字；不能再另造一套现代词，否则会切断 canonical doctrine。
