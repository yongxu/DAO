## 1. 五种结构身份 · Five Structural Identities

### 1.1 View A: $R_8 = \mathbb{F}_2^8$ (flat)

8-维 $\mathbb{F}_2$-向量空间。

元素 $w = w_0 w_1 w_2 w_3 w_4 w_5 w_6 w_7$，每 $w_i \in \{\mathrm{o}, \mathrm{x}\}$.

典则基 $\{e_0, \ldots, e_7\}$，$e_i$ 第 $i$ 位为 $\mathrm{x}$ 其余 $\mathrm{o}$.

群加法 = component-wise XOR, 阶 256, abelian, exponent 2.

### 1.2 View B: $R_8 = R_4 \oplus R_4$ (direct sum) ★

$R_8$ 是两个 $R_4$ 的直和——squaring tower 的下一层：

$$R_8 = R_4 \oplus R_4$$

- dimension: $4 + 4 = 8$ ✓
- cardinality: $|R_4| \cdot |R_4| = 16 \cdot 16 = 256 = |R_4|^2$ ✓

**Left / Right blocks**：

- $R_4^{(L)} = \{w \in R_8 : w_4 = w_5 = w_6 = w_7 = 0\} \cong R_4$
- $R_4^{(R)} = \{w \in R_8 : w_0 = w_1 = w_2 = w_3 = 0\} \cong R_4$

每 $w \in R_8$ 唯一分解 $w = (w_L, w_R)$ where $w_L \in R_4^{(L)}$, $w_R \in R_4^{(R)}$.

**这是 view 中最关键的**——所有「Left/Right pair」语义（cause/effect, past/future, state/op-extended）都基于此分解.

### 1.3 View C: $R_8 = R_2^{\oplus 4}$ (4-fold direct sum)

$R_8$ 分裂为四个 $R_2$-blocks:

$$R_8 = R_2^{(1)} \oplus R_2^{(2)} \oplus R_2^{(3)} \oplus R_2^{(4)}$$

每 $w \in R_8$ 写为 $(u_1, u_2, u_3, u_4)$ where each $u_k \in R_2$. 对应坐标：

- $u_1 = (w_0, w_1)$
- $u_2 = (w_2, w_3)$
- $u_3 = (w_4, w_5)$
- $u_4 = (w_6, w_7)$

**这是 view 中最 symplectic**——4 blocks 对应 σ 的 4 alternating blocks (§6.2).

### 1.4 View D: $R_8 = \mathrm{Hom}(R_2, R_4) = \mathrm{Hom}(R_4, R_2)$ — Hom block view ★

$R_8$ 同时等同于两种 Hom 空间——它们都是 8-dim $\mathbb{F}_2$-vector space：

#### 1.4.1 $R_8 = \mathrm{Hom}_{\mathbb{F}_2}(R_4, R_2) = \mathrm{Mat}_{1 \times 2}(R_4)$

线性映射 $R_4 \to R_2$ = $2 \times 4$ binary matrix = 「1 row × 2 columns of R_4-cells」（每 R_4-cell 是 2×2 binary block）.

flat 矩阵：

$$M_w^{(1)} = \begin{pmatrix} w_0 & w_1 & w_2 & w_3 \\ w_4 & w_5 & w_6 & w_7 \end{pmatrix} \in \mathrm{Mat}_{2 \times 4}(\mathbb{F}_2)$$

block-wise: 2 个 R_4-cells $(c_1, c_2)$ where $c_1 = w_0 w_1 w_4 w_5$, $c_2 = w_2 w_3 w_6 w_7$ (按 2x2 block 分组).

#### 1.4.2 $R_8 = \mathrm{Hom}_{\mathbb{F}_2}(R_2, R_4) = \mathrm{Mat}_{2 \times 1}(R_4)$

对偶地，$R_2 \to R_4$ = $4 \times 2$ binary matrix = 「2 rows × 1 column of R_4-cells」.

### 1.5 View E: $R_8 = \mathrm{Mat}_{m \times n}(\mathbb{F}_2)$ flat matrix forms

8 binary entries 可 arrange 为多个 flat matrix shape:

- $\mathrm{Mat}_{2 \times 4}(\mathbb{F}_2)$ (8 entries as 2×4)
- $\mathrm{Mat}_{4 \times 2}(\mathbb{F}_2)$ (8 entries as 4×2)
- $\mathrm{Mat}_{1 \times 8}(\mathbb{F}_2)$ (row vector)
- $\mathrm{Mat}_{8 \times 1}(\mathbb{F}_2)$ (column vector)

这些是 view A (flat) 的不同 reshape——结构同 view A.

### 1.6 Tensor vs Direct Sum 辨析 ★

> **重要区分**：「Squaring tower 的下一层」通过 **direct sum**（不是 tensor）实现.

$\mathbb{F}_2$-vector space 的两种主要二元运算：

| 运算 | dim 公式 | cardinality 公式 |
|------|---------|-------------------|
| Direct sum $R_N \oplus R_M$ | $N + M$ | $\mid R_N \mid \cdot \mid R_M \mid$ |
| Tensor $R_N \otimes_{\mathbb{F}_2} R_M$ | $N \cdot M$ | $2^{NM}$ |

应用于 squaring tower:

| step | direct sum | tensor |
|------|-----------|--------|
| $R_1 \to ?$ | $R_1 \oplus R_1 = R_2$ ✓ | $R_1 \otimes R_1 = R_1$ |
| $R_2 \to ?$ | $R_2 \oplus R_2 = R_4$ ✓ | $R_2 \otimes R_2 = R_4$ (巧合一致) |
| $R_4 \to ?$ | $R_4 \oplus R_4 = R_8$ ✓ | $R_4 \otimes R_4 = R_{16}$ (= End($R_4$)) |
| $R_8 \to ?$ | $R_8 \oplus R_8 = R_{16}$ ✓ | $R_8 \otimes R_8 = R_{64}$ (= End($R_8$)) |

**「Cardinality squaring」** $|R_{2N}| = |R_N|^2$ 由 direct sum 实现:

$$|R_4 \oplus R_4| = 16 \cdot 16 = 256 = |R_4|^2 = |R_8| \checkmark$$

**Tensor 产生 endomorphism algebra**:

$$R_N \otimes_{\mathbb{F}_2} R_N = \mathrm{End}_{\mathbb{F}_2}(R_N) = R_{N^2}$$

这两者只在 $N = 2$ 时巧合一致（$N+N = N \cdot N$ ⟺ $N = 2$）.

### 1.7 五种身份对照表

| View | 表达 | dim count | 关键用途 |
|------|------|-----------|----------|
| A | $\mathbb{F}_2^8$ | 8 | flat low-level |
| **B ★** | $R_4 \oplus R_4$ | $4 + 4 = 8$ | pair semantics (cause/effect, past/future) |
| C | $R_2^{\oplus 4}$ | $2 \cdot 4 = 8$ | 4-block symplectic structure |
| **D ★** | $\mathrm{Hom}(R_2, R_4) = \mathrm{Mat}_{2 \times 1}(R_4)$ 或 $\mathrm{Hom}(R_4, R_2) = \mathrm{Mat}_{1 \times 2}(R_4)$ | $2 \cdot 4 = 8$ | morphism between $R_2$ and $R_4$ |
| E | $\mathrm{Mat}_{2 \times 4}(\mathbb{F}_2)$ 等 flat shapes | $8$ | matrix-shape reshape of view A |

★ = 工程上最常用. View B 对 cause/effect 等 pair 语义关键; view D 对 morphism / 跨层映射关键.

**Squaring tower 通过 direct sum**: $R_8 = R_4 \oplus R_4$ 实现 $|R_4|^2 = 256$ cardinality squaring (详 §1.6). Tensor $R_4 \otimes R_4 = R_{16}$, 不在本层.

## 2. $R_8$ 包含 $R_0$–$R_7$ 全部为子空间 · Ceiling Property

$$\iota_N : R_N \hookrightarrow R_8, \quad (v_0, \ldots, v_{N-1}) \mapsto (v_0, \ldots, v_{N-1}, 0, 0, \ldots, 0)$$

## 6. 三层 Bilinear Pairings on $R_8$

$R_8$ 偶维 8，三层全支持，按 $R_2$ 4-block 因式化.

### 6.1 Layer 0 — Dot product

$$\langle w, w' \rangle_{R_8} := \sum_{i=0}^{7} w_i \cdot w'_i \in \mathbb{F}_2$$

在 $R_2^{\oplus 4}$ block view 下：

$$\langle w, w' \rangle_{R_8} = \sum_{k=1}^{4} \langle u_k, u'_k \rangle_{R_2}$$

对称、非退化、非 alternating.

### 6.2 Layer 1 — Symplectic σ (4 blocks) ★

$$\sigma_{R_8}(w, w') := \sum_{k=1}^{4} \sigma_{R_2}(u_k, u'_k) = \sum_{k=1}^{4} (w_{2k-2} w'_{2k-1} + w_{2k-1} w'_{2k-2})$$

展开：

$$= (w_0 w'_1 + w_1 w'_0) + (w_2 w'_3 + w_3 w'_2) + (w_4 w'_5 + w_5 w'_4) + (w_6 w'_7 + w_7 w'_6)$$

性质：alternating, $\mathbb{F}_2$-双线性, 非退化, 对称（char 2 下）.

**4 个 symplectic blocks** 是 $R_8$ 的关键结构——每 block 一个 conjugate pair（详 §10 spacetime）.

### 6.3 Layer 2 — Quadratic refinements

每 $R_2$-block 独立选 $q_0$ 或 $q_1$，共 $2^4 = 16$ quadratic refinements:

$$q^{(c_1, c_2, c_3, c_4)}(w) := \sum_{k=1}^{4} q_{c_k}(u_k), \quad c_k \in \{0, 1\}$$

每个是 $\sigma_{R_8}$ 的 quadratic refinement.

### 6.4 Arf classification

$$\mathrm{Arf}(q^{(c_1, c_2, c_3, c_4)}) = \sum_{k=1}^{4} c_k \pmod 2$$

16 个 quadratic refinements 按 Arf 分两类：

- **Arf 0** (8 元): $c_1 + c_2 + c_3 + c_4 \equiv 0 \pmod 2$ — 4 个 $c_k$ 中偶数个 1
- **Arf 1** (8 元): $c_1 + c_2 + c_3 + c_4 \equiv 1 \pmod 2$ — 4 个 $c_k$ 中奇数个 1

每类 8 元，是 $R_4$ (4 元 per class) 的 2 倍扩展.

### 6.5 Bilinear 三层的「自相似」

注意 $R_2$ 上 σ 是 1-block，$R_4$ 上是 2-block，$R_8$ 上是 4-block。$R_{2N}$ 上 σ 是 $N$-block.

类似 q 在 $R_{2N}$ 上有 $2^N$ variants, $2^{N-1}$ per Arf class.

> **这是 $R_8$ 自相似性的具体表现**：每 squaring tower 层 doubling block count.

这个性质 lift 通过：

1. **Direct-sum lift**：$P(R_{N+M}) \Leftarrow P(R_N), P(R_M)$ if $P$ 是 「sum-compatible」
2. **Tensor lift**：$P(R_N \otimes R_M)$ from $P(R_N), P(R_M)$ for tensor-compatible $P$
3. **Hom lift**：$P(\mathrm{Hom}(R_N, R_M))$ from $P(R_N), P(R_M)$

「sum / tensor / Hom-compatible」是 categorical 性质，多数代数性质满足.

### 12.7 实际工程含义

> **$R_0$–$R_8$ 内可证明的，就 lift 到所有 $R_{2^k}$**

工程意义：

- **代数验证范围**：focus on $R_0$–$R_8$，证明在 lift 下保持
- **解释器范围**：基本 ops 实现在 $R_0$–$R_8$，高层 $R_{2^k}$ 经 composition
- **memory footprint**：单 $R_8$ 元素 1 byte；$R_{16}$ 是 2 bytes；$R_{32}$ 是 4 bytes；…线性 cost

### 12.8 与 byte / word / qword 的对接

> $R_8$ 是 byte，$R_{16}$ 是 word，$R_{32}$ 是 dword，$R_{64}$ 是 qword.

每 squaring tower layer 对应计算机基本数据类型：

| Tower layer | CS 名称 | 元素数 |
|-------------|---------|--------|
| $R_8$ | byte | 256 |
| $R_{16}$ | word | 65,536 |
| $R_{32}$ | dword | $2^{32}$ |
| $R_{64}$ | qword | $2^{64}$ |
| $R_{128}$ | qword$^2$ / dqword | $2^{128}$ |

这不是巧合——计算机硬件设计因为「$\times 2$ doubling」的 algebraic 自然性而采纳这个 hierarchy.

## 13. GF(256) Field Structure — Application Layer

> $R_8$ 同时承载 $\mathrm{GF}(256)$ 有限域结构（带乘法环）——这是 R_8 algebra 之外的 application-layer 内容.

### 13.1 GF(256) 简述

$\mathrm{GF}(256) = \mathbb{F}_{2^8}$ 是 256 元 finite field. 它的加法群结构 $= R_8$ 作为 $\mathbb{F}_2$-向量空间。但 **GF(256) 有乘法**，$R_8$ 作为 additive group **没有**.

具体构造：$\mathrm{GF}(256) = \mathbb{F}_2[x] / (p(x))$ for any irreducible polynomial $p(x)$ of degree 8 over $\mathbb{F}_2$. 典型 AES 用 $p(x) = x^8 + x^4 + x^3 + x + 1$.

### 13.2 GF(256) 在 R_8 之外

GF(256) 的乘法环结构 *不是 R_8 algebra 的一部分*——它是给 $R_8$ 加上 **额外结构** (multiplicative ring).

在文 algebra 视角：
- core: $R_8$ 是 $\mathbb{F}_2^8$ (additive only)
- application: 当需要乘法时，选 specific $p(x)$ 并把 $R_8 \to \mathrm{GF}(256)$ as field

### 13.3 AES 的 GF(256)

AES 加密算法用 GF(256) 的乘法做 S-box (Substitution box). 详细在 application layer，详 §15.

## 14. Application Atlas

> **范围**：本节列出 $R_8$ 在外部传统中的对应。这些是 **application-layer**，**不是 $R_8$ algebra 的来源**.

### 14.1 Byte / 计算机基础单元

256 = 一 byte. 几乎所有现代计算机的最小数据单元.

### 14.2 Unicode UTF-8 single-byte

UTF-8 single-byte covers 256 code points (basic Latin + extensions). $R_8$-element ↔ UTF-8 byte.

### 14.3 AES (Rijndael)

AES 在 GF(256) 上操作 (§13.3). 状态是 $4 \times 4$ matrix of GF(256) elements = 16 bytes = $R_8^{\oplus 16}$.

### 14.4 Reed-Solomon codes

Reed-Solomon error-correcting codes use $\mathrm{GF}(2^k)$ for various $k$, 含 $\mathrm{GF}(256)$.

### 14.5 8-qubit register

8 qubits 的状态空间是 $\mathbb{C}^{256}$, basis state 数 = 256. 各 basis state 对应 $R_8$ 元素.

### 14.6 Pixel intensity (8-bit grayscale)

0-255 灰度值 = $R_8$ 元素.

### 14.7 DNA 编码扩展

DNA codon 4³ = 64 不直接 $R_8$, 但 8-bit DNA encoding schemes (e.g., 4-base × 2-flag) 使用 $R_8$.

### 14.8 Hex byte

一个 byte 写为 2 hex digits (`00`–`FF`)，每 hex digit 4 bit = $R_4$. 故 hex byte = $R_8 = R_4 \oplus R_4$ ✓.

### 14.9 NES sprite (8x8 grid)

经典 8x8 monochrome sprite = $R_{64}$, 但 row-wise structure 给 8 个 $R_8$. 类似各种 8-row 数据结构.

### 14.10 4-vector encoding

物理 4-vector ($x, y, z, t$) with each 2-bit precision → $R_8$ as $R_2^{\oplus 4}$ (§1.4 view D).

### 14.11 跨域共聚

$R_8$ 是 *byte-scale common ground*——许多 native digital, encoding, cryptographic, physical, biological 结构都自然地 settle 在 256-元 scale.

```lean
import Mathlib.Data.Fin.Basic

namespace 文.R8

/-- R_8 = F_2^8 — the ceiling layer -/
def R8 : Type := Fin 8 → Bool

namespace R8

/-- All 256 elements computable -/
instance : Fintype R8 := Pi.fintype
instance : DecidableEq R8 := inferInstance

/-- View B: R_8 = R_4 ⊕ R_4 -/
def R4 : Type := Fin 4 → Bool
def R2 : Type := Fin 2 → Bool

def split_LR (w : R8) : R4 × R4 :=
  (fun i => w ⟨i.val, by omega⟩, fun i => w ⟨i.val + 4, by omega⟩)

def join_LR (l r : R4) : R8 := fun i =>
  if h : i.val < 4 then l ⟨i.val, h⟩ else r ⟨i.val - 4, by omega⟩

/-- View C: R_8 = R_2 ⊕ R_2 ⊕ R_2 ⊕ R_2 (4 blocks) -/
def split_4blocks (w : R8) : R2 × R2 × R2 × R2 :=
  ( (fun i => w ⟨i.val,     by omega⟩)
  , (fun i => w ⟨i.val + 2, by omega⟩)
  , (fun i => w ⟨i.val + 4, by omega⟩)
  , (fun i => w ⟨i.val + 6, by omega⟩))

/-- View D: R_8 ≅ Hom(R_4, R_2) = Mat_{1×2}(R_4) — block view as 2×4 binary matrix -/
def as_mat_2x4 (w : R8) (i : Fin 2) (j : Fin 4) : Bool :=
  w ⟨4 * i.val + j.val, by omega⟩

/-- Group operation: component-wise XOR -/
def add (w w' : R8) : R8 := fun i => Bool.xor (w i) (w' i)

instance : Add R8 := ⟨add⟩
instance : Zero R8 := ⟨fun _ => false⟩

/-- Embedding R_N → R_8 for N ≤ 8 -/
def embed_from {N : ℕ} (h : N ≤ 8) (v : Fin N → Bool) : R8 := fun i =>
  if hLt : i.val < N then v ⟨i.val, hLt⟩ else false

/-- §2: Subspace property — R_N ⊆ R_8 for all N ≤ 8 -/
theorem subspace_property (N : ℕ) (h : N ≤ 8) :
    ∃ S : Set R8, ∀ (v : Fin N → Bool), embed_from h v ∈ S := by
  use {w | ∀ i, i.val ≥ N → w i = false}
  intro v
  intro i hLt
  unfold embed_from
  simp [Nat.not_lt.mpr hLt]

end R8
```

### 15.2 三层 Bilinear Forms

```lean
namespace 文.R8

/-- Layer 0: dot product -/
def R8.dot (w w' : R8) : Bool :=
  Finset.univ.foldr (fun i acc => Bool.xor (w i && w' i) acc) false

theorem R8.dot_symm (w w' : R8) : R8.dot w w' = R8.dot w' w := by
  unfold R8.dot
  congr 1
  funext i
  exact Bool.and_comm _ _

/-- Layer 1: symplectic σ, 4 blocks -/
def R8.sigma (w w' : R8) : Bool :=
  let block (k : Fin 4) : Bool :=
    let i0 : Fin 8 := ⟨2 * k.val,     by omega⟩
    let i1 : Fin 8 := ⟨2 * k.val + 1, by omega⟩
    Bool.xor (w i0 && w' i1) (w i1 && w' i0)
  Finset.univ.foldr (fun k acc => Bool.xor (block k) acc) false

theorem R8.sigma_alternating (w : R8) : R8.sigma w w = false := by
  unfold R8.sigma
  congr 1
  funext k
  cases (w _) <;> cases (w _) <;> rfl

/-- Diagonal linear form L on R_8 -/
def R8.L (w : R8) : Bool :=
  Finset.univ.foldr (fun i acc => Bool.xor (w i) acc) false

/-- Layer 2: quadratic refinement with mode (c_1, c_2, c_3, c_4) -/
def R8.q (c : Fin 4 → Bool) (w : R8) : Bool :=
  let block (k : Fin 4) : Bool :=
    let i0 : Fin 8 := ⟨2 * k.val,     by omega⟩
    let i1 : Fin 8 := ⟨2 * k.val + 1, by omega⟩
    let q0_val := w i0 && w i1
    let q1_val := Bool.xor (Bool.xor (w i0) (w i1)) q0_val
    if c k then q1_val else q0_val
  Finset.univ.foldr (fun k acc => Bool.xor (block k) acc) false

/-- Arf invariant: sum of c_k mod 2 -/
def R8.arf (c : Fin 4 → Bool) : Bool :=
  Finset.univ.foldr (fun k acc => Bool.xor (c k) acc) false

/-- 16 quadratic refinements split into 2 Arf classes of 8 each -/
example : (Finset.univ : Finset (Fin 4 → Bool)).card = 16 := by decide

end 文.R8
```
