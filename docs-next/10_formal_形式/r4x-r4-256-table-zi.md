# R4 x R4 256-Cell Character Table

> Status: structural working table (2026-05-12)
>
> Companion to: [`r4x-r4-256-table.md`](./r4x-r4-256-table.md). The `o/x` sheet is the audit base; this page is the character-facing view.

## Reading Policy

This page keeps the same two-dimensional carrier as the `o/x` sheet:

```text
R4_high = y1 y2 y3 y4
R4_low  = y5 y6 y7 y8
cell    = R4_high ++ R4_low
```

Character projections are read conservatively:

| Layer | Character source | Status |
|---|---|---|
| R1 | 阳 / 阴 | filled |
| R2 | 太阳 / 少阴 / 少阳 / 太阴 | filled |
| R3 | 八卦 names | filled |
| R4 | `Mian.label` single-character candidates | filled |
| R5 | `R4字 + 阳/阴`; structural composition, not a new lexical decision | filled |
| R6 | King Wen hexagram names | filled |
| R7 | `R6卦名 + 无因/有因` | filled |
| R8 | deferred | blank |

R5 is intentionally not assigned new standalone words here. It remains a combination layer until the R5+ lexicon is explicitly decided. R7 follows the existing Cell128 reading: `o = 无因`, `x = 有因`.
