## 1. 三种结构身份 · Three Structural Identities

### 1.1 View A: $R_4$ 作为 $\mathbb{F}_2^4$ (flat)

$R_4 = \mathbb{F}_2^4$，4-维 $\mathbb{F}_2$-向量空间。

元素 $w \in R_4$ 表示为长度 4 字符串：

$$w = w_0 w_1 w_2 w_3, \quad w_i \in \{\mathrm{o}, \mathrm{x}\}$$

典则基 $\{e_0, e_1, e_2, e_3\}$，$e_i$ 第 $i$ 位为 $\mathrm{x}$ 其余 $\mathrm{o}$。

群运算（加法）= component-wise XOR. 阶 16，abelian，exponent 2.

### 1.2 View B: $R_4 = R_2 \otimes R_2$ (pair of pairs)

把 $R_4$ 视为两个 $R_2$ 元素的 tensor / 直和：

$$w = (u, v), \quad u, v \in R_2$$

其中 $u = w_0 w_1$（前两位），$v = w_2 w_3$（后两位）。

- $u \in R_2$ 称 $w$ 的 **first part** 或 **α-component**
- $v \in R_2$ 称 $w$ 的 **second part** 或 **β-component**

例: $w = \mathrm{xoxx}$ 对应 $(u, v) = (\mathrm{xo}, \mathrm{xx})$.

### 1.3 View C: $R_4 = \mathrm{End}_{\mathbb{F}_2}(R_2) = \mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)$ ★

把 $R_4$ 视为 $R_2 \to R_2$ 的线性映射空间。

#### 1.3.1 矩阵表示

$w = w_0 w_1 w_2 w_3$ 对应 $2 \times 2$ 矩阵（row-major）：

$$M_w := \begin{pmatrix} w_0 & w_1 \\ w_2 & w_3 \end{pmatrix} \in \mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)$$

#### 1.3.2 线性映射的实施

矩阵 $M_w$ 作用 $R_2$ 元素 $u = (u_\alpha, u_\beta)^T$ 上：

$$M_w \begin{pmatrix} u_\alpha \\ u_\beta \end{pmatrix} = \begin{pmatrix} w_0 u_\alpha + w_1 u_\beta \\ w_2 u_\alpha + w_3 u_\beta \end{pmatrix}$$

输出仍是 $R_2$ 元素。

#### 1.3.3 对基向量的作用

- $M_w \cdot e_\alpha = M_w \cdot (1, 0)^T = (w_0, w_2)^T$ → 即「$M_w$ 第一列」
- $M_w \cdot e_\beta = M_w \cdot (0, 1)^T = (w_1, w_3)^T$ → 即「$M_w$ 第二列」

### 1.4 三 View 等价

三种 view 是同一对象的不同表达：

$$\mathbb{F}_2^4 \;\xleftrightarrow{\sim}\; R_2 \times R_2 \;\xleftrightarrow{\sim}\; \mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)$$

转换：

| flat $w$ | pair $(u, v)$ | matrix $M_w$ |
|----------|---------------|---------------|
| $w_0 w_1 w_2 w_3$ | $(w_0 w_1, w_2 w_3)$ | $\begin{pmatrix} w_0 & w_1 \\ w_2 & w_3 \end{pmatrix}$ |

**重要**：pair view 与 matrix view 看似都是「按位拆」，但解释不同：

- **pair view**：$u, v$ 是 *两个独立的 $R_2$ 元素*（如 state-pair）
- **matrix view**：$(w_0, w_1)$ 与 $(w_2, w_3)$ 是 *矩阵的两行*（如 op）

这个区分驱动 §9 的 phantom type discipline。

## 3. 群结构 · Group Structure

### 3.1 阿贝尔群

$R_4 = (\mathbb{Z}/2)^4$ — abelian, exponent 2, 阶 16.

群运算 = component-wise XOR：

$$w + w' := (w_0 \oplus w'_0,\; w_1 \oplus w'_1,\; w_2 \oplus w'_2,\; w_3 \oplus w'_3)$$

单位元 $\mathrm{oooo}$. 每元素 self-inverse.

### 3.3 关键子群

**Axis-aligned 子群** (按 coordinate fix-and-vary)：

- **Left $R_2^{(L)}$** = $\{w \in R_4 : w_2 = w_3 = 0\} = \{\mathrm{oooo}, \mathrm{xooo}, \mathrm{oxoo}, \mathrm{xxoo}\}$（前两坐标）

> **核心事实**：$R_4$ 是 $R_0$–$R_8$ 中**首层有 rank 分层**的根层。$R_0, R_1, R_2, R_3$ 都「平」（所有非零元结构等价）；$R_4$ 起按矩阵秩自然分类。

### 4.1 Rank 0 — 单元

仅 `oooo` (零矩阵 / 零映射)。

$M_w = 0$, 把所有 $R_2$ 元素映到 $\mathrm{oo}$.

`ooox, ooxo, ooxx, oxoo, oxox, xooo, xoxo, xxoo, xxxx`

### 4.3 Rank 2 — 6 元 = $\mathrm{GL}(2, \mathbb{F}_2)$ ★

6 个 rank-2 (invertible) 矩阵：

`oxxo, oxxx, xoox, xoxx, xxox, xxxo`

它们构成 $\mathrm{GL}(2, \mathbb{F}_2) \cong S_3$ —— $R_2$ 的全体自同构！见 §5 详细。

### 4.4 Rank 作为代数特征的首层涌现

为什么 $R_2$ 没有 rank 分层？因为 $R_2$ 的「endomorphism」只能内含于 $R_4$ 而非 $R_2$ 自身。$R_2$ 作为 4-元代数对象只有「群结构」，没有「内部 rank 结构」可言。

而 $R_4 = \mathrm{End}(R_2)$ 自带 matrix-rank（继承自 $\mathrm{Mat}_{2 \times 2}$）。这是「endomorphism algebra 给出 rank stratification」的最小实例。

更早的层（$R_3$ 等）虽是 $\mathbb{F}_2$-vector space，但**未被解读为 endomorphism algebra**——它们不是 $\mathrm{End}(\text{prev})$. 而 $R_4$ 恰是 $\mathrm{End}(R_2)$，所以 rank 概念自然出现。

> **First rank stratification at $R_4$** = $R_4$ 作为 $\mathrm{End}(R_2)$ 的代数特性的具体显现。

## 5. $\mathrm{GL}(2, \mathbb{F}_2) \cong S_3$ 在 $R_4$ 内 ★

### 5.1 6 个 rank-2 元素

| $w$ | $M_w$ | Aut($R_2$) 元素 | 在 $\{xo, ox, xx\}$ 的作用 | 阶 |
|-----|-------|-----------------|----------------------------|-----|
| `xoox` | $\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$ | $\mathrm{id}$ | 全 fix | 1 |
| `oxxo` | $\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$ | swap | (xo ↔ ox), fix xx | 2 |
| `xxox` | $\begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}$ | fix-xo | (ox ↔ xx), fix xo | 2 |
| `xoxx` | $\begin{pmatrix} 1 & 0 \\ 1 & 1 \end{pmatrix}$ | fix-ox | (xo ↔ xx), fix ox | 2 |
| `oxxx` | $\begin{pmatrix} 0 & 1 \\ 1 & 1 \end{pmatrix}$ | rot$_+$ | (xo → ox → xx → xo) | 3 |
| `xxxo` | $\begin{pmatrix} 1 & 1 \\ 1 & 0 \end{pmatrix}$ | rot$_-$ | (xo → xx → ox → xo) | 3 |

总：6 元，结构为 $S_3 \cong \mathrm{GL}(2, \mathbb{F}_2)$.

### 5.2 群结构 $S_3$ 在 rank-2 子集上

6 个 rank-2 元素构成 $S_3 = \mathrm{GL}(2, \mathbb{F}_2)$. 关键结构关系：

**单位元** (order 1):
- `xoox` = id

**2 阶元素 / involutions** (order 2, 3 个，self-inverse):
- `oxxo` = swap (xo ↔ ox)
- `xxox` = fix-xo (固定 xo，ox ↔ xx)
- `xoxx` = fix-ox (固定 ox，xo ↔ xx)

**3 阶元素 / 旋转** (order 3, 互逆对):
- `oxxx` = rot$_+$ = (xo → ox → xx → xo)
- `xxxo` = rot$_-$ = (xo → xx → ox → xo)

**关键关系**：

$$\text{rot}_+ \circ \text{rot}_+ = \text{rot}_-, \quad \text{rot}_- \circ \text{rot}_- = \text{rot}_+, \quad \text{rot}_+ \circ \text{rot}_- = \mathrm{id}$$

$$\text{swap}^2 = \text{fix-xo}^2 = \text{fix-ox}^2 = \mathrm{id}$$

**Involutions × 旋转 = 另一 involution**: 例如 swap ∘ fix-xo = rot$_+$ (=  $S_3$ 的标准 dihedral 关系).

**Cayley 关系**：$S_3 = \langle r, s : r^3 = s^2 = (sr)^2 = e \rangle$，取 $r = $ rot$_+$, $s = $ swap.

**完整 6×6 乘法表**：由上述生成关系机器生成，详 Lean 实现 §14.1 (Mathlib 内 `GL` lemma 可 verified by `decide`).

> **注**：手算 6×6 乘法表极易出错；建议用 Lean `decide` 验证。具体表暂不在文档中列出，避免误导。

### 5.3 这个嵌入的代数意义

> **$\mathrm{Aut}(R_2)$ 全部 6 个 group automorphisms 内嵌于 $R_4$ 作为可逆矩阵集合**

这是「**$R_2$ 自身的全体对称性，在 $R_4$ 内成为可处理的元素**」的精确实现。

哲学陈述：

- $R_2$ 的对称变换从「外部 acting」变成「$R_4$ 内部的对象」
- $R_4$ 是首层 *能容纳 prev-layer-Aut 作为内部元素* 的根层
- 这正是 endomorphism algebra 的核心功能

### 5.4 $\mathrm{Aut}(R_2)$ 嵌入 $R_4$ 的具体方式

定义嵌入 $\iota: \mathrm{Aut}(R_2) \hookrightarrow R_4$:

$$\iota(\xi) := w_\xi \quad \text{where } M_{w_\xi} = \text{matrix of } \xi$$

这是单射（不同 $\xi$ 给不同 matrix），保群结构（$\iota(\xi \circ \xi') = \iota(\xi) \cdot \iota(\xi')$，其中右边是 $R_4$ 内的矩阵乘法）。

像 $\iota(\mathrm{Aut}(R_2))$ = rank-2 子集 = 6 个可逆元 = $\mathrm{GL}(2, \mathbb{F}_2) \subset R_4$.

## 7. 三层 Bilinear Pairings on $R_4$

$R_4$ 偶维 4，三层 bilinear 全部存在。每层在 $R_4$ 上有 **2-block** 结构（$R_4 = R_2 \oplus R_2$ 反映到 pairing）。

### 7.1 Layer 0 — Dot product

$$\langle w, w' \rangle_{R_4} := \sum_{i=0}^{3} w_i \cdot w'_i \in \mathbb{F}_2$$

或在 $R_2 \oplus R_2$ block view：

$$\langle (u_1, u_2), (u'_1, u'_2) \rangle_{R_4} = \langle u_1, u'_1 \rangle_{R_2} + \langle u_2, u'_2 \rangle_{R_2}$$

对称、非退化、非 alternating。

Self-pairing 表（部分）：

- $\langle \mathrm{oooo}, \mathrm{oooo} \rangle = 0$
- $\langle \mathrm{xoox}, \mathrm{xoox} \rangle = 1 + 0 + 0 + 1 = 0$
- $\langle \mathrm{xxxx}, \mathrm{xxxx} \rangle = 1 + 1 + 1 + 1 = 0$
- $\langle \mathrm{oxxo}, \mathrm{oxxo} \rangle = 0 + 1 + 1 + 0 = 0$
- $\langle \mathrm{xooo}, \mathrm{xooo} \rangle = 1 + 0 + 0 + 0 = 1$
- $\langle \mathrm{xoxo}, \mathrm{xoxo} \rangle = 1 + 0 + 1 + 0 = 0$
- $\langle \mathrm{xxox}, \mathrm{xxox} \rangle = 1 + 1 + 0 + 1 = 1$

Self-pairing 零集 $\{w : \langle w, w \rangle = 0\}$ 含 $w$ s.t. 1 的个数为偶。共 8 个：$\mathrm{oooo}, \mathrm{ooxx}, \mathrm{oxox}, \mathrm{oxxo}, \mathrm{xoox}, \mathrm{xoxo}, \mathrm{xxoo}, \mathrm{xxxx}$。

这是 $R_4$ 的 $\ker L_{R_4}$ subgroup，order 8 = $|R_4|/2$.

### 7.2 Layer 1 — Symplectic σ (2 blocks)

$$\sigma_{R_4}(w, w') := (w_0 w'_1 + w_1 w'_0) + (w_2 w'_3 + w_3 w'_2)$$

或 block view：

$$\sigma_{R_4}((u_1, u_2), (u'_1, u'_2)) = \sigma_{R_2}(u_1, u'_1) + \sigma_{R_2}(u_2, u'_2)$$

Alternating $\sigma(w, w) = 0$ ∀ $w$（每 block 独立 alternating）。非退化。

### 7.3 Layer 2 — Quadratic refinements

每 $R_2$-block 可独立选 $q_0$ 或 $q_1$，共 4 个 quadratic refinements:

$$q^{(c_1, c_2)}(w) := q_{c_1}(u_1) + q_{c_2}(u_2), \quad c_1, c_2 \in \{0, 1\}$$

具体：

- $q^{(0,0)}(w) = w_0 w_1 + w_2 w_3$
- $q^{(0,1)}(w) = w_0 w_1 + (w_2 + w_3 + w_2 w_3)$
- $q^{(1,0)}(w) = (w_0 + w_1 + w_0 w_1) + w_2 w_3$
- $q^{(1,1)}(w) = (w_0 + w_1 + w_0 w_1) + (w_2 + w_3 + w_2 w_3)$

### 7.4 Arf 不变量分类

Symplectic basis at $R_4$: $(e_0, e_1), (e_2, e_3)$ as two blocks.

$$\mathrm{Arf}(q^{(c_1, c_2)}) = q^{(c_1, c_2)}(e_0) \cdot q^{(c_1, c_2)}(e_1) + q^{(c_1, c_2)}(e_2) \cdot q^{(c_1, c_2)}(e_3)$$

Compute:

- $q^{(0,0)}$: $e_i$ Single-coord vectors, $q^{(0,0)}(e_0) = 0$, $q^{(0,0)}(e_1) = 0$, so block 1 contributes 0. Same for block 2. **Arf = 0**.
- $q^{(0,1)}$: block 1 Arf 0; block 2 with $q_1$: $q_1(e_2) = q_1(\mathrm{xo}\text{-style}) = 1$, $q_1(e_3) = 1$. So block 2 contributes $1 \cdot 1 = 1$. **Arf = 0 + 1 = 1**.
- $q^{(1,0)}$: by symmetry **Arf = 1**.
- $q^{(1,1)}$: each block Arf 1, total $1 + 1 = 0$ in $\mathbb{F}_2$. **Arf = 0**.

总 Arf $= c_1 + c_2 \pmod 2$. 4 个 quadratic forms 分两 Arf 类，每类 2 个：

- **Arf 0**: $q^{(0,0)}, q^{(1,1)}$
- **Arf 1**: $q^{(0,1)}, q^{(1,0)}$

### 7.5 第一次出现 multiple Arf classes

$R_2$ 上只有 2 个 quadratic refinements ($q_0, q_1$)，分两 Arf 类（各 1 个）。

$R_4$ 上有 $2^2 = 4$ 个 quadratic refinements，分两 Arf 类（各 2 个）。

> **$R_4$ 是首层有 Arf-class 内部多元（每类 > 1 元）的层**——这是「rank stratification」之外的另一种「内部多样性」首次涌现。

## 8. $R_4 = \mathrm{End}(R_2)$ — Endomorphism View ★

### 8.1 核心同构

$$R_4 \stackrel{\sim}{=} \mathrm{End}_{\mathbb{F}_2}(R_2) = \mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)$$

16 个 R_4 元素 ↔ 16 个 $R_2 \to R_2$ 线性映射。

### 8.2 应用 (Application): $f(u)$

设 $w \in R_4$ 对应矩阵 $M_w$，$u \in R_2$。$w$ 应用 $u$ 给出新的 $R_2$ 元素:

$$\mathrm{apply}(w, u) := M_w \cdot u = \begin{pmatrix} w_0 u_0 + w_1 u_1 \\ w_2 u_0 + w_3 u_1 \end{pmatrix}$$

或在符号上：

$$\mathrm{apply}(w, u) = (w_0 \land u_0) \oplus (w_1 \land u_1),\; (w_2 \land u_0) \oplus (w_3 \land u_1)$$

### 8.3 复合 (Composition): $g \circ f$

两个 $R_4$ 元素 $g, f$ 复合：

$$\mathrm{compose}(g, f)_i := \text{(matrix product entry)}$$

更具体地：$M_{g \circ f} = M_g \cdot M_f$（矩阵乘法 over $\mathbb{F}_2$）。

例: 设 $g = \mathrm{oxxo}$, $f = \mathrm{xxox}$.
$M_g = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$, $M_f = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}$.
$M_g \cdot M_f = \begin{pmatrix} 0 & 1 \\ 1 & 1 \end{pmatrix} = M_{\mathrm{oxxx}}$.
所以 $\mathrm{compose}(\mathrm{oxxo}, \mathrm{xxox}) = \mathrm{oxxx}$.

### 8.4 单位元与逆元

- 单位：$\mathrm{id} = \mathrm{xoox}$ (= 单位矩阵 $\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$)
- 逆元仅对 6 个 rank-2 元素存在（=可逆元）
- 其他 10 个元素（rank 0 与 rank 1）无逆元

逆元表（rank-2 6 元）：

| $w$ | $w^{-1}$ |
|-----|----------|
| xoox (id) | xoox |
| oxxo (swap) | oxxo (self-inverse) |
| xxox (fix-xo) | xxox (self-inverse, 阶 2) |
| xoxx (fix-ox) | xoxx (self-inverse, 阶 2) |
| oxxx (rot$_+$) | xxxo (rot$_-$, 阶 3) |
| xxxo (rot$_-$) | oxxx (rot$_+$, 阶 3) |

### 8.5 满射性

$R_4$ 中 16 个元素**穷尽**所有 $R_2 \to R_2$ 线性映射（不多不少）。

证：$|\mathrm{Hom}_{\mathbb{F}_2}(R_2, R_2)| = |\mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)| = 2^4 = 16 = |R_4|$. ∎

## 9. State / Op 双重身份 ★ · State / Op Dual Identity

### 9.1 同一元素的两种解读

$R_4$ 的每个元素 $w$ **同时**承载两种身份：

| 角色 (Mode) | 元素解读 | 语义 |
|------|---------|------|
| **State** | $(u_1, u_2) \in R_2 \times R_2$ | 一对 $R_2$ 状态 |
| **Op** | $f : R_2 \to R_2$ 线性映射 | 一个 endomorphism |

#### State view 的元素读法

$w = (u_1, u_2)$ 是一对独立的 $R_2$ 状态。例 $w = \mathrm{xoxx}$ 在 State view 下 = $(\mathrm{xo}, \mathrm{xx})$ — 「first state = xo, second state = xx」.

#### Op view 的元素读法

$w$ 是一个 $R_2 \to R_2$ 的线性映射。例 $w = \mathrm{xoxx}$ 在 Op view 下表示矩阵 $\begin{pmatrix} 1 & 0 \\ 1 & 1 \end{pmatrix}$，该矩阵把 $\mathrm{xo}$ (= $e_\alpha$) 映到 $(1, 1) = \mathrm{xx}$，把 $\mathrm{ox}$ (= $e_\beta$) 映到 $(0, 1) = \mathrm{ox}$.

### 9.2 物理意义对比

| | State | Op |
|---|-------|----|
| 角色 | 数据 / 系统状态 | 变换 / 函数 |
| 操作 | 与其他 State 直和、tensor | apply 到 State，compose 与 Op |
| 范畴 | object | morphism |
| 编程类比 | data | function |
| 物理类比 | wave function | operator |

### 9.3 Phantom type discipline

为防止 State 与 Op 混用，type system 用 phantom mode 标注：

```
R_4[State]   -- 元素是 state（pair of R_2）
R_4[Op]      -- 元素是 op（endomorphism of R_2）
```

底层数据相同（都是 4-bit），但**类型不同**。

### 9.4 类型规则表

| 操作 | 输入 | 输出 |
|------|------|------|
| `apply` | `R_4[Op] × R_2` | `R_2` |
| `compose` | `R_4[Op] × R_4[Op]` | `R_4[Op]` |
| `pair` | `R_2 × R_2` | `R_4[State]` |
| `dot` | `R_4[m] × R_4[m]` | `𝔽₂` (m 任意，需相同) |
| `coerce_state_to_op` | `R_4[State]` | `R_4[Op]` ★ explicit only |
| `coerce_op_to_state` | `R_4[Op]` | `R_4[State]` ★ explicit only |

### 9.5 Coercion 的必要性

由于 State 与 Op 底层都是 $R_4$，可以「strucutrally cast」一方为另一方。**但必须显式 coerce**，避免 silent semantic drift。

例：从 State $(\mathrm{xo}, \mathrm{ox}) \in R_4[\text{State}]$ coerce 到 $R_4[\text{Op}]$ → 得到 identity map（因 State 的 first part = $e_\alpha$, second part = $e_\beta$，作为 Op 即 identity matrix）。

这个 coercion 在数值上**不变化任何 bit**，但**改变语义解释**。type system 强制 explicit 让程序员意识到这一切换。

### 9.6 Lean 4 实现

详见 §14. 在 Lean 中：

```lean
inductive Mode | State | Op deriving DecidableEq

structure R4_Phantom (m : Mode) where
  data : R 4
```

State 与 Op mode 的元素同 underlying type，但 Lean type system 区分两者. Coercion 必须 explicit (详 §14.1).

## 10. Hom Representation: $\mathrm{Mat}_{m \times n}(R_4)$ ★

### 10.1 通用 Hom 表示

> **核心结果**：$\mathrm{Hom}(R_N, R_M) \cong \mathrm{Mat}_{(M/2) \times (N/2)}(R_4)$ when $N, M$ 偶。

更精确地说，对偶 $N, M$，把 $R_N = R_2^{N/2}$ 看作 $(N/2)$ 个 $R_2$-blocks，$R_M$ 类似。$R_N \to R_M$ 线性映射由 $(M/2) \times (N/2)$ 矩阵给出，每 entry 是 $R_4 = \mathrm{End}(R_2)$.

**等价陈述**: $\mathrm{Hom}_{\mathbb{F}_2}(R_N, R_M) = R_{NM}$ 在 $R_4$-block 视图下成 matrix-shaped。

### 10.2 表示方案对比 — A, B, C

历史上有三种 Hom 表示选择：

#### 方案 A — Flat emit

$\mathrm{Hom}(R_N, R_M)$ 作为 flat $R_{NM}$ 元素——一条长度 $NM$ 的 binary 序列，无结构.

类型示意: `Hom_A : R (N * M)` (单个 R-Family 元素)

**优**: 简单；与代数表示直接对应  
**缺**: 失去 $(M/2) \times (N/2)$ 矩阵结构；surface 不可读

#### 方案 B — Function primitive

$\mathrm{Hom}$ 作为独立 function type，不归类 $R$-族.

类型示意: `Hom_B (N M : ℕ) : Type := R N → R M` (opaque function type)

提供 `apply` 与 `compose` 操作，但内部结构隐藏.

**优**: 显式 application + closure 语义  
**缺**: 失去代数对接；type system 复杂化；不可 inspect

#### 方案 C — Mat($R_4$) ✓

$\mathrm{Hom}$ 作为 $R_4$-矩阵——$(M/2) \times (N/2)$ 个 $R_4$-cells 排成矩阵.

类型示意: `Hom_C (n m : ℕ) : Type := Fin m → Fin n → R 4`

**优**: 结构化；可读；保代数；$R_4 = \mathrm{End}(R_2)$ 作 matrix cell 自然  
**缺**: 需 2D 索引语法

### 10.3 为什么文选方案 C

1. **$R_4$ 本身就是 matrix-cell 单元**：$R_4 = \mathrm{Mat}_{2 \times 2}(\mathbb{F}_2)$. 用 $R_4$ 作 cell 是「自然 minimal」选择。
2. **Surface readability**: $\mathrm{Mat}_{m \times n}(R_4)$ 直观——读 program 看见 2D 网格的 $R_4$-entries.
3. **代数兼容**: composition = 矩阵乘 (over $R_4$ field-like structure)，application = matrix-vector 乘.
4. **保 $R_N$-family closure**: 输出仍是 $R_N$-family 元素 (经 reverse flattening)，不需逃出族。

### 10.4 实例

$\mathrm{Hom}(R_4, R_4) \cong \mathrm{Mat}_{2 \times 2}(R_4) = $ $2 \times 2$ 矩阵 of $R_4$-cells. Total cardinality $16^4 = 65536 = 2^{16}$ ✓（等于 $\mathrm{End}(R_4) = R_{16}$）。

$\mathrm{Hom}(R_2, R_2) \cong \mathrm{Mat}_{1 \times 1}(R_4) = $ 一个 $R_4$-cell. 这退化到 $R_4$ 本身（$1 \times 1$ matrix of $R_4$ = just $R_4$）。✓ 与 §8.1 等价。

$\mathrm{Hom}(R_4, R_2) \cong \mathrm{Mat}_{1 \times 2}(R_4) = $ $1 \times 2$ 矩阵 of $R_4$. 即 2 个并列 $R_4$.

### 10.5 Application 与 Composition (Method C)

**Application**: $f \in \mathrm{Mat}_{m \times n}(R_4)$ 作用于 $U \in R_{2n}$ （视为 $R_2^n$）：

$$f(U)_i := \bigoplus_{j=1}^{n} \mathrm{apply}(f_{ij}, U_j) \in R_2$$

输出在 $R_2^m = R_{2m}$. (每 $f_{ij} \in R_4$ apply 到 $U_j \in R_2$ 给出 $R_2$ 结果，对 $j$ 累加（象群加法）)

**Composition**: $g \in \mathrm{Mat}_{p \times m}(R_4)$, $f \in \mathrm{Mat}_{m \times n}(R_4)$ →
$g \circ f \in \mathrm{Mat}_{p \times n}(R_4)$：

$$(g \circ f)_{ik} := \bigoplus_{j=1}^{m} \mathrm{compose}(g_{ij}, f_{jk})$$

where compose is $R_4$-endomorphism composition (§8.3).

## 11. 应用层 Atlas · Application-Layer Correspondences

> **范围**：本节列出 $R_4$ 在多个外部传统中的对应。这些是 **application-layer** 映射，不是 $R_4$ algebra 的来源。

### 11.1 16 个 Boolean functions of 2 variables

Boolean functions $f: \{0, 1\}^2 \to \{0, 1\}$ 共 $2^{2^2} = 16$ 个。这恰好与 $R_4$ 同基数。

对应：$f(x, y) = a \cdot \mathrm{const}_1 + b \cdot x + c \cdot y + d \cdot xy$ where $(a, b, c, d) \in \mathbb{F}_2^4 = R_4$. 但这是 Reed-Muller 1st-order expansion，非线性 + 线性. 真正双射是更细的。

精确对应（用 truth-table）：each Boolean function ↔ length-4 truth-table = $R_4$ element.

著名函数：

- `oooo` (false), `xxxx` (true), `xoox` (`xor`), `oxxo` (`xor` negated = `xnor`), `xxox` (?) — 详细命名 standard.

### 11.2 Single-qubit Pauli with phase

Pauli on 1 qubit modulo $\pm 1, \pm i$ phase 有 4 元 $\{I, X, Y, Z\}$ = $R_2$. 加上 phase ($\pm 1, \pm i$ 中的 phase) 给 $16 = 4 \times 4$ 元 Pauli group $\mathcal{P}_1$. 

$\mathcal{P}_1$ 不是 abelian (但 $\mathcal{P}_1 / \{\pm I, \pm iI\} = R_2$). $R_4$ 作为 abelian 不能 *群同构* $\mathcal{P}_1$, 但作为 **set** ($|\mathcal{P}_1| = 16 = |R_4|$) 双射。

### 11.3 $\mathrm{End}(V)$ for $V = \mathbb{F}_2^2$

直接 $R_4 = \mathrm{End}_{\mathbb{F}_2}(R_2)$. 这不是「应用层」对应，是 $R_4$ 的内在结构身份（§8）.

### 11.4 4-fold 张量与 Bell state

量子 Bell state 在 2-qubit Hilbert space $\mathbb{C}^4$ 上的某些结构 mirror $R_4$ 上的 entanglement 结构。具体: maximally entangled Bell states 4 个，$R_4$ 上「diagonal subgroup $\{道, 错综\}^2 = \{\mathrm{oooo}, \mathrm{ooxx}, \mathrm{xxoo}, \mathrm{xxxx}\}$」对应。

### 11.5 易学传统对应

把 $R_4$ 视为 $R_2^2$, 可在易学传统下解读为「2-fold image」结构。具体如 **double trigram** 或 "**bigram"** 结构。本对应**完全为 application-layer**（详 v4-foundation §15.4）。

### 11.6 4×4 棋盘 / Cellular Automata

256 状态的 4×4 binary 网格的某些 reduction 给 16-元 sub-class. CA Rule 等应用。

## 12. $R_4$ 作为「最小完整单元」 ★

### 12.1 完整性的五条算法证立

> **Theorem (5-fold Completeness of $R_4$)**：$R_4$ 是 $R_0$–$R_8$ 中**首层**同时满足以下五条结构性条件的层：

**Condition 1 — Rank Stratification (§4)**:

$R_4$ 是首层有 rank 分类的 (0/1/2 共 1+9+6 元). $R_0, R_1, R_2, R_3$ 都「平」。

**Condition 2 — Endomorphism Self-closure (§8)**:

$R_4 = \mathrm{End}(R_2)$ 在 $R$-族内自封闭。$R_0, R_1, R_2$ 各自的 $\mathrm{End}$ 不在族内 root layer 范围（$R_3 = \mathrm{End}(?)$ 没有自然表达；$\mathrm{End}(R_2) = R_4$ 是首个自然 self-contained）。

**Condition 3 — Prev-layer Aut Internal Embedding (§5)**:

$R_4$ 首次内嵌前一层的 Aut: $\mathrm{Aut}(R_2) = S_3 \hookrightarrow R_4$ as rank-2 elements. 这是 $R_2$ 自身对称性「成为 $R_4$ 内对象」.

**Condition 4 — Bilinear 三层完整 (§7)**:

$R_4$ 是偶维，Layer 0 (dot), Layer 1 (σ), Layer 2 (q + Arf) 全部自然定义且非退化。$R_1, R_3$ 奇维仅 Layer 0; $R_2$ 偶但 Arf 只 1 元 per class (无内部多元).

**Condition 5 — Arf Internal Diversity (§7.5)**:

$R_4$ 首次出现 Arf-class 内部多元（每类 2 元），$R_2$ 上每 Arf 类只 1 元。

### 12.2 与 $R_2$ 的对比

$R_2$ 也是「complete in some sense」（有 3 个 $\mathbb{Z}/2$ 子群、三合一刻画、Layer 0/1/2 全有），但**缺 Conditions 1, 2, 3, 5**：

- 没有 rank 分层 (3 个非恒等元结构等价)
- $\mathrm{End}(R_1) = R_1$ 不内嵌任何更小 Aut
- $R_2$ 上 Aut $S_3$ 只「外部 acting」，不内嵌为 elements
- Arf 类内每类 1 元

所以 $R_2$ 是「**基本但不完整**」，$R_4$ 才完整。

### 12.3 与更高层的对比

为什么不在 $R_5, R_6, R_7, R_8$ 上才划分「最小完整」？

- $R_5$ 奇维，**Condition 4 失败**（无 Layer 1/2）
- $R_6$ 偶维，但 $\mathrm{End}(?) = R_6$ 没有自然 self-embedding（$\mathrm{End}(R_3) = R_9 \neq R_6$）
- $R_7$ 奇维，**Condition 4 失败**
- $R_8 = R_4^2$ 比 $R_4$ 大，也完整，但**$R_4$ 是 minimum**

故 $R_4$ 是 **R-tower 内首个满足全部 5 条件的最小层**。

### 12.4 哲学含义

> $R_4$ 是「**算法的最小完整体**」。任何「需要 transformations 上 transformations」的代数结构都从 $R_4$ 开始 well-defined.

工程含义：

- $R_4$ 是 type system 的「自表达起点」——$R_4$ 元素既能描述自己也能描述 $R_2$
- $R_4$ 是 IR primitive 的「composition unit」——所有 Hom 表示用 $R_4$-cells (§10)
- $R_4$ 是「**first non-trivial category**」的实现（带 identity, composition, inverse on 6 invertibles）

## 13. $R_4$ 与其他根层的关系

### 13.1 $R_4 = R_2^2$ (squaring tower 第 3 层)

- $R_4 = R_2 \otimes R_2$ (tensor)
- $R_4 = R_2 \oplus R_2$ (direct sum, view as $R_2^{(L)} \oplus R_2^{(R)}$)
- $R_4 = \mathrm{End}(R_2)$ (endomorphism)

三种身份共生 (§1).

### 13.2 $R_4$ 与 $R_3$

$R_3 = R_1 \oplus R_2$ 是「half-step」between $R_2$ 和 $R_4$. 

dim 差：$R_4 - R_3 = 1$, 一个额外 $\mathbb{F}_2$-bit. 不构成 squaring tower 关系，是「one-bit extension」.

### 13.3 $R_4$ 与 $R_5$

$R_5 = R_1 \oplus R_4$. 一个 bit 加 $R_4$. 

注意 $R_5$ 奇维，bilinear 结构跌回 Layer 0 only.

### 13.4 $R_4$ 与 $R_8 = R_4^2$ ★

$R_8 = R_4 \otimes R_4 = R_4^2$ (next squaring layer).

详细见 `r8.md`. 关键：

- $R_8$ 是 $R_4$ 的「self-tensor square」
- $R_8 \neq \mathrm{End}(R_4)$ (后者 $= R_{16}$)
- $R_8 = R_4 \oplus R_4$ 直和分解 view

### 13.5 $R_4$ 与 $R_{16} = \mathrm{End}(R_4)$

$\mathrm{End}(R_4) = R_{16}$ — 在 root layer 之外（$R_{16}$ 不是根层）。

但 $R_{16} = R_8 \oplus R_8$ 由 $R_8$ 构造，根层范围内可表达。

### 13.6 总结关系表

| 关系 | 公式 | 备注 |
|------|------|------|
| $R_4 = R_2^2$ | tensor / squaring | squaring tower 第 3 层 |
| $R_4 = R_2 \oplus R_2$ | direct sum | 同一 cardinality |
| $R_4 = \mathrm{End}(R_2)$ | endomorphism | 自封闭 in R-family |
| $R_4 \subset R_5$ | embedding | 单 bit extension |
| $R_4 \subset R_6$ | embedding | (sub-2-block) |
| $R_4 \subset R_7$ | embedding | |
| $R_4 \subset R_8$ | embedding | (as left or right half) |
| $R_8 = R_4^2$ | tensor | next squaring layer |
| $\mathrm{End}(R_4) = R_{16}$ | 出 root range | constructible from $R_8$ |

  (fun i => w ⟨i.val, by omega⟩, fun i => w ⟨i.val + 2, by omega⟩)

def join (u v : R2) : R4 := fun i =>
  if h : i.val < 2 then u ⟨i.val, h⟩ else v ⟨i.val - 2, by omega⟩

/-- View C: 2x2 matrix -/
def matrix_entry (w : R4) (i j : Fin 2) : Bool :=
  w ⟨2 * i.val + j.val, by omega⟩

/-- Apply w as op: M_w · u -/
def apply (w : R4) (u : R2) : R2 := fun i =>
  Bool.xor (matrix_entry w i 0 && u 0) (matrix_entry w i 1 && u 1)

/-- Compose two R_4 ops -/
def compose (g f : R4) : R4 := fun k =>
  let i : Fin 2 := ⟨k.val / 2, by omega⟩
  let j : Fin 2 := ⟨k.val % 2, by omega⟩
  Bool.xor
    (matrix_entry g i 0 && matrix_entry f 0 j... -- detailed indexing
    sorry)  -- standard 2x2 matrix product over F_2

/-- The 16 elements, classified by rank -/
def rank_of (w : R4) : Fin 3 :=
  let a := w 0; let b := w 1
  let c := w 2; let d := w 3
  let det := Bool.xor (a && d) (b && c)
  if w = (fun _ => false) then 0
  else if det then 2
  else 1

/-- The 6 rank-2 elements form GL(2, F_2) ≅ S_3 -/
theorem rank_two_count : Fintype.card {w : R4 // rank_of w = 2} = 6 := by sorry

/-- The 9 rank-1 elements -/
theorem rank_one_count : Fintype.card {w : R4 // rank_of w = 1} = 9 := by sorry

/-- Layer 0: dot product on R_4 -/
def dot (w w' : R4) : Bool :=
  Bool.xor (w 0 && w' 0)
    (Bool.xor (w 1 && w' 1)
      (Bool.xor (w 2 && w' 2) (w 3 && w' 3)))

/-- Layer 1: symplectic σ on R_4 (2 blocks) -/
def σ (w w' : R4) : Bool :=
  Bool.xor
    (Bool.xor (w 0 && w' 1) (w 1 && w' 0))    -- block 1
    (Bool.xor (w 2 && w' 3) (w 3 && w' 2))    -- block 2

theorem σ_alternating (w : R4) : σ w w = false := by sorry

/-- Quadratic refinement with mode (c_1, c_2) ∈ {0,1}^2 -/
def q_refined (c1 c2 : Bool) (w : R4) : Bool :=
  let q1 := if c1 then 
    Bool.xor (Bool.xor (w 0) (w 1)) (w 0 && w 1)
    else (w 0 && w 1)
  let q2 := if c2 then 
    Bool.xor (Bool.xor (w 2) (w 3)) (w 2 && w 3)
    else (w 2 && w 3)
  Bool.xor q1 q2

/-- Phantom type for State/Op discipline -/
inductive Mode | State | Op deriving DecidableEq

structure R4Phantom (m : Mode) where
  data : R4

/-- Application requires Op mode -/
def apply_phantom (f : R4Phantom Mode.Op) (u : R2) : R2 := apply f.data u

/-- Composition requires both Op mode -/
def compose_phantom (g f : R4Phantom Mode.Op) : R4Phantom Mode.Op :=
  ⟨compose g.data f.data⟩

/-- Pair construction gives State mode -/
def pair_phantom (u v : R2) : R4Phantom Mode.State :=
  ⟨join u v⟩

/-- Explicit coercion: State → Op -/
def coerce_state_to_op (s : R4Phantom Mode.State) : R4Phantom Mode.Op :=
  ⟨s.data⟩  -- structurally same, semantically different

end 文.R4
