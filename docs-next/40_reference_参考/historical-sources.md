## 冻结原则

- 不在史料目录内修辞、校勘或删减。
- 新发现的解释进入 docs-next 对应区；旧史料只作来源指针。
- 长引用应避免；需要原文时直接回到 `史料/` 原路径。
- 自动生成索引若存在，参考 `../_generated/` 的文件清单核对路径是否仍有效。
