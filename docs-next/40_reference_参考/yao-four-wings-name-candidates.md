## 审阅规则

- 枚举顺序为 `初/二/三/四/五/上 × 彖/象/文/系`。
- `古文候选` 是短 surface；`保留别名` 使用 `爻位之翼`，更可读但更长。
- `四翼` 在此先合并十翼为四类：彖断、象观、文言释义、系辞系理；说卦/序卦/杂卦可另表追加。
- `冲突提示` 只提示与当前 operator/readings/reserved 字的静态冲突，不代表禁用。
