## 主要变更摘要（旧工作稿 → 新定本）

| 层级 | 旧 default（本工作稿原案） | 新 default（已落定） | 变更原因 |
|---|---|---|---|
| R3 乾 mode | 生 | **健** | `生` 撞 Sheng.step / T-7 / CoreAtom |
| R3 兑 mode | 开 | **悦** | `开` 已是 CoreAtom，硬撞车 |
| R3 离 mode | 显 | **显** | 不变 |
| R3 震 mode | 元 | **起** | `元` 撞 Monad root 一元 |
| R3 巽 mode | 申 | **入** | `申` 偏冷僻；`入`是《说卦》原文 |
| R3 坎 mode | 塞 | **险** | `险`比`塞`现代通用且为坎 alias |
| R3 艮 mode | 居 | **止** | `止`是《说卦》原文，更准 |
| R3 坤 mode | 守 | **顺** | `守`属 identity alias 集 |
| R4 flip4 | 待定 | **临** | 19 临卦同字 / 4 爻近君位 |
| R4 flip5 | 待定 | **主** | 5 爻君位 / 主导义 |
| R5 shiNext | 未给 | **迁** | "时迁"古义 |
| R5 shiPrev | 未给 | **溯** | "溯往"古义 |
| R1 阳/阴 义理读 | 未给 | **实 / 虚** | 邵雍《观物外篇》"阳为实, 阴为虚" |
| R2 四象单字 | 未给 | **春 / 夏 / 秋 / 冬** | 邵雍《皇极经世》先天图配四时 |
