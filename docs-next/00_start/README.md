# 起步

这里给第一次进入新文档集的读者一个短路径。完整事实索引在 `../_generated/`，人工文档只负责解释如何读。

## 三种读法

- 要验证代码：从 [形式路径](formal-path.md) 开始，再读 `../10_formal_形式/`。
- 要理解义理：从 [义理路径](theory-path.md) 开始，再读 `../20_theory_义理/`。
- 要维护文档：从 [维护路径](maintenance-path.md) 开始，再读 `../50_maintenance/`。

## 先记住的边界

- `史料/` 是 frozen archive，不是现规范。
- `docs-next/` 是 staging 区，当前不替换根 README。
- `../_generated/` 是可重建索引，数量和状态以它为准。
